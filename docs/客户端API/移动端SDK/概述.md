# 概述

企业微信分别提供了IOS、Android、Harmony sdk，企业可在自建的app引入sdk，从而可使用企业微信账号登录app，并且在app中分享消息到企业微信。

## 概述
		企业微信登录SDK
			iOS、Android、Harmony应用嵌入企业微信登录SDK，可使用企业微信账号直接登录。	
			企业微信分享SDK
			iOS、Android、Harmony应用嵌入企业微信分享SDK，可把文件、图片、链接等内容便捷的分享给企业微信用户。
