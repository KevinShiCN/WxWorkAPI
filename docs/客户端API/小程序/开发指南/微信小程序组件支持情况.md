# 微信小程序组件支持情况

# 基础组件

**视图容器(View Container)：**

        
| 组件名 | 说明 | 企业微信支持情况 |
| --- | --- | --- |
| cover-image | 覆盖在原生组件之上的图片视图 | 支持 |
| cover-view | 覆盖在原生组件之上的文本视图 | mac上部分支持 |
| match-media | media query 匹配检测节点 | mac上部分支持 |
| movable-area | movable-view的可移动区域 | 桌面端部分支持 |
| movable-view | 可移动的视图容器 | 桌面端部分支持 |
| page-container | 页面容器 | 仅移动端支持 |
| scroll-view | 可滚动视图区域 | 桌面端部分支持 |
| share-element | 共享元素 | 不支持 |
| swiper | 滑块视图容器 | 支持 |
| swiper-item | 仅可放置在swiper 组件中，宽高自动设置为100% | 支持 |
| view | 视图容器 | 支持 |

 

**基础内容(Basic Content)：**

        
| 组件名 | 说明 | 企业微信支持情况 |
| --- | --- | --- |
| icon | 图标 | 支持 |
| progress | 进度条 | 支持 |
| rich-text | 富文本 | win上部分支持 |
| text | 文字 | 支持 |

 

**表单(Form)：**

        
| 标签名 | 说明 | 企业微信支持情况 |
| --- | --- | --- |
| button | 按钮 | 支持 |
| checkbox | 多项选择器 | 支持 |
| checkbox-group | 多项选择器，内部由多个checkbox组成 | 支持 |
| editor | 富文本编辑器，可以对图片、文字进行编辑 | 支持 |
| form | 表单 | 支持 |
| input | 输入框 | 桌面端部分支持 |
| keyboard-accessory | 设置 input  textarea 聚焦时键盘上方 cover-view  cover-image 工具栏视图 | 仅移动端支持 |
| label | 标签 | 支持 |
| picker | 列表选择器 | mac部分支持 |
| picker-view | 内嵌列表选择器 | win部分支持 |
| picker-view-column | 滚动选择器子项 | 支持 |
| radio | 单项选择器 | 支持 |
| radio-group | 单项选择器，内部由多个 radio 组成 | 支持 |
| slider | 滚动选择器 | 支持 |
| switch | 开关选择器 | 支持 |
| textarea | 多行输入框 | mac部分支持 |

**导航(Navigation)：**

        
| 组件名 | 说明 | 企业微信支持情况 |
| --- | --- | --- |
| functional-page-navigator | 跳转到插件功能页 | 支持 |
| navigator | 页面链接 | 支持 |

**媒体组件(Media)：**

        
| 组件名 | 说明 | 企业微信支持情况 |
| --- | --- | --- |
| audio | 音频 | 部分支持 |
| camera | 系统相机 | 桌面端部分支持 |
| image | 图片 | 支持 |
| live-player | 实时音视频播放 | 桌面端部分支持 |
| live-pusher | 实时音视频录制 | 桌面端部分支持 |
| video | 视频 | 部分支持 |
| voip-room | 多人音视频对话 | 不支持 |

 

**地图(Map)：**

        
| 组件名 | 说明 | 企业微信支持情况 |
| --- | --- | --- |
| map | 地图 | 部分支持 |

 

**画布(Canvas)：**

        
| 组件名 | 说明 | 企业微信支持情况 |
| --- | --- | --- |
| canvas | 画布 | 部分支持 |

 

**开放能力(Open Ability)：**

        
| 组件名 | 说明 | 企业微信支持情况 |
| --- | --- | --- |
| ad | 广告 | 不支持 |
| ad-custom | 原生模板广告 | 不支持 |
| official-account | 关注公众号 | 不支持 |
| open-data | 展示微信开放的数据 | 部分支持 |
| web-view | 承载网页的容器 | 支持 |
