# 断开设备连接

## ww.closeBLEConnection(params)

> 兼容性：企业微信

### 功能描述

断开与低功耗蓝牙设备的连接。

      **
      
```ts
ww.closeBLEConnection({
  deviceId: deviceId
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | deviceId | string | 是 | 蓝牙设备 ID |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.closeBLEConnection({
  deviceId:deviceId
  success: function (res) {
    console.log(res)
  }
})
```
