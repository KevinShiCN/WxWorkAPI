# 保存选择的选项

## ww.saveApprovalSelectedItems(params)

> 兼容性：企业微信 4.0.18 开始支持

### 功能描述

保存用户选择的审批选项。

用户在网页中修改审批选项时，调用该接口保存用户的选择。

      **
      
```ts
ww.saveApprovalSelectedItems({
  key: 'key',
  selectedData: [
    {
      key: 'item-1',
      value: '选项1'
    },
    {
      key: 'item-2',
      value: '选项2'
    }
  ]
})
```

    
### 使用限制

- 本接口必须使用应用身份进行注册
- 应用必须具有审批权限

### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | key | string | 是 | 从 URL 中获取到的 key |
|  | selectedData | Object[] | 是 | 选中的选项 |
|  | 属性类型必填说明keystring是表示选项的唯一识别 IDvaluestring是选项展示给用户的名字 |  | 属性 | 类型 | 必填 | 说明 |  | key | string | 是 | 表示选项的唯一识别 ID |  | value | string | 是 | 选项展示给用户的名字 |
|  | 属性 | 类型 | 必填 | 说明 |
|  | key | string | 是 | 表示选项的唯一识别 ID |
|  | value | string | 是 | 选项展示给用户的名字 |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |

### 使用说明

- 接口仅用于审批设置外部选项场景，请勿用作其他场景
- 网页应该做好深色模式适配

## 旧版jweixin调用
