# 获取已选择的选项

## ww.getApprovalSelectedItems(params)

> 兼容性：企业微信 4.0.18 开始支持

### 功能描述

获取 saveApprovalSelectedItems 保存的审批选项。

当用户打开网页后，应该先调用一次该接口获取用户已经选择的数据作为初始数据。获取到初始数据后，应该恢复已经选择的选项。

      **
      
```ts
ww.getApprovalSelectedItems({
  key: 'key'
})
```

    
### 使用限制

- 本接口必须使用应用身份进行注册
- 所签名的应用必须具有审批权限

### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | key | string | 是 | 从 URL 获取的 key |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
|  | selectedData | Object[] | 是 | 选中的选项 |
|  | 属性类型必填说明keystring是表示选项的唯一识别 IDvaluestring是选项展示给用户的名字 |  | 属性 | 类型 | 必填 | 说明 |  | key | string | 是 | 表示选项的唯一识别 ID |  | value | string | 是 | 选项展示给用户的名字 |
|  | 属性 | 类型 | 必填 | 说明 |
|  | key | string | 是 | 表示选项的唯一识别 ID |
|  | value | string | 是 | 选项展示给用户的名字 |

### 使用说明

- 网页应该做好深色模式适配
- 接口仅用于审批设置外部选项场景，请勿用作其他场景

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.invoke('getApprovalSelectedItems', {
    "key": key, // 字符串，从 URL 中获取到的 key
}, (res)=> {
    if (res.err_msg === "getApprovalSelectedItems:ok") {
			try {
				// !!!注意：旧版中res.selectedData 为 JSON 字符串。
				const selectedData = JSON.parse(res.selectedData)
			} catch() {
				// 注意可能为空
			}
    }
});
```
