# 搜索附近设备

## ww.startBeaconDiscovery(params)

> 兼容性：企业微信

### 功能描述

开始搜索附近的 iBeacon 设备。

      **
      
```ts
ww.startBeaconDiscovery({
  uuids: ['uuid']
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | uuids | string[] | 是 | iBeacon设备广播的 uuids |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
