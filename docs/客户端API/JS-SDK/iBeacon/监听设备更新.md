# 监听设备更新

## ww.onBeaconUpdate(callback)

> 兼容性：企业微信

### 功能描述

监听 iBeacon 设备的更新事件。

      **
      
```ts
ww.onBeaconUpdate(function(event) {
  console.log(event)
})
```

    
### 参数说明

#### callback: Function

**参数**

**event: Object**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | beacons | Object[] | 是 | 当前搜寻到的所有 iBeacon 设备列表 |
|  | 属性类型必填说明uuidstring是iBeacon 设备广播的 uuidmajorstring是iBeacon 设备的主 idminorstring是iBeacon 设备的次 idproximitynumber是表示设备距离的枚举值, 0 信号太弱不足以计算距离, 或非 iOS 设备;	1	十分近; 2	比较近; 3	远;accuracynumber是iBeacon 设备的距离rssinumber是表示设备的信号强度 |  | 属性 | 类型 | 必填 | 说明 |  | uuid | string | 是 | iBeacon 设备广播的 uuid |  | major | string | 是 | iBeacon 设备的主 id |  | minor | string | 是 | iBeacon 设备的次 id |  | proximity | number | 是 | 表示设备距离的枚举值, 0 信号太弱不足以计算距离, 或非 iOS 设备;	1	十分近; 2	比较近; 3	远; |  | accuracy | number | 是 | iBeacon 设备的距离 |  | rssi | number | 是 | 表示设备的信号强度 |
|  | 属性 | 类型 | 必填 | 说明 |
|  | uuid | string | 是 | iBeacon 设备广播的 uuid |
|  | major | string | 是 | iBeacon 设备的主 id |
|  | minor | string | 是 | iBeacon 设备的次 id |
|  | proximity | number | 是 | 表示设备距离的枚举值, 0 信号太弱不足以计算距离, 或非 iOS 设备;	1	十分近; 2	比较近; 3	远; |
|  | accuracy | number | 是 | iBeacon 设备的距离 |
|  | rssi | number | 是 | 表示设备的信号强度 |
