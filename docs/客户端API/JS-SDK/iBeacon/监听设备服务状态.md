# 监听设备服务状态

## ww.onBeaconServiceChange(callback)

> 兼容性：企业微信

### 功能描述

监听 iBeacon 服务的状态变化。

      **
      
```ts
ww.onBeaconServiceChange(function(event) {
  console.log(event)
})
```

    
### 参数说明

#### callback: Function

**参数**

**event: Object**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | available | boolean | 是 | 服务目前是否可用 |
|  | discovering | boolean | 是 | 目前是否处于搜索状态 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.onBeaconServiceChange(function(event) {
  console.log(event)
})
```
