# 停止搜索设备

## ww.stopBeaconDiscovery([params])

> 兼容性：企业微信

### 功能描述

停止搜索附近的 iBeacon 设备。

      **
      
```ts
ww.stopBeaconDiscovery()
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.stopBeaconDiscovery({
    success(res) {
    }
})
```
