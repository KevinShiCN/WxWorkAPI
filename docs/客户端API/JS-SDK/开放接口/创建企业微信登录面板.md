# 创建企业微信登录面板

## ww.createWWLoginPanel(options)

### 功能描述

初始化企业微信Web登录组件，创建登录面板。

      **
      
```ts
// 初始化登录组件
const wwLogin = ww.createWWLoginPanel({
  el: '#ww_login',
  params: {
    login_type: 'CorpApp',
    appid: 'wwbbb6a7b539f2xxxxx',
    agentid: '10000xx',
    redirect_uri: 'https://work.weixin.qq.com',
    state: 'loginState',
    redirect_type: 'callback',
  },
  onCheckWeComLogin({ isWeComLogin }) {
    console.log(isWeComLogin)
  },
  onLoginSuccess({ code }) {
    console.log({ code })
  },
  onLoginFail(err) {
    console.log(err)
  },
})
```

## 特别说明

`ww.createWWLoginPanel` 使用场景：在桌面端系统浏览器中直接初始化Web登录组件，无需调用 [`ww.register`](/document/path/98560#js-%E6%8E%A5%E5%8F%A3%E9%89%B4%E6%9D%83)、`ww.checkJsApi`等接口。
