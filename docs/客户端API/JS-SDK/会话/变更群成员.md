# 变更群成员

## ww.updateEnterpriseChat(params)

> 兼容性：企业微信 iOS、Android、PC 3.0.36 开始支持

### 功能描述

变更群成员。

      **
      
```ts
ww.updateEnterpriseChat({
  chatId: 'CHATID',
  userIdsToAdd: [
    'zhangsan',
    'lisi'
  ]
})
```

    
### 使用限制

- 本接口必须使用应用身份进行注册
- 目前仅支持添加企业内部成员
- 仅支持客户群调用

### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | chatId | string | 是 | 通过企业微信创建群聊接口返回的 chatId |
|  | userIdsToAdd | string \| string[] | 是 | 参与会话的企业成员列表 |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
