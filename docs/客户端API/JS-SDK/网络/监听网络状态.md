# 监听网络状态

## ww.onNetworkStatusChange(callback)

> 兼容性：企业微信 iOS、Android

### 功能描述

监听网络状态变化。

      **
      
```ts
ww.onNetworkStatusChange(function(event) {
  console.log(event)
})
```

    
### 参数说明

#### callback: Function

**参数**

**event: Object**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | isConnected | boolean | 是 | 当前是否有网络连接 |
|  | networkType | string | 是 | 网络类型 |
|  | 合法值说明wifiwifi2g2g3g3g4g4gnone无网络unknownAndroid下不常见的网络类型 |  | 合法值 | 说明 |  | wifi | wifi |  | 2g | 2g |  | 3g | 3g |  | 4g | 4g |  | none | 无网络 |  | unknown | Android下不常见的网络类型 |
|  | 合法值 | 说明 |
|  | wifi | wifi |
|  | 2g | 2g |
|  | 3g | 3g |
|  | 4g | 4g |
|  | none | 无网络 |
|  | unknown | Android下不常见的网络类型 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.onNetworkStatusChange(function(res) {
  console.log(res.isConnected)
  console.log(res.networkType)
})
```
