# 创建文档

## ww.createDoc(params)

> 兼容性：企业微信 4.1.0 开始支持

### 功能描述

新建文档、表格或者收集表。

      **
      
```javascript
ww.createDoc({
 docType: 3
})
```

    
### 使用限制

- 本接口必须使用应用身份进行注册
- 签名应用必须具有文档使用权限
- 当前用户必须在应用的可见范围之内
- 在 Mac 端使用时，macOS 版本需 > 10.12

### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | docType | number | 是 | 文档类型 3: 文档，4: 表格, 5: 收集表, 6: PPT, 10: 智能表格，其他类型暂不支持 |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
|  | result | Object | 是 |  |
|  | 属性类型必填说明selectedFileUrlsstring是选择的在线文档的URL列表 |  | 属性 | 类型 | 必填 | 说明 |  | selectedFileUrls | string | 是 | 选择的在线文档的URL列表 |
|  | 属性 | 类型 | 必填 | 说明 |
|  | selectedFileUrls | string | 是 | 选择的在线文档的URL列表 |

### 错误信息

        
| errMsg | 说明 |
| --- | --- |
| createDoc:ok | 执行成功 |
| createDoc:fail no permission | 应用签名校验失败，或成员不在应用的可见范围内，或应用未开启文档使用权限 |
| createDoc:fail doc app closed. | 基础应用“文档”如果未启用 |
| createDoc:fail form app closed. | 基础应用“收集表”如果没有启用 |

 });
