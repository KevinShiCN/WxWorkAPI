# 批量显示功能按钮

## ww.showMenuItems(params)

### 功能描述

批量显示功能按钮。

      **
      
```ts
ww.showMenuItems({
  menuList: ['menuItem:setFont']
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | menuList | string[] | 是 | 要显示的菜单项 |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |

### 使用说明

完整功能按钮列表请参考所有菜单项列表。

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.showMenuItems({
    menuList: [] // 要显示的菜单项
});
```
