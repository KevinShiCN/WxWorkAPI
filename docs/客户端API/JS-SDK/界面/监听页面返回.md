# 监听页面返回

## ww.onHistoryBack(callback)

> 兼容性：企业微信 iOS、Android 2.2.0 开始支持，企业微信 PC、Mac 2.4.5 开始支持

### 功能描述

监听页面返回事件。

      **
      
```ts
ww.onHistoryBack(function() {
  return confirm('确定放弃当前页面的修改？')
})
```

    
### 使用限制

- 当页面左上角没有关闭按钮，不产生该事件
- iOS 系统下使用手势返回时，不产生该事件

### 参数说明

#### callback: Function

回调函数，返回 false 则表示中断此次返回操作

**返回值**

**boolean**

是否取消返回操作

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.onHistoryBack(function(){
    return confirm('确定要放弃当前页面的修改？')
});
```
