# 监听截屏事件

## ww.onUserCaptureScreen(callback)

> 兼容性：企业微信 iOS、Android 2.5.0 开始支持

### 功能描述

监听用户截屏事件。

      **
      
```ts
ww.onUserCaptureScreen(function() {
  console.log('用户截屏了')
})
```

    
### 参数说明

#### callback: Function

**参数**

**event: void**

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.onUserCaptureScreen(function(res) {
    console.log('用户截屏了')
})
```
