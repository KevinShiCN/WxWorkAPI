# 认领老师班级

## ww.claimClassAdmin([params])

> 兼容性：企业微信 3.1.8 开始支持

### 功能描述

跳转到认领班级的界面。

      **
      
```ts
ww.claimClassAdmin()
```

    
### 使用限制

本接口必须使用应用身份进行注册

### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
|  | departmentIds | string[] | 是 | 认领的班级部门 ID 列表 |

### 错误信息

        
| errMsg | 说明 |
| --- | --- |
| claimClassAdmin:ok | 执行成功 |
| claimClassAdmin:fail no permission | 应用身份鉴权失败 |
| claimClassAdmin:fail user not in allow list | 当前成员不在应用可见范围 |

## 旧版jweixin调用

### 示例代码

      
```js
wx.invoke('claimClassAdmin',{
	
	},function(res) {
        if (res.err_msg == "claimClassAdmin:ok") {
			// 回调成功
			var departmentIds = res.departmentIds;  //数组类型
			for (var i = 0; i < departmentIds.length; i++){
				var departmentId = departmentIds[i];
			}
		}
);
```
