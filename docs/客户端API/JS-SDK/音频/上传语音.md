# 上传语音

## ww.uploadVoice(params)

### 功能描述

上传语音。

      **
      
```ts
ww.uploadVoice({
  localId: localId,
  isShowProgressTips: true
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 默认值 | 必填 | 说明 |
| --- | --- | --- | --- | --- | --- |
|  | localId | string |  | 是 | 需要上传的音频的本地 ID通过 stopRecord 接口获得 |
|  | isShowProgressTips | boolean | true | 否 | 是否显示进度提示 |
|  | success | Function |  | 否 | 成功回调 |
|  | fail | Function |  | 否 | 失败回调 |
|  | cancel | Function |  | 否 | 取消回调 |
|  | complete | Function |  | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
|  | serverId | string | 是 | 音频的服务器端 ID |

### 使用说明

上传语音有效期 3 天，可以通过素材管理接口下载语音到自己的服务器，接口返回的的 `serverId` 即 `media_id`。

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.uploadVoice({
    localId: '', // 需要上传的音频的本地ID，由stopRecord接口获得
    isShowProgressTips: 1, // 默认为1，显示进度提示
    success: function (res) {
        var serverId = res.serverId; // 返回音频的服务器端ID
    }
});
```
