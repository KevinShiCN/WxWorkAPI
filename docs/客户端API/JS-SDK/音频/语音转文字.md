# 语音转文字

## ww.translateVoice(params)

> 兼容性：企业微信 iOS、Android 2.7.5 开始支持

### 功能描述

语音转文字。

      **
      
```ts
ww.translateVoice({
  localId: localId,
  isShowProgressTips: true
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 默认值 | 必填 | 说明 |
| --- | --- | --- | --- | --- | --- |
|  | localId | string |  | 是 | 需要识别的音频的本地 ID通过 stopRecord 接口获得，音频时长不能超过 60 秒 |
|  | isShowProgressTips | boolean | true | 否 | 是否显示进度提示 |
|  | success | Function |  | 否 | 成功回调 |
|  | fail | Function |  | 否 | 失败回调 |
|  | cancel | Function |  | 否 | 取消回调 |
|  | complete | Function |  | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
|  | translateResult | string | 是 | 识别结果 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.translateVoice({
    localId: '', // 需要识别的音频的本地Id，由录音相关接口获得，音频时长不能超过60秒
    isShowProgressTips: 1, // 默认为1，显示进度提示
    success: function (res) {
        alert(res.translateResult); // 语音识别的结果
    }
});
```
