# 监听播放完毕

## ww.onVoicePlayEnd(callback)

### 功能描述

监听语音播放完毕。

      **
      
```ts
ww.onVoicePlayEnd(function(event) {
  console.log(event)
})
```

    
### 参数说明

#### callback: Function

**参数**

**event: Object**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | localId | string | 是 | 音频的本地 ID |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.onVoicePlayEnd({
    success: function (res) {
        var localId = res.localId; // 返回音频的本地ID
    }
});
```
