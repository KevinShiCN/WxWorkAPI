# 获取地理位置

## ww.getLocation([params])

### 功能描述

获取地理位置。

      **
      
```ts
ww.getLocation({
  type: 'wgs84'
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 默认值 | 必填 | 说明 |
| --- | --- | --- | --- | --- | --- |
|  | type | string | wgs84 | 否 | 坐标类型如果要返回直接给 openLocation 用的火星坐标，可传入 gcj02 |
|  | 合法值说明wgs84gps 坐标gcj02火星坐标 |  | 合法值 | 说明 |  | wgs84 | gps 坐标 |  | gcj02 | 火星坐标 |
|  | 合法值 | 说明 |
|  | wgs84 | gps 坐标 |
|  | gcj02 | 火星坐标 |
|  | success | Function |  | 否 | 成功回调 |
|  | fail | Function |  | 否 | 失败回调 |
|  | cancel | Function |  | 否 | 取消回调 |
|  | complete | Function |  | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 | 最低版本 |
| --- | --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |  |
|  | errCode | number | 是 | 通用错误码 |  |
|  | latitude | number | 是 | 纬度，浮点数，范围为 90 ~ -90 |  |
|  | longitude | number | 是 | 经度，浮点数，范围为 180 ~ -180 |  |
|  | speed | number | 是 | 速度，以米/每秒计，企业微信不支持speed参数，输出固定为0 | 微信 |
|  | accuracy | number | 是 | 位置精度 |  |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.getLocation({
    type: 'wgs84', // 默认为wgs84的gps坐标，如果要返回直接给openLocation用的火星坐标，可传入'gcj02'
    success: function (res) {
        var latitude = res.latitude; // 纬度，浮点数，范围为90 ~ -90
        var longitude = res.longitude; // 经度，浮点数，范围为180 ~ -180。
        var speed = res.speed; // 速度，以米/每秒计
        var accuracy = res.accuracy; // 位置精度
    }
});
```
