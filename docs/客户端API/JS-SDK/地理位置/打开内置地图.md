# 打开内置地图

## ww.openLocation(params)

### 功能描述

使用企业微信内置地图查看位置。

      **
      
```ts
ww.openLocation({
  latitude: 0,
  longitude: 0,
  name: 'name',
  address: 'address',
  scale: 1
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 默认值 | 必填 | 说明 |
| --- | --- | --- | --- | --- | --- |
|  | latitude | number |  | 是 | 纬度范围为 90 ~ -90 |
|  | longitude | number |  | 是 | 经度范围为 180 ~ -180 |
|  | name | string |  | 否 | 位置名 |
|  | address | string |  | 否 | 地址详情说明 |
|  | scale | number | 16 | 否 | 地图缩放级别范围为 1 ~ 28 |
|  | infoUrl | string |  | 否 |  |
|  | success | Function |  | 否 | 成功回调 |
|  | fail | Function |  | 否 | 失败回调 |
|  | cancel | Function |  | 否 | 取消回调 |
|  | complete | Function |  | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.openLocation({
    latitude: 0, // 纬度，浮点数，范围为90 ~ -90
    longitude: 0, // 经度，浮点数，范围为180 ~ -180。
    name: '', // 位置名
    address: '', // 地址详情说明
    scale: 1, // 地图缩放级别,整形值,范围从1~28。默认为16
});
```
