# 进入添加客户界面

## ww.navigateToAddCustomer([params])

> 兼容性：企业微信 iOS、Android 3.0.36 开始支持

### 功能描述

在企业微信内快速跳转到添加客户的界面。

      **
      
```ts
ww.navigateToAddCustomer()
```

    
### 使用限制

- 本接口必须使用应用身份进行注册
- 应用需有客户联系功能权限
- 当前成员必须配置了客户联系功能

### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
