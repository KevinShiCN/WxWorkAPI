# 设置朋友圈封面与签名

## ww.updateMomentsSetting(params)

> 兼容性：企业微信 iOS、Android 3.1.12 开始支持

### 功能描述

设置朋友圈封面与签名。

      **
      
```ts
ww.updateMomentsSetting({
  signature: '个性签名',
  imgUrl: 'https://work.weixin.qq.com/'
})
```

    
### 使用限制

- 本接口必须使用应用身份进行注册
- 应用需有客户联系功能权限
- 当前成员必须配置了客户联系功能
- 当前成员必须在客户朋友圈使用范围
- 当前成员必须具备外部沟通管理成员使用权限

### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | signature | string | 否 | 个性签名空字符串表示清空个性签名使用限制最多支持 30 字 |
|  | imgUrl | string | 否 | 朋友圈封面封面图建议为正方形，尺寸建议为 1125*1125 像素 |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |

### 使用说明

同时设置了签名跟封面url，客户端更新顺序为先更新签名，再更新封面图url（封面图若不符合要求会让用户重新调整）。
