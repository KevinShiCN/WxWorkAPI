# 设置剪贴板内容

## ww.setClipboardData(params)

> 兼容性：企业微信 iOS、Android 2.4.16 开始支持，企业微信 PC、Mac 3.1.2 开始支持

### 功能描述

设置系统剪贴板的内容。

      **
      
```ts
ww.setClipboardData({
  data: 'data'
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | data | string | 是 | 剪贴板的内容 |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.setClipboardData({
  data: 'data',    // 设置的
  success: function(res) {
      console.log(res.errMsg)
  }
})
```
