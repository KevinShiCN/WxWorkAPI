# 调起扫一扫

## ww.scanQRCode([params])

### 功能描述

调起企业微信扫一扫。

      **
      
```ts
ww.scanQRCode({
  needResult: true,
  scanType: ['qrCode']
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 默认值 | 必填 | 说明 |
| --- | --- | --- | --- | --- | --- |
|  | needResult | boolean | false | 否 | 扫描结果是否由企业微信处理为 false 时直接返回扫描结果 |
|  | scanType | string[] |  | 否 | 扫码类型 |
|  | 合法值说明qrCode扫描二维码barCode扫描条形码 |  | 合法值 | 说明 |  | qrCode | 扫描二维码 |  | barCode | 扫描条形码 |
|  | 合法值 | 说明 |
|  | qrCode | 扫描二维码 |
|  | barCode | 扫描条形码 |
|  | success | Function |  | 否 | 成功回调 |
|  | fail | Function |  | 否 | 失败回调 |
|  | cancel | Function |  | 否 | 取消回调 |
|  | complete | Function |  | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
|  | resultStr | string | 否 | 扫码结果，needResult为true时返回 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.scanQRCode({
    needResult: 0, // 默认为0，扫描结果由企业微信处理，1则直接返回扫描结果，
    scanType: ["qrCode", "barCode"], // 可以指定扫二维码还是条形码（一维码），默认二者都有
    success: function(res) {
        // 回调
		var result = res.resultStr;//当needResult为1时返回处理结果
    },
    error: function(res) {
        if (res.errMsg.indexOf('function_not_exist') > 0) {
            alert('版本过低请升级')
        }
    }
});
```
