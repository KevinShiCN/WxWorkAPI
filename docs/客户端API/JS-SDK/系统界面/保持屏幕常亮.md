# 保持屏幕常亮

## ww.setKeepScreenOn(params)

> 兼容性：企业微信 iOS、Android 4.0.20 开始支持

### 功能描述

保持屏幕常亮。

在企业微信内打开 H5 页面时，调用该接口让屏幕保持常亮。

      **
      
```ts
ww.setKeepScreenOn({
 keepScreenOn: true,
})
```

    
### 使用限制

- 本接口必须使用应用身份进行注册
- 成员必须在应用可见范围内

### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | keepScreenOn | boolean | 是 | 是否保持常亮 |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |

### 使用说明

仅在当前页面生效，离开页面后设置失效。

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.invoke('setKeepScreenOn', 
 	{
		"keepScreenOn": true,
	},
 	function(res) {
 });
```
