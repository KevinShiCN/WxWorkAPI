# 跳转到认证界面

## ww.enterpriseVerify([params])

> 兼容性：企业微信 iOS、Android 2.8.7 开始支持

### 功能描述

跳转认证界面。

      **
      
```ts
ww.enterpriseVerify()
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.invoke('enterpriseVerify', 
 {		// 此api无需额外参数，空对象即可
 },
 function(res) {
 	 if(res.err_msg == "enterpriseVerify:ok") {
	 // 正常
	 } else {
	 // 错误处理
	 }
 });
```

    
> 备注：此接口要求企业微信版本为为2.8.7及以上
