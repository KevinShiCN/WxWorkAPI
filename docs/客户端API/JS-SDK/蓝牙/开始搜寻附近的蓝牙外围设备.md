# 开始搜寻附近的蓝牙外围设备

## ww.startBluetoothDevicesDiscovery([params])

> 兼容性：企业微信

### 功能描述

开始搜寻附近的蓝牙外围设备。

      **
      
```ts
ww.startBluetoothDevicesDiscovery({
  services: ['FEE7']
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 默认值 | 必填 | 说明 |
| --- | --- | --- | --- | --- | --- |
|  | services | string[] |  | 否 | 蓝牙设备主 service 的 uuid 列表某些蓝牙设备会广播自己主 service 的 uuid。如传入列表非空则只会搜索发出广播包有这个主服务的蓝牙设备，建议主要通过该参数过滤掉周边 不需要处理的其他蓝牙设备 |
|  | allowDuplicatesKey | boolean |  | 否 | 是否允许重复上报同一设备若允许重复上报，则 onDeviceFound 方法会多次上报同一设备，但是 RSSI 值会有不同 |
|  | interval | number | 0 | 否 | 上报设备的间隔若为 0 表示找到新设备立刻上报，否则根据传入的间隔上报 |
|  | success | Function |  | 否 | 成功回调 |
|  | fail | Function |  | 否 | 失败回调 |
|  | cancel | Function |  | 否 | 取消回调 |
|  | complete | Function |  | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
