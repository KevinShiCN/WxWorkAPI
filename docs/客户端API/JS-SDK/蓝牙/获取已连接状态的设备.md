# 获取已连接状态的设备

## ww.getConnectedBluetoothDevices(params)

> 兼容性：企业微信

### 功能描述

根据 uuid 获取处于已连接状态的设备。

      **
      
```ts
ww.getConnectedBluetoothDevices({
  services: ['FEE7']
})
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | services | string[] | 是 | 蓝牙设备主 service 的 uuid 列表 |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
|  | devices | Object[] | 是 | 搜索到的设备列表 |
|  | 属性类型必填说明namestring是蓝牙设备名称，某些设备可能没有deviceIdstring是用于区分设备的 ID |  | 属性 | 类型 | 必填 | 说明 |  | name | string | 是 | 蓝牙设备名称，某些设备可能没有 |  | deviceId | string | 是 | 用于区分设备的 ID |
|  | 属性 | 类型 | 必填 | 说明 |
|  | name | string | 是 | 蓝牙设备名称，某些设备可能没有 |
|  | deviceId | string | 是 | 用于区分设备的 ID |
