# 获取本机蓝牙状态

## ww.getBluetoothAdapterState([params])

> 兼容性：企业微信

### 功能描述

获取本机蓝牙适配器状态。

      **
      
```ts
ww.getBluetoothAdapterState()
```

    
### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
|  | discovering | boolean | 是 | 是否正在搜索设备 |
|  | available | boolean | 是 | 蓝牙适配器是否可用 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.getBluetoothAdapterState({
  success: function (res) {
    console.log(res)
  }
})
```
