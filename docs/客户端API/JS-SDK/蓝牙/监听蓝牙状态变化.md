# 监听蓝牙状态变化

## ww.onBluetoothAdapterStateChange(callback)

> 兼容性：企业微信

### 功能描述

监听蓝牙适配器状态变化。

      **
      
```ts
ww.onBluetoothAdapterStateChange(function(event) {
  console.log(event)
})
```

    
### 参数说明

#### callback: Function

**参数**

**event: Object**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | discovering | boolean | 是 | 是否正在搜索设备 |
|  | available | boolean | 是 | 蓝牙适配器是否可用 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.onBluetoothAdapterStateChange(function(res) {
  console.log(`adapterState changed, now is`, res)
})
```
