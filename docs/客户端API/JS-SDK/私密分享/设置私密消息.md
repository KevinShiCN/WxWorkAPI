# 设置私密消息

## ww.setShareAttr([params])

> 兼容性：企业微信 3.1.8 开始支持

### 功能描述

设置私密消息。

      **
      
```ts
ww.setShareAttr({
  withShareTicket: true,
  state: 'STATE'
})
```

    
### 使用限制

本接口必须使用应用身份进行注册

### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | withShareTicket | boolean | 否 | 分享消息是否携带 share ticket |
|  | state | string | 否 | 详见 私密消息 |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
