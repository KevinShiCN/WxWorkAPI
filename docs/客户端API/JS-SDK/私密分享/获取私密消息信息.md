# 获取私密消息信息

## ww.getShareInfo(params)

> 兼容性：企业微信 3.1.8 开始支持

### 功能描述

获取私密消息信息。

      **
      
```ts
ww.getShareInfo({
  shareTicket: 'xxx'
})
```

    
### 使用限制

本接口必须使用应用身份进行注册

### 参数说明

#### params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | shareTicket | string | 是 | getContext 接口返回的 shareTicket |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |
|  | encryptedData | string | 是 | 转发信息的加密数据 |
|  | iv | string | 是 | 加密算法的初始向量 |

## 旧版jweixin调用

### 示例代码

      
```javascript
wx.invoke("getShareInfo", {
      shareTicket: "xxxx",
        },function(res){
                if (res.err_msg == "getShareInfo:ok")
                {
                	encryptedData = res.encryptedData;
					iv = res.iv;
                }
        }
);
```
