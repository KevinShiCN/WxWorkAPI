# 「获客助手」如何申请合同？

### [](https://open.work.weixin.qq.com)1、管理端操作入口

【管理后台->客户与上下游->客户联系->获客助手->购买->订单记录->订单详情->申请合同】或【管理后台->高级功能->企业订单->获客助手->申请合同】。
![](https://wework.qpic.cn/wwpic3az/681813_g39F76DsSy2xAzZ_1766715614/0)
![](https://wework.qpic.cn/wwpic3az/858994_RkUnDL1vTJS7D1X_1766715641/0)

### [](https://open.work.weixin.qq.com)2、手机端操作入口

【手机端企业微信->消息页面左上角的“三”->左下角高级功能->企业/我的订单->订单记录->选择对应订单->申请合同】
![](https://wework.qpic.cn/wwpic3az/786680_JUSOxLy5QzCtapf_1766715669/0)

温馨提示：
1、如需要了解服务协议，无需双方盖章，可查看[腾讯企业微信获客助手服务协议](https://work.weixin.qq.com/nl/act/p/1e9aaaee936f4f4e?vid=1688850522866577&cst=3503B57B5E571741B5D3A9B57B2C87A823AC04CCB395545BB143DD5A678F1DA19CFB15D4A7400735FCC923558F788F87&deviceid=b8f5dcd7-befc-4de6-be63-5542135e6299&version=4.1.2.99238&platform=mac)。
2、如需线下盖章版合同，请联系客服。
