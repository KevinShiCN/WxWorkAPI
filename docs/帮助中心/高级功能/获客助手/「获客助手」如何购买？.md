# 「获客助手」如何购买？

### [](https://open.work.weixin.qq.com)一、购买入口

#### [](https://open.work.weixin.qq.com)1、手机端/电脑端

【手机端/电脑端->工作台->获客助手->购买】
![](https://wework.qpic.cn/wwpic3az/38737_56Knx2gtTwCjITE_1766714801/0)

#### [](https://open.work.weixin.qq.com)2、管理后台

【管理后台->客户与上下游->客户联系->[获客助手](https://work.weixin.qq.com/wework_admin/frame#/customer/customerAcquisition/index)->购买】
![](https://wework.qpic.cn/wwpic3az/950684_THZVtlLKQheyDhO_1753672001/0)

### [](https://open.work.weixin.qq.com)二、支付方式

移动端支持微信支付和线下转账，电脑端支持微信支付、线下转账和网银支付。

**注：**
**1、管理员和成员都可以购买，建议按需购买；管理员或者成员购买的使用量都是全企业可用。**

**2、使用时长的计算时间是从订单支付时间开始计算。**

**3、获客助手的收款账号是固定收款号。**
