# 「外部联系人规模」线下打款账号怎么获取？

可在手机端企业微信，点击工作台->客户联系与管理->客户群/客户联系->外部联系人规模->购买，或在[管理后台](https://work.weixin.qq.com)->高级功能->外部联系人规模->购买。即可采购外部联系人规模。记得选择“转账汇款”才会显示。
![](https://wework.qpic.cn/wwpic3az/449073_mhBwgOrLSGKRpNc_1753408258/0)

**温馨提示：**
1、对公支付要留意预留的付款人（企业全称）必须跟实际付款人（企业全称）一致；
2、付款时，请务必将本次订单页面显示的识别码填写至“用途/备注/摘要”区域，否则将影响订单正常生效；
3、腾讯的对公账号以本次订单为准，所有信息需要准确无误（如打款金额、打款账号等）；
4、对公支付有2个小时左右的延迟，打款后切记不要取消订单，耐心等待即可，如长时间未生效辛苦联系客服反馈处理；
**温馨提示：以上4点务必准确无误，否则会导致订单无法生效，要走退款流程的呢，退款流程预计需要25个工作日的时间退回。**
