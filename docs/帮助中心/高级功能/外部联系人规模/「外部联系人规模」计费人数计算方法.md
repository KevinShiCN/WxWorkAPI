# 「外部联系人规模」计费人数计算方法

「外部联系人规模」计算方法是包含了企业内成员已添加好友和群聊没有添加好友的客户数合计，计算时系统已经做了去重。外部联系人规模购买您只需要参考「已服务X位」（企业已服务的外部联系人）即可。

管理员可在【手机端->工作台->客户联系/客户群->外部联系人规模->已服务X位】查看企业已服务的外部联系人。
![](https://wework.qpic.cn/wwpic3az/122007_xMbRFqHHT5ymc2M_1766714754/0)

或在【[管理后台](https://work.weixin.qq.com)->高级功能->外部联系人规模】查看企业已服务的外部联系人。
