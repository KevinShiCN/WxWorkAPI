# 「数据保障计划」如何申请合同？

购买「数据保障计划」后，管理员可通过【[管理后台](https://work.weixin.qq.com)->安全与管理>管理工具->数据保障计划->查看全部订单->点击订单后的“更多操作”->申请合同】提交申请。可以申请电子版合同或纸质合同。

**温馨提示：申请合同流程的“甲方签约主体”需填写和开通主体一致的名称。**
