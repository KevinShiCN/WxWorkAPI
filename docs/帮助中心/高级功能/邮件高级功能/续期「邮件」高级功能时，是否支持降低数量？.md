# 续期「邮件」高级功能时，是否支持降低数量？

「邮件」高级功能操作续期时，支持灵活调整购买模式、购买数量、购买时长。

### [](https://open.work.weixin.qq.com)一、如何选择购买模式

购买模式可选：指定范围、自定义数量。
![](https://wework.qpic.cn/wwpic3az/345894_M2Efd5bSR5Oig8S_1766715354/0)
温馨提示：任一购买模式下续费数量最低起购值为5。
若购买过程有其他问题，管理员可在【手机端->消息页面左上角的三->设置->帮助与建议】页面，下拉到底部点击进入【在线客服】联系我们。

### [](https://open.work.weixin.qq.com)二、如何选择购买数量

#### [](https://open.work.weixin.qq.com)1、购买模式为指定范围

选择**指定范围**续费时，支持选择为哪些成员进行续费（即预先分配高级功能账号）。
续期数量不可少于已选择要续期的成员数量，如需续期减购，需先在[为成员续期]中剔除无需续期的的成员。续费成功后，如需修改生效成员范围，可在待生效订单的生效成员范围内调整（订单生效之前都可以手动调整）。
![](https://wework.qpic.cn/wwpic3az/962928_S2YnfcwFSDWO_sR_1766715367/0)

#### [](https://open.work.weixin.qq.com)2、购买范围为自定义数量

选择**自定义数量**续费时，最低起购值为5。
续费时不可预先设置生效成员范围，需要续费成功后，在待生效订单中设置分配高级功能生效成员，如没有在待生效订单中设置高级功能使用成员的情况下，则按高级功能分配的顺序（从早到晚）保留高级功能账号；

注：新版/融合版/协作版续期数量≥原高级功能数量，购买生效后也会自动带上原设置的高级功能成员（包括公共邮箱），无需重新分配；待分配列表为空实际上也不需要重新分配。
![](https://wework.qpic.cn/wwpic3az/76424_63Kp7JOySW2YMYz_1766715390/0)

### [](https://open.work.weixin.qq.com)三、如何选择购买时长

支持在续期页面中，选择购买时长，页面可查看到续期开始时间和结束时间。
![](https://wework.qpic.cn/wwpic3az/398977_o40PBZXKTGqri8y_1766715404/0)
