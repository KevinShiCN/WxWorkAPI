# 什么是「邮件」高级功能？有什么作用？

购买「邮件」高级功能可以获得无限邮箱容量，支持分配更多业务邮箱和提高对外发信数量限制，同时可以开通：邮件自动备份、邮件审核、IP限制和撤回成员发信等邮件管理能力。
