# 「会议」高级功能如何购买？有什么注意事项？

在手机端企业微信，左上角三个短横线->左下角高级功能->会议高级功能->了解更多->购买，即可下单购买「会议」高级功能。

管理员也可以点击[会议高级功能](https://work.weixin.qq.com/mall/txmeeting?wws=19&from=kefu#/txmeeting?wws=19&from=kefu)进入购买页面。

温馨提示：
1、对公支付要留意预留的付款人（企业全称）必须跟实际付款人（企业全称）一致；
2、付款时，请务必将本次订单页面显示的识别码填写至“用途/备注/摘要”区域，否则将影响订单正常生效；
3、腾讯的对公账号以本次订单为准，所有信息需要准确无误（如打款金额、打款账号等）；
4、对公转账/打款后预计2小时左右生效，可在2小时后再查看订单状态和申请发票，若超过2小时还未生效，可联系人工客服核实。请勿取消订单，会导致打款失效。

以上务必准确无误，否则会导致订单无法生效，退款申请提交后，我们将在15个工作日内完成审核。审核通过后，退款金额会退回至您的原支付账户。
在进行支付之前，我们建议您详细阅读支付页面的支付条款。
