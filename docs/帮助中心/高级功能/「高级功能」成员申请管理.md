# 「高级功能」成员申请管理

管理员可在【管理后台->高级功能】管理成员申请的审批模板，也可通过API方式配置，自定义申请流程。

### [](https://open.work.weixin.qq.com)一、申请模板管理

管理员可在【管理后台->高级功能->成员申请管理】自定义设置成员的申请模板内容和审批流程。模板可见范围默认全公司可见，暂不支持调整。
![](https://wework.qpic.cn/wwpic3az/891155_A3wQhr96SFq5R--_1753406904/0)
![](https://wework.qpic.cn/wwpic3az/193757_xglBn3nSRnCox2y_1753406907/0)

### [](https://open.work.weixin.qq.com)二、通过[API](https://developer.work.weixin.qq.com/document/53457)设置申请流程

通过接口可在成员申请中接入企业自建应用的审批流程、回调通知、获取成员申请内容，灵活管理成员申请。
![](https://wework.qpic.cn/wwpic3az/641939_7B_MclFWTSqQco1_1753406912/0)

### [](https://open.work.weixin.qq.com)三、可以关闭高级功能申请入口吗？

暂不支持关闭高级功能申请入口。成员操作申请使用高级功能后，如需取消申请，可撤销审批单，审批单撤销成功后，即可成功取消申请。

操作入口：
【手机端->消息页面左上角的“三”->高级功能->点击已申请的高级功能->高级功能申请详情->左下角的更多->撤销】或【手机端->工作台->审批->已提交】中，找到高级功能申请单进行撤销审批单。
