# 如何修改企业内资料展示

### [](https://open.work.weixin.qq.com)一、如何修改企业内资料展示

管理员支持修改企业内成员在聊天和通讯录等界面显示的成员信息。
支持勾选显示，或取消勾选隐藏对应字段，姓名、部门字段不支持隐藏。还可点击“添加自定义信息”，添加个性字段。

操作路径：
【管理后台->我的企业->通讯录管理->企业内资料展示->全体成员信息显示】

![](https://wework.qpic.cn/wwpic3az/12448_-Va6X7G_Q42rE0l_1753251399/0)

### [](https://open.work.weixin.qq.com)二、如何设置邮箱信息来源

操作路径：【管理后台->我的企业->通讯录管理->成员资料显示->邮箱】勾选邮箱后，邮箱信息来源可选择「企业邮箱」、「邮箱」。
![](https://wework.qpic.cn/wwpic3az/425809_uNDkR9Q5Q7qhqB8_1753251777/0)
![](https://wework.qpic.cn/wwpic3az/272738_qYXBC0TIRC6NUyh_1753251914/0)
1、若同时勾选，会优先展示「企业邮箱」字段；

2、新创建的企业，会默认勾选邮箱字段，邮箱信息来源，会默认勾选「邮箱」；

3、若勾选的是「企业邮箱」，当成员使用的是wecom域onexmail.com的体验域名、成员不在邮件的可见范围/使用范围内，则不会显示。

4、成员可在【手机端消息页面点击左上角“≡”->点击头像进入个人信息页面->邮箱】查看邮箱字段详情来确认个人信息展示的是「企业邮箱」还是「邮箱」。
（1）若提示“管理员已设置显示XXXX”，则展示的是「企业邮箱」；
（2）若显示可修改内容或不可修改，则展示的是「邮箱」。
![](https://wework.qpic.cn/wwpic3az/370840_0_CqQbUWTWqC4bx_1765942288/0)

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、成员资料显示自定义字段支持添加多少个？

对内自定义信息/全体成员信息显示自定义字段：支持添加20个。

对外自定义信息/对外专属信息：文本、网页、小程序类型一共支持添加6个；图文/附件类型支持添加4个。

#### [](https://open.work.weixin.qq.com)2、成员资料显示未勾选邮箱字段，但在客户端通过邮箱可以搜索到成员。

在成员资料显示中取消沟通邮箱之后，客户端可能不会实时更新到；可让成员重新登录企业微信客户端后确认是否恢复。

#### [](https://open.work.weixin.qq.com)3、手机端搜索成员不显示部门信息。

一般情况搜索成员后，姓名的下方会展示部门和职务，当管理员在企业内资料展示中设置了不展示“职务”字段时，在搜索成员时连同部门字段也不会展示出来。
若需要展示部门字段，可在【管理后台->我的企业->通讯录管理->成员资料显示->企业内资料展示->全体成员信息显示】中勾选职务字段即可。
注：取消勾选职务后，电脑端搜索成员姓名依旧会显示部门信息；勾选职务后，查看成员信息页会显示职务。

#### [](https://open.work.weixin.qq.com)4、个人信息页没有别名入口。

（1）企业在【管理后台->我的企业->通讯录管理->成员资料显示->企业内资料展示->全体成员信息显示】中，需勾选展示别名。
（2）若在“企业内资料展示”未勾选别名，但“对外资料显示”中勾选了别名时，需要成员在【手机端->消息页面左上角的三->点击名字->个人信息->对外信息显示->名字显示->选择别名】点击保存后，在个人信息页会有别名入口。
