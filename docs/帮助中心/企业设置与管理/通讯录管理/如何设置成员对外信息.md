# 如何设置成员对外信息

### [](https://open.work.weixin.qq.com)一、什么是对外信息

对外信息指的是对外部联系人展示的成员信息。

### [](https://open.work.weixin.qq.com)二、如何设置对外信息

#### [](https://open.work.weixin.qq.com)1、配置成员展示信息

管理员可在【管理后台->我的企业->通讯录管理->对外资料显示】勾选是否对外展示对应字段信息。
![](https://wework.qpic.cn/wwpic3az/839764_wGEH8aM9TfiiwQg_1753252744/0)
![](https://wework.qpic.cn/wwpic3az/213618_oY9ZJVEvTeCCMyn_1753252870/0)
设置后，成员在【手机端->我左上角“≡”->姓名->个人信息->对外信息显示】可以选择是否开启对外展示对应字段，如职务、手机、姓名显示等。
![](https://wework.qpic.cn/wwpic3az/425949_U--ON0aXRxmE4Gn_1765963546/0)

#### [](https://open.work.weixin.qq.com)2、对外职务

（1）管理员可在【通讯录->组织架构->个人信息->编辑成员信息->对外信息】进行设置。

（2）对外职务可选择同步公司内职务或者自定义，成员也可在客户端自行修改对外职务。
成员可在【手机端消息页面点击左上角“≡”->点击头像进入个人信息页面->对外信息显示】中查看/编辑自己的对外职务。
成员在编辑自己的对外信息时，可选择：「同步公司内职务」或「使用专属对外职务」

- 同步公司内职务：如果该成员有内部职务，则可选择切换为该选项
- 使用专属对外职务：若成员有对外职务编辑权限，可自行填写或编辑对外职务；若无编辑权限，则不可填写或编辑对外职务

![](https://wework.qpic.cn/wwpic3az/425045_I4Wz-NQDRq6GRcT_1765963582/0)

（3）管理员可在【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->对外资料显示】里统一查看和管理成员填写的对外职务。
查看时会展示当前「成员自己填写的对外职务信息」，且管理员可点击「编辑成员信息」去修改成员的对外职务
注：管理员填写的信息不会展示在对外资料显示该页面里，成员自己填写的才会出现在这个页面。
![](https://wework.qpic.cn/wwpic3az/836976_1YyvS3qGSz69J-N_1753253112/0)

（4）管理员可在【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->成员可修改字段】中设置成员是否能修改对外职务。
![](https://wework.qpic.cn/wwpic3az/807788_4qcanTewQkK5pbg_1753253343/0)

#### [](https://open.work.weixin.qq.com)3、设置专属对外信息

管理员可在【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->成员资料显示->对外资料显示->修改->添加自定义信息】中添加专属对外信息；

添加之后可以在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->个人信息->编辑成员信息->对外信息】编辑成员的专属对外信息。
![](https://wework.qpic.cn/wwpic3az/225334_2jJHVf5yRsu_vyx_1753252980/0)
成员可在【手机端->消息页面左上角的三->点击名字->个人信息->对外信息显示】中查看/设置自己的对外信息。
![](https://wework.qpic.cn/wwpic3az/296987_kx-Es_f7TIeOKbp_1765963609/0)
