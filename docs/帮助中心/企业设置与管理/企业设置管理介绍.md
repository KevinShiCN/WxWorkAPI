# 企业设置管理介绍

管理员支持在【管理后台->我的企业】对企业基本信息进行设置，如[企业logo](https://open.work.weixin.qq.com/help2/pc/15024)、[企业地址](https://open.work.weixin.qq.com/help2/pc/15022)/[域名](https://open.work.weixin.qq.com/help2/pc/15023)等。以及管理权限设置，如[管理员权限](https://open.work.weixin.qq.com/help2/pc/17353)、[通讯录权限](https://open.work.weixin.qq.com/help2/pc/20383)、[沟通权限](https://open.work.weixin.qq.com/help2/pc/20402)等。还可以通过【管理后台->安全与管理->管理工具】一系列工具为企业内部管理提升效率，如[使用分析](https://open.work.weixin.qq.com/help2/pc/17362)、[员工服务](https://open.work.weixin.qq.com/help2/pc/17364)、[一周小结](https://open.work.weixin.qq.com/help2/pc/17366)等。
