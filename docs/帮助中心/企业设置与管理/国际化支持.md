# 国际化支持

v4.1.32版本开始，新增「国际化支持」入口，可通过该入口查看「国际化支持」功能介绍，也可以点击[链接](https://work.weixin.qq.com/wework_admin/international?bar_style_type=5&wecomi18n=10)了解「国际化支持」功能。
