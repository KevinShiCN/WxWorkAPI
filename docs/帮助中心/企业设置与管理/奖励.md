# 奖励

## [](https://open.work.weixin.qq.com)**奖励功能目前已下线。**

### [](https://open.work.weixin.qq.com)一、什么是奖励功能

为奖励企业邀请成员使用企业微信，每个新注册的企业将可统一领到10000元红包。成员邀请了同事加入企业后，可领到相应的奖励红包。

### [](https://open.work.weixin.qq.com)二、奖励规则

1、红包有效期
(1) 企业领到红包有效期为2年
(2) 员工超24小时未领取的红包，将退回给企业微信。

2、领取红包规则
(1) 企业领取到10000元红包后，员工才能抢「邀一得一红包」。

(2) 全新注册的员工在激活企业微信账号的时候可以领到一个红包。

(3) 员工邀请同事加入企业（该同事必须是全新注册），邀请人在被邀请同事激活企业微信账号的时候可以领到一个红包，同时被邀请人由于满足规则1可以拿到红包。一个员工最多可以通过邀请同事拿到10个红包。

(4) 企业的10000元红包领完；后续加入的员工无法领取到红包，企业暂时也无法继续领取新的红包 ，（若开通企业支付功能，可充值邀一得一红包。）

(5) 红包统一通过「企业微信团队」下发，员工24小时内不领取将会退回。

### [](https://open.work.weixin.qq.com)三、如何充值奖励红包

企业的奖励红包余额为0时，若企业有开通了企业支付功能，可以在【[管理后台](https://work.weixin.qq.com)->安全与管理->管理工具->奖励】内进行充值。充值之后不支持开发票和退款。

### [](https://open.work.weixin.qq.com)四、邀请同事加入没有领到红包

1、若企业未领取奖励红包，成员无法得到红包；
2、只有邀请全新注册用户加入企业，对方登录后，双方才能获得红包奖励；
3、每个成员最多可领取10个红包。
4、若管理员已邀请过该同事加入企业(即已经在组织架构)，其他同事再邀请也无法领取到邀请红包。
5、目前通过企业微信【[管理后台](https://work.weixin.qq.com)->管理工具->成员加入->复制邀请链接中的链接来邀请成员】，是不会有邀请红包的。

### [](https://open.work.weixin.qq.com)五、常见问题

1、领邀请红包时，显示红包已过期。
企业微信的红包和微信红包一样，有效期为24小时，超过后则会被自动退回，不可再领取，若重新点击已过期红包，则会重新补发。

2、邀请后得到红包，但提示“红包已被领完”。
出现该同事，是因为：您的企业红包已被领完，如需继续领取，可联系管理员在【管理后台->安全与管理->管理工具->奖励】进行红包充值。

3、领取邀请红包后，企业微信绑定的微信没有到账。
需确认近期是否有更换企业微信绑定的微信。若领取红包时，绑定的是A微信，后续换绑到B微信，红包还是实际到账的是A微信。可在之前绑定的微信账号中进行确认。

4、点击邀请同事加入企业共享10000元企业红包卡片的继续邀请，再领红包，提示“处理失败，请联系应用开发者/服务商”。
需确认企业通讯录是否已经满人。目前企业通讯录人数达到上限，无法邀请更多同事。

5、邀请同事获取到的红包，领取后封面显示的简称不是自己的企业名称。
目前企业邀请红包和激活红包的信息是从管理端的企业红包封面设置中获取的，请确认企业红包封面是否为该名称。若企业更改过企业简称，已生成的红包封面中的企业名称不会自动变更，企业管理员可以在【管理后台->应用管理->红包封面】中，查看和修改当前的红包封面信息。
