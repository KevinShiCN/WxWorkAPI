# 一周小结

### [](https://open.work.weixin.qq.com)一、什么是一周小结

一周小结是为管理员、管理者、成员汇总的一周小结，总结使用企业微信的情况。
内容包括成员一周小结、部门使用小结、企业使用小结。

### [](https://open.work.weixin.qq.com)二、如何使用一周小结

设置路径：【管理后台->安全与管理->管理工具->一周小结】
![](https://wework.qpic.cn/wwpic3az/615197_S1At_ruRR-CzIJZ_1753336476/0)
![](https://wework.qpic.cn/wwpic3az/157527_aojwW-9QRDuD4Lt_1753336488/0)

#### [](https://open.work.weixin.qq.com)1、一周小结

● 成员在一周内有登录过企业，在周五18:00左右，在手机端会收到一周小结的推送。
● 一周小结的内容主要包括处理工作聊天次数、一共花了多长时间、最晚工作时间。
注：
工作聊天次数是指进入会话次数。
一共花了多长时间是指处理会话的时间。
最晚时间是指打开企业微信的最晚时间。

#### [](https://open.work.weixin.qq.com)2、部门使用小结

● 企业成员是部门的负责人的时候，每周一会收到系统推送的上周的部门使用小结。
● 部门小结会显示上周部门使用率以及未使用人数，未使用成员指的是一周内未登录过企业微信APP的成员，点击通知未使用的成员进入可以查看具体名单；
● 若不想要接收部门使用小结，超级管理员可以在【管理后台->安全与管理->管理工具->一周小结】中关掉部门使用小结。

#### [](https://open.work.weixin.qq.com)3、企业使用小结

● 超级管理员可以在每周一收到上周的企业使用小结。企业小结主要是统计上周企业内使用率以及未使用人数。
● 管理后台需要同时打开“一周小结”以及‘“企业使用小结”，系统才会下发企业使用小结。
● 未使用成员指的是一周内未登录过企业微信APP的成员，点击通知未使用的成员进入可以查看具体名单。

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、为什么没有收到《气泡君》推送？

（1）需要打开《气泡君》推送；
（2）气泡君是不定时进行推送的。
![](https://wework.qpic.cn/wwpic3az/652529_6xh5GFasSjSWiV__1753336548/0)

#### [](https://open.work.weixin.qq.com)2、为什么没有收到一周小结？

（1）企业“一周小结”的开关需要开启状态。
开启入口：【管理后台->安全与管理->管理工具->一周小结】
（2）一周小结会在每周五下午6点左右推送，推送的时候在登录状态的企业才会收到推送，如果在这个推送时间未登录到这个企业的成员将不会收到。
（3）下发推送的时候，如果成员使用的企业微信版本不是v2.4.22以上的话，则不会收到推送。
![](https://wework.qpic.cn/wwpic3az/548093_2edCRyQJSkuxmcB_1753336513/0)

#### [](https://open.work.weixin.qq.com)3、为什么没有收到企业使用小结？

（1）“一周小结”和“企业使用小结”需同时开启，超级管理员才能收到“企业使用小结”；
（2）企业使用小结会在每周一进行推送，确认无法收到推送人员在周一时使用的版本是否为最新版本；
（3）若管理员太久未登录过企业不会收到推送，一般在一周内有登录过企业的会收到，即使在推送的时候没有登录到对应的企业。
（4）若同时是超级管理员和部门负责人的身份，仅会给超级管理员身份下发企业使用小结。
（5）若未收到企业使用小结的超级管理员被设置了通讯录查看权限（限制查看外部门、隐藏部门/成员、限制查看所有人），则不会推送企业使用小结给该管理员。
注：若关闭了一周小结，但开启了企业使用小结，可能会出现短期内继续收到企业使用小结的情况，但后续没有继续收到。

#### [](https://open.work.weixin.qq.com)4、支持统计一周小结的场景

（1）工作小结在周一0点清零，目前一周小结一般会在周五18:30至21:30之间推送。
（2）电脑端和手机端的在线时间。
（3）会话中发起语音/视频通话，会算进一周小结处理会话时间中
（4）发起直播后，如果直播一直在前台直播会算进一周小结的处理时间中。
（5）一周小结统计的会话包含与文件传输助手发送的内容，计算为1个会话。
（6）一周小结统计的会话包含与企业微信的在线客服对话。

#### [](https://open.work.weixin.qq.com)5、不支持统计一周小结的场景

（1）周末不统计工作时长。
（2）微信插件中的使用情况不会统计。
（3）一周小结的统计不包含员工服务处理时间、会话数。
（4）电脑端打开会话之后最小化客户端，不会算进一周小结的处理时间中。
（5）操作打卡不会计算到处理工作会话中。
