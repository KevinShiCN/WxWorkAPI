# 企业信息中的联系电话作用

### [](https://open.work.weixin.qq.com)一、联系电话用途

添加企业电话，在外部联系人查看成员企业信息的时候，会展示对应的电话信息。
![](https://wework.qpic.cn/wwpic3az/521290_Ch1GDtSiTwKZSer_1730704964/0)

### [](https://open.work.weixin.qq.com)二、联系电话设置入口

可以在【管理后台->我的企业->企业信息->联系电话】后点击添加或修改。
![](https://wework.qpic.cn/wwpic3az/786230_cinC4S-tSOuG-1D_1753238190/0)
