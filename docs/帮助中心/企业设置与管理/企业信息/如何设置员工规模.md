# 如何设置员工规模

### [](https://open.work.weixin.qq.com)一、员工规模的作用

不同企业需要的功能和服务可能不一样，补充行业类型和人员规模后，有助于获取我们提供的不同功能和服务。

### [](https://open.work.weixin.qq.com)二、如何修改员工规模人数

注册企业微信的时候，需要选择对应的员工规模人数，选择后可以在【管理后台->我的企业->企业信息->员工规模】或【手机端->工作台->管理企业->企业信息->员工规模】入口进行修改。

注：三十天内只能修改一次，员工规模人数目前暂时不会影响[企业成员人数扩容](https://open.work.weixin.qq.com/help2/pc/14936)。
