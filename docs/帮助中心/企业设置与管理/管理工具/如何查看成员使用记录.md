# 如何查看成员使用记录

1、成员使用记录内可以看到成员最后一次使用企业微信的时间以及使用的平台是什么；记录的是成员最近一次登录，或者从休眠、锁屏激活的时间。（离职成员不记录）

2、成员使用记录的平台有：ios(iPhone)、android、Mac、windows、微信插件。
注：使用鸿蒙系统将会记录为android。

3、成员使用记录可以支持查询近45天内的使用情况。

4、查看路径：【管理后台->安全与管理->管理工具->使用分析->成员使用记录】
