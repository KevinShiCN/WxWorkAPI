# 如何限制分级管理员的“可查看的成员信息”

1、管理员可限制分级管理员在管理端通讯录查看成员时，可查看的成员信息。
该设置仅在查看“具有查看权限”范围的成员时生效，查看“具有管理权限”范围的成员则不生效。

2、分级管理员被设置了通讯录的不可见字段后，在通讯录中查看成员信息时，不可见的字段会显示“无权限查看”。

3、姓名、别名、账号、部门是默认可见的，其他的字段可由超级管理员自主选择可见哪些信息。

4、操作路径：【管理后台->我的企业->权限管理->点击分级管理员右侧的“编辑”按钮】。
