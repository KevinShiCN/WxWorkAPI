# 管理员的管理范围与通讯录权限范围有什么不同？

### [](https://open.work.weixin.qq.com)1、管理范围

**管理范围**为管理员在进行管理操作时，所能影响到的成员范围。对管理员权限的设置均不得超出管理范围。支持设置“特定部门”和“管理员所在部门及下级部门”。

### [](https://open.work.weixin.qq.com)2、通讯录权限范围

管理员的**通讯权限范围**是在“管理范围”内设置可管理通讯录的范围，是“管理范围”的子集。管理员可查看或管理、在其通讯录权限范围内成员的通讯录信息。
