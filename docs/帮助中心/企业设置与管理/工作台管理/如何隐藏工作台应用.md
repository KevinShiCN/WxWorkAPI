# 如何隐藏工作台应用

### [](https://open.work.weixin.qq.com)一、隐藏工作台应用方式

#### [](https://open.work.weixin.qq.com)1、在应用分组中隐藏应用

管理员可在【管理后台->应用管理->工作台管理/我的企业->工作台管理->应用分组->修改分组与排序】中隐藏应用。被隐藏的应用可在【手机端->工作台->更多】里查看。

（1）被企业隐藏的应用，成员也可以自己操作恢复显示。

（2）当成员原本有操作过隐藏的应用又恢复了显示，此时企业操作隐藏，则不会隐藏这个应用。

（3）工作台->其他里面的应用不支持管理后台设置隐藏。（仅成员自己设置隐藏，企业无法统一操作隐藏）

![](https://wework.qpic.cn/wwpic3az/276354_Ojg01hoQQd-NyDB_1753257454/0)

#### [](https://open.work.weixin.qq.com)2、设置使用范围隐藏应用入口

管理应用时，支持设置应用可见范围，不在可见范围内的成员，查看工作台时将不展示对应应用。
【管理后台->应用管理->点击需设置的应用->可见范围】
![](https://wework.qpic.cn/wwpic3az/509364_Ei7H25IzTfOLXw7_1753257748/0)
【手机端->管理企业->应用管理->点击需设置的应用->可见范围】
![](https://wework.qpic.cn/wwpic3az/881345_4OV0LmMWThq_4gl_1765267300/0)

#### [](https://open.work.weixin.qq.com)3、管理成员可见入口

可在“成员可见”设置中，关闭上下游、第三方应用、上门服务的展示入口。
（1）上下游：关闭后，将对未加入任何上下游的成员隐藏“上下游”入口，超级管理员可见。
（2）第三方应用：关闭后，将对普通成员隐藏“第三方应用”入口，仅具有创建应用与小程序权限的管理员可见。
（3）上门服务：关闭后，将对所有工作台内的普通成员隐藏“上门服务”入口，仅超级管理员可见

【管理后台->我的企业->工作台管理->成员可见】
![](https://wework.qpic.cn/wwpic3az/843625_Qf7D2JXNSGeRE6v_1753257615/0)
【手机端->工作台->管理企业->工作台显示->工作台应用显示】
![](https://wework.qpic.cn/wwpic3az/109297_I8gHVrsoQ9qBtv4_1765267326/0)

#### [](https://open.work.weixin.qq.com)4、 对可见范围内成员隐藏自建应用

管理员登录管理后台在【应用管理->自建应用->企业微信授权登录】左下角可勾选“在工作台隐藏此应用”，设置后仅使用“企业微信登录”，在工作台会看不到对应的应用，该设置不影响接收应用消息。
![](https://wework.qpic.cn/wwpic3az/766738_GKoeBi-ASMGrXgM_1753258069/0)
![](https://wework.qpic.cn/wwpic3az/443320_yttRhMABSFSY50X_1753258113/0)

### [](https://open.work.weixin.qq.com)二、成员隐藏自己的工作台应用

在【手机端->工作台->右上角】进入编辑状态，选择对应的应用隐藏。被隐藏的应用可以在“更多”入口查看。
（仅操作者账号生效，管理员不可通过该操作隐藏成员侧应用）
