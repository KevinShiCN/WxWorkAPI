# 如何设置应用排序

可以添加多个应用分组，把需要分类的应用按照自己设置的分组进行添加排序，调整排版布局，便于使用。

### [](https://open.work.weixin.qq.com)一、如何在管理后台调整应用排序

管理员可在【[管理后台](https://work.weixin.qq.com)->我的企业->工作台管理->修改应用分组排序】选择对应的应用分组右侧三点，通过上移或下移进行应用分组排序，或编辑分组->点击应用左侧三横拖动应用排序。
![](https://wework.qpic.cn/wwpic3az/333824_3qCQ_hIpTNCG5sr_1753256235/0)
![](https://wework.qpic.cn/wwpic3az/968438_KMyXXeTxQAi_W6Q_1753256278/0)
![](https://wework.qpic.cn/wwpic3az/490429_O6UeqakqTAaL7So_1753256637/0)

### [](https://open.work.weixin.qq.com)二、如何在手机端调整应用排序

手机端操作路径：【手机端->工作台->管理企业->工作台显示->应用分组排序->编辑->点击需要调整顺序的应用右侧三横拖动排序】
![](https://wework.qpic.cn/wwpic3az/901206_UorDKqrBTBGeHNY_1762332357/0)

### [](https://open.work.weixin.qq.com)三、如何设置常用应用

支持在工作台中将应用添加到常用栏，使用应用更方便。
注：没有添加任何常用应用的情况下，工作台是没有“常用”这个区域的。

1、手机端操作方法：
【手机端->工作台->长按应用】即可添加至常用，添加后长按应用即可从常用中移除。
![](https://wework.qpic.cn/wwpic3az/725644_WTtPrXAxRMe9bvN_1765963280/0)
或是在【手机端->工作台->右上角->编辑工作台】页面中，操作添加至常用，添加后可以在编辑工作台页面中操作从常用中移除。
![](https://wework.qpic.cn/wwpic3az/622055_2M4KmE2QRB2rukV_1765963297/0)
同时支持在【手机端->第三方应用内的右上角“…”->添加至常用】
![](https://wework.qpic.cn/wwpic3az/76715_uFwH3yVQSr-LT7m_1765963330/0)

2、电脑端操作方法：
电脑端可以在工作台中，将鼠标移动到需要添加的应用上，点击右上角的“···”即可操作添加至常用，添加后可以按照相同的方法点击应用右上角的“···”操作从常用中移除。
![](https://wework.qpic.cn/wwpic3az/407141_UEUpytKhQwi-SmS_1750132108/0)
![](https://wework.qpic.cn/wwpic3az/380674_pqITpQg9Q2qwNR1_1750132124/0)
或是在【电脑端->工作台->打开应用后，顶部栏右键此应用tab->添加至常用】
![](https://wework.qpic.cn/wwpic3az/196501_dLx0P850QjOoiiY_1750132155/0)
也可以在【电脑端->工作台->应用内页的右上角“…”->添加至常用】
![](https://wework.qpic.cn/wwpic3az/306119_qT1Y-gifTSWXFuS_1750132177/0)
