# 如何设置工作台显示模式

企业微信的工作台应用默认显示模式为“宫格模式”，管理员支持根据企业使用习惯，将显示模式切换为“列表模式”。

设置路径：【[管理后台](https://work.weixin.qq.com)->我的企业->工作台管理->显示模式】
![](https://wework.qpic.cn/wwpic3az/798111_3EfBvoFIRbyAO5J_1753254957/0)

【手机端->工作台->管理企业->工作台显示->显示模式】
![](https://wework.qpic.cn/wwpic3az/797497_YbO0lxfpQT2H3f5_1765942330/0)
