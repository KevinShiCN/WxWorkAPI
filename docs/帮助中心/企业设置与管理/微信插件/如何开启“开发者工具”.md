# 如何开启“开发者工具”

管理员登录【管理后台->我的企业->微信插件->最底端->开发者工具】中进行勾选开启。开启后，当前管理员可在工具内模拟微信网页，方便开发与调试。

每个超级管理员、分级管理员都可以查看该入口，若需使用该工具，则需要使用人登录后台进行勾选开启。
