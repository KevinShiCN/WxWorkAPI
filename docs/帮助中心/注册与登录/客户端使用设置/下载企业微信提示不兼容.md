# 下载企业微信提示不兼容

#### [](https://open.work.weixin.qq.com)一、IOS（苹果）版

苹果（ios）系统需要iOS 11或更高以上的版本；iPhone5不支持下载企业微信；
目前只支持64位的设备，可与苹果客服确认当前使用的设备是否属于64位的设备。
不支持下载设备下载企业微信会出现“无法下载App，企业微信与此设备不兼容”的相关提醒。

#### [](https://open.work.weixin.qq.com)二、Android（安卓）版

需要android5.0或以上。
若安卓系统是32位的，需在应用商店中搜索企业微信进行下载，暂不支持在官网中下载。
