# 如何进入工作台

**注:** [团队形态](https://open.work.weixin.qq.com/help2/pc/21059)没有“工作台”入口。

### [](https://open.work.weixin.qq.com)一、手机端工作台入口

【手机端登录企业微信->下方菜单栏->工作台】。
![](https://wework.qpic.cn/wwpic3az/829534_Tjw3dqFORgC1D_T_1764559655/0)

### [](https://open.work.weixin.qq.com)二、电脑端工作台入口

【电脑端登录企业微信->左边菜单栏->工作台】。
![](https://wework.qpic.cn/wwpic3az/429462_8c8LNc3iQqK7dJR_1773905510/0)

### [](https://open.work.weixin.qq.com)三、如何在工作台搜索/添加应用

1、工作台页面上方支持搜索，查找应用更快捷。
![](https://wework.qpic.cn/wwpic3az/142210_X9kbuhWRRJ6QD5h_1764559664/0)
2、支持将应用添加到常用栏，使用应用更方便。
![](https://wework.qpic.cn/wwpic3az/62047_YLG945-_SWyW2bD_1764559667/0)
