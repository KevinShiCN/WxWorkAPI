# 注册企业微信

### [](https://open.work.weixin.qq.com)一、如何在官网注册企业微信

在[官网](https://work.weixin.qq.com/)点击“立即注册”即可注册企业。
![](https://wework.qpic.cn/wwpic3az/680200_FSAioBTTRIGzRzA_1755662112/0)
注册时，需填写企业信息（企业名称、行业类型、员工规模），同意并遵守[《腾讯企业微信服务协议》](https://work.weixin.qq.com/nl/eula)[《隐私政策》](https://work.weixin.qq.com/nl/privacy)，使用手机微信扫码补充管理员信息（管理员姓名、管理员手机号、短信验证码）即可提交注册。
![](https://wework.qpic.cn/wwpic3az/471465_rhe9syxtQHiHefw_1772517550/0)

### [](https://open.work.weixin.qq.com)二、如何在APP注册企业微信

#### [](https://open.work.weixin.qq.com)1、创建企业

在【手机端->消息界面->左上角三横->创建/加入企业->全新创建企业】
![](https://wework.qpic.cn/wwpic3az/88241_o95LYMvqQQOoeA8_1755662551/0)
或【手机端->消息页面->左上角三横->设置->管理企业->全新创建企业】。
![](https://wework.qpic.cn/wwpic3az/165916_J4dpMHrJQ6Cdva5_1755663017/0)
注册时需要选择企业类型、补充企业信息（企业名称、行业类型、员工规模、创建人姓名）
![](https://wework.qpic.cn/wwpic3az/516493_acUiObjIQm-iXv8_1755678100/0)

#### [](https://open.work.weixin.qq.com)2、个人组建团队

如果需要个人组建团队，可以在手机端创建企业页面底部点击“个人组建团队”入口，补充团队名称以及注册人姓名，即可注册成功（团队模式暂不支持登录mac端、扩容微盘容量、登录管理后台）。
![](https://wework.qpic.cn/wwpic3az/461538_a2QGsNSURSKNa_L_1755662392/0)

### [](https://open.work.weixin.qq.com)三、企业微信创建/注册/认证/验证上限是多少

1、一个企业微信账号可以作为5个企业的创建人。
2、企业微信账号支持同时绑定一个微信号和一个手机号。
3、一个微信号/手机号不支持同时绑定多个企业微信账号。
（例如，用户张三已经在企业微信中，操作绑定了手机号123/微信abc，那么李四在注册/绑定时，就无法绑定手机号123/微信abc）
4、同一个主体证件能验证/认证5个企业微信。

### [](https://open.work.weixin.qq.com)四、企业微信注册是否收费

**企业微信目前免费使用**。其他自费项：
1、300元或以上企业认证费用，由腾讯收取；
温馨提示： 若申请企业微信认证功能，有效期为一年。
2、企业采购接入的第三方软件、企业公费电话话费、微盘扩容等企业业务费用

腾讯保留本软件的任何服务或功能后续收费的权利。腾讯有权提前七（7）日通知用户收费标准，若用户继续使用则需按腾讯公布的收费标准支付费用。
详细说明请留意[服务协议](https://work.weixin.qq.com/nl/eula)。

### [](https://open.work.weixin.qq.com)五、注册/创建企业的企业名称的命名要求

1、由3-30个中文、英文、数字及合法字符组成。
2、当前在web端注册企业微信时填写的企业名称只有英文才允许有空格，中文名称不允许有空格，但是在手机端注册企业微信时中文名称可以带空格。
3、手机端、web端在注册/创建企业微信时，企业名称不支持包含“,” 字符。
4、目前企业名称暂不支持生僻字，这个情况我们将在后续优化。

### [](https://open.work.weixin.qq.com)六、常见问题

#### [](https://open.work.weixin.qq.com)1、创建企业提示“创建失败”

（1）需确认微信是否有实名，若没有实名需完成实名后再操作。
（2）需检查创建页面的内容是否全部填写完毕，例如姓名没有填写，也会创建失败。

#### [](https://open.work.weixin.qq.com)2、创建企业提示“为验证你的身份，请用微信绑定本人银行卡后再进行注册”

创建人的微信实名和创建人姓名需一致，微信侧需绑定中国大陆已实名的银行卡。
