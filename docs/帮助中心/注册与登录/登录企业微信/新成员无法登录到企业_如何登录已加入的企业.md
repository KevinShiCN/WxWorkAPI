# 新成员无法登录到企业/如何登录已加入的企业

### [](https://open.work.weixin.qq.com)一、新成员无法登录到企业微信

1、新成员需要先加入企业，管理员可在[通讯录添加成员账号](https://open.work.weixin.qq.com/help2/pc/16490)后，成员在登录时，可以通过手机号或邮箱匹配到对应企业后成功登录企业微信。

2、企业内的成员或管理员可以[邀请新成员加入企业](https://open.work.weixin.qq.com/help2/pc/16490)。新成员点击确认加入后，可下载企业微信app登录到对应的企业。

3、登录时输入的手机号需和管理员在[通讯录录入](https://open.work.weixin.qq.com/help2/pc/16490)的手机号一致，若不一致，则无法找到对应的企业进行登录。

### [](https://open.work.weixin.qq.com)二、已登录过对应的企业，登录时找不到这个企业

1、可以联系管理员在通讯录检查对应的账号是否在通讯录中，登录时找不到对应的企业，有可能是被管理员误删除了。
未参与 “数据保障计划” 的企业，对应的成员可能需管理员重新添加后才能继续登录。
若企业提前参与 “数据保障计划” 的情况下，有可能找回原账号的记录和数据，详情见[“数据保障计划”功能说明](https://open.work.weixin.qq.com/help2/pc/18154)。

2、可以联系管理员在通讯录检查对应的账号是否为“[已退出](https://open.work.weixin.qq.com/help2/pc/14962)”状态。若是主动退出状态，需管理员点击邀请后，需在30天内登录到企业中。若管理员未在通讯录进行邀请加入或已经邀请，成员未重新登录到对应的企业，30天后账号会被系统自动删除，系统自动删除的账号数据不支持恢复。

3、登录时输入的手机号需和管理员在[通讯录录入](https://open.work.weixin.qq.com/help2/pc/16490)的手机号一致，若不一致，则无法找到对应的企业进行登录。

### [](https://open.work.weixin.qq.com)三、登录企业时提示“让身边的1位同事扫码确认你的身份即可加入企业”

当账号没有加入任何企业，在加入企业时会出现该提示，建议联系已加入企业的同事，让同事在他的企业微信手机端的消息页面，点击右上角“+” ->扫一扫，扫描页面显示的二维码，协助您确认身份。

若是已在通讯录的成员使用微信登录时仍显示“让身边的同事扫码确认你的身份即可加入企业”，请确认成员登录时使用的微信与管理后台->通讯录成员详情页中的微信是否为同一个。需使用企业微信绑定的微信进行登录。
