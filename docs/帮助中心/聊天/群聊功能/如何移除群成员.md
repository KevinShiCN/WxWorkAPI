# 如何移除群成员

群主或群管理员支持移除群成员。

### [](https://open.work.weixin.qq.com)一、手机端移除群成员入口

【手机端->进入群聊->右上角三个点->减号->选择需要移出的成员】
![](https://wework.qpic.cn/wwpic3az/817779_FW-FMC9DRhieN-o_1764828993/0)

### [](https://open.work.weixin.qq.com)二、电脑端移除群成员入口

【电脑端->群成员列表->右键群成员名字->移出群聊】
![](https://wework.qpic.cn/wwpic3az/557126_EroEvFDASGuzX0A_1750324125/0)

### [](https://open.work.weixin.qq.com)三、外部群移出成员提示“移出失败”

1、若单次移出的群成员超过40人，建议减少单次移出成员的数量到40人以下后再尝试操作。

2、若未超过40人，可让群主在【手机端->左上角“≡” ->设置->退出登录】后，再使用企业微信绑定的微信账号重新登录重试。
