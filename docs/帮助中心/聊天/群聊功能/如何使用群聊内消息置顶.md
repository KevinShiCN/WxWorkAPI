# 如何使用群聊内消息置顶

1、在多人聊天时，可以长按某一条消息气泡，选择“置顶”，即可把当前消息（文字消息、文件、图片等）置顶到群聊窗口顶部，方便群聊成员查看，避免遗漏。

2、群聊成员可以移除消息置顶，不影响其他成员查看。

3、置顶创建者移除，会将所有群成员的消息置顶移除。

4、消息置顶不会自动过期，每个群聊同一时间内最多展示5条消息置顶。如果已经有5条置顶消息，再操作新的置顶，会提示将替换最早置顶的消息。

5、被置顶的消息，即使这个消息记录被删除/撤回了，这个消息也依然会保持置顶的，但是点击这个置顶消息会提示“无法定位到原消息”。

6、外部群目前暂不支持除群主/群管理员身份以外的人操作置顶，普通群成员身份会看不到置顶入口。

7、文本卡片消息（text card消息）、分享的历史消息（邀请其他人进群时附带的消息）暂不支持置顶。

8、开启“仅群主可管理”之后，群员不能操作置顶，只能群主和群管置顶消息。

9、群主/群管支持移除他人置顶的消息，且移除置顶消息后，群内全员生效。

10、微信联系人支持查看群聊置顶消息。

11、旧版本企业微信客户端看不到置顶消息，并且会有提示“暂不支持此类型消息”。建议更新至最新版本查看。

12、暂不支持设置“仅自己置顶”。

13、若群内被设置了禁言，群主或群管理员仍可以操作置顶消息。
