# 如何设置语音和视频通话提醒

手机端支持设置语音和视频通话提醒，语音和视频通话提醒这个开关不会受到接收新消息通知这个开关设置的影响。

v5.0.2版本开始，苹果手机在锁屏、非锁屏状态下可快捷接听视频来电，无需解锁进入企微。

设置入口：【手机端->消息页面左上角“≡”->设置->消息通知->开启语音和视频通话提醒】同时可开启语音通话用弹窗快捷接听。
