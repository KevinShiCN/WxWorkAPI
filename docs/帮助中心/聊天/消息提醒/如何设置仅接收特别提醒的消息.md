# 如何设置仅接收特别提醒的消息

支持在【手机端->消息界面左上角“≡”->设置->新消息通知】中设置“仅接收特别提醒的消息”。

设置仅接收特别提醒的消息后，只对被设置了特别提醒的人发送的消息会有角标显示以及消息通知，非特别提醒的人不会有角标显示以及消息通知。
