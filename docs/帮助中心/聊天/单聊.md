# 单聊

单聊是指跟单成员进行沟通的聊天（一对一聊天），企业微信单聊有内部单聊和外部单聊两种类型。

### [](https://open.work.weixin.qq.com)一、内部单聊

发起会话时，若选择了本企业的人员进行对话，那么这个会话是一个内部单聊。
![](https://wework.qpic.cn/wwpic3az/246779_NZjOReRrSYKCasc_1764664186/0)

### [](https://open.work.weixin.qq.com)二、外部单聊

发起会话时，若选择了外部企业的人员或微信联系人对话，那么这个会话属于外部单聊。
![](https://wework.qpic.cn/wwpic3az/691223_uVHbaO1uTOG3tSH_1764664215/0)
