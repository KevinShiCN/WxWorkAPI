# 如何修改群名

### [](https://open.work.weixin.qq.com)一、谁可以修改群名

若群主未开启「仅群主或群管理可管理」开关，群成员支持修改群名。若开启了「仅群主或群管理可管理」，那么仅群主或群管理可修改群名。

外部群可通过「禁止改群名」限制群成员修改群名，所有外部群都有该选项；开启后群主和群管理员才能成功变更群聊名称，如果是成员修改群名，成员修改群名后，将会自动以群主身份将群名修改回原来的群名。
注：mac端暂不支持设置「禁止改群名」。

### [](https://open.work.weixin.qq.com)二、操作入口

#### [](https://open.work.weixin.qq.com)1、手机端修改群名入口

【群聊右上角三个点->点击群聊名称->输入名称->确定】
![](https://wework.qpic.cn/wwpic3az/236251_BZg2AGcFT3KcXgX_1764830424/0)

#### [](https://open.work.weixin.qq.com)2、电脑端修改群名入口

在群聊会话页面，可点击群聊名称直接修改，或在点击群聊右上角“…”->群聊名称，操作修改群名。
![](https://wework.qpic.cn/wwpic3az/517387_fjYEAcnHQGm6YVk_1750325733/0)

### [](https://open.work.weixin.qq.com)三、群聊的群名称字数上限和更改次数上限

1、内部群聊的群名称字数上限：50个字。
2、外部群聊的群名称字数上限：42个字。
3、修改群名称频率：暂无上限。

### [](https://open.work.weixin.qq.com)四、常见问题

#### [](https://open.work.weixin.qq.com)1、win端群聊的群名称展示不完整

需确认群聊名称是否带有换行；当前群名只支持单行显示，可调整为单行后重新设置。

#### [](https://open.work.weixin.qq.com)2、群聊名称未到50字被截断，名称显示不完整

群名称包含文字、符号等只要满50个字符就会被截断，超过50个字符的则不会显示，符号也会计算在字符数中。

#### [](https://open.work.weixin.qq.com)3、电脑端修改群名称不生效

当前PC端在群聊名称位置直接输入/修改群名称后，鼠标点击群名称旁边的空白区域（title/标题区域），群名称会修改不生效，可输入群名称后点击会话区域/回车键。

#### [](https://open.work.weixin.qq.com)4、群名称字体显示异常/群聊名称字体粗细不一致

请确认电脑系统是否有安装“微软雅黑”字体，若有，需重新安装“微软雅黑”字体后重试。
