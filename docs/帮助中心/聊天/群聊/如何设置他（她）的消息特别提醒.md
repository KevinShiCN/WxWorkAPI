# 如何设置他（她）的消息特别提醒

设为消息特别提醒后，对应的联系人发来消息有特别的提醒音效，也会在消息列表特别提醒。

目前有2种方式可以设置消息特别提醒。可在【设置->新消息通知】中查看已设置的联系人和修改提醒音效。

1、点击成员或客户，进入【个人信息页面->右上角三个点->他（她）的消息特别提醒】选择开启。
![](https://wework.qpic.cn/wwpic3az/108837_PB3btM8KQMC-bpG_1764837243/0)

2、在【手机端->消息页面左上角“三”->设置->新消息通知->他（她）的消息特别提醒】进行设置。
![](https://wework.qpic.cn/wwpic3az/695706_5IbFkPSKQJa1O1L_1764836844/0)
