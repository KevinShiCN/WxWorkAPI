# 如何设置群禁言

### [](https://open.work.weixin.qq.com)一、如何设置群禁言

仅**内部群**支持设置群禁言。外部群暂不支持设置群禁言。
群主和群管理员支持设置群内禁言，可开启【全员禁言】或选择部分群成员禁言。

手机端设置入口：
【群聊会话右上角三个点->群管理->设置群内禁言】
![](https://wework.qpic.cn/wwpic3az/101590_uJsC_KmSTb-CA1K_1764831389/0)

电脑端设置入口：
【群聊会话右上角三个点->群管理->群成员禁言】
![](https://wework.qpic.cn/wwpic3az/188973_WcflX1xUQiCuQyj_1750325111/0)

### [](https://open.work.weixin.qq.com)二、常见问题

1、开启群禁言设置了可发言成员，操作移出可发言成员时列表不显示需要被移出的成员
需确认需被移出的成员是否是群管理员，目前群管理员不能被禁言。

2、在内部群->群管理->群成员禁言先添加成员为可以发言的人，再在群管理员中添加该成员，成员不会显示在群管理员中
出现该情况是因为群管理员暂不支持被禁言，故一个成员不能被同时禁言且同时被设置成管理员。
