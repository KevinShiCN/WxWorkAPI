# 如何设置群管理员

企业微信侧发起的群聊仅支持设置同企业的人员为群管理员，其中，客户群设置的群管理员需为同企业且有客户联系使用权限的成员；
微信侧发起的外部群支持设置微信联系人或企业微信联系人为群管理员。

如何设置多个群管理员？[点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488326713827072_5)

### [](https://open.work.weixin.qq.com)一、设置群管理员路径

#### [](https://open.work.weixin.qq.com)1、电脑端设置群管理员入口

群主在【会话列表右击群聊->点击右上角“…”->群管理->在群管理页面往下拉->在群管理员处点击+号】即可添加群管理员。
![](https://wework.qpic.cn/wwpic3az/568593_BX3b6OyQSF2muI3_1764829650/0)

#### [](https://open.work.weixin.qq.com)2、手机端设置群管理员入口

群主在【消息界面打开群聊->点击右上角“…”->点击群管理->点击群管理员->点击+号】即可添加群管理员。
![](https://wework.qpic.cn/wwpic3az/864872_mslYOYnPS5qRONx_1764829682/0)

### [](https://open.work.weixin.qq.com)二、群管理员上限

每个群聊仅支持添加3个群管理员。

注：在企业微信侧发起的外部群，仅支持设置为企业内账号为群管理员，不支持设置微信账号或企业外账号为群管理员。

### [](https://open.work.weixin.qq.com)三、常见问题

1、选择群管理员时，无法选择某个成员
若群主或群成员被设置了通讯录查看权限限制，群主在通讯录中无法查看到需要设置的群管理员，群主在设置群管理员时，可以在添加群管理员列表中看到该成员，但勾选按钮会置灰，无法操作选择。
