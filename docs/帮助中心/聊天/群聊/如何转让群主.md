# 如何转让群主

### [](https://open.work.weixin.qq.com)一、手机端转让群主入口

1、企业微信用户创建的群聊，支持在企业微信手机端操作转让群主：
【群聊->点击右上角“…”->群管理->转让群主】
![](https://wework.qpic.cn/wwpic3az/402800_EUMtw0-DQsqnz8r_1764829990/0)

2、微信用户创建的外部群，支持在微信手机端操作转让群主：
【群聊->点击右上角“…”->群管理->群主管理权转让->选择新群主】
![](https://wework.qpic.cn/wwpic3az/424169_nhFv2uYmQWCYuBc_1739786025/0)

### [](https://open.work.weixin.qq.com)二、电脑端转让群主入口

【群聊->点击右上角“…”->群管理->转让群主】

#### [](https://open.work.weixin.qq.com)3、如何转让群主给外部联系人

转让群主，目前有以下三种情况：
（1）客户群：群主仅支持转让给同企业有[客户联系权限](https://open.work.weixin.qq.com/help2/pc/14690)的成员。
（2）内部群和普通外部群：群主仅支持转让给同企业内的成员。
（3）微信用户创建的外部群：暂无群主转让限制。

#### [](https://open.work.weixin.qq.com)4、管理员可以操作转移群主吗

非“学前/初中等教育”行业类型的企业，可以由管理员操作[在职](https://open.work.weixin.qq.com/help2/pc/20569)/[离职](https://open.work.weixin.qq.com/help2/pc/14706)继承来转让成员客户群的群主身份。

行业类型为“学前/初中等教育”的学校支持由管理员在【管理后台->家校沟通->班级群->群管理->修改群主】。

注：
1、内部群仅支持群主操作转让群主身份，暂不支持管理员操作。
2、非客户群的外部群（即成员不在[客户联系使用范围内](https://open.work.weixin.qq.com/help2/pc/14690)创建的外部群）、微信用户创建的外部群，目前仅支持由群主操作转让群主身份，暂不支持由管理员操作。

#### [](https://open.work.weixin.qq.com)5、转让后能收回群主权限吗

暂不支持收回，需由新群主主动转让群主身份。
