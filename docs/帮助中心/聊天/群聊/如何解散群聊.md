# 如何解散群聊

### [](https://open.work.weixin.qq.com)一、如何解散群聊

**仅群主**可以解散群聊，暂不支持普通群成员解散群聊。

注：全员群、部门群、群主是员工服务人员的群聊暂不支持解散。

操作路径如下：
【手机端->群聊右上角三个点->群管理->解散群聊】
![](https://wework.qpic.cn/wwpic3az/7014_7Z2xGZfBR2OrogM_1764830103/0)

【电脑端->群聊右上角三个点->群管理->解散群聊】
![](https://wework.qpic.cn/wwpic3az/156994_NF2XvaJdTa63O45_1750326091/0)

### [](https://open.work.weixin.qq.com)二、常见问题

#### [](https://open.work.weixin.qq.com)1、群聊只剩下1个人的时候，还可以解散群聊吗

仅剩群主1人在群里时，支持点击“解散群聊”按钮操作解散。

#### [](https://open.work.weixin.qq.com)2、Mac群聊的群管理中没有解散群聊入口

出现该情况，可能是因为窗口太大挡住入口导致的，可操作隐藏dock栏重试。
具体操作如下：主菜单左上角苹果图标->系统偏好设置->程序坞与菜单栏->去掉勾选【自动隐藏和显示程序坞】

#### [](https://open.work.weixin.qq.com)3、解散外部群/客户群，提示“解散群聊失败”

请在手机端企业微信退出登录后，重新使用企业微信绑定的微信号授权登录后重试。

#### [](https://open.work.weixin.qq.com)4、管理员可以解散成员创建的群聊吗

暂不支持管理员解散成员的群聊，需要群主操作。

#### [](https://open.work.weixin.qq.com)5、已解散的群聊出现在消息列表

当前已解散的群聊不会从会话列表/消息列表中移除。

#### [](https://open.work.weixin.qq.com)6、解散群聊后，聊天记录会被删除吗

解散群聊后仍会保留群聊聊天记录，群聊记录并不会被清除。
