# 聊天功能介绍

会话支持[单聊](https://open.work.weixin.qq.com/help2/pc/17446)、[群聊会话](https://open.work.weixin.qq.com/help2/pc/17545)，单聊是指跟单成员进行沟通的聊天，群聊是指三个人或以上的会话。成员还支持对会话消息进行管理、设置消息提醒等。

发起单聊支持对内部同事发起会话，也支持给外部联系人发起会话。

群聊包含外部群聊、内部群聊、[全员群](https://open.work.weixin.qq.com/help2/pc/17547)、[部门群](https://open.work.weixin.qq.com/help2/pc/17546)、课程群、[上下游群](https://open.work.weixin.qq.com/help2/pc/19414)、[客户群](https://open.work.weixin.qq.com/help2/pc/17923)、[班级群](https://open.work.weixin.qq.com/help2/pc/17798)、[居民群](https://open.work.weixin.qq.com/help2/pc/19267)、[学员群](https://open.work.weixin.qq.com/help2/pc/18303)等。
