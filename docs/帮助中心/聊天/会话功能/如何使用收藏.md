# 如何使用收藏

### [](https://open.work.weixin.qq.com)一、支持收藏的内容有哪些

目前支持收藏文字、语音、图片、视频、文件、地理位置以及邮件中的附件。

### [](https://open.work.weixin.qq.com)二、如何操作收藏会话中的消息

1、手机端：长按需要收藏的消息操作收藏。
![](https://wework.qpic.cn/wwpic3az/531007__9QvKw8eSkaiR58_1764733646/0)
2、电脑端：鼠标右键点击需要收藏的消息操作收藏。
![](https://wework.qpic.cn/wwpic3az/819761_PEb2yxyxSeWQ9rL_1750408434/0)

### [](https://open.work.weixin.qq.com)三、如何查看已收藏的内容

查看路径：
1、【手机端->消息页面左上角“三”->点击姓名->收藏】
![](https://wework.qpic.cn/wwpic3az/432845_e85lD3_iThieMDX_1764734825/0)

2、【电脑端->左上角头像->收藏】
![](https://wework.qpic.cn/wwpic3az/786906_88bgGx_JQVac9l4_1750408468/0)

### [](https://open.work.weixin.qq.com)四、如何转发收藏的内容

点击【消息页面左上角“三”->头像->收藏】选择需要分享或转发的内容，长按转发给其他成员。
![](https://wework.qpic.cn/wwpic3az/32228_FWXIJncTTCePKTf_1764734853/0)

### [](https://open.work.weixin.qq.com)五、如何/在哪里搜索收藏的内容

点击【消息页面左上角“三”->头像->收藏】在页面上的搜索框中根据输入关键字搜索。
![](https://wework.qpic.cn/wwpic3az/631094_R6wbivvdRuWcV7p_1764734936/0)

### [](https://open.work.weixin.qq.com)六、如何删除收藏内容

若需要删除/取消某条收藏，请点击【消息页面左上角“三”->头像->收藏】在需要删除的某条收藏内容点击右上方三个点，取消收藏。
![](https://wework.qpic.cn/wwpic3az/461963_1hi8QgugSai9HAD_1764734876/0)

### [](https://open.work.weixin.qq.com)七、点击收藏提示“空间不足，收藏失败”

目前表情包加上收藏空间上限一共是2G，当前收藏夹和表情的存量大小已经达到2G会出现这个提醒。
