# 如何使用截屏功能

### [](https://open.work.weixin.qq.com)一、如何操作截屏

1、在电脑端会话中可直接点击“剪刀”图标进行截图，支持勾选【隐藏当前窗口截图】，勾选后截取屏幕时，将自动隐藏企业微信窗口。
![](https://wework.qpic.cn/wwpic3az/523743_pcqeUcE3RCurxFJ_1750400982/0)

2、v4.1.32版本开始，在电脑端截图打码时，支持框选某个区域填充马赛克。
![](https://wework.qpic.cn/wwpic3az/648597_QOBIwNDwQf-G_Pm_1750401036/0)

### [](https://open.work.weixin.qq.com)二、截屏快捷键

在【电脑端->左上角头像->设置->快捷键->截屏】可设置截屏快捷键。设置快捷键后，可以在键盘中按住对应快捷键触发截屏。
![](https://wework.qpic.cn/wwpic3az/433710_GhojvN0TR7y5TB7_1750399944/0)
![](https://wework.qpic.cn/wwpic3az/988035_GLMclWpnRdeQvT2_1750400151/0)

### [](https://open.work.weixin.qq.com)三、翻译截图中的文字

v5.0.6版本开始，在电脑端截图时，支持翻译截图中的文字，翻译的语言为企业微信设置的语言。
可以在【电脑端->头像->设置->通用设置->语言】中查看当前客户端设置的语言。
![](https://wework.qpic.cn/wwpic3az/209029_RfdzjTq-Sr-OqjT_1770609193/0)
![](https://wework.qpic.cn/wwpic3az/436186_jpI-m_L1TTGsJNR_1770609205/0)

### [](https://open.work.weixin.qq.com)四、常见问题

#### [](https://open.work.weixin.qq.com)1、Mac截图时，无法截图到除企业微信以外的页面？

请确认是否有勾选允许企业微信录屏权限，可在苹果菜单->“系统设置”,在边栏中点击“隐私与安全性”，找到“屏幕与系统音频录制”，勾选允许企业微信使用。
![](https://wework.qpic.cn/wwpic3az/146777_9-6T6nHrTd-6Pwi_1710811382/0)

若是开启了录屏权限后，还是无法录屏/无法截图到除企业微信以外的页面，可参考以下方法操作后重试：
（1）找到系统设置
（2）点击下面的“—”，先删除企业微信
![](https://wework.qpic.cn/wwpic3az/977845_RPTX3keuT_e5yJJ_1710811541/0)
（3）重新从应用程序中进行拖拽
![](https://wework.qpic.cn/wwpic3az/947882_Q6cAbfLXTaOuWiG_1710811609/0)

#### [](https://open.work.weixin.qq.com)2、在Mac端企业微信全屏模式下，点击“隐藏当前窗口截图”会显示黑屏

目前企业微信是一个全屏窗口时，会独立占据一个桌面。当隐藏窗口截屏后，这个桌面上已经没有其他内容了，所以不显示其他内容。 建议在不全屏企业微信下，使用隐藏当前窗口截图或用快捷键（command+shift+a）。

#### [](https://open.work.weixin.qq.com)3、在企业微信侧操作截图界面会闪烁/闪屏

（1）请确认在微信端使用微信截图功能，是否会出现该问题；
（2）请确认是否有外接显示屏，若有，可不外接显示屏进行尝试，可能是显示屏出现问题。

#### [](https://open.work.weixin.qq.com)4、如果企业开启了文件防泄漏功能，还可以使用翻译截图中的文字吗？

目前开启了文件防泄漏功能，不影响翻译截图中的文字。
