# 如何把会话标记未读/已读

手机端和电脑端均支持把已读聊天标为未读/已读，标记未读后，会话会展示小红点，可等有空后处理对应的会话。
暂不支持一键操作标记会话为未读/已读，需要单个会话操作。

#### [](https://open.work.weixin.qq.com)1、电脑端操作入口

电脑端右键会话选择“标为未读/已读”
![](https://wework.qpic.cn/wwpic3az/184258_d2G-J1eLSrSmJUa_1750408030/0)

#### [](https://open.work.weixin.qq.com)2、IOS系统操作入口

IOS系统手机端【从右往左滑会话->标为未读/已读】
![](https://wework.qpic.cn/wwpic3az/264255_wYPy-FJRR6i1UWO_1764732433/0)

#### [](https://open.work.weixin.qq.com)3、Android系统操作入口

Android系统手机端【长按需要标记未读的会话->标为未读/已读】
![](https://wework.qpic.cn/wwpic3az/811271_3ML9hddVSVyy78n_1764732469/0)
