# 审批如何提交电子签名

### [](https://open.work.weixin.qq.com)一、如何设置审批人同意时需要签名

管理员可在审批模板设置中勾选「审批人同意时需要签名」，勾选后审批人在操作「同意」的时候则需要签名。
打印审批单可以看到电子签名，导出的pdf支持查看电子签名。

操作路径如下：
【[管理后台](https://work.weixin.qq.com)->应用管理->审批->选择对应的模板->规则设置->电子签名】
![](https://wework.qpic.cn/wwpic3az/114426_prKiAJp_QNGL-z2_1725523384/0)
【手机端->工作台->审批->管理->申请模板管理->选择对应的模板->编辑->规则->同意时需电子签名】
![](https://wework.qpic.cn/wwpic3az/364437_-Mk2xRyqQj-DCib_1765857151/0)

### [](https://open.work.weixin.qq.com)二、如何设置下次同意时继续使用此签名

审批人在操作「同意」时（手机端和电脑端均可操作），同时可勾选「下次同意时继续使用此签名」，勾选后同意下个审批单时将不需要重新签名，可以通用该签名（不同模板可以通用同一个签名）。
![](https://wework.qpic.cn/wwpic3az/973500_vxAyu6dSTXaXwb5_1765857170/0)

### [](https://open.work.weixin.qq.com)三、如何修改电子签名

填写电子签名后，已审批的审批单不支持修改电子签名。
若在审批的时候选择了「下次同意时继续使用此签名」，则下次操作审批时，页面会默认显示上一次的签名，若需重新签名可点击「重新签名」。
![](https://wework.qpic.cn/wwpic3az/711880_Ey_RaGtZQnKRTdS_1765857199/0)
