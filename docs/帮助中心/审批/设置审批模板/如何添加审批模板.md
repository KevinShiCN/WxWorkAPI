# 如何添加审批模板

### [](https://open.work.weixin.qq.com)一、如何新增/修改审批模板

#### [](https://open.work.weixin.qq.com)1、管理员在管理后台设置审批模板

管理员在【[管理后台](https://work.weixin.qq.com)->应用管理->审批】中，点击「添加模板」可以新建模板，可选择新增自定义模板、从推荐模板添加或是从已有模板复制；
![](https://wework.qpic.cn/wwpic3az/277873_ABKt4UBaSuey_vO_1753770953/0)
管理员在【[管理后台](https://work.weixin.qq.com)->应用管理->审批】中单击「已有模板」进入模板设置，即可编辑已有模板。
![](https://wework.qpic.cn/wwpic3az/204293_3gOHxju8Rs-qXlj_1753770983/0)

#### [](https://open.work.weixin.qq.com)2、管理员和成员在手机端设置审批模板

管理员支持在【手机端->工作台->审批->管理->单据模板管理】添加模板，或是单击「已有模板」进行编辑。
有创建审批模板权限的成员，支持在【手机端->工作台->审批->管理->申请模板管理】新增审批模板，或是单击自己创建的模板进行编辑。
![](https://wework.qpic.cn/wwpic3az/112778_z6jDpDZZR7SVGEC_1765791191/0)

#### [](https://open.work.weixin.qq.com)3、审批模板分组和排序

管理员可在【[管理后台](https://work.weixin.qq.com/)->应用管理->审批->分组和排序】中添加分组、调整模板顺序。
![](https://wework.qpic.cn/wwpic3az/307269_uqevwZE-Q2Wsoy1_1753770899/0)

### [](https://open.work.weixin.qq.com)二、如何通知成员使用模板

新增/编辑模板后，可在规则设置选择开启或关闭「通知成员」，开启后将会在发布模板时，给可见范围内的人员推送通知，让成员查看并使用审批模板。

注：新创建模板时是默认打开的，编辑更新模板时是默认关闭的。删除模板暂不涉及通知。
![](https://wework.qpic.cn/wwpic3az/360878_0WwMRjadRyueNXT_1753771169/0)
