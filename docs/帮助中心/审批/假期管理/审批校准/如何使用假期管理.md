# 如何使用假期管理

目前超级管理员、有审批应用管理权限且管理范围是整个企业的分级管理员支持在管理后台、手机端的审批应用中查看到假期管理入口。
在假期管理中，可设置假期类型并查看假期余额，员工请假后根据请假审批单自动扣假。

---

### [](https://open.work.weixin.qq.com)一、如何添加/编辑假期

成员在填写请假审批单时，需要先选择请假类型，可选择的请假类型对应假期管理中已有的请假类型，企业微信自带假期类型有：年假、事假、病假、调休假、婚假、产假、陪产假、其他。
企业可选择已有模板进行编辑，或是创建新的假期类型使用。

**1、添加路径：**
在【[管理后台](https://work.weixin.qq.com)->应用管理->审批->假期管理->假期类型】或【管理后台->应用管理->打卡->假期管理->设置】中，可添加/编辑请假类型。
![](https://wework.qpic.cn/wwpic3az/465371_kMlSw9oMTnW-l7X_1765788630/0)
【手机端->工作台->审批->管理->假期类型管理】
![](https://wework.qpic.cn/wwpic3az/430377_Y21mJbDPTUO1dcu_1765787102/0)
【手机端->工作台->打卡->申请->假勤管理->假期类型管理】
![](https://wework.qpic.cn/wwpic3az/504532_4xH7E39ZRo-aXjI_1765787130/0)

2、进入假期编辑页面后，可按需设置「假期基础信息」、「请假时长核算规则」、「假期余额发放规则」。
![](https://wework.qpic.cn/wwpic3az/777265_gXjr-h57QPGAqmK_1765788967/0)
**提示：**
修改请假类型后，所有成员的该类型的假期余额将按最新设置重新发放，且不可恢复，可在确认修改的弹窗中勾选“导出该假期余额”进行备份。
[查看问题：如何添加/编辑假期](https://open.work.weixin.qq.com/help2/pc/18004)

### [](https://open.work.weixin.qq.com)二、如何设置假期余额发放规则

1、如果是「限额」假期，可根据需要选择余额发放方式，当前支持「按年自动发放」、「按月自动发放」、「手动导入余额」、「加班自动调休」4类发放方式。
![](https://wework.qpic.cn/wwpic3az/820727_XTWZK6h-R6-TZKC_1765789227/0)

2、选择按年/按月自动发放后，也可随时通过手动导入的方式调整员工假期。
[查看问题：如何设置假期余额发放规则](https://open.work.weixin.qq.com/help2/pc/18005)

### [](https://open.work.weixin.qq.com)三、如何批量设置或备份假期余额

在【[管理后台](https://work.weixin.qq.com)->应用管理->审批/打卡->假期管理设置->假期余额】中，可导出/导入余额。
![](https://wework.qpic.cn/wwpic3az/697825_EXsqsPT9R-yOzW7_1765789374/0)
[查看问题：如何批量设置或备份假期余额](https://open.work.weixin.qq.com/help2/pc/18006)

### [](https://open.work.weixin.qq.com)四、如何查看指定员工使用记录并修改假期余额

1、在【[管理后台](https://work.weixin.qq.com)->应用管理->审批/打卡->假期管理设置->假期余额】中，点击具体某成员所在行，可查看此人假期余额详情。
![](https://wework.qpic.cn/wwpic3az/788668_KsNzgQ4CSj64KDS_1765789671/0)
2、对于限额且已发放的假期类型，可以手动修改其余额。
![](https://wework.qpic.cn/wwpic3az/519045_0egsQKWuQcqysHg_1765789756/0)
[查看问题：如何修改指定成员的假期余额](https://open.work.weixin.qq.com/help2/pc/18007)
