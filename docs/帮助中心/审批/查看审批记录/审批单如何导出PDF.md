# 审批单如何导出PDF

目前在手机端、电脑端、管理后台查看审批详情时，支持把对应的审批单导出为PDF。
注：目前支持在导出的pdf中查看审批人电子签名。
入口：
1、手机端：【工作台->审批->审批详情->右上角】
![](https://wework.qpic.cn/wwpic3az/935863_Pxr-HtbEQfGJxIg_1765857252/0)
2、电脑端：【工作台->审批->审批详情->右上角】
![](https://wework.qpic.cn/wwpic3az/953152_vZTkxDoiTnKw-qt_1753775438/0)
3、管理后台：【应用管理->审批->选择对应的审批模板->申请记录->选择对应的审批单->在审批单详情页面右上角点击“导出”】
![](https://wework.qpic.cn/wwpic3az/656034_8nQSnPCaTKSAqAZ_1753774935/0)
