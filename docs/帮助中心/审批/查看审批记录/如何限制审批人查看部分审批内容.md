# 如何限制审批人查看部分审批内容

在后台编辑审批流程，选择某个审批人节点-右上角选择权限设置，可以设置该节点的审批人能否查看、编辑申请人的审批内容，审批模板内的每个控件都可以分别设置权限，可设置隐藏、仅查看、可编辑权限。

- **隐藏：**审批人处理申请人的申请时，看不到这个被隐藏的控件，控件名称、申请人填写内容都看不到
- **仅查看：**审批人处理申请人的申请时，能看到对应控件及内容（不能改）
- **可编辑：**审批人处理申请人的申请时，可以修改对应控件的内容

注：假勤组件不支持设置隐藏、可编辑，只能是仅查看
