# 如何扣除假期余额

1、超级管理员在管理后台【应用管理->审批/打卡->假期管理->编辑/添加假期类型->时长计算】选择自然日或工作日后，请假时长将会判断是否扣除周末和法定节假日。
● 当选择自然日则不会自动扣除周末和国家法定节假日；
● 当选择工作日则会自动扣除周末和国家法定节假日。
![](https://wework.qpic.cn/wwpic3az/114392_QG3NxCfGTByfOAw_1753780449/0)

2、管理员在管理后台给成员设置了对应的假期时长后，成员使用相应的请假，则会扣除对应的假期时长。
