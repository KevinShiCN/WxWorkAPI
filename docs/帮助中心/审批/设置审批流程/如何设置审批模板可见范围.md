# 如何设置审批模板可见范围

管理员可以在审批模板中设置模板可见范围，在审批模板可见范围中的成员，才能使用对应的模板提交审批。

操作路径：
【管理后台->应用管理->审批->选择对应模板->规则设置->可见范围】
![](https://wework.qpic.cn/wwpic3az/134779_x_rIhAI5RM2647d_1753771396/0)
【手机端->工作台->审批->管理->申请模板管理->选择对应模板->编辑->规则->可见范围】
![](https://wework.qpic.cn/wwpic3az/530129_6VUucoj2STuSqzg_1765856430/0)
