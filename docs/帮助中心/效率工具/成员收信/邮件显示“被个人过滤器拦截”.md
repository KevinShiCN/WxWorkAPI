# 邮件显示“被个人过滤器拦截”

当您发现收不到邮件，并在企业邮箱的【自助服务->收信查询】中查到某封邮件的状态为“被个人过滤器拦截”时，通常是因为邮箱设置了自定义的收信规则或自动转发功能，系统根据个人设置将邮件进行了拦截处理。
![](https://wework.qpic.cn/wwpic/243305_i_deOaQmSGONVYE_1692238900/0)

### [](https://open.work.weixin.qq.com)一、问题原因与排查步骤

#### [](https://open.work.weixin.qq.com)1、检查【邮件自动转发】设置

**路径：**网页端登录企业邮箱【设置->常规->邮件自动转发】
**可能原因：**若开启了“自动转发功能”，并且同时勾选了“删除已自动转发的邮件”选项。系统在执行自动转发操作后，会根据此规则将原邮件从原邮箱的收件箱中删除，导致无法直接看到该邮件。
![](https://wework.qpic.cn/wwpic/336378_IAJNP0JrRmWsUYy_1692239093/0)

#### [](https://open.work.weixin.qq.com)2、检查【收信规则】设置

**路径：**网页端登录企业邮箱【设置->收信规则】
**可能原因：**创建的某条或多条收信规则中，设定的执行操作包含了“直接删除”或“转发后删除”。当收到的邮件符合设定的规则条件（如特定的发件人、主题关键词、收件地址等）时，系统便会自动执行删除操作。
![](https://wework.qpic.cn/wwpic/371754_qwI1stkNR9ymrNT_1692239211/0)

#### [](https://open.work.weixin.qq.com)3、注意事项

（1）建议登录企业邮箱网页版，仔细检查并删除或修改可能导致邮件被删除的过滤规则或自动转发设置。
（2）显示为“被个人过滤器拦截”的邮件不支持取回，建议登录QQ邮箱网页版，仔细检查并删除或修改可能导致邮件被删除的过滤规则或自动转发设置后，联系发件方重新发送邮件。
