# 550 SPF check failed

**出错原因：**SPF检查未通过验证

**改善建议：**发信域名设置了SPF，但检查未通过验证。如有新增ip，请及时更新SPF记录。
为减少DNS查询失败对入信带来不可控的影响，企业邮箱对DNS记录将缓存最长不超过12小时。如果有IP变更，请提前更新SPF。

详情可参考：https://open.work.weixin.qq.com/help2/pc/19817
