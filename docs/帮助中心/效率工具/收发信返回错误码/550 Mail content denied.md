# 550 Mail content denied

**出错原因：**该邮件内容涉嫌大量群发，并且被多数用户投诉为垃圾邮件。

对于具有群发性质的邮件，如果出现用户普遍表示反感或集中投诉的情况，腾讯邮箱将禁止类似此邮件内容继续发送。

**改善建议：**如果您是此群发邮件的管理员，请停止群发对用户造成骚扰的广告及推广类邮件。

如果您是个人邮箱用户，请向您的电子邮件提供商报告此情况。
