# 新增其他联系人

可通过文档、会议、日程、邮件等工具间协作行为产生其他联系人，具体产生方式如下：
（1）企业微信用户发送文档、日程、会议卡片到会话中，如果对方是非同事、上下游、好友身份，且对方打开了卡片，此时对方会被记录为“其他联系人”身份；
（2）在邮件输入收件地址，收件地址非同事、上下游、好友身份也会记录到“其他联系人”中。

### [](https://open.work.weixin.qq.com)一、文档

1、可将文档分享给其他企业的企业微信用户，也可将文档分享给微信用户
2、当满足一定条件时，打开文档的成员将成为文档分享者的其他联系人，条件如下：

- 打开文档的成员与文档创建者不存在同事、上下游和微信客户的关系
- 用户打开文档，并且用户是文档成员

### [](https://open.work.weixin.qq.com)二、日程

1、支持在“日程详情页->分享日程”到聊天
![](https://wework.qpic.cn/wwpic/340571_76OpM8ydRxGsr12_1644571965/0)
2、若通过分享的日程卡片接受日程邀约，并且该用户不是创建者的同事、上下游或微信客户，那么就会成为分享者的其他联系人。
![](https://wework.qpic.cn/wwpic3az/722655_tNkI-5PzRFC9r_H_1764831911/0)

### [](https://open.work.weixin.qq.com)三、会议

1、支持进入“会议房间->分享会议”到聊天
![](https://wework.qpic.cn/wwpic/438187_DuR1X0WgROihtfL_1644572020/0)
2、若通过分享的会议邀约卡片进入会议，并且该用户不是创建者的同事、上下游或微信客户，那么就会成为分享者的其他联系人。
![](https://wework.qpic.cn/wwpic3az/94939_7j-ZDWfAQgW1Sx7_1764831923/0)

### [](https://open.work.weixin.qq.com)四、邮件

使用企业微信发送邮件后，非同企业的收件人将成为发信者的其他联系人
