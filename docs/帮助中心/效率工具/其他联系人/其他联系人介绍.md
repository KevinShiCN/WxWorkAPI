# 其他联系人介绍

其他联系人是基于效率工具（文档、日程、会议、邮件）产生的协作者，包括微信用户、企业微信用户、邮箱用户三种不同的类型。
成员在使用企业微信的文档、日程、会议和邮件功能时，可选择其他联系人进行协作。
成员可以在【手机端/电脑端->通讯录->其他联系人】中查看已有的其他联系人。
