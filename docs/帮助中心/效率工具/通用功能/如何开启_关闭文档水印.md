# 如何开启/关闭文档水印

1、水印信息可设置为访问者姓名和自定义内容，支持同时勾选。
访问者姓名：勾选后，水印会显示为访问者在企业通讯录中的姓名
自定义内容：勾选后，需要输入自定义内容，水印会显示自定义设置的内容。

2、支持设置三种水印样式：密集、标准、宽松。

3、操作路径：
打开文档，点击文档顶部菜单栏“三” ->权限管理->文档水印->开启/关闭。
