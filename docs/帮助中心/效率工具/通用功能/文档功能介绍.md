# 文档功能介绍

快速上手在线文档，效率倍增->[点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488324968827073_1)

文档是无需购买办公软件，直接新建在线文档和表格的应用，支持多种内容格式操作，云端实时保存，随时取用。
1、可通过会话与企业内同事共享文档，一起编辑，比传统文档、邮件往来更方便，提高工作效率。
2、可按企业组织架构来设置文档访问权限，也可设置企业外临时访问有效期，保证企业资料安全。
3、支持通过API进行文档管理，点击[查看api文档](https://developer.work.weixin.qq.com/document/43883)。
4、支持设置可调用接口的应用，设置后企业自建应用可调用文档相关接口。
