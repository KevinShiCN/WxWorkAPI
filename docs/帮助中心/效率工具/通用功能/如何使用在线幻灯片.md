# 如何使用在线幻灯片

### [](https://open.work.weixin.qq.com)一、在线幻灯片功能介绍

在文档应用界面新建文档时，支持新建幻灯片。可在线编辑演示文稿，多端实时同步，轻松满足汇报需求。

### [](https://open.work.weixin.qq.com)二、创建幻灯片入口

#### [](https://open.work.weixin.qq.com)1、电脑端入口

【左边菜单栏->文档->上方菜单栏->更多->新建幻灯片】
![](https://wework.qpic.cn/wwpic3az/438953_IQ5R6pRvRQeP9oQ_1763518373/0)
或在【模板库->右上角全部->幻灯片】选择合适的模板创建使用。
![](https://wework.qpic.cn/wwpic3az/306316_ee4xPTNDRPOs2H4_1763518377/0)
![](https://wework.qpic.cn/wwpic3az/739397_RAkT_OnqS1Kk_wh_1763518379/0)

#### [](https://open.work.weixin.qq.com)2、手机端入口

【手机端->文档->右上角“+”->更多->新建幻灯片】
![](https://wework.qpic.cn/wwpic3az/145037_u8_XQjg0RHOQjkq_1763518171/0)
或在【模板库->右上角筛选->幻灯片】选择合适的模板创建使用。
![](https://wework.qpic.cn/wwpic3az/179286_Z4pPYeZFTgCF_Z6_1763518174/0)

### [](https://open.work.weixin.qq.com)三、幻灯片支持的功能

#### [](https://open.work.weixin.qq.com)1、插入功能

在线幻灯片支持插入文本框、形状、图片、表格、链接、页脚、文档、文件等。
![](https://wework.qpic.cn/wwpic3az/14870_tJ35fUNdTQK6Zle_1763518581/0)

#### [](https://open.work.weixin.qq.com)2、效果设置

支持对文稿内容进行效果设置，包括阴影、映像、发光、柔化边缘等更多设置，优化演示文稿内容展示形式。
![](https://wework.qpic.cn/wwpic3az/833490_c-vP5WRYS8-TyhO_1763518734/0)

#### [](https://open.work.weixin.qq.com)3、动画效果

可根据需求添加动画效果，提升演示效果。
![](https://wework.qpic.cn/wwpic3az/566291_wuZLR7KLRxykXag_1763518800/0)

#### [](https://open.work.weixin.qq.com)4、在线演示

幻灯片支持在线演示，电脑端支持激光笔功能。
![](https://wework.qpic.cn/wwpic3az/38466_nONXil0vRLut_2V_1763518908/0)
![](https://wework.qpic.cn/wwpic3az/451387_3OcJZmv2QPilY9P_1734335776/0)

#### [](https://open.work.weixin.qq.com)5、快捷美化

购买文档高级功能后，有文档高级功能账号的成员，支持在幻灯片中使用快捷美化功能，包括一键换色、统一字体、批量设置图片尺寸、批量图片布局、批量图片下载。
![](https://wework.qpic.cn/wwpic3az/822974_qQln387hRR-0rs5_1763519042/0)

#### [](https://open.work.weixin.qq.com)6、备注

幻灯片支持备注功能，可以对幻灯片内容进行备注。
![](https://wework.qpic.cn/wwpic3az/107716_e8B6IqpvQJ-iOD0_1763519137/0)
