# 如何停止收集表

### [](https://open.work.weixin.qq.com)一、自动停止收集

管理员可在【工作台->收集表->统计->选择对应的收集表->设置->设置截止时间】设置定时结束，到了设置的结束时间自动结束收集。
![](https://wework.qpic.cn/wwpic3az/129618_7Auoi3qeRGyKRL0_1765262177/0)

### [](https://open.work.weixin.qq.com)二、手动停止收集

1、收集表的发起人在【手机端->工作台->收集表->统计中找到对应的收集表->右上角“三”->停止收集】
![](https://wework.qpic.cn/wwpic3az/827604_8py-qoeqSvehfAE_1765262201/0)

2、收集表的发起人在【电脑端->文档找到对应的收集表打开->顶部“三”->停止收集】可结束收集。
![](https://wework.qpic.cn/wwpic3az/40300_KYpuMIrqSoWGtli_1750739406/0)

### [](https://open.work.weixin.qq.com)三、常见问题

1、已停止收集表，但成员还可以填写收集表。
可确认是否有成员手动又重新开启了收集表。
