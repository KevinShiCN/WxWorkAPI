# 如何开启会议字幕和转写同时显示双语

v4.1.32版本开始，会议支持字幕和文字转写同时显示双语，支持翻译17种语言，帮助参会者理解和记录会议内容。

**若是会议基础功能和会议高级功能专业版账号，目前仅支持配简体中文、英文、日语。
目前仅会议高级功能账号（商业版、企业版）支持配置17种语言。**

支持的语言（17种）：简体中文、英语、日语、韩语、俄语、泰语、印尼语、越南语、马来语、菲律宾语、葡萄牙语、土耳其语、阿拉伯语、西班牙语、印地语、法语、德语。

![](https://wework.qpic.cn/wwpic3az/853160__8mAt6HgT2e5zlf_1736132915/0)

**1、会前设置入口**
【客户端->会议应用->右上角“…”->会议设置->字幕和转写->翻译显示】
![](https://wework.qpic.cn/wwpic3az/39984_UNMIe6MsSWCMNsf_1750751792/0)
![](https://wework.qpic.cn/wwpic3az/44550_xYwp7l-DSxq9CpI_1750751797/0)

**2、会中设置入口**
【会中界面->应用/设置】开启字幕、文字转写。
![](https://wework.qpic.cn/wwpic3az/729720_xYdO1XQBRF-wJk3_1769411164/0)
3、实时转写功能，当会议中开启过实时转写时，会后支持查看转写，以及导出转写。
【客户端->会议应用->历史会议->点开需要查看的会议记录->文字转写】，查看页面右上角可导出。
![](https://wework.qpic.cn/wwpic3az/791333_ImAH4jzoSzajBKp_1769411176/0)
![](https://wework.qpic.cn/wwpic3az/401440_vVeQSDVmQ-iLNrd_1769411186/0)
