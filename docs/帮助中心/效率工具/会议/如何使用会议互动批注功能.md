# 如何使用会议互动批注功能

1、发起共享屏幕后参会人可参与互动批注，需要发起[共享屏幕](https://open.work.weixin.qq.com/help2/pc/18174?person_id=)之后才能点击设置，若未发起，在应用中的“互动批注”会置灰。

2、互通批注设置：批注跟随PowerPoint（默认勾选）、允许其他成员添加互动批注（默认勾选）、显示批注者名称（默认勾选）。
