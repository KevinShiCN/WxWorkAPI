# 会议高级功能介绍

快速了解「会议」高级功能 [点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488324038827078_2)

腾讯企业微信会议作为企业微信协同办公套件之一，深度融合了腾讯会议功能，为企业提供高清、稳定、流畅的开会体验，并集成屏幕共享、电子白板、主持会控、会议纪要等功能。
在侧边栏、工作台、群聊里，都可以快速发起或加入一个会议。

### [](https://open.work.weixin.qq.com)一、如何购买会议高级功能

若需将基础功能会议升级为高级功能会议，管理员可在电脑端【登录[管理后台](https://work.weixin.qq.com/)->协作->会议->购买高级功能】，在跳转页面选择需要购买的数量。
![](https://wework.qpic.cn/wwpic3az/6555_ARm4K4HbSbuA1oX_1753690813/0)
或是在【管理后台->高级功能】选择【会议】进行购买。
![](https://wework.qpic.cn/wwpic3az/780372_pPuLTdJdRVOejPL_1753690771/0)
手机端可在【消息页面左上角的“三”->高级功能->会议高级功能->了解更多】进行购买。
![](https://wework.qpic.cn/wwpic3az/940331_fpkInCcCQ6Ck1zy_1765350835/0)
