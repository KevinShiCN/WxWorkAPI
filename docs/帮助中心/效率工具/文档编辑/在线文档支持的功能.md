# 在线文档支持的功能

#### [](https://open.work.weixin.qq.com)1、查找与替换、拓展插件

在线文档支持查找与替换、拓展插件

![](https://wework.qpic.cn/wwpic3az/119275_eQUfyrxTSO2EFwS_1765419148/0)

#### [](https://open.work.weixin.qq.com)2、待办事项

![](https://wework.qpic.cn/wwpic3az/525415_0vEQf058Tw6evDx_1765418322/0)

#### [](https://open.work.weixin.qq.com)3、切换web模式或页面模式

可在左下角将文档切换为web模式、分页模式、连页模式。文档为分页模式时，支持在页面中插入页码（页眉、页脚）。
![](https://wework.qpic.cn/wwpic3az/100133_MBEzcdt4Th6DgDY_1765418332/0)
![](https://wework.qpic.cn/wwpic3az/457848___HkFA5aRKiDcHL_1768181443/0)

#### [](https://open.work.weixin.qq.com)4、调整文档格式

可在文档编辑中调整文档格式、底纹等基本功能。
![](https://wework.qpic.cn/wwpic3az/36861_Fc3cdOdwSsyxRRG_1765418901/0)

#### [](https://open.work.weixin.qq.com)5、插入表格

可在文档中插入表格，支持快捷增删行列以及调整表格位置（左对齐/居中/右对齐），不支持设置带有斜线的单元格。
