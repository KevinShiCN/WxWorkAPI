# 如何解除原邮箱的 IP 登录限制

解除原邮箱的 IP 登录限制，需要登录原来的邮箱设置。

## [](https://open.work.weixin.qq.com)● 以网易企业邮箱为例

管理员从【首页->安全中心->登录管理->IP登录管理】路径进入IP登录管理功能，找到需要解除的“IP登录限制规则”，点击删除即可。如下图所示。
![](https://wework.qpic.cn/wwpic3az/926337_NPDKAJvfRxagmPp_1741668208/0)

## [](https://open.work.weixin.qq.com)● 以 35 互联企业邮箱为例

管理员：登录管理后台，点击【域管理->IP登录限制->用户IP登录限制->解除所有IP限制】，或勾选需要解除IP登录限制的成员， 点击“解除IP限制”
![](https://wework.qpic.cn/wwpic3az/461974_Rok_kd7xSqOP7xy_1741668239/0)

成员：登录网页端邮箱后【账号设置（右上角）—>更多账号设置—>账户登录保护】，删除所设置的IP限制/地理位置设置。
![](https://wework.qpic.cn/wwpic3az/831883_vUDLJ3TIQzmkqV7_1741668260/0)
![](https://wework.qpic.cn/wwpic3az/877827_CPkphSfxQ8Wlk1l_1741668271/0)

## [](https://open.work.weixin.qq.com)● 以阿里企业邮箱为例

1、使用管理员账号登录企业邮箱，默认进入 域管后台，点击【安全管理->IP登录限制->添加策略】，找到需要删除的ip登录限制策略后，点击“删除”即可。
![](https://wework.qpic.cn/wwpic3az/808873_-zgrId8mQeynyAl_1741668223/0)
