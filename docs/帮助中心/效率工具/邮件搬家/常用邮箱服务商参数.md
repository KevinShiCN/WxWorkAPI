# 常用邮箱服务商参数

邮件搬家支持通过IMAP或者POP两种协议，因此在搬家前，请先确认原来的邮箱服务商是否支持该协议，如果原服务商支持IMAP服务的情况下，请优先IMAP协议来启动搬家。避免搬家后发现搬家邮件不完整的问题。
