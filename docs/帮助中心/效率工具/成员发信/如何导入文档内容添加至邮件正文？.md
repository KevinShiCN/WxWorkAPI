# 如何导入文档内容添加至邮件正文？

### [](https://open.work.weixin.qq.com)一、网页端编辑邮件如何导入本地文档

网页端登录企业邮箱，编辑邮件时支持选择本地文档进行导入至邮件正文，导入方式：【登录邮箱->写信->插入正文->文档】可选择电脑本地文档进行导入至邮件正文编辑发送邮件。
注：仅支持导入小于3M的doc、docx 、xls 、xlsx格式文档。
![](https://wework.qpic.cn/wwpic/728191_fKL4ddXkTzi9GHf_1692237652/0)

### [](https://open.work.weixin.qq.com)二、企业微信PC端编辑邮件如何导入在线文档

通过【企业微信PC端->邮件】中编辑邮件支持通过以下方式将企业微信在线文档内容导入至邮件正文：
注：目前通过企业微信PC端编辑邮件选择“导入文档”功能，支持选择企业微信“文档功能”的在线word文档，不支持选择excel格式文档或导入电脑本地文档。
1、【企业微信PC端->邮件->新建邮件->下拉菜单】可以选择“导入文档新建”。
![](https://wework.qpic.cn/wwpic3az/802235_cvQ3QADjSHiBPcr_1750242353/0)
2、【企业微信PC端->邮件->新建邮件->工具栏】点击“导入文档”。
![](https://wework.qpic.cn/wwpic3az/610512_mLdz3-n0QvilH1C_1750242816/0)
3、通过【企业微信PC端->邮件->新建邮件】点击编辑邮件正文，右键菜单选择“导入文档”。
![](https://wework.qpic.cn/wwpic3az/2163_PbaWpZh3TLeaW_7_1750242717/0)
