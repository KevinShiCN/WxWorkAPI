# 【企业微信-邮件】编辑邮件插入黄脸表情

企业微信PC端：【邮件->写邮件】选择“表情”入口可选择对应黄脸表情添加至邮件正文。
![](https://wework.qpic.cn/wwpic3az/307446_2Nr458HiT0eEPfg_1749719304/0)

企业微信移动端：【邮件->写邮件】点击邮件正文进入编辑，选择“表情”即可选择黄脸表情添加至邮件正文。
![](https://wework.qpic.cn/wwpic3az/639048_OnglVlxpQk2LDni_1758168874/0)
