# 回复邮件如何带上原邮件中的普通附件？

企业邮箱收到带有附件的邮件，如回复邮件需带上原邮件的普通附件，可通过以下方式添加：
一、网页端登录邮箱查看收到的邮件，点击【回复->添加附件->添加原邮件附件】。
注：若是属于超大附件，需下载后手动添加本地附件。
![](https://wework.qpic.cn/wwpic/864799_8_jLZR-HQbqB1XC_1696755368/0)

二、企业微信客户端【邮件】中点击回复邮件时，在收件人栏增加新收件人后可以按需一键添加原邮件附件。
![](https://wework.qpic.cn/wwpic3az/857993_mPmCVITETJGYTlx_1758174472/0)
