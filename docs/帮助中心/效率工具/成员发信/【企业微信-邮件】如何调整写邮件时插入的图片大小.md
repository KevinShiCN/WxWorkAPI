# 【企业微信-邮件】如何调整写邮件时插入的图片大小

在企业微信客户端【邮件】撰写邮件时，可以直接对插入的图片进行大小调整。具体操作步骤如下：
**一、插入图片**
打开企业微信电脑桌面端，进入【邮件->新建邮件】。点击编辑器中的“插入图片”按钮，选择需要添加的图片。

**二、调整图片大小**
在正文中已插入的图片上右键单击，在弹出的菜单中选择【调整图片大小】。

**三、选择缩放比例**
可以选择系统预设的缩放比例（如 100%、50%、25%），也可在“自定义”选项中输入具体数值，灵活控制图片显示尺寸。
