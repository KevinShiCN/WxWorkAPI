# 「分别发送」功能介绍

让您「省时省力」发邮件->[点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488323705827174)

如果您想发封邮件给多个好友，但又希望对方看到收件人里只有自己的邮件地址，让每个人都感觉到是您单独发送的，那么可以使用【写信->分别发送】；“分别发送”可以帮您快速的群发邮件给多个人，每个人收到的都是单独的“一对一”的邮件。
