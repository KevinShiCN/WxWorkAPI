# 「个性群发」功能

### [](https://open.work.weixin.qq.com)一、 「个性群发」介绍

当需要发送大量内容相似、但关键信息因人而异的邮件时，例如员工工资条、候选人面试邀请、商务函等，可使用“个性群发”功能。该功能能够将邮件模板中的变量（如姓名、公司、日期）自动替换为收件人对应的真实信息，实现“一人一邮”的个性化发送。

**使用路径：**网页端登录邮箱，进入写信页，点击【个性群发】应用。

温馨提示：
1、目前该功能仅支持在网页端操作使用。
2、“个性群发”属于邮件高级账号权限，需确保账号已被分配邮件高级功能。普通账号或学生高级功能邮箱暂不支持使用。

### [](https://open.work.weixin.qq.com)二、两种发送方式详解

「个性群发」功能提供 “发表格”和 “发图文”两种模式，以适应不同场景需求。
![](https://wework.qpic.cn/wwpic/620040_xUqEbzbbTaK453h_1679624258/0)

#### [](https://open.work.weixin.qq.com)1、发表格 [模板文件](https://rescdn.qqmail.com/node/wwqy/qymng/style/images/sass/independent/81ac2c0d1cc044c8976ead90b5979a2c.xlsx)

**适用场景：**适用于直接发送表格内容的场景，如工资条发放。
**操作流程：**将已统计好的表格文件（需包含“邮箱地址”列）上传到个性群发中。系统将自动识别表格内容并生成邮件正文。
![](https://wework.qpic.cn/wwpic/152093_3dXc2S8sTUyRJxB_1679624268/0)

**预览与发送：**

- 点击“预览并发送”，可在左侧栏选择不同收件人姓名，查看其对应的真实邮件内容。
- 支持“把此邮件发到我的收件箱”进行效果测试。
- 确认内容无误后，即可点击“确认发送”，每个收件人将会收到各自的邮件内容。

![](https://wework.qpic.cn/wwpic/438001_iew_DZ1SS6Sxs_4_1679624279/0)

#### [](https://open.work.weixin.qq.com)2、发图文 [模板文件](https://rescdn.qqmail.com/node/wwqy/qymng/style/images/sass/independent/TextTemplate.xlsx)

**1、适用场景：**适用于需要定制化正文模板的场景，如面试邀约、商务信函。
**2、表格文件要求：**
（1）标题行：第一行为标题行，其列名将成为邮件模板中可调用的变量名。
（2）格式限制：文件内不能含有合并单元格。

（2）邮箱列：必须包含命名为 “邮箱地址”或 “地址”的列，且每个地址均需合法有效。
（3）数据行：从第二行起，每一行代表一位收件人的信息。

（4）标题命名规范：标题（变量名）不能包含 ‘ “ < > ( ) . []?\^{|等特殊英文字符，建议改用 ‘“《》（）。【】等对应的中文字符。

**3、编辑与发送：**

- 上传表格后，编辑邮件正文，通过插入【变量】来实现内容个性化。
- 编辑完成后可进行预览，检查变量替换效果。

温馨提示：支持将编辑好的正文保存为模板，方便后续类似场景快速调用。

### [](https://open.work.weixin.qq.com)三、功能限制与说明

1、收件人数限制：单封个性群发邮件最多支持添加 1000个收件人。
2、邮件查看：

- 发送后，可长期在网页端【已发送】邮件中，查看每一位收件人收到的具体内容。
- 在其他邮件客户端（如Foxmail、Outlook等）的“已发送”中，仅能查看邮件模板或第一位收件人的内容。

### [](https://open.work.weixin.qq.com)四、常见问题

#### [](https://open.work.weixin.qq.com)1、网页端使用「个性群发」发信时提示系统繁忙

请检查上传的表格文档中是否已正确填写了变量数据。如果文档中变量列为空，请补充相应数据后重新尝试上传和发送。

#### [](https://open.work.weixin.qq.com)2、个性化群发是否会受到发信量限制？

会。个性化群发与保密邮件一样，均受企业邮箱发信总量限制。

#### [](https://open.work.weixin.qq.com)3、「个性群发」是否支持添加抄送人？

不支持。如需添加抄送人，建议使用企业微信客户端内的【邮件】功能，选择「群发邮件」进行操作，详情可参考：[【企业微信-邮件】群发邮件功能介绍](https://open.work.weixin.qq.com/help2/pc/20601)

#### [](https://open.work.weixin.qq.com)4、上传时提示 “表格文件中部分行的邮箱地址缺失或不合法”

请检查邮箱地址中是否误输入了空格，或地址格式是否符合规范（如包含“@”和正确域名）。
