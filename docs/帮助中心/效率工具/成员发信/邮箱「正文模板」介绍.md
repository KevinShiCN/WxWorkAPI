# 邮箱「正文模板」介绍

「商务函」模板功能内置了常见的企业公文模板，可帮助您高效、规范地撰写商务邮件。
网页端登录企业邮箱【写信->正文模板】页面点击选择模板类型，邮件正文将自动替换为对应的商务函内容，点击模板中的文字区域，输入相应信息即可完成替换。
