# 邮件「普通附件」与「超大附件」区别

企业微信邮箱支持两种附件发送方式：普通附件和超大附件，

### [](https://open.work.weixin.qq.com)一、普通附件

#### [](https://open.work.weixin.qq.com)1、大小限制：

单个附件最大支持 50MB。

#### [](https://open.work.weixin.qq.com)2、上传方式:

**网页端：**【写信->附件】点击“从本地选择”或“拖拽上传”
![](https://wework.qpic.cn/wwpic3az/910769_-KhMpol7TKipjz7_1770623255/0)
**企业微信电脑端：**【邮件->新建邮件->附件】，可选中上传本地文件、微盘文件或选择聊天中的文件。
**企业微信移动端：**【邮件->新建邮件->附件】点击文件，可选择上传本地文件、微盘文件或聊天中的文件。

**温馨提示**：若选择的文件超过50M，文件将存入微盘「我的空间-文件中转站」后上传以超大附件方式发送，「文件中转站」有着文件保存时间限制，所以超大附件对应也会有保存期限，接收者需要在文件保存期内进行下载。
![](https://wework.qpic.cn/wwpic3az/422637_PgzegDuTSg2Neap_1770621868/0)

### [](https://open.work.weixin.qq.com)二、超大附件

#### [](https://open.work.weixin.qq.com)1、大小与存储说明

（1）使用「微盘」的企业微信邮箱，上传超过50M的文件将存入微盘「我的空间->文件中转站」后上传以超大附件方式发送。
（2）邮件高级功能账号网页端编辑邮件上传单个超大附件的大小不能超过 3G；企业微信客户端编辑邮件单个超大附件大小不能超过20G.
（3）单封邮件里可添加多个文件，但所有文件的总大小不能超过微盘「我的空间->文件中转站」总的剩余可用空间限制。
（4）若企业使用「企业网盘」的企业邮箱，单封超大附件大小不能超过 2GB。总大小亦不可超过「文件中转站」的剩余空间。

**微盘「我的空间-文件中转站」容量说明：**
（1）普通成员邮箱的「文件中转站」容量上限为 20GB。
（2）邮件高级功能账号「文件中转站」容量上限为 20GB。

**温馨提示：**您可登录网页端邮箱：https://exmail.qq.com/login ，在邮箱左侧【邮箱->功能栏】查看使用的是「企业网盘」还是「微盘」。

#### [](https://open.work.weixin.qq.com)2、上传方式

**网页端：**在【写信】页面，点击【附件】按钮，并选择“从微盘选择”。
![](https://wework.qpic.cn/wwpic3az/51897_j_Tr3RmxQoiEzvh_1770623222/0)
**企业微信电脑端：**进入【邮件->新建邮件】，点击【超大附件】，可选择上传本地文件、微盘文件或聊天中的文件。
**企业微信移动端：**目前暂不支持发送超过50MB的文件。

**温馨提示：**
「文件中转站」有着文件保存时间限制，所以超大附件对应也会有保存期限，接收者需要在文件保存期内进行下载；若超过文件有效期，文件将无法下载。
![](https://wework.qpic.cn/wwpic3az/794874_Z_6-ReLKTfq8MQS_1770621887/0)

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、企业微信客户端如何下载超大附件？

登录企业微信电脑端，进入【邮件】打开包含目标超大附件的邮件，点击附件旁的下拉箭头（⌄），选择【另存为】即可下载至本地。
**温馨提示：**超大附件有保存期限，需在有效期内下载，过期后将无法下载。
