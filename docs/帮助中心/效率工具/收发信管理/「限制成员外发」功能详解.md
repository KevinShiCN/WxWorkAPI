# 「限制成员外发」功能详解

### [](https://open.work.weixin.qq.com)一、功能介绍

为防止员工通过企业邮箱向外部联系人发送邮件，导致公司敏感资料与文档等信息外泄，管理员可通过此功能灵活设置成员的外发邮件权限。

温馨提示：该功能为企业高级功能特权，需购买5个或以上邮件高级功能账号才可启用。

### [](https://open.work.weixin.qq.com)二、设置流程

#### [](https://open.work.weixin.qq.com)1、设置路径：

企业微信管理员登录到管理后台【协作->邮件->安全管理->内控管理->限制外发】
![](https://wework.qpic.cn/wwpic3az/261508_m_7vmQWuQ7K6xv4_1756274823/0)

#### [](https://open.work.weixin.qq.com)2、配置规则

**限制名单：**选择需要禁止外发邮件的成员账号。添加到限制名单的成员将无法向公司域名以外的邮箱地址发送邮件。
温馨提示：所有成员账号（不区分是否分配高级功能）均可被添加至限制名单。

**白名单：**被添加至白名单的成员，外发权限不受影响。
温馨提示：仅支持已分配邮件高级功能的成员账号可被添加至白名单。

![](https://wework.qpic.cn/wwpic3az/331769_LFlK3r7HTBen_Cq_1753072258/0)

#### [](https://open.work.weixin.qq.com)3、生效效果

**受限成员：**可向公司内部成员发送邮件，向外部邮箱发信的操作会失败并收到系统提示。
**白名单成员：**可正常与内、外部联系人进行邮件通信，不受限制规则影响。
