# 如何设置成员邮箱退出到指定地址？

可登录【企业微信管理后台->协作->邮件->配置->退出时跳转】输入指定地址（比如公司网站、个性化登录页面等），那么成员网页端登录邮箱，点击邮箱右上角“退出登录”便会则会跳转到管理端设置的指定地址。
注意：
1、该功能属于邮件高级功能，企业需购买≥5个高级功能账号才可设置。
2、设置指定网址须为已添加至腾讯企业邮箱管理端的已备案域名。
