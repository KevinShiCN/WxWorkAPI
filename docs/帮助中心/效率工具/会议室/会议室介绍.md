# 会议室介绍

会议室功能，可以帮助企业更高效的开会交流、合理使用企业会议室，支持审批预约、选择对应位置会议室预约等。
使用企业微信会议室功能，首先需要企业微信管理员在企业微信管理后台，进行[会议室配置](https://open.work.weixin.qq.com/help2/pc/15207)，设置会议室预定时间范围，添加预定审批，手动添加会议室或是批量导入公司会议室。设置完成后，员工就可以在企业微信[预定会议室](https://open.work.weixin.qq.com/help2/pc/15209)了。
