# 管理员如何查看会议室使用明细

目前管理员支持在[管理后台](https://work.weixin.qq.com)【应用管理->会议室->会议室预定明细】导出会议室使用明细。

导出的信息包括：会议室名称、位置、容量、预定人、预定人所属部门、日程标题、日程参与人数、日程接受人数、日程开始时间、日程结束时间、预定开始结束时间、预定时长等。导出格式为Excel文件。
