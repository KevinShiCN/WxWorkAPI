# 如何创建日历

#### [](https://open.work.weixin.qq.com)一、 个人/团队日历

支持点击【日程应用->添加日历->订阅公共日历->右上角“三”->添加日历】，选择个人/团队日历完成创建，方便管理个人工作安排，也可共享给团队成员协作。
![](https://wework.qpic.cn/wwpic3az/301225_HzC4yjiMS8K2ccg_1732175019/0)

#### [](https://open.work.weixin.qq.com)二、 公共日历

点击【日程应用->右上角“三”->添加日历->订阅公共日历】，可以创建一个公共日历，选择日历本的共享成员和可订阅范围，创建后成员可[订阅公共日历](https://open.work.weixin.qq.com/help2/pc/18211)。
![](https://wework.qpic.cn/wwpic3az/445495_CF6EdbkpQ_WswvX_1732175023/0)

#### [](https://open.work.weixin.qq.com)三、 全员日历

管理员可在【管理后台->协作->日程应用->添加全员日历】，选择全员日历的自动订阅范围，自动订阅范围内的成员将在自己的日程应用中看到全员日历本中的日程。
![](https://wework.qpic.cn/wwpic3az/392983_ussCL5mRTUGQaBG_1753693387/0)
