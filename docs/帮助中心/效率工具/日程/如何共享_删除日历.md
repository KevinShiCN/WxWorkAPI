# 如何共享/删除日历

### [](https://open.work.weixin.qq.com)一、删除日历

只有创建者可以删除日历，删除后，给所有（不包括创建人）参与者推送日程消息。
![](https://wework.qpic.cn/wwpic3az/781865_EyNLb8b7QGeQcY8_1732175565/0)

### [](https://open.work.weixin.qq.com)二、共享日历

日历创建人可将日历共享给上级、下属、项目同事等成员，可以设置是否允许可被查看日历内容。

可设置3种权限：可查看、可编辑、可管理（默认可查看）。其中可编辑权限的人员增/删/改日历的话，暂无增/删/改的记录，也不会推送提醒通知给日历创建人。

（1）可查看：支持查看该日历本下的日程、会议事件。同时可查看支持设置“仅查看闲忙状态”，给对应成员开启仅查看闲忙状态后权限后，成员在查看日历时，该日历中的日程会被打码处理/日程名称只显示星号/日程名称无法正常显示文字。

（2）可管理：成员和日历本创建者拥有一致权限，可以管理该日历本下的日程、会议事件。

（3）可编辑：成员可编辑该日历本下的日程、会议事件。

【电脑端->日程->点击日历旁边的“…”->添加共享成员】添加后可以给成员设置权限。
![](https://wework.qpic.cn/wwpic3az/293241_y_2mV1SGQ6S0RUA_1750831529/0)
![](https://wework.qpic.cn/wwpic3az/841066_89K0HQmHTwOosam_1750831531/0)
【手机端->日程->右上角的“≡”->点击目标日历->添加共享成员】
![](https://wework.qpic.cn/wwpic3az/778594_uWvLTslWQkGuZn__1732175644/0)

### [](https://open.work.weixin.qq.com)三、退出日历

共享成员可以自动退出该日历，退出后需要推送通知给创建人。退出后，共享人列表需删除该人。
