# 如何订阅日历

1、在日历中可以看到可订阅范围内的全部公共日历，点击订阅后，在自己的日历视图中会展示该公共日历本及日程内容。

操作路径如下：
【手机端->日程应用->右上角三横/三点->添加日历->订阅公共日历】
![](https://wework.qpic.cn/wwpic3az/986750_FdmK9xvpQqGR7fO_1732175224/0)
【电脑端->日程->添加日历->订阅公共日历】
![](https://wework.qpic.cn/wwpic3az/300200_jOP-SunXT_So2YC_1765262273/0)

2、已订阅日历的成员，支持退订日历。在日历列表里找到已订阅的日历，点开日历旁边的三个点即可看到“取消订阅”按钮。
![](https://wework.qpic.cn/wwpic3az/180266_3XArAjlBQKiDWBr_1765262300/0)
