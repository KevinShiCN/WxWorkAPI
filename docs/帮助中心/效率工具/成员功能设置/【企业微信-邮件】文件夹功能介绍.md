# 【企业微信-邮件】文件夹功能介绍

【企业微信->邮件】功能支持创建文件夹归类邮件、同时也支持设置常用文件夹。可对已创建的文件夹重命名、调整文件夹排序、删除等操作。

### [](https://open.work.weixin.qq.com)一、创建文件夹

入口：【企业微信->邮件->我的文件夹】点击”+”选择新建文件夹。
![](https://wework.qpic.cn/wwpic3az/776652_WCAnXJ1QQwG40BN_1750143297/0)

### [](https://open.work.weixin.qq.com)二、重命名文件夹名称

入口：【企业微信->邮件->我的文件夹】选择已创建的文件夹，右击点击“：重命名。”
![](https://wework.qpic.cn/wwpic3az/332005_CRo2PUUfT1K5CqC_1750143532/0)

### [](https://open.work.weixin.qq.com)三、文件夹排序

操作方式：【企业微信->邮件->我的文件夹】选择已创建的文件夹进行拖拽排序。

### [](https://open.work.weixin.qq.com)四、邮件排序方式

支持筛选未读、审批、日程邮件；同时也支持按时间、发件人、由旧到新等排序方式，并且支持选择按分组显示。
![](https://wework.qpic.cn/wwpic3az/354215_AjPMtwIbSnmWbvf_1750143834/0)
![](https://wework.qpic.cn/wwpic3az/12190_GOOpG2bTRRuwIUY_1750144420/0)

### [](https://open.work.weixin.qq.com)五、设置常用文件夹

【企业微信->邮件->我的文件夹】选择文件夹，右击点击“添加到常用文件夹”，设置常用文件夹后，文件夹将会在【企业微信->邮件】上方显示。
![](https://wework.qpic.cn/wwpic3az/908210_cP_B8_HCQlWrENz_1750144583/0)

### [](https://open.work.weixin.qq.com)六、删除文件夹

入口：【企业微信->邮件->我的文件夹】选择文件夹，右击点击“删除文件夹”删除文件夹可选择是否邮件移动回收件箱或同时删除邮件。
注：删除文件夹勾选了同时删除邮件，邮件将彻底删除，不支持恢复，请谨慎操作。
![](https://wework.qpic.cn/wwpic3az/747171_W4TqKXyyQAixZ1__1750144917/0)
