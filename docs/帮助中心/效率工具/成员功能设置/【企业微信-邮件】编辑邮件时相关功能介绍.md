# 【企业微信-邮件】编辑邮件时相关功能介绍

让你的邮件写得「更专业」->[点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488323705827174_2)

### [](https://open.work.weixin.qq.com)一、如何设置邮件字体样式

1、企业微信pc端写信支持设置默认字体，可在企业微信桌面客户端打开【邮件->邮件设置->写邮件->字体】中选择写信默认字体、字号、颜色、行距。
注：企业微信客户端设置写信默认字体样式只针对本地客户端使用，不支持同步到网页端、移动端或者其他客户端。
![](https://wework.qpic.cn/wwpic3az/937950_CNAsO_JHTM-FLOG_1750146162/0)

2、企业微信PC端写信页面支持快速搜索“写信字体”，通过【企业微信-邮件-写邮件】时点击“字体”，可快速搜索写信字体编辑邮件内容。
![](https://wework.qpic.cn/wwpic3az/295712_dGSXZ50xTgyN4zq_1750146075/0)

### [](https://open.work.weixin.qq.com)二、编辑邮件时如何设置段落格式

1、通过【企业微信-邮件-写邮件】支持设置邮件正文内容的段落格式，例如：行间距、前后段落的特殊缩进等。
设置方法：选择需要设置段落格式的正文内容，然后鼠标右键选择段落格式进行设置。
![](https://wework.qpic.cn/wwpic3az/450447_mqnSanOnSkGGtnB_1750146399/0)
![](https://wework.qpic.cn/wwpic3az/579605_lIimT0IOTWmZuBc_1750149503/0)

### [](https://open.work.weixin.qq.com)三、如何添加表格至邮件正文

企业成员邮箱通过【企业微信PC端->邮件】新建、回复或转发邮件时，点击“插入表格”即可在邮件正文中添加表格，同时支持在大表格中嵌入小表格。
![](https://wework.qpic.cn/wwpic3az/39604_W5AdB89tRhmMbb2_1750149863/0)
