# 如何更换邮箱绑定的微信？

目前暂不支持在邮箱中直接操作更换绑定的微信，可通过【企业微信手机端->消息页面左上角“三”->设置->账号与安全->微信】可以点击更改企业微信绑定的微信号，更换后会自动同步更新至企业微信对应企业邮箱所绑定的微信。
注：暂不支持解绑微信，仅支持换绑。
