# 企业微信邮件「智能写信」 介绍

### [](https://open.work.weixin.qq.com)一、邮件「智能写信」说明

通过「邮件」入口唤起「智能总结」功能，输入需要编写的邮件内容指令生成完整邮件框架。「智能总结」会基于邮箱过往的邮件内容、已有的会话记录、文档和会议录制内容等进行深度总结生成邮件内容，选择“使用内容”即可插入到邮件正文进行编辑发送，提升写信效率。

### [](https://open.work.weixin.qq.com)二、邮件「智能写信」入口

电脑端：企业微信客户端【邮件->新建邮件->智能总结】
![](https://wework.qpic.cn/wwpic3az/873818_xZNZf__SR8ezUhs_1755827385/0)

手机端：企业微信APP【邮件】点击“+”选择智能总结。
![](https://wework.qpic.cn/wwpic3az/918893_tnKW-pdKTja7PsD_1755827258/0)

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、是否支持关闭「智能写信」功能？

目前暂不支持关闭该功能入口。
