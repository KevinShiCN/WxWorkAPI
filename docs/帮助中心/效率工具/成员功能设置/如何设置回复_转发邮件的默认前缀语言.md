# 如何设置回复/转发邮件的默认前缀语言

### [](https://open.work.weixin.qq.com)一、网页端设置方法

**设置路径：**网页端登录企业邮箱【设置->常规->回复/转发->回复/转发时主题】

**可选语言前缀**
1、中文前缀：回复：/转发：/自动回复：/自动转发：
2、英文前缀：Re:/Fw:/AutoReply:/AutoForward：

![](https://wework.qpic.cn/wwpic3az/655361_l7kYzrNMREab-Ss_1721983289/0)

### [](https://open.work.weixin.qq.com)二、企业微信客户端设置方法

**设置路径：**企业微信客户端【邮件->邮件设置->写邮件->回复/转发前缀】

**可选语言前缀**
1、跟随企业微信语言：通过企业微信【设置->通用->语言】查看客户端当前语言设置。
2、中文前缀：回复:/转发：
3、英文前缀：Re:/Fw:
