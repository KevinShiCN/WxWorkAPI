# 表格提示「粘贴内容过多」应该怎么办？

1、从其他表格工具复制时，请选择实际有内容的表格范围（如下图所示）。部分表格工具选择整行整列时会包含大量额外单元格，导致无法粘贴。
![](https://wework.qpic.cn/wwpic/816781_MDOViV1cS6KvrJX_1678678423/0)
2、如果粘贴内容超过 1000 行，请尝试从「文档->导入->导入本地文档」来导入本地文件。
![](https://wework.qpic.cn/wwpic3az/349876_co8FwJ3kTjaiPa3_1765351279/0)
