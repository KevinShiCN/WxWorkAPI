# 管理员如何加入微盘共享空间

为成员分配微盘高级功能后，管理员可加入该成员创建的共享空间、管理空间内的所有文件。

管理员可在管理端【协作->微盘->空间文件管理->空间内容管理->查看->加入空间】，管理员加入空间之后，该空间的管理员会在微盘通知收到一条提醒“企业管理员XXX以管理员身份加入了空间”。
