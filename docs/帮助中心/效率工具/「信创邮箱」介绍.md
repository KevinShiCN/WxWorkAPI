# 「信创邮箱」介绍

### [](https://open.work.weixin.qq.com)一、「信创邮箱」说明

1、全面适配国产化的主流CPU、操作系统、数据库、中间件等产品，为企业提供安全合规、稳定可靠的邮箱运行环境。
2、企业所有成员的邮箱在开启后将自动迁移至适配信创的环境中运行。

### [](https://open.work.weixin.qq.com)二、启用「信创邮箱」条件

1、启用「信创邮箱」，企业全体成员邮箱账号需分配邮件高级功能。
注：不包含通讯录未加入的成员、通讯录被删除的成员、主动退出企业的成员、企微通讯录被禁用成员、邮箱账号被禁用、该部门邮箱被禁用成员, 账号为非生效专属域名邮箱的成员。

### [](https://open.work.weixin.qq.com)三、启用「信创邮箱」须知

1、启用「信创邮箱」后，全体成员邮箱将迁移至基于国产化基础设施构建的环境中运行，以符合企业信创合规要求。
2、「信创邮箱」生效范围：企业下所有邮箱
3、使用企业微信「邮件」功能的成员，启用「信创邮箱」后无需额外操作，将自动登录信创邮箱。
4、使用「邮件网页版」的成员将会被登出，需要重新登录邮箱。
5、使用「第三方客户端」的成员将会被登出，在重新登录邮箱时需修改IMAP/POP 配置信息。

### [](https://open.work.weixin.qq.com)四、企业微信邮箱如何开启/关闭「信创邮箱」

企业微信创建人/超级管理员登录【企业微信管理后台->协作->邮件->信创邮箱】开启/关闭「信创邮箱」
注：
1、邮件「应用负责人」不支持管理「信创邮箱」。
2、开启「信创邮箱」后，若企业下存在未分配邮件高级功能账号，企业微信【管理后台->首页】会下发“当前企业有成员未分配邮箱高级功能，信创邮箱将无法使用，请尽快分配”提醒，7天后系统会自动关闭「信创邮箱」，需及时为企业下邮箱分配邮件高级功能。

### [](https://open.work.weixin.qq.com)五、「信创邮箱」客户端配置参数

**POP3/SMTP协议**
接收邮件服务器：xcpop.exmail.qq.com ，使用SSL，端口号995
发送邮件服务器：xcsmtp.exmail.qq.com ，使用SSL，端口号465

**IMAP协议/SMTP协议**
接收邮件服务器：xcimap.exmail.qq.com ，使用SSL，端口号993
发送邮件服务器：xcsmtp.exmail.qq.com ，使用SSL，端口号465

电子邮件地址：企业邮箱完整邮件地址
密码：企业邮箱密码；若邮箱已开启安全登录需【网页端登录邮箱->设置->邮箱绑定】中获取客户端授权码。
同时【网页端设置->收发信设置】中需开启客户端服务协议。
**温馨提示：**企业开启信创邮箱后，客户端不支持配置exchange 协议。

### [](https://open.work.weixin.qq.com)六、关闭「信创邮箱」须知

1、停用后，全体成员邮箱将不再在信创环境中运行。
2、使用企业微信「邮件」功能的成员，启用后无需额外操作，将自动登录邮箱。
3、使用「邮件网页版」的成员将会被登出，需要重新登录邮箱。
4、使用「第三方客户端」的成员将会被登出，在重新登录邮箱时需修改IMAP/POP 配置信息。

### [](https://open.work.weixin.qq.com)七、常见问题

#### [](https://open.work.weixin.qq.com)1、开启「信创邮箱」是否需要额外收费？

开启「信创邮箱」不需要额外收费，但需为企业通讯录全体成员购买邮件高级功能且分配高级功能账号。
注：不包含通讯录未加入的成员、通讯录被删除的成员、主动退出企业成员、企微通讯录被禁用成员、邮箱账号被禁用、该部门邮箱被禁用成员, 账号为非生效专属域名邮箱的成员。

#### [](https://open.work.weixin.qq.com)2、启用信创邮箱需多久？

启用「信创邮箱」一般几分钟即可完成。

#### [](https://open.work.weixin.qq.com)3、启用信创邮箱后，通讯录新增成员未分配高级功能，是否有影响？

开启「信创邮箱」后，若企业有成员未分配邮箱高级功能，信创邮箱将无法使用；同时企业微信【管理后台->首页】会有相关提醒“当前企业有成员未分配邮箱高级功能，信创邮箱将无法使用，请尽快分配”，7天内未为全体成员分配「信创邮箱」，「信创邮箱」会自动关闭。

#### [](https://open.work.weixin.qq.com)4、启用信创邮箱后，Mac Foxmail客户端配置信创邮箱，未有启动超大附件选项？

信创邮箱在Mac Foxmail暂不支持超大附件，可通过网页端登录邮箱或企业微信【邮件】使用超大附件。

#### [](https://open.work.weixin.qq.com)5、启用信创邮箱后，成员邮箱是否会收到提醒？

目前管理端开启「信创邮箱」，成员侧不会收到相关提醒。
