# 如何使用苹果电脑（macOS）自带邮件程序配置企业邮？

### [](https://open.work.weixin.qq.com)一、什么是Mac邮件程序?

Mac邮件程序是指内置在MacBook/MacBook Pro/MacBook Air/iMac(这些设备这里统称为Mac)中的邮件应用程序Mail，方便用户在本地管理服务器上的电子邮件。

### [](https://open.work.weixin.qq.com)二、如何在苹果电脑邮件程序中配置企业邮?

下面向大家介绍如何使用Mac上的邮件程序Mail配置企业邮账户 ，这里以macbook(版本：OS10)为例:
1、在Mac打开邮件程序，左上角点击【邮件->添加账户】
![](https://wework.qpic.cn/wwpic/304702_gON2-J_JTQ2Raj__1678332220/0)

2、选择其他“邮件”账户。
![](https://wework.qpic.cn/wwpic/651422_hALkWkTZTSqLeO2_1678332231/0)

3、进入添加账户页面，输入您的企业邮箱地址和密码（如已开启安全登录需要使用客户端专用密码：[如何获取客户端专用密码](https://open.work.weixin.qq.com/help2/pc/19902)）以及账户类型，点击“登录”。
![](https://wework.qpic.cn/wwpic/291059_DhRVJf80SMqTmsT_1678332241/0)
![](https://wework.qpic.cn/wwpic/140327_xavxHVb_RDG7TJB_1678332250/0)
**其他说明：**
用户名：（填写您的账号全称或邮件地址）
账户类型：IMAP
接收邮件服务器：imap.exmail.qq.com ，使用SSL，端口号993
发送邮件服务器：smtp.exmail.qq.com ，使用SSL，端口号465
账户类型：POP
接收邮件服务器：pop.exmail.qq.com ，使用SSL，端口号995
发送邮件服务器：smtp.exmail.qq.com ，使用SSL，端口号465

4、完成并收取邮件。
![](https://wework.qpic.cn/wwpic/817499_f7CLgTZGS8O2K5g_1678332271/0)
