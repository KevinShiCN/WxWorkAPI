# 如何转交文档所有权

### [](https://open.work.weixin.qq.com)一、可转交文档所有权的条件

如需转让文档的“超级管理员”权限，文档需要**同时**满足以下两个条件，才会出现“转交所有权”的入口。
1、自己创建的文档
2、文档归属“我的空间”下面

### [](https://open.work.weixin.qq.com)二、如何转交所有权

#### [](https://open.work.weixin.qq.com)1、手机端如何转交所有权

手机端可在【工作台->微盘->我的空间->点击对应文档右侧的“···”->权限管理->转交所有权】选择需转交的成员。
![](https://wework.qpic.cn/wwpic3az/572198_PZq0pxmvS5KUJf8_1765258804/0)

#### [](https://open.work.weixin.qq.com)2、电脑端如何转交所有权

电脑端可在【文档->我的空间->右键对应文档->转交所有权】；
![](https://wework.qpic.cn/wwpic3az/931555_X4lh5DlARZKVFrA_1750836647/0)
或是【打开对应文档->右上角“三”->权限管理->转交所有权】。
![](https://wework.qpic.cn/wwpic3az/798890_l9lsWCjQS2q0PVE_1750836651/0)

#### [](https://open.work.weixin.qq.com)3、如何批量转交所有权

可在【电脑端->文档】中，按住鼠标左键拖动选择多个文档后，右键文档选择转交所有权。
![](https://wework.qpic.cn/wwpic3az/191906_3OrsUC4oQfe23Co_1750836655/0)
或是在【微盘->我的空间】中，创建文件夹，将需要转交所有权的文档移动到该文件夹中，右键文件夹操作转交所有权。
![](https://wework.qpic.cn/wwpic3az/852855_w-jw2XfVRkCP322_1750836847/0)

### [](https://open.work.weixin.qq.com)三、常见问题

1、文档转接所有权提示转交失败。
成员操作转交后，对方需要在7天内确认，否则会导致转交失败。

2、文档发起转让所有权，但新的接替人没有“与我相关”列表没有收到通知。
若转让时选择新的接替人未激活账号，则新的接替人不会收到转让所有权的通知，建议新接替人激活账号后，文档所有者再重新发起转交。
