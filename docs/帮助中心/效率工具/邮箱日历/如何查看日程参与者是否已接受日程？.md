# 如何查看日程参与者是否已接受日程？

通过【企业微信->邮件->日程邀约】发送日程邮件后，参与者接受日程，系统会推送邮件通知参与人已接受日程；
也可以通过网页版【登录邮箱->已发送】找到日程邮件后，查看日程邮件参与者接受详情。
