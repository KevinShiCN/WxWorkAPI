# 如何同步打卡数据到智能表格

v5.0.2版本开始，支持将「打卡」考勤月报数据同步到智能表格，每月统计更轻松。可通过考勤分析仪表盘，直观查看考勤异常和趋势，及时了解团队考勤情况。

### [](https://open.work.weixin.qq.com)一、在智能表格中同步打卡数据

【电脑端->打开智能表格->同步应用中的数据->打卡】
支持将打卡应用数据同步到智能表格中，可以手动操作同步数据。
![](https://wework.qpic.cn/wwpic3az/190453_OKIYiP-CQNemym4_1763449897/0)
在同步时，可以选择时间范围。
![](https://wework.qpic.cn/wwpic3az/208152_TqUoXdJeRPOKipf_1763449945/0)
同步完成后，可以在智能表格中查看到同步表示的工作表。
![](https://wework.qpic.cn/wwpic3az/486278_2oqg-uRmTz-OYlJ_1763450103/0)
可点击工作表“…”->数据同步中，操作手动同步数据，也可以在“同步设置”中，开启自动同步，启用后将每小时自动同步一次数据。
![](https://wework.qpic.cn/wwpic3az/906988_7Ki0grr0QvCWtSM_1763450297/0)
![](https://wework.qpic.cn/wwpic3az/797149_5GK4jk01TO2r-00_1763450300/0)

### [](https://open.work.weixin.qq.com)二、考勤分析仪表盘

【手机端->工作台->打卡->统计->团队统计->考勤分析仪表盘】
同步后，工作表中的数据会和打卡保持同步，仪表盘内容实时更新，若不需要同步了，可以手动取消同步，取消同步后，工作表不会再获取打卡应用数据，已同步的数据会正常保留，支持成员正常编辑。
![](https://wework.qpic.cn/wwpic3az/82747_GfRTrGH6RyupAlL_1763449825/0)
v5.0.2版本开始，在【智能表格->连接应用中的数据】新增了“考勤分析仪表盘”，可按照页面向导创建和新建自动化提醒。
![](https://wework.qpic.cn/wwpic3az/379018_BUSrx14xRJ6jyOt_1763517014/0)
![](https://wework.qpic.cn/wwpic3az/126521_r9ThtljDTd27c6X_1763517017/0)

### [](https://open.work.weixin.qq.com)三、导出打卡记录在智能表格中查看

【管理后台->应用管理->打卡->上下班打卡->统计->汇总报表->导出日报/月报】
导出打卡记录后，可使用智能表格查看，自动生成打卡统计仪表盘。
![](https://wework.qpic.cn/wwpic3az/471949_FGOnGdKcSyej45Q_1763450424/0)

### [](https://open.work.weixin.qq.com)四、常见问题

1、同步打卡月报数据到智能表格中，最多可以选择哪个时间范围的数据？
目前同步打卡月报数据到智能表格中，同步的时间范围只能选择包括本月在内的最近12个月，超出此范围的月份将置灰无法选择。
