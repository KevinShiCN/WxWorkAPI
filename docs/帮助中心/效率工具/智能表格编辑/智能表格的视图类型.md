# 智能表格的视图类型

### [](https://open.work.weixin.qq.com)一、如何展开/关闭视图侧边栏

（1）当视图侧边栏展开时，点击表格左上方的视图栏，即可隐藏侧边栏。
![](https://wework.qpic.cn/wwpic3az/209586_2diaddudR9uevzL_1750904419/0)
（2）当视图侧边栏关闭时，点击表格左上方的视图栏，即可展开侧边栏。
![](https://wework.qpic.cn/wwpic3az/571501_UJwIjY8_T7uGmxb_1750904422/0)

### [](https://open.work.weixin.qq.com)二、如何新建/切换视图

#### [](https://open.work.weixin.qq.com)1、电脑端

（1）可在视图侧边栏中切换视图或点击左下方创建工作表/仪表盘/收集表单/导入Excel文件/从模板新建视图；
![](https://wework.qpic.cn/wwpic3az/333190_v_XMYunAQd-tosN_1750904426/0)
也可在表头中点击“新建视图”选择视图类型进行创建视图。
![](https://wework.qpic.cn/wwpic3az/613117__qJAg_sQSNyrQhv_1750904429/0)
（2）创建视图后，可在视图侧边栏中点击已有的不同视图切换查看。
（3）在表头中，鼠标左键长按某个视图即可拖拽调整视图顺序。
![](https://wework.qpic.cn/wwpic3az/310073_XHhM49bgQgqgg-h_1750904433/0)

#### [](https://open.work.weixin.qq.com)2、手机端

（1）手机端支持操作新建视图。
![](https://wework.qpic.cn/wwpic3az/194144_2Um4MUxVQxuIbMg_1765262358/0)
（2）可通过滑动表头唤出视图列表，点击需要查看的视图进行切换，或点击“V”操作切换。
![](https://wework.qpic.cn/wwpic3az/668592_dAP8tg5tSBmTxWF_1765262379/0)

### [](https://open.work.weixin.qq.com)三、不同视图适用场景与使用

#### [](https://open.work.weixin.qq.com)1、表格视图

（1）“表格视图”适用于规整收集、分组统计等场景，是传统的表格使用。
![](https://wework.qpic.cn/wwpic3az/265952_ondxHY_YTDqqPhz_1750904454/0)
（2）点击工具栏的【字段管理】按钮，在窗口可以隐藏列和恢复显示隐藏的列，还可以调整列的顺序。
![](https://wework.qpic.cn/wwpic3az/294675_hpRgwEJjQr-oWDF_1750904459/0)

#### [](https://open.work.weixin.qq.com)2、看版视图

（1）“看板视图”适用于任务分工、状态流转等场景，比如在项目管理场景下，“看版视图”可以在不同的分组中可视化工作流程，任务指派给多个人时也能在看板中分别展现，更方便地进行项目管理。
![](https://wework.qpic.cn/wwpic3az/526618_nkEV-KTXSdmEPGL_1750904502/0)
（2）看板视图的分组依据只能是单选类型的列，或者是人员类型且未开启多选的列，暂时还不支持其他列作为分组依据。
（3）点击工具栏的【看板设置】按钮，将会弹出侧边栏，在侧边栏可以隐藏列和恢复显示隐藏的列，开启/关闭隐藏列名，调整列的顺序，还可以调整封面内容为需要展示的图片列。
![](https://wework.qpic.cn/wwpic3az/133190_9rq62BtpTnytADD_1750904506/0)

#### [](https://open.work.weixin.qq.com)3、甘特视图

（1）“甘特视图”适用于跟进排期、监督进度等场景，可以更方便地看到每个项目的时间线，设置里程碑，统计工时等。
![](https://wework.qpic.cn/wwpic3az/140902_uph3JUESRWGI9LW_1750904510/0)
（2）点击工具栏的【字段管理】按钮，在窗口可以隐藏列和恢复显示隐藏的列，还可以调整列的顺序。
![](https://wework.qpic.cn/wwpic3az/505110_vBL0OR5BS76blsZ_1750904529/0)
（3）点击工具栏的【甘特设置】按钮，将会弹出侧边栏，可以调整时间条起点/终点、时间条颜色以及假期设置。
![](https://wework.qpic.cn/wwpic3az/661763_gJz0MrZLTCib7E1_1750904540/0)

#### [](https://open.work.weixin.qq.com)4、画册视图

（1）“画册视图”适用于展示图文、宣传产品等场景，可将记录以卡片的形式展示，配合表格公告和记录详情效果更佳。
![](https://wework.qpic.cn/wwpic3az/772805_Bmm-t-0DR3KgM3A_1750904554/0)
（2）点击工具栏的【画册设置】按钮，将会弹出侧边栏，在侧边栏可以隐藏列和恢复显示隐藏的列，开启/关闭隐藏列名，调整列的顺序，还可以调整封面内容为需要展示的图片列。
![](https://wework.qpic.cn/wwpic3az/412396_6-yrU1uYSPuEGrf_1750904559/0)

#### [](https://open.work.weixin.qq.com)5、日历视图

（1）“日历视图”适用于日程管理、事项排期等场景，可以从月的维度总览事项的进展，方便地在智能表中管理自己的日程；在团队协作中，则可以集体用同一个日历视图进行事项规划和排期。
![](https://wework.qpic.cn/wwpic3az/100549_oPDvTHXtQpK8I-d_1750904564/0)
（2）点击工具栏的【日历设置】按钮，将会弹出侧边栏，在侧边栏可以调整时间条起点/终点、时间条颜色。
（3）起点和终点可以是同一列，这样的话每个日程都只能在一个日期下。
![](https://wework.qpic.cn/wwpic3az/889869_I-mDCL5lS82axUV_1750904568/0)
（4）点击工具栏的【日历高度】按钮，支持设置日历显示高度，可自行设置为按月自适应、中等、高、超高，调整合适的固定高度后，完整展示同一天内的多个记录。
![](https://wework.qpic.cn/wwpic3az/997380_T-mMx1bqSIei2Nr_1750904574/0)

#### [](https://open.work.weixin.qq.com)6、表单视图

（1）“表单视图”形式类似于问卷，可以用来收集信息和数据。收集到的信息将自动汇总到表格中，结合智能表格的筛选、分组、数据仪表盘等功能，便捷进行后续的整理与分析。
![](https://wework.qpic.cn/wwpic3az/699028_M782dWAjRyysc_S_1750904578/0)
（2）支持通过已有字段一键生成表单来收集信息，收到的数据将自动汇总到表格中。
![](https://wework.qpic.cn/wwpic3az/566034_W0wbM40ASVuJ_TC_1750904582/0)
（3）分享表单链接给他人填写，填写人不会看到其他的提交结果，避免信息泄露。
![](https://wework.qpic.cn/wwpic3az/732907_K6UlpAi7QHyR6m3_1750904586/0)
（4）表单视图可以一键设置自动化提醒，当有新表单提交时，自动发送消息提醒相关人员。
![](https://wework.qpic.cn/wwpic3az/262825_4BgFWt71RsG5wZA_1750904590/0)
