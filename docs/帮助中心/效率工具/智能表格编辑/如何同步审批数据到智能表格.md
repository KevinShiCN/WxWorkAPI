# 如何同步审批数据到智能表格

v5.0.2版本开始，支持将审批数据同步到智能表格中，支持汇总查看各部门、成员的审批数据，自动统计金额分布、审批单状态等，实时掌握企业审批情况。

### [](https://open.work.weixin.qq.com)一、操作路径

【电脑端->打开智能表格->左下角“同步应用中的数据”->审批】
![](https://wework.qpic.cn/wwpic3az/543155_TuFWOF8URziIXaX_1763446029/0)

### [](https://open.work.weixin.qq.com)二、同步说明

1、支持选择需要同步的审批模板，只能查看到自己有管理权限的审批模板，如果没有对应可管理的模板，会出现“暂无可同步的审批模板，请联系企业管理员设置”的提示。
![](https://wework.qpic.cn/wwpic3az/468357_2tzaEInvTHi8_MN_1763446215/0)

2、支持选择发起时间，默认选择最近一个月发起的审批，可自定义调整时间，最多可选择近一年的审批数据。
若选择开始日期早于审批首次发起的时间，则以审批发起时间为准。
![](https://wework.qpic.cn/wwpic3az/288931_mNa2m9Y-Trupnic_1763446284/0)
3、一个智能表格中可以添加多个同步工作表，一个工作表只能与一个审批模板建立同步关系，同步完成后可以在工作表中查看到同步的标识。
![](https://wework.qpic.cn/wwpic3az/176502_aNJ1MnmjRP-hYOc_1763446761/0)
4、一个审批模板可以同步到多个工作表。

5、审批单同步到智能表后，如果审批单删除了，同步后智能表格会同步删除该记录。

6、支持同步的记录：已通过/驳回、未完成、已撤销、再次发起的审批记录都会同步到智能表格。

7、同步的工作表名称为审批模板名称，支持自定义修改，不影响智能表格文件名称。

8、数据同步
操作同步审批数据到智能表格后，支持在智能表格左下角“数据同步”中查看到上次同步时间，并手动操作“立即同步数据”。
![](https://wework.qpic.cn/wwpic3az/544709_uBFccGUzQW6CNkf_1763446567/0)
支持配置“同步设置”，可查看到同步人、设置同步字段（全部字段、部分字段）。
![](https://wework.qpic.cn/wwpic3az/630718_64mMa1WERo-8jys_1763446630/0)
若操作“取消同步”，该工作表会变成普通工作表。
![](https://wework.qpic.cn/wwpic3az/485838_Q8bZ4F9NQ9CMJgg_1763446664/0)

9、同步的字段内容
系统字段：审批单链接、审批单编号、申请人、申请人部门、申请人账号、提交时间、完成时间、提交方式、当前审批状态；
![](https://wework.qpic.cn/wwpic3az/633019_TSV4mGW4R2-POTa_1763446885/0)
审批模板字段：按照模板中的控件顺序展示；
![](https://wework.qpic.cn/wwpic3az/8078_vW-YUL1GRNmp-sz_1763446961/0)
系统字段：审批流程、备注。
![](https://wework.qpic.cn/wwpic3az/993309__G3CpreAQOOKyhJ_1763446963/0)

### [](https://open.work.weixin.qq.com)三、同步到仪表盘

v5.0.2版本开始，在【智能表格->连接应用中的数据】新增了“审批仪表盘”，可按照页面向导创建和新建自动化提醒。
![](https://wework.qpic.cn/wwpic3az/557284_3_yoq4ZNS5SToqM_1763515804/0)
![](https://wework.qpic.cn/wwpic3az/928747_voLdXIfpSSGZlh7_1763515806/0)
在【手机端/电脑端->工作台->审批->数据统计】中，支持查看到“同步到智能表格”入口，可以将审批数据同步到仪表盘中进行查看。
![](https://wework.qpic.cn/wwpic3az/640110_-YSfMwSgSMSL9wR_1763516063/0)
![](https://wework.qpic.cn/wwpic3az/805953_YhnU1Xr_RFixDY5_1763516065/0)
