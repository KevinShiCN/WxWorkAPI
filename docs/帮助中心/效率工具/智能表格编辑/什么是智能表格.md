# 什么是智能表格

2分钟告诉您，到底什么是「智能表格」？[点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488324837827078)

### [](https://open.work.weixin.qq.com)一、智能表格介绍

文档智能表格，是一款新型的数据库电子表格。有别于传统的EXCEl、在线表格等，它具备丰富的列类型和多维的视图展示，可以让您从功能繁复的在线表格中解脱出来，专注于处理数据本身。

### [](https://open.work.weixin.qq.com)二、智能表格的优势

在线表格是以单元格为核心的表格，允许用户对单元格进行任意的数据修改、样式修改，保留了极高的编辑自由度；但数据格式难以控制，容易出现不同成员填写的内容格式也不相同的情况。而对表格添加一些限制来更好地应对信息统计、信息收集的需求，就是智能表格诞生的契机。

和在线表格相比，由于智能表格严格限制了每列只能是一种固定的列类型（可以理解为在线表格的数据验证），因此操作起来更为简单便捷，在协作时也会更少出现误写、乱写数据的情况；同时由于每列内容相同、行列格式规整，就可以方便地进行分组和统计。

智能表格还可以转换为各种各样的视图，可以通过看板视图查看人员和模块分工，通过甘特和日历视图查看项目进展，这些视图的数据都是同源的，只需要修改一处，其他处都可以同时修改。

### [](https://open.work.weixin.qq.com)三、智能表格侧边栏标签

1、智能表格是一个新型业务数据管理工具，内置了丰富的模板，各行各业可以在关键业务环节中使用智能表格，提升全流程的协作效率。
从内部协作的「项目管理、业绩管理、进销存」场景，到对外服务客户的「销售跟进、客户服务管理」等场景，智能表格都有对应的模板，并且还可以根据「人事、行政、财务」等不同岗位来查找合适的模板，所有模板都可以便捷地使用，提升协作效率。
为此，我们在企业微信侧边栏中提供了智能表格的功能模块，方便大家便捷地找到和启用智能表格。

2、在电脑端侧边栏独立显示智能表格标签，需满足以下条件：

（1）企业已开启文档应用功能。（路径：管理后台（https://work.weixin.qq.com）->协作->文档）

（2）企业微信版本为4.1.36.99603(mac) / 4.1.36.6011(win) 或更高版本。

3、企业微信手机端底部栏目前不会有智能表格单独显示，手机端可以通过文档顶部的智能表格按钮来使用智能表格。

4、电脑端侧边栏智能表格图标的顺序暂不支持调整，该情况我们将在后续优化。

5、目前智能表格标签属于文档的功能模板，可通过「管理后台->协作->文档」页面来统一查看与管理，企业可在这个页面中，了解使用情况信息、容量占用情况，并根据自己的使用情况和需求来调整开关、权限等配置。
