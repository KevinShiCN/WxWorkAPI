# 什么是域名？

域名用来表示一个企业或组织在网上的地址，便于快速查找。例如“qq.com”就是一个常见的域名。企业邮箱地址由用户名和域名构成，例如[hr@qq.com](https://open.work.weixin.qq.commailto:hr@qq.com), 便于员工的邮件收发。

#### [](https://open.work.weixin.qq.com)一、企业邮箱为什么需要域名？

企业邮箱地址由用户名和域名构成，域名有助于提升公司的形象。

#### [](https://open.work.weixin.qq.com)二、如何获取域名？

1、注册腾讯企业邮后，可页面选择【企业没有官网】跳转购买企业域名使用；
2、您也可以前往万网、新网等自行购买。

#### [](https://open.work.weixin.qq.com)三、所有域名都能申请腾讯企业邮箱吗？

1、二级域名如果能够设置MX记录，也是可以申请企业邮箱的，请直接在输入框中输入即可；
2、目前不支持中文域名。
