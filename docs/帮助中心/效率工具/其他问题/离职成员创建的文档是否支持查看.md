# 离职成员创建的文档是否支持查看

离职成员的文档是否可查看，取决于文档本身的权限设置。
例如文档权限设置为「企业内、企业外可编辑或仅浏览」，则对应成员仍可继续查看；如果设置为「需申请权限」，则新加入文档的成员需先向文档管理员申请权限，若文档所有管理员均已离职，则会出现无人审批的情况，新成员无法查看文档。

针对离职成员的文档，管理员可在【管理后台->协作->文档->文档管理工具->离职成员文档转交】将离职成员的文档转交给其他在职成员，接替者可以在文档应用中查看到该离职员工创建的所有文档。

注：开通[文档高级功能](https://work.weixin.qq.com/mall/wedoc?wws=19&from=kefu#/wedoc?wws=19&from=kefu)后，即可自动转交分配了高级功能的成员的文档资产。开启自动转交后，分配了高级功能的成员离职时，创建的文档将自动交接给其直属上级。
