# 如何获取在文档中插入的第三方内容链接

### [](https://open.work.weixin.qq.com)一、 figma链接

1、直接复制顶端地址栏的链接即可
2、点击右上角的Present按钮后，复制顶端地址栏即可

![](https://wework.qpic.cn/wwpic/172783_oRXDPtNrRsCVQbR_1645172873/0)
![](https://wework.qpic.cn/wwpic/772902__aOwIytoSvGclce_1645172967/0)

### [](https://open.work.weixin.qq.com)二、 腾讯视频链接

1、点击视频左下角【分享】按钮
![](https://wework.qpic.cn/wwpic/32743_dahiQXzPTgSYPMq_1645173003/0)
2、点击【复制页面地址】，即可成功复制链接地址到剪切板
![](https://wework.qpic.cn/wwpic/859342_0p9UnwrxScaE-d0_1645173023/0)

### [](https://open.work.weixin.qq.com)三、 bilibili链接

1、点击视频下方【分享】按钮
![](https://wework.qpic.cn/wwpic/91864_q9KdN0QjRIaNpnk_1645173045/0)
2、点击【视频地址】右侧的【复制】，即可成功复制链接地址到剪切板
![](https://wework.qpic.cn/wwpic/623428_ypWoUZv-Q7iiffC_1645173143/0)

### [](https://open.work.weixin.qq.com)四、 ProcessOn链接

1、在ProcessOn右上角，找到【分享】按钮并点击
![](https://wework.qpic.cn/wwpic/395341_zz5PpLg8QeygvUp_1645173172/0)
2、在【分享】面板中，点击【嵌入其他站点】，复制【链接】即可成功复制链接地址到剪切板
![](https://wework.qpic.cn/wwpic/498813_2_9f00PUTZycv4y_1645173202/0)

### [](https://open.work.weixin.qq.com)五、常见问题

1、在文档中插入第三方内容（processon）时，提示“文档不存在或无权限访问”。
需核实填写的链接是否为processon.com/embed/，目前需要在第三方应用processon获取嵌入第三方的链接。
