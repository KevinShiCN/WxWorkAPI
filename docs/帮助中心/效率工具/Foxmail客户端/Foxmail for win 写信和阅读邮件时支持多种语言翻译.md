# Foxmail for win 写信和阅读邮件时支持多种语言翻译

为邮件内容是外语的企业，提供写信实时翻译、取词翻译和全文翻译。
1、写信时勾选实时翻译。
![](https://wework.qpic.cn/wwpic/384683_1X-to9pMRxqaW69_1692241460/0)
![](https://wework.qpic.cn/wwpic/71715_gFUPLgqcRxGGX1i_1692241555/0)
2、发出或收到邮件取词翻译或全文翻译。
![](https://wework.qpic.cn/wwpic/647381_d8QNUCl6RrKAI-I_1692242456/0)
![](https://wework.qpic.cn/wwpic/675483_MioZi1YcT8GS6yb_1692242251/0)
3、取词翻译/翻译全文/写信，增加“纠错”功能。
![](https://wework.qpic.cn/wwpic/593846_Tl08wJJ4TiGH9V1_1692241924/0)
注：VIP成员支持多种语言翻译，普通成员仅支持中英文翻译。
