# 如何查看复学码统计

1、启用复学码之后，可在【手机端->工作台->复学码】中查看统计，老师只可以看到自己可见范围内的统计详情（取决于家校设置的「老师可查看的班级」范围），超管可以查看所有家长的统计详情。

2、复学码的统计，是以学生维度计算的，会统计正常、异常、未更新的人数，支持查看未更新的学生、导出全部明细；
统计页会有单独显示异常学生，若有异常学生，可以看到「导出名单」按钮，该按钮可以导出具体的异常名单，也可以直接在统计页面点击异常的学生查看异常详情。
