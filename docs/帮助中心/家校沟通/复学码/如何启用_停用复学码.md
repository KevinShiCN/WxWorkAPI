# 如何启用/停用复学码

1、启用路径：【客户端->工作台->复学码】
仅超级管理员可以启用复学码，非超级管理员进入到该应用会提示需要联系管理员启用。
![](https://wework.qpic.cn/wwpic3az/732157_N3J6MjEjRBO8Nyg_1766039203/0)

2、启用复学码后，学生版的复学码将会通过学校通知发送给使用范围内的家长；也可以通过【手机端/电脑端->工作台->复学码->统计->学生->邀请微信中的家长使用】。
![](https://wework.qpic.cn/wwpic3az/861425_dIAU_ubhTFeVv7H_1766039247/0)

3、教职工版的复学码，是通过【消息界面->复学码】的应用消息推送提醒；也可通过【手机端/电脑端->工作台->复学码->统计->教职工->查看未更新的教职工->发送更新提醒】。
![](https://wework.qpic.cn/wwpic3az/729893_B3tLK4KcRwOVOy8_1766039270/0)

4、停用路径：【客户端->工作台->复学码->配置->停用复学码】，停用后，可在“统计”页面中重新启用。
![](https://wework.qpic.cn/wwpic3az/937740_mWE16_89Ra-OvEY_1766039287/0)
