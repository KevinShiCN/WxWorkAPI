# 学生如何提交班级作业

### [](https://open.work.weixin.qq.com)一、在哪里可以查看作业通知提醒

老师在工作台布置的作业，家长会在家校通知微信插件中会收到推送，作业会以小程序形式在家校通知微信插件展示，家长点击小程序可以提交作业。

提交作业需使用微信提醒。若使用企业微信账号打开作业小程序会出现“仅微信家长可提交作业”的提示。
![](https://wework.qpic.cn/wwpic/778307_UGxp-8sIQNa1pD8_1662533344/0)
如果老师是在班级群布置作业，家长会在对应的班级群聊里收到推送，作业会以小程序形式在群聊中展示，家长点击小程序可以提交作业。
![](https://wework.qpic.cn/wwpic/983367_AJk0zSpES-S4n-X_1662533457/0)
注：使用班级作业小程序，需要打开小程序的麦克风权限。
![](https://wework.qpic.cn/wwpic/688680_Yk-ctS8cTcWU8uk_1662533715/0)

### [](https://open.work.weixin.qq.com)二、提交作业时支持上传哪些内容

家长在交作业时候，可以添加图片/录音/视频（必填）、备注（选填）。

- 图片支持选择相机进行拍摄、从照片中选择，大小不可以超过10M；
- 录音最长10分钟，在最后10秒将会显示倒计时提醒。
- 视频：大小不可超过25M；可以从视频选择，所选视频时长不能超过10分钟，也支持选择相机录制视频，录制最长30秒。
- 图片/录音/视频加起来上限12个，备注上限5000字。
- 如果已经提交了作业，家长可以修改作业内容，重新打开作业->点击修改->编辑作业内容提交即可。
![](https://wework.qpic.cn/wwpic/7615_FmC78XGrRMWfoKB_1644573041/0)

### [](https://open.work.weixin.qq.com)三、如何查看待提交的作业

如果家长已经提交了作业，重复点击已提交的作业，可以查看已提交的作业内容，在已提交作业底部若有待提交的作业，会显示“X份待交作业”，点进去可以查看待提交的作业有哪些。
![](https://wework.qpic.cn/wwpic/162044_BG2BUOwuTg6IP0H_1662606231/0)

### [](https://open.work.weixin.qq.com)四、家长如何给同个班级的两个孩子提交作业

支持给同个班级的多个孩子分别提交作业，家长在点击提交作业，页面可以选择对应的孩子进行提交作业。
![](https://wework.qpic.cn/wwpic/524931_hqoUbpJcRVWmhd0_1662606650/0)

### [](https://open.work.weixin.qq.com)五、如何修改已提交的作业

打开作业详情，点击“修改”可修改已提交的作业。
![](https://wework.qpic.cn/wwpic/723432_-EHx6QJwS-CpHP4_1662606990/0)

### [](https://open.work.weixin.qq.com)六、家长/学生打开作业提示“不在作业的布置范围内，无法查看”

1、提交作业的家长需在老师布置作业的范围内，如果不在范围内的学生点击提交作业，会出现这个提示。

2、如果是班级群里发起的班级作业，这个班级作业的范围是：
班级作业的范围跟[管理后台](https://work.weixin.qq.com)【家校沟通->配置->家校通讯录设置】有关：

- 若选择「全部班级」老师在家校通讯录中可以看见家校通讯录的所有学生，班级群布置作业范围则是在家校通讯录且在这个群里的学生。
- 若选择「仅负责范围内的班级」老师在家校通讯录中只能看见负责范围内的学生，班级群范围是在布置作业的老师负责的班级学生；

3、在群里历史已发布的班级作业，新入群的学生，即使满足班级群班级作业范围条件，点击进群之前已发布的班级作业，也会不在作业的布置范围内。需要老师重新发布班级作业。
