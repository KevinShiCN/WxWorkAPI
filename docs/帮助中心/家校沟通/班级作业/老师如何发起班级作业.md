# 老师如何发起班级作业

### [](https://open.work.weixin.qq.com)一、老师可通过以下方式布置作业：

1、【手机端/电脑端->群聊工具栏->班级作业->布置作业】（mac暂不支持）
注：班级群内的群作业入口，只有群内同企业的企业微信用户可以看到；工作台的班级作业入口，企业内人员均可看到入口。
![](https://wework.qpic.cn/wwpic3az/949155_IdTycDinSt-D2wO_1766038049/0)
2、【手机端/电脑端->工作台->班级作业->布置作业】。
![](https://wework.qpic.cn/wwpic3az/194336_yswboSWfSI-G1l5_1766038070/0)
![](https://wework.qpic.cn/wwpic3az/875080_xuRRVsE9RziESSP_1750153313/0)

### [](https://open.work.weixin.qq.com)二、作业内容

1、布置作业页面右上角可以选择「科目」，每次布置作业只能选一个科目。
在选择科目的页面，可以添加科目，添加的科目只有添加者可以查看使用，也可以在选择科目的页面删除科目。
注：添加科目后，被添加的科目仅保存在本地客户端，在其他客户端不可见，例如在A手机操作添加科目，添加的科目在电脑端/B手机等设备不可见。
![](https://wework.qpic.cn/wwpic3az/673505_J5p6iIRwSsuhiB8_1750153306/0)
2、布置作业时，作业内容、作业对象和科目必须全部填写后才可以点击「布置」按钮，否则布置按钮会置灰。

3、老师布置内容可以填写文字，支持添加视频、文件、录音等多种类型的附件。作业内容支持填写5000字符，最多支持9个附件（含各种类型）、录音最长支持录制10分钟，视频最大25M。
视频支持的格式：rm、rmvb、wmv、avi、mpg、mpeg、mp4、mov。
 注：若布置作业上传的音频是从“添加文件”入口上传的，当前以文件类型布置仅支持doc、docx、xls、xlsx、ppt、pptx、pdf格式。

![](https://wework.qpic.cn/wwpic3az/854736_fi3mVlb1SB-W_tC_1766038095/0)

4、已布置的作业，可以通过【工作台->班级作业】或者在【班级群聊->群作业】中查看每个作业的作业对象，以及统计详情。

5、管理员暂不支持查看到其他老师布置的作业。

6、支持修改/删除已布置的作业，从布置作业的路径找到作业列表->点开需要修改/删除的作业->点击右上角->修改/删除。
（1）删除作业后会直接通知家长XX作业已被删除；而修改作业可以选择“保存并通知家长”或者“仅保存”，修改作业之后家长打开作业会直接展示最新的作业内容。
（2）修改/删除作业的通知路径取决于作业是哪里布置的，如果是在班级群里布置的作业会通知到班级群里，如果是工作台里布置的作业会通知到家校通知微信插件里。
