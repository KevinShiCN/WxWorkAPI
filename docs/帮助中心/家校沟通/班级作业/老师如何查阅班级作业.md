# 老师如何查阅班级作业

### [](https://open.work.weixin.qq.com)一、班级作业布置入口

只有布置作业的老师可以批改作业，老师可以在班级群或工作台查看作业详情进行修改班级作业。
在【作业详情->已提交】中点击查看和批改作业，支持对已提交的作业图片进行批注评分和添加评语（评语支持5000字），批注后的图片支持删除批注。
![](https://wework.qpic.cn/wwpic3az/920791_8aOUtkLCSdam-zR_1766039790/0)
![](https://wework.qpic.cn/wwpic/43742_nXKFxGIkS26HrYU_1665473081/0)

### [](https://open.work.weixin.qq.com)二、提醒家长查看作业

老师批改作业后，如果需要对多个家长提醒查看批改，可以在已布置作业的页面->点击其中一份作业->点击最底端的“提醒家长查看”；如果需要对单个家长进行提醒，可以已布置作业的页面->点击其中一份作业->点击某个学生的作业->点击右上角的“提醒家长查看批改”。
![](https://wework.qpic.cn/wwpic3az/462513_D-zAmv1cRCCmcqe_1766039817/0)

### [](https://open.work.weixin.qq.com)三、作业人数和家长群里的家长数不一致

一般有以下3个原因：
1、班级作业的范围跟管理后台【家校沟通】->【配置】->【家校通讯录设置】有关：

- 若选择【全部班级】老师在家校通讯录中可以看见家校通讯录的所有家长，班级群布置作业范围则是在家校通讯录且在这个群里的学生。
- 若选择【仅负责范围内的班级】老师在家校通讯录中只能看见负责范围内的家长，班级群范围是在布置作业的老师负责的班级；

2、历史群里已发布的班级作业，新入群的家长，即使满足班级群班级作业范围条件，也不会算进提交统计中，只有进群后下一次新布置作业的时候才会把这个家长算进去。需确认在发布作业后是否有新的家长加入群聊。

3、班级作业统计的人数是按照学生维度来统计的，且需要学生同时在班级群里以及家校通讯录架构里。

### [](https://open.work.weixin.qq.com)四、作业范围人数和对应的班级人数不一致

1、布置作业范围选择的班级和查看通讯录的班级需一致；
2、班级作业的提交人数统计是统计家校通讯录中的学生数，不是家长数。
