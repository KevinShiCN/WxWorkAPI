# 如何发布党史学习/德育学习内容

### [](https://open.work.weixin.qq.com)一、发布权限设置

管理员可以在【手机端->工作台->德育学习->右上角配置->可发布学习内容的老师】，中按需调整范围。
![](https://wework.qpic.cn/wwpic3az/491928_7el2khoYSOiw9KE_1766039329/0)

### [](https://open.work.weixin.qq.com)二、如何发布

拥有发布权限的成员支持在此板块进行发布学习任务，支持搜索「微信看一看」的相关文章添加到发布内，也可以由发布者自行编辑内容进行发布，学习内容标题默认使用「XX月XX日党史学习内容」，支持发布者手动修改辩题，支持选择学生或老师作为学习对象，选择学生或老师都可以选择全部学生/老师或部分学生/老师，点击发布即可发布学习任务。

学习对象是老师时，老师将通过德育学习/党史学习应用号收到学习任务通知，学习对象是学生时，学生家长将通过学校通知收到学习任务通知。

若发布者被设置了通讯录限制，在发布学习任务的时候选择了老师为学习对象，在选择范围时，会过滤掉看不到的老师。

【手机端->工作台->德育学习->去发布学习内容/发布】
![](https://wework.qpic.cn/wwpic3az/229842_nK0IAqgFSre6zK__1766039345/0)

### [](https://open.work.weixin.qq.com)三、编辑和删除已发布内容

发布了党史学习内容后，在「发布」中找到对应发布的内容，点击查看详情，右上角“…”可以进行编辑或删除已发布的内容。（mac暂不支持）
![](https://wework.qpic.cn/wwpic3az/415981_MnlkJvHBS26QYIW_1766039369/0)
