# 如何使用党史学习/德育学习

#### [](https://open.work.weixin.qq.com)1、手机端操作入口：

【手机端->工作台->德育学习/党史学习】
![](https://wework.qpic.cn/wwpic3az/918202_j9O-OlNYSo6TWLQ_1766038902/0)

#### [](https://open.work.weixin.qq.com)2、电脑端操作入口：

【电脑端->工作台->德育学习/党史学习】
![](https://wework.qpic.cn/wwpic3az/683906_05-YOjtURRm3VzN_1750225394/0)

#### [](https://open.work.weixin.qq.com)3、设置可见范围：

超级管理员可以在【手机端->工作台->管理企业->应用管理->德育学习/党史学习】中设置应用可见范围（默认全员可见）。
注：管理后台暂无此应用。
![](https://wework.qpic.cn/wwpic3az/904103_DOMT_TWlQVKLdO3_1766038919/0)
