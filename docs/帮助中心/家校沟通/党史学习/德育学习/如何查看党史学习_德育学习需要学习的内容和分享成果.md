# 如何查看党史学习/德育学习需要学习的内容和分享成果

1、企业内成员可以在【手机端->工作台->德育学习/党史学习->学习/成果】看到目前自己需要学习的内容和分享成果。
![](https://wework.qpic.cn/wwpic3az/196794_ORxBPkDmQHm1f43_1766039930/0)
2、学生家长可通过学校通知收到学习任务通知。
![](https://wework.qpic.cn/wwpic/691953_kqkPnPPPRc-CdBU_1662623882/0)
