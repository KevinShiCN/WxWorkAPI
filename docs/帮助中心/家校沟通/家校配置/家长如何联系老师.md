# 家长如何联系老师

1、家长可在「学校通知->联系老师」中联系学校为班级配置的班主任和任课老师，添加老师的企业微信并与之联系。
「联系老师」入口中会展示孩子所在班级对应的班主任和任课老师。
![](https://wework.qpic.cn/wwpic/423243_xFmn5RnPS8Ob6mz_1662361625/0)

2、当管理员未开启「家长联系老师需验证」家长点击联系，则可直接跟老师发起会话；若开启「家长联系老师需验证」开关，则老师会收到家长的好友申请，待老师通过好友申请后，家长可以和老师进行沟通。

「家长联系老师需验证」开关可以在【管理后台->家校沟通->配置->家校通讯录设置】中设置。
![](https://wework.qpic.cn/wwpic3az/665757_0uC6-IK7TBmKbOF_1753772147/0)
![](https://wework.qpic.cn/wwpic3az/222932_0EL4DDqDRJWOn0G_1753772150/0)

或在手机端【工作台->家校通讯录->右上角->家校通讯录设置】入口进行设置。
![](https://wework.qpic.cn/wwpic3az/603565_Xa4oWh8ZQECmqS8_1766039413/0)
