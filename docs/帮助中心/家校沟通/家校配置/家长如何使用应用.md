# 家长如何使用应用

1、超级管理员登录管理后台，进入【家校沟通->配置->家长可使用的应用】中进行添加家长可使用的应用。
支持添加主页型的自建应用和主页型的第三方家校应用。
注：主页型应用，即设置了工作台应用主页的应用；第三方应用市场中的「教育行业方案」已经上线，欢迎前往挑选试用。
![](https://wework.qpic.cn/wwpic3az/842003_lvhy_AB9QqGcK1k_1753772303/0)

2、在手机端，管理员可进入“家长通知->配置家长应用”中配置这些应用。
![](https://wework.qpic.cn/wwpic3az/200368_-AMeoNkGTimXgEh_1766039103/0)

3、家长可以在【微信->通讯录->学校通知->学校应用】中使用学校配置的校园应用，接收这些应用所推送的消息，提交学生请假、缴纳学费等申请；
![](https://wework.qpic.cn/wwpic/491333_fUIe3UcPTn6x2Qc_1662347233/0)
