# 企业微信关于学校通知的解决方案

- 家长在微信上扫码关注「学校通知」，接收学校消息，使用学校应用，联系学校老师。
- 管理员在企业微信上通过「家校通讯录」将老师配置进相应班级，老师即可向配置范围内的班级家长发送通知。
- 老师在企业微信上通过「家长通知」发送作业、事件等通知，查看通知的已读未读情况。
