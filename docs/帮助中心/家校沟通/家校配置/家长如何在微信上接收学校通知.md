# 家长如何在微信上接收学校通知

1、家长需[关注对应学校的学校通知](https://open.work.weixin.qq.com/help2/pc/19114)。
2、需在学校通知开启接收企业消息（设置入口：微信->通讯录->学校通知->右上角”+”->设置->接收企业消息。）
![](https://wework.qpic.cn/wwpic/133967_8u75l_vMSVqO005_1662363466/0)
3、家长需要在[学校发送通知范围](https://open.work.weixin.qq.com/help2/pc/15062)里。不在通知范围里的家长不会收到学校的推送。
