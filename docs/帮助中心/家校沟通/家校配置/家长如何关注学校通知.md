# 家长如何关注学校通知

家长无需下载企业微信APP，只需通过下述方式邀请家长关注学校通知，关注成功后，家长在微信上就能接收学校消息、使用学校应用、联系学校老师。

### [](https://open.work.weixin.qq.com)一、老师录入家长信息，邀请家长关注学校通知

老师在家校[录入了家长](https://open.work.weixin.qq.com/help2/pc/15047)手机号后，家长微信绑定了手机号，已验证企业的学校，老师支持在家校通讯录邀请未关注的家长关注学校通知。家长会在微信侧服务通知收到邀请关注的提醒。

### [](https://open.work.weixin.qq.com)二、老师获取邀请二维码，家长扫码关注学校通知

管理员或老师支持获取家校通知二维码给家长微信扫码关注。
管理后台获取入口：【家校沟通-> 家校沟通->添加学生家长】
![](https://wework.qpic.cn/wwpic3az/156824_dvMEcqCATg-wQDH_1753771651/0)
手机端获取入口：【工作台->家校通讯录->右上角“邀请家长”或在工作台->家长通知->邀请家长】
![](https://wework.qpic.cn/wwpic3az/290768_jmPbXGKzQfSMLLi_1766040204/0)

### [](https://open.work.weixin.qq.com)三、常见问题

**1、家长收不到老师下发的邀请关注家校通知**
（1）未验证企业暂时不支持邀请家长发送。（没有邀请加入的入口，在家校通讯录添加了家长，家长也不会收到关注通知）

（2）家长微信绑定的手机号，需和老师在家校通讯录中录入的手机一致。若不一致，家长会收不到。

（3）同一个家长微信，一天最多支持推送三次邀请关注通知。这个3次通知会包含“邀请加入企业”通知和“关注学校通知”的推送。当同个企业发送第四次的时候，则家长侧无法继续发送，如果是其他企业已经给这个家长的微信发送了3次“邀请加入企业”通知或“关注学校通知”，在当天未发送的企业点击发送，家长也是收不到的。

（4）若企业是新注册的，第一次进行邀请，需要审核学校资质后才能发送，家长会延迟收到学校邀请，家长可以晚点重新查看。

**2、关注家校微信插件提示“手机号被占用，无法关注，当前手机号已被其他家长占用，加入失败”**
若企业已经有另外的家长的微信使用了这个手机号关注了家校通知微信插件，当另外一个微信扫码输入这个手机号会出现这个提示。

可让家长使用新手机号进行关注，或者管理员可在【[管理后台](https://work.weixin.qq.com)->家校沟通->家校通讯录】通过搜索手机号查看已使用该手机号的家长，可以删除这个家长信息后重试。
