# 如何使用学校通知/家长通知

### [](https://open.work.weixin.qq.com)一、学校通知/家长通知发送入口

1、管理员支持在【管理后台->家校沟通->配置->学校通知】发送学校通知给家长
![](https://wework.qpic.cn/wwpic3az/253257_gzdlglbMSGKG6Xp_1753772866/0)

2、管理员和老师也支持在手机端【工作台->家长通知】发送家长通知
![](https://wework.qpic.cn/wwpic3az/50429_PkeGLXdvSgWPyyQ_1766037186/0)

### [](https://open.work.weixin.qq.com)二、学校通知/家长通知发送范围

1、老师默认可见「家校通讯录」中的完整架构和所有学生家长信息，并且给所有的班级家长发送「家长通知」。
2、超级管理员可以前往「家校通讯录配置->老师可查看的班级」，调整为“仅负责范围的班级”。设置后，老师仅可查看自己被配置到的班级以及向配置范围内的班级家长发送通知。
3、超级管理员发送「家长通知」的范围是全部家长，权限不受上述开关控制。

![](https://wework.qpic.cn/wwpic3az/424490_PkL-j2GoSxuTQ-W_1766037216/0)

### [](https://open.work.weixin.qq.com)三、学校通知/家长通知发送内容说明

1、通知家长的内容，支持输入文字、图片、视频、文件、网页、小程序。
注：文本消息，单条通知最长5000字符； 文件单个不能超过25m，图片不得超过10m，安卓录视频可录150s，大小70m；ios录屏支持120s ，大小25m左右；电脑端和管理后台操作时发送视频压缩后不能超过25m；发送单个通知最多10个附件；小程序需要是企业已关联的小程序，并且发送人在小程序的可见范围内。
2、发送通知时可勾选须家长确认收到通知，若家长不点确认，老师可点提醒，再次推送，每个老师一天10次提醒。
3、发送学校通知的推送频率暂无限制。
4、发送通知支持选择“需家长点击按钮确认”、“仅需家长阅读”、“需家长签字确认”。

![](https://wework.qpic.cn/wwpic3az/386844_-zyqjxAyRFicpak_1753772876/0)

### [](https://open.work.weixin.qq.com)四、如何查看已发送的学校通知/家长通知

在管理后台【家校沟通->配置->发送学校通知->历史通知】和手机端【工作台->家长通知->已发送的通知】可查看发送记录。
![](https://wework.qpic.cn/wwpic3az/909232_Dd_5HBGaRtyWMst_1753772879/0)

### [](https://open.work.weixin.qq.com)五、如何修改或撤回学校通知/家长通知

1、学校超级管理员支持修改和撤回其他管理员和老师发送的通知；班主任支持支持修改和撤回自己发送的通知，当管理员发送的通知范围中含班主任负责的班级，班主任支持查看这个通知，并且提醒自己负责的班级家长确认。
2、在管理后台【家校沟通->配置->发送学校通知->历史通知】和手机端【工作台->家长通知->已发送的通知】可修改已发送的通知。
3、超管支持撤回老师发出的通知（24小时内）。撤回后则不支持再次提醒家长。撤回后的家长通知不支持删除。

### [](https://open.work.weixin.qq.com)六、家长会在哪里收到学校通知/家长通知

家长可在微信侧「学校通知」收到提醒，若通知需“需家长签字确认”，则家长在阅读通知后，需要签字确认。签字确认时页面会默认勾选“下次继续使用此签名”，若下次需要重新签名，则点击“清除”即可重新签名。
