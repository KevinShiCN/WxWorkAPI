# 新学期如何分班

### [](https://open.work.weixin.qq.com)一、新学期分班介绍

针对学校大规模重新分班的场景，企业微信·教育版在管理后台上线了「新学期分班」工具，帮助学校快速、批量、准确地调整家校通讯录，创建新的班级群。
应用场景：全年级重新分班
如果只需要调整个别学生的所在班级，请前往「家校通讯录」手动操作。功能入口：【管理后台->家校沟通->配置->新学期分班】
操作权限：仅管理员可操作
需要准备：分班后，全年级各班的班主任及学生名单

### [](https://open.work.weixin.qq.com)二、「新学期分班」作用

按照模板填写分班表，导入后即可：
学生调班：根据分班表，批量调整学生在家校通讯录中的班级；
换班主任：根据分班表，快速设置家校通讯录中各班级的班主任；
建班级群：分班后将自动创建新的班级群，并邀请学生家长进群。

#### [](https://open.work.weixin.qq.com)1、开始分班

登录学校[企业微信管理后台](http://work.weixin.qq.com/login)，前往「家校沟通->配置->新学期分班」，点击「开始分班」。

![](https://wework.qpic.cn/wwpic3az/627401_-PAQrHk1T0qPraM_1753172404/0)

#### [](https://open.work.weixin.qq.com)2、下载分班表

点击「下载分班表」，按学校实际情况选择需要分班的年级后，下载对应的分班表模板：
● 每次分班只能选择一个年级；
● 分班时，家校通讯录可能尚未完成[自动升年级](https://open.work.weixin.qq.com/help2/pc/15054?person_id=1)，请根据入学年份选择合适的年级名称。
![](https://wework.qpic.cn/wwpic3az/242554_nFTXkN3vQe2fqJ4_1753172525/0)

#### [](https://open.work.weixin.qq.com)3、填写分班表

请仔细阅读分班表顶部的七条填写须知，根据要求完整填写分班后各班级的班主任和学生。尤其注意：
● 不要自行修改或删除第1-3行、第1列，改动表中的年级或班级；
● 在每个班级下方填写该班级在新学期的全部学生名单，不漏一人；
● 如果新学期分班年级的班级数量有所增加，请先在家校通讯录中添加新的班级，再重新下载分班表。

#### [](https://open.work.weixin.qq.com)4、上传分班表

填写好分班表后，回到管理后台上传即可。如果遇到以下情形，还需要手动选择某些班级的班主任：
● 某班主任尚未加入企业微信内部通讯录；
● 某班主任的名字在内部通讯录中存在重名现象；
● 上传的分班表中，未填写某个班级的班主任；
● 上传的分班表中，某班主任的名字写错了。

#### [](https://open.work.weixin.qq.com)5、确认分班信息

成功上传分班表后，需要检查并确认参与分班的班级名单，以及这些班级在分班后的班主任和学生名单。

#### [](https://open.work.weixin.qq.com)6、正式进行分班

确认分班后，系统会按照分班表中填写的信息，自动完成以下设置：
● 将学生从家校通讯录的原班级移入新班级中；
● 移除这些班级的原有班主任和任课老师，设置新的班主任；
● 取消这些班级的原有班级群，将其转为普通外部群；
● 以新班主任为群主，为这些班级创建新的班级群；
● 通过学校通知和原班级群，通知学生家长扫码入群。

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、上传分班表时，遇到“请完整填写各班级的学生名单”提示怎么办？

请检查上传的分班表中，是否存在只填写了班主任，未填写任何学生的班级。

#### [](https://open.work.weixin.qq.com)2、为什么会遇到分班失败？

上传分班表到确认分班期间，如果有人修改了家校通讯录的组织架构，如删除了某个班级，就会导致分班整体失败。分班失败不会造成任何影响，请重新下载分班表后再进行分班。

#### [](https://open.work.weixin.qq.com)3、为什么有部分学生分班失败？

以下几种情形会导致某学生分班失败：
①该学生尚未加入家校通讯录的对应年级中；
②在家校通讯录中存在同年级的重名学生；
③该学生的名字写错了，在家校通讯录的对应年级中找不到。

#### [](https://open.work.weixin.qq.com)4、如何处理分班失败的学生？

①对于未能在「家校通讯录」对应年级中找到的学生，你可以在家校通讯录中直接添加该学生，并设置其所在班级；
②对于同年级重名的学生，你可以在家校通讯录中通过搜索找到该学生，并修改其所在班级。
