# 如何配置老师

1、可以看见家校通讯录的成员或管理员在【手机端->工作台->家校通讯录->在对应的校区、学段、年级、班级中设置各级的负责人和任课老师】
![](https://wework.qpic.cn/wwpic3az/906201_Rzken43RSi2gfMW_1766030071/0)

2、超级管理员在【管理后台->家校沟通->家校通讯录->在对应的班级中配置老师】
![](https://wework.qpic.cn/wwpic3az/699621_LsAr184ITGSKX3Q_1753775062/0)

或是在【管理后台->家校沟通->配置->班级老师管理】中进行统一配置。
已有班级群的情况下，新配置的班主任/任课老师将会被邀请加入班级群，但是被移除的老师不会自动退群，群主可以移除该群成员。
![](https://wework.qpic.cn/wwpic3az/89011_SHSXX5nSRguujGk_1753775068/0)
![](https://wework.qpic.cn/wwpic3az/974957_GLjcTD46RDuVVvW_1753775070/0)
