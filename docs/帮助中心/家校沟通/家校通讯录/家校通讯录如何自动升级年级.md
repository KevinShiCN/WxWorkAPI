# 家校通讯录如何自动升级年级

### [](https://open.work.weixin.qq.com)一、基本说明

●  开启自动升级年级，开启后在对应的升级日期将会自动升年级，如「一年级」会升为「二年级」。
● 自动升年级只对标准年级产生影响，不会对自定义年级产生影响。
● 今年已经自动升级过年级，修改了默认升级年级的时间，在今年也不会再次升级，需要等到明年。
● 家校通讯录被授权给服务商进行管理之后，服务商可以调用接口修改自动升年级的时间，也可以关闭自动升年级的功能。

### [](https://open.work.weixin.qq.com)二、开关入口

1、管理员可在【管理后台->家校沟通->配置->自动升年级】中开启或关闭。
![](https://wework.qpic.cn/wwpic3az/309177_PNszSTPCT3SRN9u_1753773703/0)
![](https://wework.qpic.cn/wwpic3az/452498_E4cP9NFfQ_CXHbR_1753773706/0)
或在手机端【工作台->家校通讯录->右上角->家校通讯录设置->自动升年级】选择开启或关闭。
![](https://wework.qpic.cn/wwpic3az/140486_0x5E0McITMG2CaP_1766036358/0)

2、如有个别年级不需要自动升级，管理员可在【管理后台->家校通讯录->找到对应的年级->修改年级信息->改为自定义年级】
![](https://wework.qpic.cn/wwpic3az/997897_noiEfIxPTpO3iv2_1753773710/0)

### [](https://open.work.weixin.qq.com)三、自动升级年级的特点

● 所有年级的显示，年级都会提升，入学年份不变；比如说2019级一年级变成2019级二年级
● 新建年级的时候，入学年份+1；比如2020级一年级，变成2021级一年级；
● 当前学段中最高的一个年级，将会变成毕业年级，比如2019级高中毕业，2020级初中毕业，2021级国际部毕业；
● 没改过名字的班级群名称修改，改过名字的班级群名字不做变动，例如一年级1班·班级群，会自动修改为二年级1班·班级群。
