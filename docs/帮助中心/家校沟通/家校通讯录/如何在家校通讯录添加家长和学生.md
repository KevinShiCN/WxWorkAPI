# 如何在家校通讯录添加家长和学生

### [](https://open.work.weixin.qq.com)一、手动添加

管理员可以通过【手机端->工作台->家校通讯录->选择对应的班级进入->添加家长】手动添加家长/学生，添加家长/学生时需要设置学生所在班级。
![](https://wework.qpic.cn/wwpic3az/913410_54qVFdW2RBeZdQr_1766029533/0)
【管理后台->家校沟通->家校通讯录->选择对应的班级进入->添加学生家长】
![](https://wework.qpic.cn/wwpic3az/612379_s5dZgUMGQIelUyj_1753775830/0)

### [](https://open.work.weixin.qq.com)二、批量添加

#### [](https://open.work.weixin.qq.com)1、批量添加的作用

● 批量导入家长后，家长和对应的学生中都被导入到家校通讯录中；
● 批量导入家长后，会自动给家长发送邀请，邀请家长使用微信加入家校通讯录，加入后可以收到学校和老师发送的通知以及联系老师；
注：已验证/认证企业在添加家长时会默认勾选“邀请家长使用微信加入”，未验证/认证企业不支持发送邀请。

#### [](https://open.work.weixin.qq.com)2、批量添加入口

超级管理员支持在【管理后台->家校沟通->家校通讯录->选择对应的班级进入->批量导入/导出】，手机端暂不支持。
必填项：学生姓名、学生账号、家长关系、家长手机号
选填项：学生手机号
已导入的家长列表以每个条目的“学生账号”为唯一识别码。新导入的Excel列表和已导入列表中识别码相同的条目，会被新列表的信息覆盖；若出现新的识别码，则新增一条信息。

![](https://wework.qpic.cn/wwpic3az/375112_iXp3z99sSGK4MOV_1753775834/0)

#### [](https://open.work.weixin.qq.com)3、批量导入失败原因

如果上传的Excel表格中某些信息没有按照要求填写，其所处条目则会导入失败。具体原因分为以下四种：

| 错误原因 | 错误提示语 |
| --- | --- |
| “手机号码”一项不符合大陆手机通用格式 | 家长手机格式有误 |
| “所在班级”一项被强制填入了班级标签之外的值 | 其他用户已经限定了可以输入单元格的数值 |
| 学生姓名、年级-班级、称呼、家长手机号都不能为空 | 家长信息缺失 |
| 多个条目的“学生账号”一致，但其他信息却不同 | 学生账号已存在 |

### [](https://open.work.weixin.qq.com)三、家长主动关注

#### [](https://open.work.weixin.qq.com)1、管理后台获取加入二维码

管理员可以在【管理后台->家校沟通->家校通讯录->邀请加入】或【管理后台->家校沟通->添加学生家长】
这两个路径获取加入二维码，家长使用微信扫码关注，可关注「家校通知」插件，同时加入家校通讯录。
![](https://wework.qpic.cn/wwpic3az/824355_5k1rgM8ZRlSkE3r_1753775840/0)
![](https://wework.qpic.cn/wwpic3az/169679_37qkOSbeRCWxnBt_1753775843/0)

#### [](https://open.work.weixin.qq.com)2、手机端获取加入二维码

在手机端可以看见家校通讯录的成员，在手机端可以获取二维码邀请家长加入家校通讯录，入口：工作台->家校通讯录->添加家长->家长加入二维码 。
家长扫码关注后，需要选择自己的孩子所在的班级。
![](https://wework.qpic.cn/wwpic3az/974440_M9svkcaBQXmz5xG_1766029559/0)

#### [](https://open.work.weixin.qq.com)3、一个家长多个孩子加入方式

家长首次扫码关注家校的时候，在提交页面底部会出现“我有多个孩子”入口，可通过该入口继续填写其他孩子的班级信息进行加入；
当家长已经关注家校后，需联系老师进行添加。
![](https://wework.qpic.cn/wwpic/755052_IZCgeDYLQHOhjdu_1662343628/0)

#### [](https://open.work.weixin.qq.com)4、家长扫码加入家校通讯录权限控制

● 如果学校希望严格管控能够加入家校通讯录中的家长，可以在家校通讯录配置中，关闭掉“可扫码填写资料加入”的开关；
● 关闭后，只有被手动输入添加，或者excel导入添加的家长，才能够加入到家校通讯录中，关注“学校通知”
