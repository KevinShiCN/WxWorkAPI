# 家校通讯录如何扩容

### [](https://open.work.weixin.qq.com)一、家校通讯录人数上限

未验证企业最多可以添加1000名家长，家长数超1000则企业需要进行验证后申请扩容；最高支持申请扩容100000名家长。

### [](https://open.work.weixin.qq.com)二、申请扩容入口

目前在家校通讯录人数即将达到上限的时，暂无提醒管理员去扩容。

管理员可在管理后台进行扩容申请，审核时间1个工作日左右。
扩容结果会通过短信、企业微信团队或者管理后台的首页，通知操作扩容的管理员。

操作路径：【管理后台->家校沟通->配置->家校通讯录设置->家校通讯录人数扩容】

### [](https://open.work.weixin.qq.com)三、扩容的条件

申请的主体需要是幼儿园、小学、初中、高中、职高等国家规定的学前教育、初中等教育等K12学校；
培训机构申请扩容将会被驳回，培训机构推荐使用 教育-培训机构子行业的相关功能。
