# 企业微信教育版的核心功能有哪些

### [](https://open.work.weixin.qq.com)一、家校通讯录

- 学校可以统一管理学生家长信息，老师可以随时随地查看并联系家长。
- 可快捷导入家长资料或邀请家长使用微信扫码填写，换班毕业等管理更加高效。

![](https://wework.qpic.cn/wwpic3az/191549_uNUxHEmHTi2JX_C_1766048012/0)

### [](https://open.work.weixin.qq.com)二、班级群

- 基于家校通讯录，可自动组建班级群并邀请同班家长和老师加入，家校沟通更有效率。
- 班级群人数上限提升至500，满足更多家校互动场景。

![](https://wework.qpic.cn/wwpic3az/737571_OSVs20dXRbeEZsY_1766047987/0)

### [](https://open.work.weixin.qq.com)三、家长通知

- 老师可使用「家长通知」，向家长发布作业、校园事件通知，家长在微信「通讯录->学校通知」中接收通知的同时，还可直接联系相关老师、发起会话。
- 老师可勾选“需家长确认收到通知”选项，重要通知的已读情况一目了然；老师还可以对尚未阅读通知的家长发起“再次提醒”，确保重要通知触达率。

![](https://wework.qpic.cn/wwpic3az/74581_KLsJQ2cIQgGl4QA_1766048036/0)

### [](https://open.work.weixin.qq.com)四、与微信上的家长沟通

 老师可以使用工作身份添加家长微信，在企业微信上和微信上的家长沟通，将工作和生活分开，保护双方隐私。

![](https://wework.qpic.cn/wwpic/351485_OnMMROWfSkeYx4Q_1615800260/0)

### [](https://open.work.weixin.qq.com)五、家长驻留在微信

 家长无需下载企业微信，可直接在微信中扫描「家长加入二维码」二维码加入「家校通讯录」，接收通知、联系老师，还可以使用学校应用进行学生请假、校园留言等操作。

![](https://wework.qpic.cn/wwpic/797740_STdsaayiTGWnkRu_1615800302/0)

### [](https://open.work.weixin.qq.com)六、校园管理应用

- 老师可使用「审批」应用，便捷完成老师关于调课、请假、公文流转、报修、场地预定、物品领用、报销等内部事务的审批和存档。

- 老师可使用「汇报」应用，便捷完成老师关于家访报告、行政巡检、值周日志、听课评课等事务的汇报和记录。

![](https://wework.qpic.cn/wwpic3az/819004_7YYhFXvmThmAGTs_1766048079/0)
