# 如何设置家校通讯录

### [](https://open.work.weixin.qq.com)一、添加/修改校区、学段、年级、班级信息

可在【管理后台->家校沟通->家校通讯录->点击对应学段、年级、班级进行添加或修改】
注：家校通讯录中一个班级的学生人数上限为200人。
![](https://wework.qpic.cn/wwpic3az/180581_PUdXHjdnRfqdCCJ_1753774279/0)

点击搜索框旁的“+”可新建校区。
![](https://wework.qpic.cn/wwpic3az/700105_ek7V-fAWRZm32Np_1753774286/0)

### [](https://open.work.weixin.qq.com)二、编辑家长信息、删除家长

【管理后台->家校沟通->家校通讯录->在对应的班级】中可以编辑或删除家长。
![](https://wework.qpic.cn/wwpic3az/380909_um88q7q-TpOwXvE_1753774290/0)

### [](https://open.work.weixin.qq.com)三、其他配置

1、「家长和学生可自己填写学生资料」
未被手动输入添加或批量导入的家长和学生可自己填写学生资料

2、「家长联系老师需验证」
家长在「学校通知->联系老师」加老师为联系人时需要老师验证，开关默认关闭。

3、「老师可查看的班级」
可以选择全部班级、仅负责范围内的班级，默认选择“全部班级”。
若选择了全部班级，则全部的老师可查看全校范围内的班级并向家长发通知。
若选择了仅负责范围内的班级，则老师仅可查看负责范围内的班级并向家长发通知（若是校区/学段/年级的负责人，则仅可以查看负责的校区/学段/年级范围内的班级并向家长发通知）。

4、「[班级群创建方式](https://open.work.weixin.qq.com/help2/pc/15095?person_id=1)」
创建班级群有2种方式，支持自动创建、手动创建方式。
生成班级群时，老师和对应班级的学生家长是外部好友关系的情况下，家长会自动加入到班级群中。

5、「一键设置年级、班级」
重新调整学校架构，设置后已添加学生和老师的班级将会保留。

6、「[家校通讯录人数扩容](https://open.work.weixin.qq.com/help2/pc/15053?person_id=1)」
学校验证后可根据使用人数申请扩容。
