# 重新分班时，如何妥善处理原有班级群

### [](https://open.work.weixin.qq.com)问题背景

学校大范围重新分班时，班级学生和教师都会发生变动，原有班级群的成员、群主与新班级的人员构成不相匹配。为此，管理员应提前进行相关配置。

### [](https://open.work.weixin.qq.com)情形一：如果您想要解散原有班级群，重新建群，操作如下：

1、在自动创建班级群模式下：
（1）首先，解散所有需重新分班的班级群；
（2）然后，在管理后台利用「导出家校通讯录」进行分班；
（3）最后，重新调整班级老师的配置。

2、在手动创建班级群模式下：
（1）首先，解散所有需重新分班的班级群；
（2）然后，在管理后台利用「导出家校通讯录」进行分班；
（3）最后，重新调整班级老师的配置，手动创建班级群。

### [](https://open.work.weixin.qq.com)情形二：如果您不想解散原有班级群，操作如下：

1、首先，告知分班后需退群的家长自行退出班级群；（或者，您也可以让群主将这部分家长直接移出群聊）

2、然后，在管理后台利用「导出家校通讯录」进行分班，重新调整班级老师的配置；

3、若有需要，让原群主将群主转让给新老师；（或者，您也可以在「工作台->班级群」的群管理中直接设置新群主）

4、最后，群主需要在班级群设置里的“邀请家长进群”中，邀请还未进群的家长进入群聊。（或者，您也可以告知家长在微信的「学校通知->联系老师」中，申请加入新的班级群）
