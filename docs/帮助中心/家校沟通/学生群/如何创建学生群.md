# 如何创建学生群

1、创建学生群时，当前班级有班级群的话，会在班级群下发通知提醒邀请学生入群，若没有班级群，会在家长微信的学校通知下发提醒。

2、学生群创建成功后，学生群的群名默认格式为「班级名·学生群」，例如：「中班1班·学生群」。

3、1个班级只能有1个班级群和1个学生群。

4、支持在学生群添加家长，家长加入群聊也计算为学生数，在群成员中会提示含有多少位家长。

5、目前支持在Win端、手机端、管理后台中创建学生群，暂不支持在Mac中创建，具体操作路径如下：
【手机端/电脑端->工作台->学生群->创建学生群】
