# 如何创建班级群

### [](https://open.work.weixin.qq.com)一、创建班级群介绍

创建班级群有2种方式，支持自动创建或手动创建。
生成班级群时，老师和对应班级的学生家长是外部好友关系的情况下，家长会自动加入到班级群中。

### [](https://open.work.weixin.qq.com)二、自动创建班级群

「班级群创建方式」开关可以在管理后台或手机端进行设置。可选择「自动创建」或「手动创建」，默认开启自动创建。
若选择了「自动创建」班级群，当管理员给对应的班级设置了班主任、任课老师后，会生成对应的班级群。
班级群创建成功后，班级群名称默认格式为【班级名->班级群】，例如：【中班1班->班级群】，群主默认是老师或班主任，当班级同时存在任课老师和班主任，则群主在任课老师和班主任直接随机生成，老师在群里的名称格式默认为【班主任->xx老师】或【语文->xx老师】。

管理后台操作路径：【家校沟通->配置->家校通讯录设置->班级群创建方式】中设置。
![](https://wework.qpic.cn/wwpic3az/575408_FkWjF_OOSyWQASe_1753770937/0)
![](https://wework.qpic.cn/wwpic3az/48278_vkWbEszcRiGcBg0_1753770940/0)
手机端操作路径：【工作台->家校通讯录->右上角->家校通讯录设置->班级群创建方式】进行设置。
![](https://wework.qpic.cn/wwpic3az/979883_Y95ems6pSwC3ViK_1766037433/0)

### [](https://open.work.weixin.qq.com)三、手动创建班级群

1、老师支持在手机端：【工作台->班级群】选择创建新的班级群。
![](https://wework.qpic.cn/wwpic3az/889783_5kK_ADnqSKuNDJ4_1766037481/0)
2、在手机端：【工作台->家校通讯录->点击班级目录->班级群->创建新的班级群/迁移微信中的班级群】
![](https://wework.qpic.cn/wwpic3az/760317_2uLJeg-yTyed9I__1766037522/0)
3、在电脑端：【工作台->家校通讯录->点击班级目录->立即创建】
![](https://wework.qpic.cn/wwpic3az/261077_vqRBEQD_T9KoPiL_1750152904/0)
4、管理后台：【家校沟通->家校通讯录->对应班级->立即创建】
![](https://wework.qpic.cn/wwpic3az/541766_MoqZuzSGQzCeASi_1753770945/0)

### [](https://open.work.weixin.qq.com)四、微信上已有班级群，可以直接迁移到企业微信上

可以升级微信群为企业微信班级群，方便老师更加高效地进行家校沟通。
升级的微信群会自动创建班级节点，统一收纳在家校通讯录的「未知年级的班级」中。用户可以通过移动班级或者重新标记班级群的方式，重新规整班级。
针对有家校沟通功能的企业，成员都可以看到「迁移微信中的班级群」入口，通过该入口跳转到微信侧选择对应的群聊可以迁移为班级群。

操作路径：手机端会话页面中「迁移微信中的班级群」。
![](https://wework.qpic.cn/wwpic3az/709910_c10SgqKhRkusKDj_1766037651/0)
或在【工作台->班级群->迁移微信中的班级群】。
![](https://wework.qpic.cn/wwpic3az/16500_eR6mhivTR-2Q-MJ_1766037674/0)

### [](https://open.work.weixin.qq.com)五、标记已有外部群作为班级群

发起一个外部群后，群主支持在「群管理」标记班级群，点击标记会出现家校通讯录，需选择一个没有班级群的班级进行标记。
由微信用户创建的外部群暂不支持标记为班级群，会提示「非外部群无法标记」。
![](https://wework.qpic.cn/wwpic3az/474224_hHfvCvtsQ0mk0wa_1766037697/0)

### [](https://open.work.weixin.qq.com)六、移动班级至正确的年级下

适用情况：家校通讯录不完整（班级节点没有对应的班级群、大部分家长没有加入家校通讯录）
操作者权限要求：管理员或者有家校通讯录可见权限的成员

操作路径1：工作台->家校通讯录->未知年级的班级->班级层级->顶部banner「选择班级」 为微信迁移群选择正确的班级节点；
![](https://wework.qpic.cn/wwpic3az/824110_i4CcjGmSTViY7q-_1766037762/0)
或在底部tab「更多管理」->移动当前班级，选择正确的学段年级，将班级移动过去。
![](https://wework.qpic.cn/wwpic3az/44641_9eJWim-CRmCymfx_1766037931/0)

路径2: 升级群内，顶部有「班级未指定到正确的年级」banner，点击「选择班级」，即可将该班级群移动到对应的班级节点。
操作者权限要求：升级群的群主且对家校通讯录有可见范围权限
![](https://wework.qpic.cn/wwpic3az/494080_cb3YwJsRSDaoW9c_1766037950/0)
注：按照以上路径将班级移动到正确的年级下后，如果同一个年级下有出现重复班级（比如：一个是企业微信默认预设的班级、另一个是微信升级来产生的班级节点），则可以自行操作，将较不饱满的班级节点删除（班级层级->更多管理->删除班级）

### [](https://open.work.weixin.qq.com)七、重新标记新的班级群

为了更加方便班级管理，1个班级仅可以对应1个班级群。如果需要重新创建一个班级群，可通过下述步骤操作。
适用情况：家校通讯录的完善饱满程度较高（家校通讯录中的班级，基本都有对应的企业微信班级群）

操作步骤：
1、若微信升级群对应的班级节点下，已经存在企业微信班级群，则该群群主需要先进入该企业微信班级群的会话界面，【点击会话界面右上角->群详情页->群管理->标记的班级->点击「取消标记」】。确认后，该群和班级节点的绑定关系随即解除。
操作者权限要求：需要是已存在的企业微信班级群的群主
![](https://wework.qpic.cn/wwpic3az/875342_BIhn5z73S-irWSC_1766037975/0)
2、群主进入需标记的升级群，在「群管理」标记班级群，点击标记会出现家校通讯录，需选择一个没有班级群的班级进行标记。
操作者权限要求：需要是升级群的群主且对家校通讯录有可见范围权限
![](https://wework.qpic.cn/wwpic3az/488181_i5Z0O5-tTOKktj5_1766037995/0)
