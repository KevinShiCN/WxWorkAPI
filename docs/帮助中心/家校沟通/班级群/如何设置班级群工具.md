# 如何设置班级群工具

### [](https://open.work.weixin.qq.com)一、设置班级群工具栏

配置应用页面到班级群工具栏，可方便成员在班级群聊中查看和使用，提高家校沟通效率。
可配置工具有：班级相册、上课直播、通知、快速会议、家长收款、健康上报、收集表、班级作业、课外打卡、自建应用、第三方应用。

**配置入口：**

#### [](https://open.work.weixin.qq.com)1、手机端入口

【工作台->班级群->聊天工具栏->配置成员工具栏/我的班级群工具栏】
![](https://wework.qpic.cn/wwpic3az/509941_oWVbIVXIRm6lZE9_1766039647/0)
【班级群->工具栏->添加】（这个入口可配置我的班级群工具栏）
![](https://wework.qpic.cn/wwpic3az/600375_L7CHEv9CQCq8m5H_1766039672/0)
注：超级管理员支持给企业成员配置班级群工具栏的应用，因此超级管理员会有配置成员工具栏，非超级管理员仅可编辑我的班级群工具栏。

#### [](https://open.work.weixin.qq.com)2、管理后台入口

【家校沟通->配置 ->班级群工具栏】
![](https://wework.qpic.cn/wwpic3az/577800_Zs4A80xVSR6cVeU_1753770306/0)
![](https://wework.qpic.cn/wwpic3az/320250_JIKe1c89S_emyAN_1753770311/0)

### [](https://open.work.weixin.qq.com)二、群发班级通知

管理员或者成员可以群发通知到全校班级群或者各自负责的班级群，省去通知层层下达的繁琐，发布者可知通知的已读情况，还可快速提醒未读家长。

**发送方式：**
1、老师可在「工作台->班级群应用->工具」，找到「通知」；
2、通知支持文字和图片形式，确认方式支持：

- 仅需家长阅读：家长收到通知，无需任何操作；
- 需家长手动点击按钮确认：家长收到通知，需点击按钮进行确认；
- 需家长签字确认：家长收到通知，需签字确认；

3、点击发送后，各班级群的群主将收到通知，确认后即可发送到相应的班级群；
4、群发通知创建者和各班级群群主均可看到通知的已读情况。
![](https://wework.qpic.cn/wwpic3az/833484_p-idBaZORmG2ZLE_1766039693/0)

### [](https://open.work.weixin.qq.com)三、敏感信息警告

学校管理员和群主可在「班级群」应用内，为班级群设置敏感信息警告规则，降低广告、刷屏、诈骗风险。成员也可以为自己是群主的班级群统一配置。

**配置方式：**
1、管理员可以在「工作台->班级群应用->工具」内，找到「班级群敏感信息警告」功能；
![](https://wework.qpic.cn/wwpic3az/931225_aQau3xQNQ9Ojyur_1766039723/0)
2、管理员可以点击「全校班级群配置规则」，警告规则的内容包括以下，用户可按需进行选择，规则将对群中的微信用户生效：

- 警告不文明信息：消息命中敏感词、群昵称命中敏感词；
- 警告无关信息：警告发图片、视频、文件、网页、小程序、名片；
- 警告刷屏：消息长度超过800字、消息换行超过80行、10秒内连续发10条消息（数据均支持调整）；

3、警告方式包括：

- 踢出群聊：当微信用户触碰了规则，则会被自动移出群聊；
- 警告并踢出群聊：当微信用户触碰了规则，则会被班级群小助手警告，并自动移出群聊；
- 警告三次后，再触碰规则时踢出群聊：当微信用户触碰了规则，则会被班级群小助手警告最多三次，再次触碰时将被自动移出群聊；
- 仅发警告：当微信用户触碰了规则，则会被班级群小助手警告，警告内容支持用户自定义；

4、系统提供了三个模板，供用户快速创建：

- 防广告
- 防刷屏
- 防诈骗

5、成员也可以为自己是群主的班级群配置敏感信息警告规则，维护班级群舆论环境，家校沟通更安全。
**禁止加入班级群的微信用户：**当班级群群主设置了个别用户为「禁止加入班级群的微信用户」时，对应的微信用户会在群主所在的班级群退出群聊。
**移出禁止加入班级群的微信用户：**在【工作台->班级群->敏感信息警告->禁止加入班级群的微信用户】中可以移出被禁止加入班级群的微信用户。

### [](https://open.work.weixin.qq.com)四、班级相册

1、只有对应班级的老师才能创建相册，老师创建班级相册后，班级老师和家长可以上传图片到班级相册，上传的图片将储存到微盘相册，每个班级会有一个对应的相册。（Mac端暂不支持）
注：若微盘设置了「禁止成员创建共享空间」，老师会无法创建班级相册。企业可关闭「禁止成员创建共享空间」或者在禁止成员创建共享空间中添加对应老师为白名单。

2、老师可在班级群的工具栏看到班级相册入口，老师上传图片后，班级机器人会在群里推送卡片，家长可通过推送的卡片进入班级相册。

3、老师进入班级相册后，可以点击「邀请家长上传」，选择会话发送邀请上传卡片，家长可在群聊内点击卡片上传图片。家长也可以在【微信->通讯录->家校通知->应用->班级相册】进行上传。需要家长在家校通讯录内才能上传图片。
注：目前家长上传图片以后，班级群里会有灰字提醒，班级机器人不再推送卡片提醒。

4、家长上传图片的容量将占用微盘的「外部联系人」容量，并非相册创建人的容量。
