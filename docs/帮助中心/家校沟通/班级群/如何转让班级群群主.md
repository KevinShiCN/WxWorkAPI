# 如何转让班级群群主

系统默认第一个被配置进「家校通讯录->班级」的老师或班主任是班级群的群主。群主身份在创建群聊后，可以点击班级群会话页面的右上角进入群背后，群主点击「群管理->转让群主」后即可设置其他老师为群主。
同时管理员也支持操作转让班级群群主，详情可查看：[管理班级群](https://open.work.weixin.qq.com/help2/pc/17807)。

当班级同时存在任课老师和班主任，则群主在任课老师和班主任直接随机生成，老师在群里的名称格式默认为【班主任-xx老师】或【语文-xx老师】。群主离职后，系统将随机分配给企业内的其他成员作为群主。
