# 家长如何加入班级群

### [](https://open.work.weixin.qq.com)一、加入班级群的方式

目前家长加入班级群共有3种方式，具体如下：
1、 班级群老师或家长获取班级群二维码发送给对应家长扫码进群。
![](https://wework.qpic.cn/wwpic3az/44665_NZsBmPo9TkmfrDI_1766039138/0)

2、老师若能在微信侧和家长沟通的情况，可选择「邀请微信家长进群」，点击后，会在微信会话中发送班级群二维码，家长可扫码进群。
![](https://wework.qpic.cn/wwpic3az/849560_DgYxl1FeSei6-1S_1766039158/0)
3、加入对应班级的家长，若已关注家校通知，可以在【家校通知->联系老师】选择进入对应的班级群，点击「进入」后页面会出现班级群二维码。
![](https://wework.qpic.cn/wwpic/748313_0Kc9zRptRyGonJn_1662433470/0)

### [](https://open.work.weixin.qq.com)二、家长被自动踢出班级群

1、请确认是不是被群主/群管理手动移出的群聊。
2、请让群主在“群管理”里确认这个班级群是不是开启了“仅家校通讯录中的家长可进群”，开启了这个功能之后，非家校通讯录的家长进群后会立马被踢出群。
3、请让群主确认被踢出群的家长是不是被群主加入了黑名单，查看路径：【手机端->工作台->班级群->敏感信息警告->禁止加入班级群的微信用户】。可以在禁止加入班级群的微信用户里移除家长后，重新邀请家长进群。
4、请让群主点击群聊右上角三个小点->敏感信息警告->确认已配置的敏感信息警告规则，并对比规则确认被退出群聊的用户是否命中了规则。
