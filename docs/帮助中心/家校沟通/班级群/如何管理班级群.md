# 如何管理班级群

### [](https://open.work.weixin.qq.com)一、管理班级群入口

1、群主：【班级群->右上角“…”->群管理】
支持设置「禁止改群名、仅家校通讯录中的家长可进群、群聊邀请确认、群管理员、转让群主、取消班级群标记、解散群聊」。
![](https://wework.qpic.cn/wwpic3az/358988_JlXrmo1QQKKeILg_1766040059/0)

2、超级管理员可在管理后台：【家校沟通->班级群->点击某个班级群 ->群管理】
支持设置「禁止改群名、进入群聊权限、成员邀请权限、群管理员、修改群主、取消班级群标记、解散群聊」。
![](https://wework.qpic.cn/wwpic3az/771279_JcdCAqJMRBKFrF8_1753769423/0)
![](https://wework.qpic.cn/wwpic3az/112896_grYV6IRsRSOIIeV_1753769427/0)
或在手机端入口：【工作台->班级群->学校班级群->点击任意班级群->群管理】
![](https://wework.qpic.cn/wwpic3az/551495_dtSYTHKhTt-Z0Mk_1766040080/0)

### [](https://open.work.weixin.qq.com)二、班级群的管理权限

- 禁止改群名：开启后，群成员会无法修改群名，仅群主或群管理员可以修改。
- 进入群聊权限：开启后，仅家校通讯录中的家长可进群。
- 成员邀请权限：开启后，群成员需要群主确认才能邀请成员进群。扫描二维码进群将同时停用。
- 群管理员：可为班级群设置群管理员，单个群聊仅支持添加3个群管理员。
- 修改群主：管理员可修改班级群的群主。
- 标记为班级群：管理员可标记/取消班级群。取消后原班级群将不能使用班级群功能。
- 解散群聊：解散群聊后，所有群成员都会被移出群聊。

### [](https://open.work.weixin.qq.com)三、管理员如何批量管理班级群

管理员支持为全校的班级群打开以下设置：禁止改群名、仅家校通讯录中的家长可进群、群聊邀请确认。
1、管理后台入口：【家校沟通->班级群->批量设置群管理】
![](https://wework.qpic.cn/wwpic3az/327429_pk7prEjVR1OcO8-_1753769432/0)
2、手机端入口：【工作台->班级群->学校班级群->批量管理】
![](https://wework.qpic.cn/wwpic3az/838424_f0Nimdi3QWCSbZO_1766040114/0)

### [](https://open.work.weixin.qq.com)四、取消标记班级群

一个班级仅支持对应一个班级群，若不需要班级群功能，可以在群管理中选择取消标记班级群或解散群聊。
![](https://wework.qpic.cn/wwpic3az/355496_h6rFwNEvS3eqgEB_1766040154/0)
