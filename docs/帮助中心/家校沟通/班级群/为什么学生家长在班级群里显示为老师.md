# 为什么学生家长在班级群里显示为老师

当家长使用企业微信账号加入班级群时，对应的家长会出现在群成员列表“老师”一栏。目前群成员列表“老师”一栏会统计企业微信账号的群成员，“家长”一栏则统计微信身份加入群聊的群成员。

暂不支持标记微信用户为老师，建议老师使用企业微信账号加入班级群/学生群，便于管理班级群。

暂不支持标记企业微信用户为家长/学生，建议家长/学生使用微信账号加入班级群/学生群，便于提交班级作业、健康上报等。
