# 家校沟通介绍

行业类型选择了「学前教育」、「初中等教育」企业新增「家校沟通」功能，使用家校沟通，学校可统一管理「家校通讯录」并快速组建班级群。家长无需下载企业微信，可使用微信接收学校通知，使用学校应用，并添加老师的企业微信进行沟通。

如果想快速了解和入门「家校沟通」，您可以阅读：[「家校沟通」使用指引](https://work.weixin.qq.com/wework_admin/eduContacts/guide)
