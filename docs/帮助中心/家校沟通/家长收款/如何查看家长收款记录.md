# 如何查看家长收款记录

管理员可以在客户端以及管理后台查看收款记录并导出包含年级、班级、学生姓名、缴费与否、缴费金额、缴费时间的明细表。若选择收款至学校的对公账户，班主任则无需在中间收款记账，仅需起到督促收款的作用，管理员在后台明细表中即可掌握全校收缴情况。

家长收款的收款记录有两种：“指定范围收款”和“二维码收款”（区别在于是否是通过二维码收款）

操作路径如下：
【管理后台->应用管理->家长收款->收款记录->指定范围收款/二维码收款】
![](https://wework.qpic.cn/wwpic3az/213802_D7AziEMsRCORlyy_1753768628/0)
【手机端->工作台->家长收款->收款记录/二维码收款】
