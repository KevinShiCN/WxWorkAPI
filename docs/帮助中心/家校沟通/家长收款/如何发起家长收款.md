# 如何发起家长收款

### [](https://open.work.weixin.qq.com)一、如何设置收款账户

1、教育版支持两种收款方式：学校对公账户收款和微信零钱账户收款。

（1）老师的个人微信零钱账户收款：学校验证后，已实名的老师可以使用个人微信零钱账户收款，无需申请商户号，学校管理员还可统一查看和管理收款记录。

（2）学校的对公账户收款：开通此功能需提前申请微信支付商户号，收到的钱会进入到企业的微信支付商户号中，可提取至企业对公银行卡或者个体户的法人银行账户。

（3）**操作路径如下：**
【管理后台->应用管理->家长收款->收款账户管理】
![](https://wework.qpic.cn/wwpic3az/225576_BQeLNdinSfyEuII_1753768678/0)

【手机端->工作台->家长收款->右上角三个点->应用管理->收款账户管理】
![](https://wework.qpic.cn/wwpic3az/829320_xrpKBfoLTt-w197_1766038756/0)

2、学校可根据需要配置家长收款能力的老师范围，可以开放给全部老师，也可以指定部分老师。您可以在【手机端->工作台->家长收款->右上角三个点->应用管理->收款账户管理->使用范围】中管理。
![](https://wework.qpic.cn/wwpic3az/485349_P128kqKNQN2L43j_1766038782/0)

### [](https://open.work.weixin.qq.com)二、收款对象

支持向微信外部联系人收款，包括单聊、外部群聊和班级群。老师发起后，外部联系人在微信中收到付款提醒，可点击进入付款。

### [](https://open.work.weixin.qq.com)三、收款入口

老师可以通过以下两个入口发起家长收款：
1、【手机端->工作台->家长收款->发起收款 / 二维码收款】
![](https://wework.qpic.cn/wwpic3az/286054_NrCF7ExKTfKUUTg_1766038801/0)

2、【手机端->班级群->群聊工具栏 / 点击+号->家长收款】
![](https://wework.qpic.cn/wwpic3az/610513_T9VhhrqpQvqX484_1766038819/0)
![](https://wework.qpic.cn/wwpic3az/752225_8K1s7W6jSk2ew-1_1766038843/0)

### [](https://open.work.weixin.qq.com)四、发起收款入口区别

1、班级群入口中的家长收款，仅支持向班级群内的全部家长（或指定家长）发布收款小程序。这个入口更适合班主任发布家长收款任务。

2、工作台入口中的家长收款，既可以从「家长通知」渠道发送收款小程序到指定年级、班级与家长，也可以通过生成收款二维码的方式下发至班主任继续分发收款任务并督促收款。这个入口更适合学校管理员发布家长收款的任务。

### [](https://open.work.weixin.qq.com)五、个人账户收款条件

学校必须同时满足以下条件，才可以使用个人微信零钱账户收款：
1、学校验证
管理员可以前往【工作台->管理企业->企业信息->学校验证】中，进行学校验证，推荐学校使用「[主体资料验证](https://open.work.weixin.qq.com/help2/pc/16541?person_id=1&searchData=)」的方式。

2、老师个人实名
老师可以在【手机端消息页面左上角「三」->点击姓名->个人信息->-姓名】中，进行实名认证。

3、「我的微信零钱账户」使用范围
管理员支持在【管理后台->应用管理->对外收款->收款账户管理->查看】设置「我的微信零钱账户」账号使用范围，不在使用范围内的成员不支持收款到个人微信零钱。

### [](https://open.work.weixin.qq.com)六、如何停止收款

可在【手机端->工作台->家长收款->收款记录->点击需要停止收款的记录->右上角->停止收款】

### [](https://open.work.weixin.qq.com)七、常见问题

#### [](https://open.work.weixin.qq.com)1、家长收款，如何限制老师仅能对负责的班级收款？

目前暂不支持限制老师仅能对负责的班级收款。
如果老师是负责是1班，1班班级群里非1班的学生，1班的老师发起收款后，收款统计会统计老师负责范围里且在群里的学生，不在负责范围里的学生目前是支持点击付款的。

#### [](https://open.work.weixin.qq.com)2、家长付款时提示【暂时无法支付，请联系对方前往“小程序设置>收款二维码”开启收款开关】

收款方未开启二维码支付，则微信用户会看到弹窗“暂时无法支付，请联系对方前往“小程序设置>收款二维码”开启收款开关”。

需进行以下操作
（1）让收款方通过会话消息发送收款卡片。
（2）微信用户点击卡片进行付款，此时由于未开启二维码支付，会出现提示开启的弹窗（暂时无法支付，请联系对方前往…..）。
（3）微信用户关闭弹窗后，收款方会话中会推送一条灰字消息。
（4）收款方点击灰字消息中的“去开启”，就可以按页面提示开启后再让微信用户操作。
（5）请确保家长的微信客户端是最新版本，请确保成员的企业微信客户端是最新版本。

#### [](https://open.work.weixin.qq.com)3、家长收款订单如何退款？

家长收款订单暂不支持申请退款。
