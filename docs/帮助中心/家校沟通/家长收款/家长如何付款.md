# 家长如何付款

家长在班级群会话中或学校通知中收到收款推送，点击推送即可进行付款。

注：
1、同个学生绑定了两个家长，其中一个家长付款之后，另个家长则无法继续付款。
2、家长在付款时，暂不支持填写“付款备注”。
3、家长收款订单暂不支持退款。
