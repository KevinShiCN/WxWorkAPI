# 如何设置局校通讯录

### [](https://open.work.weixin.qq.com)一、创建局校互联

超级管理员登录【管理后台->左下角->局校互联】进入局校互联。一个企业支持创建5个局校互联。
![](https://wework.qpic.cn/wwpic3az/723340_ncfO3SzcTauIMA0_1753767985/0)

### [](https://open.work.weixin.qq.com)二、添加学校

添加辖区学校到局校互联，统一管理互联通讯录并和学校成员共同使用应用，局校沟通更方便。
教育局支持为未注册企业微信的学校创建企业微信。教育局管理员登录企业微信管理后台，点击【左下角局校互联->首页->邀请学校加入】，支持单个创建和批量创建。
已经注册了企业微信的学校，可通过扫码加入局校通讯录。二维码3天内有效
![](https://wework.qpic.cn/wwpic3az/67883_KOPaaRMPQviPyCs_1753768002/0)
学校加入教育局的局校互联后，教育局可在局校通讯录管理学校。由教育局创建的学校，教育局的超级管理员支持进入学校的管理后台。

### [](https://open.work.weixin.qq.com)三、学校侧使用

学校加入了局校互联后，在对局校通讯录可见的情况下，在客户端通讯录可以看见局校互联，可对局校互联内的人员进行沟通。
![](https://wework.qpic.cn/wwpic3az/921778_O5N2CRjeTQKsAXj_1766036523/0)

### [](https://open.work.weixin.qq.com)四、局校通讯录标签

1、发送公告的时候可以选局校互联通讯录的标签；
2、设置局校互联通讯录查看权限的时候可以选局校互联企业的标签。
