# 如何设置局校管理权限

### [](https://open.work.weixin.qq.com)一、通讯录查看权限

局端超级管理员支持设置「可查看互联通讯录的学校」、「相互可见的成员」
【[管理后台](https://work.weixin.qq.com)->左下角->局校互联->管理->通讯录查看权限】
![](https://wework.qpic.cn/wwpic3az/588439_R6exAw5PTe6yyT6_1753756250/0)

### [](https://open.work.weixin.qq.com)二、验证学校

教育局管理员登录企业微信管理后台，点击【左下角局校互联->首页->批量验证学校】或【管理->验证学校】多选或全选学校申请验证，可以给下属学校批量验证企业微信。（一般审核时间是3到5个工作日。）
![](https://wework.qpic.cn/wwpic3az/949386_E8Uy3n5WRHK0iC6_1753756359/0)

### [](https://open.work.weixin.qq.com)三、查看互联信息

在「管理」中支持查看互联信息，可查看局校互联名称、局校互联简介、创建者、创建时间、局校互联ID。
![](https://wework.qpic.cn/wwpic3az/904256_2XBX4zzFQY6EgP9_1753756363/0)
