# 老师如何发起课外打卡

### [](https://open.work.weixin.qq.com)一、 发起入口

老师可以通过工作台、班级群或者学生群创建课外打卡任务，支持全新创建，或者选择已有模板创建。仅支持在手机端查看、发起，电脑端暂不支持。

操作路径如下：
【手机端->工作台->课外打卡】
![](https://wework.qpic.cn/wwpic3az/979427_5rIT1ubBT5KUyPJ_1766038143/0)

【班级群/学员群->聊天工具栏->课外打卡】
![](https://wework.qpic.cn/wwpic3az/91175_bkXyjoK1RxOyXBm_1766038160/0)

### [](https://open.work.weixin.qq.com)二、课外打卡设置

支持添加以下内容：
**「打卡内容」**：支持填写标题和正文内容，支持添加图片/视频/录音；上限：标题20字、正文1000字、图片9张、视频1个，视频不可大于100M；
**「参与打卡」**：会唤起家校通讯录，可以选择班级或者某个学生；
**「持续时间」**：显示任务从起始到结束的总天数，起始时间为当天，默认总天数30天，点击可设置持续天数。起始时间到截止时间最多可覆盖100天；
**「打卡频次」**：默认每天，可设置星期几的方式设置打卡频次；
**「提醒打卡」**：默认是每天19点，点击可设置具体提醒时间，支持选时分；
**「生效时间」**：发布课外打卡时老师可以选择立即生效、明日生效；立即生效是当天就生效，明天生效是第二天生效。

### [](https://open.work.weixin.qq.com)三、频率限制

发布课外打卡次数暂无上限。
