# 老师如何查阅课外打卡

### [](https://open.work.weixin.qq.com)一、查看/导出课外打卡

老师可以在班级群、学生群或工作台->课外打卡中查看打卡的完成情况并提醒未完成家长去打卡。
老师支持导出打卡明细，包括哪些学生有进行打卡，没有打卡的日期以及已完成打卡的日期。
![](https://wework.qpic.cn/wwpic3az/658277_Knr58YVoSayCmJM_1766039593/0)

### [](https://open.work.weixin.qq.com)二、班级群发起课外打卡统计有误

班级群发起课外打卡，打卡范围统计跟管理后台【家校沟通->配置->家校通讯录设置】有关：
1、选择「全部班级」老师在家校通讯录中可以看见家校通讯录的所有学生，班级群布置作业范围则是在家校通讯录且在这个群里的学生；

2、选择「仅负责范围内的班级」老师在家校通讯录中只能看见负责范围内的学生，发起课外打卡的范围是在布置作业的老师负责的班级学生。

若出现统计人数和班级群学生数不一致的情况，可确认老师是否有家校通讯录的查看权限，或对应班级群的学生是否有在该老师的负责范围内。
