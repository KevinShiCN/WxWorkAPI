# 学生如何填写课外打卡

### [](https://open.work.weixin.qq.com)一、填写入口

家长可以在「学校通知」、班级群或者学生群中收到打卡提醒并完成打卡。

若家长有多个孩子，在课外打卡的页面支持选择对应的孩子进行打卡，已完成打卡的孩子无法重复打卡。

![](https://wework.qpic.cn/wwpic/380337_UOQc5WGnQJ-U0sV_1662607727/0)

### [](https://open.work.weixin.qq.com)二、课外打卡支持上传哪些内容

课外打卡支持添加语音、视频、图片、录音。
1、家长打卡时至少需要输入一个字符或者上传一个附件后才能提交打卡。
2、每次打卡最多可添加9个附件，一行最多展示4个附件（语音最长是10分钟/单个图片大小不超过10M/单个视频时长不超过10分钟、大小不超过25M，相机拍摄视频最长30s）
3、支持输入200个字符（即100个中文汉字）。
