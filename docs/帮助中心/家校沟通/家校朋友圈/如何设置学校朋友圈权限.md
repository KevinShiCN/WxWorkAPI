# 如何设置学校朋友圈权限

### [](https://open.work.weixin.qq.com)一、设置朋友圈使用权限

超级管理员在【管理后台->家校沟通->配置->家校朋友圈->配置使用范围】
![](https://wework.qpic.cn/wwpic3az/89472_mETpp16DSG60AWk_1753754647/0)
![](https://wework.qpic.cn/wwpic3az/819568_LSCssxtpQiWExNx_1753754722/0)
或【手机端->工作台->家校朋友圈->配置->配置使用范围】入口给成员添加朋友圈的使用权限。
![](https://wework.qpic.cn/wwpic3az/24483_XTF0-CVoQ4iKgCq_1766040563/0)
家校朋友圈默认是全员可以使用，支持修改使用范围，需要有朋友圈使用权限的成员才能在手机端使用朋友圈功能。

### [](https://open.work.weixin.qq.com)二、关闭成员的家校朋友圈入口

超级管理员可在【手机端->工作台->管理企业->工作台显示->成员可见->家校沟通】处打开开关，打开后，仅超级管理员和具有使用权限的成员可见家校朋友圈入口、无权限的成员将无法在工作台上看到该入口。
![](https://wework.qpic.cn/wwpic3az/497922_sL0lE0ItSt-rWFz_1766040585/0)
