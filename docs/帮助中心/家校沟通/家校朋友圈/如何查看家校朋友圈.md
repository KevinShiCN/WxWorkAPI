# 如何查看家校朋友圈

### [](https://open.work.weixin.qq.com)一、家长查看朋友圈

#### [](https://open.work.weixin.qq.com)1、查看入口

成员发布朋友圈后，微信家长可在朋友圈或查看成员个人详情时查看成员推送的朋友圈内容。
![](https://wework.qpic.cn/wwpic3az/959794_mQ0mA2TCSluR7_t_1705455048/0)

#### [](https://open.work.weixin.qq.com)2、查看不到家校朋友圈的原因/为什么家长看不到朋友圈

（1）一个家长每天只能看到同个人发送的3条朋友圈，这三条朋友圈可以是企业发表的，也可以是成员发自己发表的。若管理员对这个成员的家长在当日推送超过3条，则成员可以发送成功，家长则看不到这个朋友圈。

（2） 若企业中的多个成员添加了同一个家长，管理员对这些添加了同一个家长的成员都推送了企业朋友圈，则家长侧会看见最先点击确认发送成员推送的企业朋友圈内容。后面点击确认的成员会显示发送成功，实际用户还是只能看见第一个点击确认推送的成员发表的内容。

（3）微信家长在查看到企业微信联系人发送的朋友圈后，对应的朋友圈消息右上角可「不感兴趣」：点击后对同一个服务人员的圈第一次点7天不可见，第二次点30天不可见，第三次点一年(自然年)不可见。

（4）发表朋友圈后，可能会有10分钟的延迟。

（5）家长的微信需要更新到微信最新版本，Android版本7.08版本或以上，ios版本需要7.09版本或以上。

（6）需要检查看不到朋友圈的这个微信是否与成员是双向好友，单向好友的情况看不到朋友圈。

（7）发布的朋友圈图片或视频大小不支持超过10M，若超过大小在微信侧朋友圈会看不到该内容。

（8）发送朋友圈时可通过输入#号加文字，添加微信话题，如果客户使用的微信版本低于7.0.17，则#号的话题内容不会显示蓝色。

### [](https://open.work.weixin.qq.com)二、成员查看家校朋友圈内容

成员支持在【手机端->工作台->家校朋友圈】页面查看已发表的朋友圈。
![](https://wework.qpic.cn/wwpic3az/199832_BD4Pq0njRnW5EC7_1766040447/0)

### [](https://open.work.weixin.qq.com)三、管理员查看家校朋友圈内容

超级管理员支持在管理后台和手机端查看企业成员发表的朋友圈：
【管理端->家校沟通->配置->学校发表到家长的朋友圈->学校发布内容】
![](https://wework.qpic.cn/wwpic3az/466517_z_zbFM5KS7KTuG3_1753754802/0)
![](https://wework.qpic.cn/wwpic3az/835995_5aDevnX3TJChAQp_1753754805/0)
【手机端->工作台->家校朋友圈->统计】
![](https://wework.qpic.cn/wwpic3az/551890_duYQ6ft9R8Wj0IF_1766040466/0)
