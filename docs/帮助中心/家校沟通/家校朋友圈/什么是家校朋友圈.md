# 什么是家校朋友圈

使用家校朋友圈可将活动信息、产品动态、专业知识等内容发表到家长的微信朋友圈，并与家长评论互动。

服务消息高效分发，企业可统一创建朋友圈，成员确认后、即可直接将内容发表到家长的微信朋友圈。

监管成员的发表内容和数据，保障服务质量，企业可查看成员发表的内容以及与家长的互动详情，确保服务质量。

目前除了行业类型为「教育-初中等教育」、「教育-其他」这两类企业还未支持使用客户朋友圈，其余企业均支持使用客户朋友圈功能。

注：行业类型为「教育-学前教育」支持使用家校朋友圈功能。
