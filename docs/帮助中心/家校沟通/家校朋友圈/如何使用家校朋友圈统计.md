# 如何使用家校朋友圈统计

### [](https://open.work.weixin.qq.com)一、如何查看朋友圈统计

超级管理员在【手机端->工作台->家校朋友圈->统计】中可以查看管理范围内当天成员发布朋友圈概况。
![](https://wework.qpic.cn/wwpic3az/984368_fcQWRl4iR6mNq59_1766048178/0)

### [](https://open.work.weixin.qq.com)二、如何接收成员每日发表概况

仅超级管理员可在【手机端->工作台->家校朋友圈->配置】开启【接收成员每日发表概况】开关，开关默认关闭。
开启后超级管理员可以在每天18:00后在【朋友圈】应用收到成员每日发表概况推送。
![](https://wework.qpic.cn/wwpic3az/828829_eeD5WS_fQiSOnWj_1766048191/0)
