# 如何发表家校朋友圈

### [](https://open.work.weixin.qq.com)一、发送发表家校朋友圈说明

● 超级管理员支持创建朋友圈内容通知成员发表，并可查看成员发表数据情况，支持将活动信息、产品动态、专业知识等内容发表到家长的微信朋友圈，并与家长评论互动。
● 目前除了行业类型为「教育-初中等教育」、「教育-其他」这两类企业还未支持使用客户朋友圈，其余企业均支持使用客户朋友圈功能，行业类型为「教育-学前教育」支持使用家校朋友圈功能。

### [](https://open.work.weixin.qq.com)二、家校朋友圈的发送入口

1、超级管理员支持在【管理后台->家校沟通->配置->学校发表到家长的朋友圈】进行新增企业朋友圈。
![](https://wework.qpic.cn/wwpic3az/187245_hWIe0kRuSr6qrvF_1753754864/0)
![](https://wework.qpic.cn/wwpic3az/784074_ZAnml7JNTGSkVpK_1753754866/0)
2、有家校朋友圈使用权限的账号支持在【手机端->工作台->家校朋友圈->发表到家长的朋友圈】自己发布内容，管理员也可在该入口创建内容，通知成员发表。
![](https://wework.qpic.cn/wwpic3az/901862_BHneYEhhTeynu_d_1766040284/0)
注：电脑端暂不支持发表家校朋友圈。

### [](https://open.work.weixin.qq.com)三、家校朋友圈可展示家长限制/发送朋友圈提示「当前没有满足发表条件的家长」

1、**朋友圈发送频率：：**一个家长每天只能看到同个人发送的3条朋友圈，这三条朋友圈可以是企业发表的，也可以是成员发自己发表的。

2、**家校朋友圈可展示的家长数：**v4.1.26版本开始，所有企业朋友圈发表的内容，可展示给所有已添加的家长。

3、若发送朋友圈提示“当前没有满足发表条件的家长”，请更新企业微信到最新版本后重试。
若更新版本后仍未恢复，请确认：
（1）有可能是超过可发送的限制，一个家长每天只能看到同个人发送的3条朋友圈，这三条朋友圈可以是企业发表的，也可以是成员发自己发表的。（发表成功后再删除也会占用次数）
（2）成员需要有添加微信联系人，可查看【手机端->通讯录->外部联系人】是否有微信联系人。
（3）成员需要在[学校朋友圈权限](https://open.work.weixin.qq.com/help2/pc/20596?person_id=1)使用范围内。

### [](https://open.work.weixin.qq.com)四、设置朋友圈封面

● 支持在朋友圈页面设置封面及个性签名。
![](https://wework.qpic.cn/wwpic3az/988668_PPdDTyRzTRC105I_1705394217/0)
● 手机端->工作台->家校朋友圈，新增封面素材库和签名素材库，有朋友圈管理权限的成员可配置素材，成员在设置封面及签名时可直接使用。
![](https://wework.qpic.cn/wwpic3az/443502_-YtkeT3VTG6HcQb_1766040308/0)
● 朋友圈背景推荐尺寸是1028 x1028。

### [](https://open.work.weixin.qq.com)五、家校朋友圈如何停止发表

企业群发的朋友圈支持停止发表，未发表成员不会再收到发表提醒。

管理员可在【管理后台->家校沟通->配置->学校发表到家长的朋友圈->学校发布内容->选择对应的朋友圈记录->点击右侧“…”->停止发送】（仅支持停止发表一年内的朋友圈，大于一年的，需在手机端操作）
![](https://wework.qpic.cn/wwpic3az/481550_W8iAZxCAQkWcZ6I_1753754874/0)
![](https://wework.qpic.cn/wwpic3az/258268_y3xGZ_QWRsmorY9_1753754877/0)
![](https://wework.qpic.cn/wwpic3az/676382_TZ6QcIAVSECml4x_1753754915/0)

【手机端->工作台->家校朋友圈->统计->待成员发表的->选择对应的朋友圈记录进入->右上角“…”->停止发表】
![](https://wework.qpic.cn/wwpic3az/416154_ZVPs94A3ScuWHB8_1766040333/0)
