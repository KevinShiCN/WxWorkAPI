# 如何发起上课直播

### [](https://open.work.weixin.qq.com)一、发起入口

1、【手机端/电脑端->工作台->上课直播】（mac暂时不支持上课直播）
![](https://wework.qpic.cn/wwpic3az/904736_neDCN1klTwqLqcU_1766038957/0)
![](https://wework.qpic.cn/wwpic3az/757226_1YnUljqcQhC0bie_1750225836/0)
2、【手机端/电脑端->班级群的会话窗口->上课直播】
![](https://wework.qpic.cn/wwpic3az/674930_5QxE0AwaQ0qB0LE_1766038975/0)
![](https://wework.qpic.cn/wwpic3az/103470_wLWRtsPFTtiksVy_1750225842/0)

### [](https://open.work.weixin.qq.com)二、观看直播

选择上课班级发起上课直播后，会向所选择的班级群下发直播卡片。
在群里的家长可以点击直播卡片进行观看直播。
家长支持申请发言，同个直播目前最多支持8个家长同时发言。
![](https://wework.qpic.cn/wwpic/750124_3NuHHwkGQbyPkHa_1662624908/0)

### [](https://open.work.weixin.qq.com)三、分享直播

直播过程中，主播可选择「分享到微信」，邀请微信好友观看直播。微信好友打开直播链接，可选择某个班级群去扫码加入群聊，进群后点击「班级群小助手」下发的直播小程序来观看直播。
![](https://wework.qpic.cn/wwpic/122685_ag-f3R2RQ1mMG7s_1662624979/0)
