# 如何查看上课直播记录

1、主播可以在【手机端/电脑端->工作台->上课直播->直播记录】 查看直播详情，可查看微信观看人员明细，查看详情中暂时不支持查看企业微信联系人的观看明细。直播人数统计包含企业微信的观看人数和微信的观看人数。
![](https://wework.qpic.cn/wwpic3az/823965_lc98FnbIQV6KlNH_1766039970/0)

2、管理员在【管理后台->应用管理->上课直播】中可查看直播记录，以及导出直播明细。
直播记录，仅保留了半年内的记录。时间选择范围目前仅支持选择时间跨度在45天内的直播记录。
![](https://wework.qpic.cn/wwpic3az/443357_oJHf_4fiRnS2OgY_1753768462/0)
