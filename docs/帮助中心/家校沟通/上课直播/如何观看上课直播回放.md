# 如何观看上课直播回放

1、在发起直播时开启了课堂回放，则可查看直播回放记录。
![](https://wework.qpic.cn/wwpic3az/267304_3XXAYBKXRg6S2sN_1750227011/0)

2、在【手机端/电脑端->工作台->上课直播】查看直播记录中可查看回放。
![](https://wework.qpic.cn/wwpic3az/476515_JUSks2oGQcaiZYF_1750227017/0)

3、管理员在【管理后台->应用管理->上课直播】可以设置回放储存时间。
回放存储时间支持选择临时存储15天或永久存储。
临时存储15天，不占用企业微盘容量，过期后自动删除；永久存储，会占用企业微盘容量。
![](https://wework.qpic.cn/wwpic3az/41965_hkp9cMCNR2S_V2J_1753768317/0)
