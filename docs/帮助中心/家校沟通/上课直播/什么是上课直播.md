# 什么是上课直播

1、上课直播可以选择「大班课」、「小班课」、「通用直播」三种直播方式。

（1）小班课：进入直播后，会默认开启音视频设备（若直播人数超过50人，则会默认关闭麦克风和摄像头）

（2）大班课：学生需要举手发言。

（3）上课直播支持在班级群和工作台发起直播，班级群发起的直播观看范围默认群里的学生，工作台发起的直播可选择观看范围。
