# 上课直播相关设置

### [](https://open.work.weixin.qq.com)一、如何发起上课直播签到

1、主播点击直播间内签到图标可向学生发起签到；
2、发起签到时支持选择30秒、1分钟、2分钟内签到。
![](https://wework.qpic.cn/wwpic/541384_0y5ScXgaTwi5sij_1662625102/0)

### [](https://open.work.weixin.qq.com)二、老师如何关闭直播麦克风

发起直播后，老师可点击麦克风图标进行开启/关掉麦克风。
![](https://wework.qpic.cn/wwpic/655142_rFKl4L2mRwqdZdq_1662430458/0)

### [](https://open.work.weixin.qq.com)三、如何一键关闭全员语音、视频和评论

发起直播后，老师可点击正在上课的界面，关闭全员语音、视频和评论。
关闭权限后，正在连麦的观众也会被关闭麦克风、视频和评论，所有观众都会无法评论、申请连麦。重新开启后观众可正常评论和连麦。
![](https://wework.qpic.cn/wwpic/148128_VFEoq9L2RpK-FyD_1670825030/0)

### [](https://open.work.weixin.qq.com)四、上课直播观看权限

开启直播前、直播中，都可以设置观看权限，支持选择「仅家长和学校内成员可观看」、「学校内、外人员都可观看」、「仅上课群内的成员可观看」三种方式，在线上课更安全。
![](https://wework.qpic.cn/wwpic/855640_baadp00zTWqVFRA_1670825827/0)
