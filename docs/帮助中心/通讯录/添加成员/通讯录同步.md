# 通讯录同步

### [](https://open.work.weixin.qq.com)一、什么是通讯录同步

通讯录同步是企业微信提供的方便企业用于与其他系统进行通讯录双向同步的工具，可通过API接口或第三方应用同步通讯录。

### [](https://open.work.weixin.qq.com)二、设置入口

设置路径：【[管理后台](https://work.weixin.qq.com)->安全与管理->管理工具->通讯录同步】可开启/关闭。
![](https://wework.qpic.cn/wwpic3az/497688_zdrynhuEQiOd2pi_1753407199/0)
开启后，企业可以根据需要设置相应的权限，如读取并编辑通讯录的权限。

![](https://wework.qpic.cn/wwpic3az/114131_7bOVGR3yRNK2G8__1753407771/0)

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、管理后台没有通讯录同步入口

目前仅超级管理员有权限，若是分级管理员权限会看不到通讯录同步的入口。

#### [](https://open.work.weixin.qq.com)2、通讯录同步没有secret选项

若企业使用了第三方应用同步通讯录，则不会有secret入口。若需要看到入口，可关闭同步。
在通讯录同步页面左下角可“关闭接口同步”，点击关闭后则无法继续同步第三方通讯录。企业需自行评估是否需要取消同步。
