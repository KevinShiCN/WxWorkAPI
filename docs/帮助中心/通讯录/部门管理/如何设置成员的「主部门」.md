# 如何设置成员的「主部门」

### [](https://open.work.weixin.qq.com)一、主部门的作用

主部门会影响客户端和管理后台成员信息展示部门的排序。具体有以下三个情况：
1、通讯录页面：搜索成员时优先显示主部门，再显示其他组织架构。
![](https://wework.qpic.cn/wwpic3az/817351_fidDLvHMTlS2Q3s_1753422866/0)

2、管理员在【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->企业内资料展示】开启了桌面端信息显示部门后，电脑端单人会话界面展示的部门会展示主部门。
![](https://wework.qpic.cn/wwpic3az/57570_8G4xqLH-TbSTcQr_1753422896/0)

3、电脑端查看成员个人信息页面上的部门信息，排序在首行的为主部门。
![](https://wework.qpic.cn/wwpic3az/144211_djZcsnLGSp2aNW5_1750147295/0)

### [](https://open.work.weixin.qq.com)二、设置入口

若成员有多个部门，管理员在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构】设置成员信息时，支持为其设置主部门。
![](https://wework.qpic.cn/wwpic3az/909998_hYTKlzECRFmHTeX_1753422918/0)
