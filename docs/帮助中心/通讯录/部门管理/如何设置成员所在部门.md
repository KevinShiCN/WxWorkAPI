# 如何设置成员所在部门

### [](https://open.work.weixin.qq.com)一、管理后台设置成员所在部门

#### [](https://open.work.weixin.qq.com)1、编辑成员信息入口修改

在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->编辑成员信息->找到部门字段->修改】可为成员设置部门。
![](https://wework.qpic.cn/wwpic3az/711971_JqQjXzU1SOylR6z_1753417863/0)

#### [](https://open.work.weixin.qq.com)2、批量设置成员部门

在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->批量设置成员信息】进行批量修改。
![](https://wework.qpic.cn/wwpic3az/845694_rxPbwyZdSda8F5z_1753417582/0)

#### [](https://open.work.weixin.qq.com)3、接口修改

通过接口更新成员所在部门：[参考文档](https://work.weixin.qq.com/api/doc/90000/90135/90197)

### [](https://open.work.weixin.qq.com)二、手机端设置成员所在部门

#### [](https://open.work.weixin.qq.com)1、批量移动入口修改

入口：【手机端->通讯录->右上角->通讯录管理->更多管理】，批量修改成员信息。
![](https://wework.qpic.cn/wwpic3az/883035_mv6PRdTAR2CxRAj_1765786306/0)

#### [](https://open.work.weixin.qq.com)2、编辑成员信息入口修改

入口：【手机端->通讯录->右上角编辑按钮->选择成员->编辑成员->设置部门】
![](https://wework.qpic.cn/wwpic3az/297099_ru6iOLBORtq1OPp_1765786327/0)

### [](https://open.work.weixin.qq.com)三、已为员工设置部门，但在手机端企业通讯录中成员显示在部门上方

若需企业通讯录中仅显示部门，管理员可以在【 [管理后台](https://work.weixin.qq.com/)->通讯录->组织架构->点击成员->编辑成员】，设置成员所在的部门，可以将成员移出根部门；
或在【手机端->工作台->管理企业->成员与部门管理】，点击成员进入编辑页面，将成员移出根部门。
设置后成员不会显示在部门上方。
