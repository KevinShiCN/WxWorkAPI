# 通讯录介绍

企业可依据组织架构创建通讯录，支持快速批量导入，统一管理；同事信息准确完善，方便查阅。
在通讯录管理模块，管理员可对通讯录的可见范围、通讯录的字段、以及通讯录安全进行统一管理；可根据企业内部人员构成情况，设置通讯录可见范围，保护成员信息不外泄；对成员通讯录卡片的多个字段进行可见、隐藏的设置。
