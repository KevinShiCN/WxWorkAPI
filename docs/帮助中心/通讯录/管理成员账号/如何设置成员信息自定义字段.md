# 如何设置成员信息自定义字段

### [](https://open.work.weixin.qq.com)一、新增通讯录自定义信息

1、管理员可以在【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->成员资料显示->企业内资料展示/对外资料显示】中修改/添加自定义信息。

对内自定义信息/全体成员信息显示自定义字段：支持添加20个。

对外自定义信息/对外专属信息：文本、网页、小程序类型一共支持添加6个；图文/附件类型支持添加4个。
![](https://wework.qpic.cn/wwpic3az/218528_GsgVHwyhTMe3zlp_1753412544/0)
![](https://wework.qpic.cn/wwpic3az/523515_glZ42wxbQRCYWc2_1753413041/0)
![](https://wework.qpic.cn/wwpic3az/494923_ZY2sw_XNS3aHDIN_1753412600/0)

2、在企业内资料展示/对外资料显示中添加自定义信息后，自定义信息的内容可在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->编辑成员信息】中设置。
![](https://wework.qpic.cn/wwpic3az/957145_9oCjdB26QJK1Xk8_1753413172/0)

### [](https://open.work.weixin.qq.com)二、批量设置成员的通讯录信息

可批量修改的信息为：部门、地址、职务、企业简称（企业有申请多个简称情况下才有该选项，企业只有一个简称则不可见）、视频号、对外职务、自定义信息。
批量修改成员信息，单次操作最大批量为1000人。

可在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->批量设置成员信息】进行批量修改。
![](https://wework.qpic.cn/wwpic3az/171123_U_yi6RcPQ3mlC98_1753414276/0)
或在【手机端->通讯录->右上角->通讯录管理->更多管理】，批量修改成员信息。
![](https://wework.qpic.cn/wwpic3az/105762_KQmCFsDNStK6W9F_1765782469/0)
