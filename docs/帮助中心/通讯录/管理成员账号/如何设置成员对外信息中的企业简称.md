# 如何设置成员对外信息中的企业简称

### [](https://open.work.weixin.qq.com)一、对外简称应用场景

当企业**[认证了多个简称](https://open.work.weixin.qq.com/help2/pc/14643)**后，管理员在企业微信管理端的Web端和手机端、[接口](https://work.weixin.qq.com/api/doc/90000/90135/92230)中可以对不同成员，设置不同的企业简称。
设置后，可在企业成员和外部联系人沟通的场景，显示设置的成员简称。
包含企业微信成员在微信、企业间的沟通等场景。

### [](https://open.work.weixin.qq.com)二、设置入口

1、[管理后台](https://work.weixin.qq.com)点击【通讯录->组织架构->搜索成员、部门、标签】搜索到对应的成员，点击【成员详情->编辑->企业简称】

![](https://wework.qpic.cn/wwpic3az/939941_6mteSUiTSFWbpCH_1706604270/0)
2、管理员可在手机端点击【工作台->管理企业->成员与部门管理】或【通讯录->右上角->管理通讯录】找到对应成员，点击成员信息编辑企业简称。
![](https://wework.qpic.cn/wwpic3az/413692_qpLJsxd5R5yn7NV_1765782161/0)
![](https://wework.qpic.cn/wwpic3az/113663_FD9yBKFbTgauQW0_1765782180/0)

### [](https://open.work.weixin.qq.com)三、设置成员对外信息中的企业简称未生效

1、目前修改企业简称后，在微信侧需要96小时左右同步。若未超过96小时可后续留意更新。

2、可以尝试关闭微信/企业微信后台运行，重新查看。

3、若是在微信侧群聊里或企业微信外部群查看的成员姓名@简称的简称未更新，则请该微信用户点击下这位企业微信成员，进入成员个人信息页面查看下，进行刷新数据确认是否可以正常显示。
