# 如何修改成员的职务

管理员支持在管理后台和手机端修改成员职务；成员暂不支持修改自己的对内职务，支持修改[对外职务](https://open.work.weixin.qq.com/help2/pc/17428)。
管理后台设置入口：【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->编辑成员信息->职务】，也可设置自定义对外职务，默认对外职务同步公司内职务。
![](https://wework.qpic.cn/wwpic3az/281400_GvsFy746Si-CGaF_1765785316/0)

手机端设置入口：【通讯录->右上角编辑按钮->选择对应成员->编辑成员->职务】，也可在对外职务中设置专属对外职务。
![](https://wework.qpic.cn/wwpic3az/99640_sdcUm7G7QlCcnce_1765785068/0)
