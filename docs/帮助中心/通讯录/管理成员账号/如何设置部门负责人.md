# 如何设置部门负责人

### [](https://open.work.weixin.qq.com)一、部门负责人的应用场景

在通讯录中设置了某个成员为部门负责人后，成员在对应通讯录会置顶显示。
在审批人、汇报人选择了部门负责人后，对应的处理人是通讯录中被标记为部门负责人的人员。

### [](https://open.work.weixin.qq.com)二、管理后台如何设置部门负责人

#### [](https://open.work.weixin.qq.com)1、查看部门页面设置部门负责人

在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->点击部门右侧的“···” ->设置负责人/在部门右上角设置负责人】，部门负责人暂无设置上限。
![](https://wework.qpic.cn/wwpic3az/203481_QUhL4aCuQECHL7y_1753414730/0)

#### [](https://open.work.weixin.qq.com)2、编辑成员信息入口修改部门负责人

在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->编辑成员信息->部门负责人】进行设置。
![](https://wework.qpic.cn/wwpic3az/355029_RD6CIlG4QNO3xgn_1753414637/0)

#### [](https://open.work.weixin.qq.com)3、接口修改部门负责人

通过接口更新部门负责人：[参考文档](https://work.weixin.qq.com/api/doc/90000/90135/90197)

### [](https://open.work.weixin.qq.com)三、手机端设置部门负责人

在【手机端->通讯录->右上角编辑按钮->选择对应成员->编辑成员->部门负责人】
![](https://wework.qpic.cn/wwpic3az/440392_bzB2I3PCReee_GO_1765785490/0)

### [](https://open.work.weixin.qq.com)四、通讯录中部门负责人如何排序

若通讯录设置[成员排序规则](https://open.work.weixin.qq.com/help2/pc/17422)是姓名A-Z排序，身份为「部门负责人」的成员会在通讯录中置顶显示。

若是设置的是自由排序，身份为「部门负责人」的成员则不会置顶显示。管理员可以在通讯录页面自由拖动成员进行排序。

### [](https://open.work.weixin.qq.com)五、常见问题

#### [](https://open.work.weixin.qq.com)1、成员在多个部门中，可以作为多个部门的部门负责人吗？

支持根据成员所在的部门设置对应的部门负责人，比如成员在1、2、3部门，可以单独把成员设置为是2、3部门的部门负责人。
![](https://wework.qpic.cn/wwpic3az/35752_IJ8-ICWlSdOb043_1749003193/0)

#### [](https://open.work.weixin.qq.com)2、为什么已经设置了成员为部门负责人，但是在通讯录中查看成员时，不显示“负责人”的标识

姓名旁边的“负责人”logo表示的是成员当前所在部门的身份，如果成员存在A、B部门，在A部门不是部门负责人，在B部门是部门负责人，则在名字旁边不会显示“负责人”logo，但是部门负责人一列会显示“是”。
