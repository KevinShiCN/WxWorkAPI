# 如何查询退出企业的员工

1、管理员可在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构】，在通讯录成员信息菜单中选择“已退出”，可筛选出主动退出企业的成员。
![](https://wework.qpic.cn/wwpic3az/772713_eyVHPfk-Sa2FmL4_1753411241/0)

2、如果成员退出企业不超过30天，且管理员未删除成员账号的，管理员可在【[管理后台](https://work.weixin.qq.com/) ->通讯录->组织架构->点击已退出企业的成员->邀请重新加入】。

管理员操作邀请后，成员可以用绑定的微信在微信侧点击管理员邀请加入企业的链接，或扫管理员提供的加入企业二维码加入企业，会出现“你已退出该企业，是否重新加入”的提示，点击“重新加入”后，成员需在账号被删除前，在企业微信app使用绑定的手机号或微信授权登录重新进入企业。
![](https://wework.qpic.cn/wwpic3az/205482_s2F0LPwQT-u6bM8_1753411846/0)
