# 如何退出企业/删除企业微信账号

### [](https://open.work.weixin.qq.com)一、如何主动退出企业

若是已离职/加入了陌生企业，需要退出加入的企业。可在手机端点击【消息页面左上角“三”->设置->管理企业->点击对应企业“>” ->退出企业】进行退出。
温馨提示：主动退出企业后，30天内若管理员未在通讯录进行邀请加入或已经邀请，成员未重新登录到对应的企业，30天后账号会被系统自动删除，被系统删除的账号不可恢复。
![](https://wework.qpic.cn/wwpic3az/642882_4QLDJ8ZdQGCgpNV_1758521403/0)

### [](https://open.work.weixin.qq.com)二、如何删除企业微信账号/离职成员

可以联系管理员在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构】找到需要[删除](https://open.work.weixin.qq.com/help2/pc/17403)的成员，勾选后，点击删除。

温馨提示：
1、删除成员后，成员所有信息和日志将被清除不可恢复，将自动从企业微信全员群退出。一般在通讯录被删除的成员会称为“离职成员”。
2、被删除的成员账号不可恢复（删除后，成员的企业微信消息记录将完全被清除）；若企业提前参与“数据保障计划”的情况下，可以联系数据服务团队工作人员咨询。详情见[“数据保障计划” 功能说明。](https://open.work.weixin.qq.com/help2/pc/18154)
![](https://wework.qpic.cn/wwpic3az/789154_9ZzUKf1tQnqPSSQ_1752818903/0)

### [](https://open.work.weixin.qq.com)三、如何解散企业

#### [](https://open.work.weixin.qq.com)1、管理后台如何解散企业微信

创建人可在【[管理后台](https://work.weixin.qq.com)->我的企业->设置->[解散企业](https://open.work.weixin.qq.com/help2/pc/14984)】进行解散。
![](https://wework.qpic.cn/wwpic3az/584027_kyOGdMSdReuYOAc_1752818518/0)

#### [](https://open.work.weixin.qq.com)2、手机端如何解散/注销企业微信

创建人可在【手机端->消息页面左上角“三”->设置->管理企业->点击对应企业“>”->解散企业】进行解散。
![](https://wework.qpic.cn/wwpic3az/610071_Y15ucjpVTnyicDM_1758521413/0)

### [](https://open.work.weixin.qq.com)四、解散企业的影响

解散企业需谨慎操作，解散后无法恢复。若企业提前参与 “数据保障计划” 的情况下，有可能找回原账号的记录和数据，详情见[“数据保障计划” 功能说明。](https://open.work.weixin.qq.com/help2/pc/18154)

### [](https://open.work.weixin.qq.com)五、常见问题

#### [](https://open.work.weixin.qq.com)1、解散企业微信提示“因为系统升级，该企业暂不支持解散，请联系客服处理”

出现该情况是由于系统升级，目前贵企业暂不支持解散企业微信，若有不需要使用企业微信的成员，管理员可登录【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->点击对应成员】禁用成员，禁用后成员将无法登录到对应的企业微信。

若不想使用禁用成员的方式，且确认企业不再使用需要解散，可用微信关注“企业微信”公众号，通过【找我们】->【联系客服】或在企业微信我的客服进行咨询。

#### [](https://open.work.weixin.qq.com)2、退出企业了还可以重新加入吗？

若主动退出企业不超过30天，且管理员未删除成员账号，管理员可在【 [管理后台](https://work.weixin.qq.com/)->通讯录->组织架构->[查看已退出企业的成员](https://open.work.weixin.qq.com/help2/pc/14962)】点击“重新邀请加入”，管理员操作邀请后，成员可使用绑定的手机号或微信授权登录到企业中。

#### [](https://open.work.weixin.qq.com)3、被禁用的成员可以退出企业吗？

被禁用成员仅限制成员登录当前企业，若成员登录到同个企业微信账号的其他企业中，在【手机端 ->消息页面左上角“三”->设置->管理企业】，可看见被禁用的企业，可操作主动退出。
