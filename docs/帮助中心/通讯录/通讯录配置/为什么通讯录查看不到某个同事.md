# 为什么通讯录查看不到某个同事

可能存在以下几种情况，请联系管理员确认：
1、请确认是否被设置限制查看所有人，被限制的部门或成员，不能看到企业所有通讯录。
【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->限制查看所有人】
![](https://wework.qpic.cn/wwpic3az/207560_bOCc9awHQcyKFkx_1753424866/0)

2、请确认是否被设置限制查看外部门，被限制的部门或成员，只能查看本部门的通讯录。
【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->限制查看外部门】
![](https://wework.qpic.cn/wwpic3az/767594_mhZWlGezRX259BB_1753424846/0)

3、请确认查不到的同事是否被设置为隐藏部门/成员，被设置为隐藏部门/成员的部门或成员无法被搜索到，也不会显示在组织架构中。
【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->隐藏的部门/成员】
![](https://wework.qpic.cn/wwpic3az/421182_HFoZj2JWS4eTLK-_1753424831/0)
