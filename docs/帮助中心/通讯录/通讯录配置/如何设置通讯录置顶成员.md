# 如何设置通讯录置顶成员

1、超级管理员在【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->[成员排序规则](https://open.work.weixin.qq.com/help2/pc/17422)】选择了【姓名A-Z排序】后，在通讯录支持设置置顶成员。若选择【自由排序】则不能设置置顶成员。
![](https://wework.qpic.cn/wwpic3az/946582_HuozzVOuQEWZSYj_1753424511/0)

2、【通讯录->组织架构->成员列表右边三个点】，可以选择置顶或取消置顶。被置顶的成员，会有浅灰色底色。
![](https://wework.qpic.cn/wwpic3az/947773_YPfVyJifTUmMEWe_1753424535/0)
