# 为什么看不到同事的手机或邮箱等信息字段

以下几种情况会导致看不到同事的手机或邮箱等信息字段：
1、若该成员信息中的手机/邮箱字段为空，则在通讯录看不见他的手机/邮箱信息；
2、有可能管理员在后台取消了企业通讯录中成员手机或邮箱等字段信息的显示，可联系管理员在【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->企业内资料展示->全体成员信息显示】查看。
![](https://wework.qpic.cn/wwpic3az/358719_vHhjO924QVufOYe_1753424915/0)

3、有可能该同事被企业管理员设置为了[敏感成员](https://open.work.weixin.qq.com/help2/pc/17426)，可联系管理员在企业微信管理后台进行查看，若是需核实敏感信息成员是否隐藏了手机或邮箱等字段。
![](https://wework.qpic.cn/wwpic3az/667916_pkJX2CWKRZq3Ytm_1753424935/0)
