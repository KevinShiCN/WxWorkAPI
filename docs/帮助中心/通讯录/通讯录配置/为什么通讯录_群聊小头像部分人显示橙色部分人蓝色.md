# 为什么通讯录/群聊小头像部分人显示橙色部分人蓝色

小头像代表性别，蓝色代表成员性别男，橙色代表成员性别女。
若性别有误可以[修改性别](https://open.work.weixin.qq.com/help2/pc/14946)。
