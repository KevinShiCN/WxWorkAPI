# 如何自定义通讯录列表的显示字段

1、创建人/超级管理员支持在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->右侧“☰”图标->列表显示字段】设置列表显示字段。分级管理员暂不支持该设置。
![](https://wework.qpic.cn/wwpic3az/430385_cM4Ic6LTQ5KGf0w_1753425380/0)

2、列表显示字段最多只能选择6个字段，其中姓名字段为必选；可在列表显示设置页面按住“☰”拖动图标进行排序字段。
![](https://wework.qpic.cn/wwpic3az/315319_2UkVloRkTtmngHc_1753425394/0)

3、**每页显示成员数**指通讯录每页可显示的成员的数量，可选择20、50、100。
![](https://wework.qpic.cn/wwpic3az/38248_GL7pKxrMR3-KIqc_1753425451/0)
![](https://wework.qpic.cn/wwpic3az/43749_a7yawcvRQzya-8y_1753425422/0)
