# 新增部门

### [](https://open.work.weixin.qq.com)一、如何在管理后台新增部门

超级管理员或有通讯录管理权限的分级管理支持在管理后台新增部门/子部门。

操作路径如下：
【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->点击加号->添加部门】
![](https://wework.qpic.cn/wwpic3az/119702_qcbegku9Q5yf0YG_1753155710/0)
【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->点击部门旁边“…”->添加子部门】；或通讯录页面右上方添加子部门。
![](https://wework.qpic.cn/wwpic3az/426674_6S6Ptlm_Sf2xH2V_1753155759/0)

### [](https://open.work.weixin.qq.com)二、如何在手机端新增部门

超级管理员或有通讯录管理权限的分级管理员，支持在【手机端->通讯录->右上角编辑按钮->添加子部门】。
![](https://wework.qpic.cn/wwpic3az/442949_wfjgcchkSQilajO_1765419281/0)

### [](https://open.work.weixin.qq.com)三、如何通过接口新增部门

可参考[文档](https://work.weixin.qq.com/api/doc/90000/90135/90205)。

### [](https://open.work.weixin.qq.com)四、部门数量上限是多少

1、部门最大层级为15层；
2、部门总数不能超过3万个；
3、每个部门下的节点不能超过3万个。
4、每个成员加入部门的数量暂无限制。
5、部门名称字数上限64个字符。

### [](https://open.work.weixin.qq.com)五、如何调整部门排序

管理员可在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构】点击部门右侧的“···”，点击「上移」或「下移」移动部门的位置，或者用鼠标左键按住部门上下拖动来调整排序。
![](https://wework.qpic.cn/wwpic3az/108824_RrucD69LR5G0m04_1753155791/0)

### [](https://open.work.weixin.qq.com)六、如何删除部门

超级管理员或拥有通讯录管理权限的分级管理员可以在通讯录中删除部门。
注：部门需为空，即部门下不包含任何成员和子部门，才能删除该部门。

操作路径：【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->点击部门右侧的“···”->删除】
![](https://wework.qpic.cn/wwpic3az/763896_ZCuuHw8lQQOC5LQ_1760938751/0)

### [](https://open.work.weixin.qq.com)七、常见问题

#### [](https://open.work.weixin.qq.com)1、通讯录添加部门/修改部门名称提示“部门已存在”/添加子部门失败

目前同级部门的部门名称暂不支持重复。

#### [](https://open.work.weixin.qq.com)2、部门id是否可以自定义

管理后台新增的部门会默认部门id，部门ID是随机生成的。暂不支持自定义。
通过api同步通讯录，创建的部门支持自定义看，可参考[文档](https://work.weixin.qq.com/api/doc#90000/90135/90205)。

#### [](https://open.work.weixin.qq.com)3、手机端删除部门时，提示“该部门下尚有被隐藏的成员或部门，无法删除”

请先检查企业是否有在【管理后台->我的企业->通讯录管理】中设置隐藏的成员或部门，若无，请超级管理员在管理端后台查看该部门下是否有成员。

#### [](https://open.work.weixin.qq.com)4、什么是根部门

通讯录中第一个部门为根部门，管理员可在【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构】查看。
