# 连接微信介绍

### [](https://open.work.weixin.qq.com)一、什么是连接微信

连接微信能力，支持添加客户的微信，在单聊、群聊、朋友圈、视频号中向客户提供持续的服务，还可结合小程序、支付等能力；更有「微信客服」，为客户提供临时咨询服务。

### [](https://open.work.weixin.qq.com)二、与微信用户可以做什么

1、添加微信用户为客户后，可以像普通微信好友之间一样沟通聊天。
![](https://wework.qpic.cn/wwpic3az/215636_bhajMgcxTS-AKbi_1764924021/0)

2、在发起群聊时，支持同时选择企业通讯录人员以及外部联系人/客户/微信联系人，此时创建的群聊为外部群。
![](https://wework.qpic.cn/wwpic3az/954117_dwpiIjyRSYquh7h_1764924060/0)

3、有客户朋友圈使用权限的账号支持在【手机端->工作台->客户朋友圈->发表到客户的朋友圈】自己发布内容，管理员也可在该入口创建内容，通知成员发表。
![](https://wework.qpic.cn/wwpic3az/294589_ANNLWNpDT765btw_1764924920/0)

4、超级管理员支持操作绑定视频号，视频号管理员扫码确认后可在成员对外信息中展示。（团队形态暂不支持绑定视频号。）
成员支持在对外资料展示页面中开启或关闭视频号显示。
![](https://wework.qpic.cn/wwpic3az/338561_XScA4WejTjeqtak_1764925384/0)

5、企业在企业微信内开通商户号后，客户即可在微信主动发起转账，员工也可使用收款单、收款码发起收款，资金入账企业账户统一管理，支付费率低至0.2%。
![](https://wework.qpic.cn/wwpic3az/156223_jI7_2MZ8QliONQd_1764924216/0)

6、企业使用企业微信即可由超级管理员在【管理后台->应用管理->微信客服】中启用微信客服应用进行使用。
![](https://wework.qpic.cn/wwpic3az/281691_ARe7ZjKGRHig-Lu_1764924711/0)
