# 如何生成客户群

如何新建客户群高效服务客户？[点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488326713827072_2)

### [](https://open.work.weixin.qq.com)一、主动创建客户群

在[客户联系使用范围内](https://open.work.weixin.qq.com/help2/pc/14690)的成员，发起的外部群会默认为客户群。

方式一：在【消息界面右上角“+” ->发起群聊->选择我的客户】
![](https://wework.qpic.cn/wwpic3az/179269_dfW6ITLtR5az0sp_1765174896/0)

方式二：在【手机端->工作台->客户群->创建一个客户群】
![](https://wework.qpic.cn/wwpic3az/830948_CfhjN6rTShKuazr_1760925256/0)

### [](https://open.work.weixin.qq.com)二、已有群聊确认为客户群

在企业微信侧创建的外部群，当群主把群主身份转让给在[客户联系使用范围](https://open.work.weixin.qq.com/help2/pc/14690)内的成员，新群主可以在把群聊确认为客户群，设置入口【群管理->标记为客户群】。
若成员不在[客户联系使用范围](https://open.work.weixin.qq.com/help2/pc/14690)，创建了外部群，然后给成员添加了客户联系功能使用权限后，成员支持把这类型的群确认为客户群，设置入口【群管理->标记为客户群】。
**注：目前从微信迁移的群聊暂不支持标记为客户群。**
![](https://wework.qpic.cn/wwpic3az/801510_Kf4POE9dREGlDd2_1765178489/0)

### [](https://open.work.weixin.qq.com)三、创建客户群数量上限

未验证未认证的企业，所有外部群聊（包括客户群）加起来的总人数上限为100人；
未满100人时，也仅支持创建20个群聊；
已验证/认证的企业，群聊个数无限制。
关于认证，可以参考：[企业微信认证介绍](https://open.work.weixin.qq.com/help2/pc/19729)
温馨提示：认证审核需要收取服务费，在支付之前，我们建议您详细阅读支付页面的相关条款。
