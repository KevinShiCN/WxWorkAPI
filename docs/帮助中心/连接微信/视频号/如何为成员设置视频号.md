# 如何为成员设置视频号

管理员支持在编辑成员通讯录资料时，为不同的成员设置不同的视频号，或[批量设置信息](https://open.work.weixin.qq.com/help2/pc/14956)：

1、【手机端->通讯录->右上角->选择成员->视频号】
![](https://wework.qpic.cn/wwpic3az/408662_FyuRxaqZS_uOmPK_1765357050/0)

2、【[管理后台](https://work.weixin.qq.com)->通讯录->组织架构->选择成员->编辑->视频号】
![](https://wework.qpic.cn/wwpic3az/446876_C3vUPyl1S82d4Ic_1753683470/0)

3、成员支持在对外资料展示页面中开启或关闭视频号显示：
【手机端->消息界面左上角“三”->头像->个人信息->对外信息显示->视频号】
![](https://wework.qpic.cn/wwpic3az/655175_WeXajCx2Se-LeXp_1765357079/0)

4、点击查看视频介绍：[如何设置不同员工对外展示不同视频号？](https://work.weixin.qq.com/nl/learning/24488323138827074_2)
