# 外部联系人介绍

1、外部联系人是指非本企业微信人员，企业内的成员可通过加好友的方式添加其他企业的联系人或微信联系人进行沟通。

2、外部联系人名称后会带有@后缀，同企业内账号则不会有@后缀显示。

3、添加外部联系人后，可以进行单人聊天或者多人群聊。目前已支持文本、图片、语音、名片、文件、聊天记录、地理位置、收藏、H5、小程序的收发等。
