# 如何在广告落地页中使用「获客助手」

可在腾讯及其他广告平台的落地页中使用，有两种模式：
模式一：广告落地页一键加好友。链路短、转化高，有效提升获客转化。
模式二：表单提交后加好友。不改变原有广告转化链路的基础上，零成本增加新触点，放大投放效果。

在广告落地页中使用可查看[配置指引](https://work.weixin.qq.com/nl/act/p/92d55c1116c44db9)。
