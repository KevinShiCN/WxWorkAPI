# 「获客助手」添加客户后自动扣款

### [](https://open.work.weixin.qq.com)一、什么是自动扣款

4.1.22版本开始，除家校行业和政务行业，其他企业的获客助手支持添加客户后自动扣款（即用即付），无需提前预估使用量，使用更安心。

### [](https://open.work.weixin.qq.com)二、如何开启/关闭自动扣款

#### [](https://open.work.weixin.qq.com)1、如何开启自动扣款

在获客助手页面中点击“购买”，管理员可以选择“成功添加客户后，自动扣款”进行开通，开通后，企业需要到高级功能->[企业充值](https://open.work.weixin.qq.com/help2/pc/20897)中操作充值，把钱充到企业微信钱包里，之后每自动添加一个好友就会立刻从余额里扣一次款。
![](https://wework.qpic.cn/wwpic3az/893041_GJt1m9MiQWW8VOJ_1753672672/0)
![](https://wework.qpic.cn/wwpic3az/491716_NRjlNfetSu2lHjm_1753672789/0)

#### [](https://open.work.weixin.qq.com)2、如何关闭自动扣款

在获客助手页面中点击“购买”，管理员可以选择“成功添加客户后，自动扣款”进行关闭自动扣款服务。
关闭后，通过获客链接添加时:
（1）将不再通过企业余额自动扣款；
（2）若企业未购买使用量，客户将无法继续添加成员。
![](https://wework.qpic.cn/wwpic3az/681664_cKPzY1xCQHiy0qV_1753672927/0)

### [](https://open.work.weixin.qq.com)三、如何操作[企业充值](https://open.work.weixin.qq.com/help2/pc/20897)

**1、充值路径**
【管理后台->高级功能->企业充值】。
![](https://wework.qpic.cn/wwpic3az/469037_TatvNws6SgieodQ_1753673175/0)

**2、充值方式**
支持通过对公转账汇款或在线充值（微信支付、QQ钱包）的方式完成充值。
**3、成员申请开通**
成员支持联系管理员申请开通；点击「余额支付」出现弹窗，告知暂无余额支付的权限，需要找管理员开通权限。
**4、计费方式**
开通自动扣款后，计费方式为实时扣款。

### [](https://open.work.weixin.qq.com)四、常见问题

#### [](https://open.work.weixin.qq.com)1、如何开具发票

自动扣款后，可在【手机端企业微信>左上角三个短横线>左下角高级功能>企业/我的订单>订单记录>选择对应订单>申请发票】。
注：需消耗之后才可以开发票。暂不支持自定义开票金额，会直接根据企业当前消耗的使用量的情况去进行开票。

#### [](https://open.work.weixin.qq.com)2、如何申请自动扣款合同

暂不支持申请双方签章的充值合同。目前只有《[腾讯企业微信充值服务协议](https://work.weixin.qq.com/nl/act/p/95a9be4f05624098)》。

#### [](https://open.work.weixin.qq.com)3、如何查看获客助手使用情况

开通自动扣款后，支持在【管理后台->客户与上下游->客户联系->获客助手】或【管理后台->安全管理->获客助手】中，查看获客助手使用概况。

#### [](https://open.work.weixin.qq.com)4、已购买过获客助手，开通自动扣款后，如何计算使用量

若企业之前已购买过获客助手且还有剩余的使用量，此时开通自动扣款，会优先已购买的剩余使用量后，再消耗自动扣款的使用量。
剩余的使用量是受到使用时长限制的，购买获客助手后，使用量有效期为90天，过期失效。请合理评估需求，避免浪费。
