# 「获客助手」的价格

### [](https://open.work.weixin.qq.com)一、「获客助手」价格

企业按需购买使用量，使用量为可通过获客链接成功添加的客户数量，不是获客链接数量，也不是链接的曝光或点击量。

内测期间，每个企业可免费领取50次使用量，用于测试使用，使用量限时优惠价格为1元/次，内测结束后的正式定价待公布。

### [](https://open.work.weixin.qq.com)二、「获客助手」价格调整说明

**自2026年3月27日(含)起，获客助手的限时优惠价格在部分经营类目下将进行调整，企业届时可在「高级功能-获客助手」中查看最新价格。**

**以上变化，从2026年3月27日(含)起生效，特提醒您提前做好规划。在2026年3月27日之前的待支付订单，在订单有效期内支付，不受以上调整影响。企业已购买的使用量不受以上调整影响。**

### [](https://open.work.weixin.qq.com)三、关于价格调整的常见问题

1、什么是「获客助手」？
「获客助手」是企业微信推出的帮助企业线上获客的高级功能。具有无需扫码，客户点链接可 1 步添加、可用于各类广告场景，支持广告数据回传等产品亮点。

2、「获客助手」是如何收费的？
获客助手是一个企业自愿选用的高级功能，企业可按需采购。
自2026年3月27日(含)起，获客助手的限时优惠价格将进行调整，企业届时可在「高级功能-获客助手」中查看最新的限时优惠价格，由于企业的获客需求有所差异，不同经营类目的企业获客助手的功能、单价可能会存在差异，以下单页展示为准。
此外，企业微信为每个企业免费提供50次使用量，供企业试用。若企业评估效果优，可按需采购。

3、不用「获客助手」，可以添加客户吗？
可以。企业可继续使用二维码、搜索手机号等方式[添加客户](https://open.work.weixin.qq.com/help2/pc/20169)。

4、为什么「获客助手」的价格变了？
自2026年3月27日(含)起，获客助手的限时优惠价格在部分经营类目下将进行调整，企业届时可在「高级功能-获客助手」中查看最新价格。
以上变化，从2026年3月27日(含)起生效，特提醒您提前做好规划。在2026年3月27日之前的待支付订单，在订单有效期内支付，不受以上调整影响。企业已购买的使用量不受以上调整影响。

5、为什么我的企业看到的价格跟其他企业不一样？
由于企业的获客需求有所差异，不同经营类目的企业获客助手的功能、单价可能会存在差异，以下单页展示为准。

6、为什么我没有收到定价预告？是不是意味着我的价格不变？
调价预告会面向调价范围的经营类目的企业下发，是否收到影响以下单页展示为准，自2026年3月27日(含)起，企业可在「高级功能-获客助手」中查看最新的限时优惠价格。

7、每个类目的价格是多少/功能是什么？
不同经营类目的企业获客助手的功能、单价可能会存在差异，以2026年3月27日(含)下单页展示为准。

8、你们为什么不对外公布每个类目的价格？
获客助手功能处于内测期，在企业的积极参与和反馈中，产品能力不断完善，感谢您的反馈，我们会逐渐完善。
