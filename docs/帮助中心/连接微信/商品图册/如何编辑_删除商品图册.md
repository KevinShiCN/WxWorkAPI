# 如何编辑/删除商品图册

### [](https://open.work.weixin.qq.com)一、管理员如何编辑/删除共享使用的商品图册

超级管理员可在【[管理后台](https://work.weixin.qq.com)->客户与上下游->客户联系->聊天工具->商品图册配置->成员的商品图册->管理】找到对应商品图册进行编辑修改或删除图册。
![](https://wework.qpic.cn/wwpic3az/441879_Oq-m0P-FRvSvG7A_1753339771/0)
![](https://wework.qpic.cn/wwpic3az/501303_z83Cw5GkTsGzkxb_1753339774/0)
![](https://wework.qpic.cn/wwpic3az/119230_be4817DXRquIUyk_1753339777/0)
超级管理员/商品图册负责人，还可通过进入【工作台->客户联系/客户群->商品图册->成员的->已配置的商品图册】，点击某个图册之后继续点右上角进行编辑/删除，也可以在“已配置的商品图册”页面点右上角“选择”进行批量删除。
![](https://wework.qpic.cn/wwpic3az/906799_LqnJyw-7QO2JMxE_1765353329/0)

### [](https://open.work.weixin.qq.com)二、成员如何编辑/删除仅自己使用的商品图册

成员可在【工作台->客户联系/客户群->商品图册】，点击某个图册之后继续点右上角进行编辑/删除，也可以在“我的商品图册”页面点右上角三个点选择“编辑图册”进行批量删除。
![](https://wework.qpic.cn/wwpic3az/327037_6XuYZvoRTU6m0QS_1765353355/0)
