# 查看好友

### [](https://open.work.weixin.qq.com)一、 外部好友查看入口

电脑端或手机端【通讯录->外部联系人】 可查看已添加的外部联系人。
![](https://wework.qpic.cn/wwpic3az/988609_DRqBvFOySDusUWW_1765177702/0)

### [](https://open.work.weixin.qq.com)二、 如何设置外部好友备注

在查看外部联系人个人信息页面，支持给好友设置备注和描述。
![](https://wework.qpic.cn/wwpic3az/773390_6naN9xC3S3Sr3jP_1760941670/0)
