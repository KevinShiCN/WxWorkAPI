# 「外部联系人规模」计费人数的计算方法？

外部联系人规模计算方法是包含了企业内成员已添加好友和群聊没有添加好友的客户数合计，计算时系统已经做了去重。
外部联系人规模购买只需要参考「已服务X位」（企业已服务的外部联系人）即可。

查看「企业已服务的外部联系人」，可通过手机端企业微信，点击「工作台」>「客户群」或「客户联系」>左上角「外部联系人规模：XXX位」>中间「已服务XX位」。
