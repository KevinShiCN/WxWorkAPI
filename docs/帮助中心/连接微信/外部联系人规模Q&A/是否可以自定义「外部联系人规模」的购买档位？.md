# 是否可以自定义「外部联系人规模」的购买档位？

外部联系人规模的购买档位是固定的，无法购买当前显示档位以外的规模。可以根据公司当前已添加的客户加上未来一年内即将添加的总和客户数来进行评估。

注：高档位对低档位是包含关系，不是叠加关系。（例如您已经购买到了5W档位之后又购买到了20W档位，您的总量是20W。）

可在【手机端企业微信->点击左上角”≡”->左下角高级功能->外部联系人规模->了解更多->购买】查看档位。
