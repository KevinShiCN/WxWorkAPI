# 账号封禁的成员，名下的客户是否还在企业的外部联系人规模中？

客户是企业的重要资产。成员被封禁后，客户仍会在企业的外部联系人规模中。

如果成员被「短期封禁」，建议在封禁期间及时调整运营策略，封禁结束后，即可继续联系和服务客户；如果成员被「长期封禁」，将无法继续联系和服务客户，企业可按需选择是否将对应的客户和客户群从企业的外部联系人规模中移除。
