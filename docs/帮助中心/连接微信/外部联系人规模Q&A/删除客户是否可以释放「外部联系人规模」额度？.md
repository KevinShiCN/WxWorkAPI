# 删除客户是否可以释放「外部联系人规模」额度？

### [](https://open.work.weixin.qq.com)一、如何释放「企业已服务的外部联系人」额度？

（1）企业成员主动删除客户**可以释放名额占用**。若多个成员添加了同一个客户，则需所有成员把这个客户删了才能释放额度。
例如：A客户添加了B、C俩个企业成员，则需要B、C都操作删除了A客户才会释放额度。
温馨提示：删除客户或成员后不支持恢复，建议您谨慎操作。

（2）加入群聊的客户/外部联系人，在对应企业的客户群或外部群全部退出/群主把人员移出群聊/群主解散了对应客户群后，可释放客户数额度。
例如：A客户加入了企业的D、F俩个群，则需要A客户在D、F群都退出或者群主解散了这两个群了才会释放额度。

注：客户单方面删除企业成员好友不会释放；企业成员在通讯录被删除（离职），也不会释放额度。

### [](https://open.work.weixin.qq.com)二、常见问题

#### [](https://open.work.weixin.qq.com)1、释放额度是实时更新的吗？

释放额度一般是实时更新，若企业客户数量较多，会存在延迟10分钟左右。

#### [](https://open.work.weixin.qq.com)2、如何释放离职成员占用的外部联系人规模名额？

客户是企业的重要资产。成员离职后，客户仍然在企业的外部联系人规模中。建议通过离职继承功能将客户分配给其他员工，继续提供服务。

若客户单方面删除了离职成员，操作离职继承该客户将会继承失败，当对应客户没有加入其他企业的客户群、没有添加其他企业成员这个客户会自动从企业的外部联系人规模中移除。

非客户身份的外部联系人，对应成员在通讯录被删除后，外部联系人还是会占用外部联系人规模的名额，不会释放外部联系人规模的使用名额。

#### [](https://open.work.weixin.qq.com)3、如何查看/删除单删好友的客户？

客户单方面删除企业成员好友后，成员可在【手机端->通讯录->我的客户->全部客户】选择单向客户进行筛选查看，可点击【右上角“编辑”->删除】删除单向好友。支持批量删除单向好友，单次删除上限为50个。

#### [](https://open.work.weixin.qq.com)4、离职成员离职前，客户群/离职成员账号被封禁了，这些客户占用了外部联系人规模名额如何释放？

目前在成员离职时，如果离职成员的客户群/离职成员账号被封禁的情况，系统会在客户端【消息->离职继承】中给超级管理员和业务负责人推送卡片提醒，点击卡片后可以把这些封禁的群从企业客户群/封禁成员的客户从企业客户列表移除，移除这些群/客户后会释放对应外部联系人规模的名额。

注：这个卡片会在成员离职瞬间立即推送。
