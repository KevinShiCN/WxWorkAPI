# 「外部联系人规模」购买量是全企业可使用还是仅限购买人？

外部联系人规模购买后，默认全企业成员可用，如果需要控制范围，管理员可在【管理后台->我的企业->外部沟通管理】中限制成员对外联系权限。

注：不在对外联系权限范围内的成员，将不支持添加外部联系人、发起外部群、对外部联系人发单聊消息。
