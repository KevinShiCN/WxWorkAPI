# 外部沟通权限管理

### [](https://open.work.weixin.qq.com)一、什么是外部沟通权限

企业管理员可设置外部沟通成员使用权限，名单内的成员可添加微信用户或企业微信用户为外部联系人。

### [](https://open.work.weixin.qq.com)二、如何设置外部沟通使用权限

手机端：
【工作台->管理企业->外部联系人权限设置】
![](https://wework.qpic.cn/wwpic3az/449369_D8DxRb1CRuuv63X_1765178241/0)
[管理后台](https://work.weixin.qq.com)：
【我的企业->外部沟通管理->对外联系权限】
![](https://wework.qpic.cn/wwpic3az/766596_zp7z1tQQTCaB9ha_1753320461/0)

### [](https://open.work.weixin.qq.com)三、如何设置加好友需验证/无需通过好友验证

在【手机端->左上角“三”->设置->联系人权限->加我为联系人时】可开启/关闭“需要验证”，关闭“需要验证”时，可选择是否自动添加对方到通讯录。

对于以前已开启无需验证的用户，首次打开和单向好友的聊天界面时，会出现弹窗提醒免验证可以自动加对方好友。
![](https://wework.qpic.cn/wwpic3az/543877_xSa_3xnoSQeFT8R_1765174649/0)
