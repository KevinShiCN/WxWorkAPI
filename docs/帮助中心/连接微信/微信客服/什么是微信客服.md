# 什么是微信客服

1分钟认识微信客服，[点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488325441827073)

[「微信客服」](https://work.weixin.qq.com/kf/)由腾讯微信团队为企业打造，用于满足企业的客服需求，帮助企业做好客户服务。企业可以在微信内、外各个场景中接入微信客服，用户可以在微信里免加好友发起咨询。
