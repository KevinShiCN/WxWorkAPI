# 如何使用微信客服聊天工具栏

### [](https://open.work.weixin.qq.com)一、微信客服聊天工具栏的作用

微信客服聊天工具栏类似客户联系聊天工具栏，企业配置好聊天工具栏后，方便接待人员在接待过程查看使用。

### [](https://open.work.weixin.qq.com)二、设置微信客服聊天工具栏

设置路径：管理员或负责人支持在【管理后台->应用管理->微信客服->聊天工具栏】进行设置。
![](https://wework.qpic.cn/wwpic3az/803025_XcsjAU01TCqS7rj_1753688050/0)

或手机端【工作台->微信客服->聊天工具栏->配置接待人员的工具栏】入口中进行配置。
![](https://wework.qpic.cn/wwpic3az/21476_p6EkXvkWTquFWpQ_1765421856/0)

### [](https://open.work.weixin.qq.com)三、使用微信客服聊天工具栏

1、接待人员支持在手机端【工作台->微信客服->聊天工具栏->我的聊天工具栏】进行添加自己使用的工具栏，也可以自定义工具栏排序。
![](https://wework.qpic.cn/wwpic3az/409308_DAy0ZcahTRmjFq7_1765421863/0)
2、自带三个工具栏：「升级服务」、「快捷回复」、「客户详情」，可停用，不可删。
![](https://wework.qpic.cn/wwpic/819345_iafSDor1Ska-7KT_1670897002/0)
4、接待人员可以在工具栏的“客户详情”内查看到客户来源，包括可以查看客户具体是从哪个视频号、小店或App发起咨询的。
![](https://wework.qpic.cn/wwpic3az/400549_feCbO0m-QOW48D9_1765421879/0)
3、接待人员可以在工具栏的「客户详情」内为客户添加描述，支持添加20个字。全企业支持添加客户描述的上限是50W条。
后续客户再次接入时，将会在会话界面推送“接待人员已给该客户添加了描述”，点击即可查看前坐席添加的内容。

注：需要接入相同客服账号的才能看到，其他客服账号的描述将不会同步。
![](https://wework.qpic.cn/wwpic3az/999128_sGTlkVgESmqstzS_1765421885/0)
4、成员需在对应应用可见范围内，才可见聊天工具栏配置的应用。
