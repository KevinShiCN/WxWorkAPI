# 如何在微信中搜索到我的企业微信账号

目前企业微信暂时没有“企业微信号”或“微信号”。微信用户支持在微信中通过搜索成员在企业微信中绑定的手机号的方式搜索到企业微信用户。

1、成员需要允许别人找到当前企业身份，请在企业微信侧【手机端->消息页面左上角“三”->设置->联系人权限->打开允许别人找到当前企业身份】。
![](https://wework.qpic.cn/wwpic3az/332390_luOzo_Z-QJSYO2U_1764925835/0)
2、在微信侧，请确保微信已更新到最新版本。可在手机端微信->我->设置->关于微信中查看版本。
3、上述操作确认完成后，在微信端【通讯录->右上角添加朋友->企业微信联系人】，即可通过手机号搜索到企业微信用户。
![](https://wework.qpic.cn/wwpic/777747_bPopwTN5S_GbDFt_1659928699/0)
