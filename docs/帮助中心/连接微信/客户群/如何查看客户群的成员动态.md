# 如何查看客户群的成员动态

在【手机端/电脑端企业微信->工作台->客户群】中可查看最近有新增客户群的10位同事（头像、姓名、客户群数、群成员数）。
![](https://wework.qpic.cn/wwpic3az/807336_pYtgk1foShC3yuE_1765172261/0)

管理员还可以在【手机端企业微信->工作台->客户群->右上角的三个点->显示成员动态】中自定义开启或关闭显示。开启后，成员可以在客户群中查看到最近创建了客户群的同事。
![](https://wework.qpic.cn/wwpic3az/914932_lJ66VGzWSMSmWvt_1765172353/0)
