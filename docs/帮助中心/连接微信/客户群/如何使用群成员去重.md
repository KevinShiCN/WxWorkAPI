# 如何使用群成员去重

如何用“去重”功能筛选重复入群者？[点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488326713827072_4)

### [](https://open.work.weixin.qq.com)一、群成员去重操作入口

【管理后台->客户与上下游->客户联系->安全管控->群成员去重】支持按群主筛选或按群名包含的关键词筛选去重。
![](https://wework.qpic.cn/wwpic3az/870322_TiGc3JYpS3WL4N3_1753327013/0)
【手机端->工作台->客户群->群成员去重】
![](https://wework.qpic.cn/wwpic3az/770068_w8Zi700yRneS0mw_1765172472/0)

### [](https://open.work.weixin.qq.com)二、支持筛选关键词给对应的群成员去重

超级管理员、业务负责人支持根据群主和群名关键词筛选最近活跃的2000个客户群中是否有同一个客户，筛选后选择要踢出该客户的客户群通知服务人员。

### [](https://open.work.weixin.qq.com)三、管理员点击了去重后成员侧需点击确认去重

成员在【客户端->消息栏->客户群】收到通知去重，点击进入后可以操作移出客户。
![](https://wework.qpic.cn/wwpic3az/755862_g6Ds0luFQWKAMzV_1765172506/0)

### [](https://open.work.weixin.qq.com)四、常见问题

1、群聊群客户去重，同时选择多个部门去重拉取到的重复数比分别选择多个部门拉取到的重复数要少。
目前每次只能对最近活跃的2000个群去重；若选择的范围较大，导致群数量超过2000，则范围内会有部门群聊无法去重，建议缩小范围分批进行操作。
