# 如何使用离职继承分配成员的客户群

### [](https://open.work.weixin.qq.com)一、什么是离职继承分配客户群

若在成员[创建了客户群](https://open.work.weixin.qq.com/help2/pc/14732)，群主离职的情况下，客户群需由管理员分配新群主（新群主需是同企业人员且是客户联系和客户群的使用范围内的成员），若客户群群主离职，管理员未分配新的群主，则这个群的群主不会自动转让群主。

### [](https://open.work.weixin.qq.com)二、离职继承分配路径

管理端入口：【管理后台->客户与上下游->客户联系->离职继承】，选择需要查看的离职人员，点击待分配的群聊。
![](https://wework.qpic.cn/wwpic3az/349804_anS-2rfKTY2N_b8_1753323421/0)
![](https://wework.qpic.cn/wwpic3az/754277_bf-McetGQy-zL3t_1753323243/0)
手机端入口：管理员/负责人可在【手机端->工作台->离职继承】操作分配。管理员可以分配所有离职人员的客户群，负责人只能分配自己管理范围内离职人员的客户群。
![](https://wework.qpic.cn/wwpic3az/50295_InkAlR1kRzSxef6_1761029494/0)

电脑端入口：【工作台->客户群->去分配】，仅客户联系负责人可在电脑端操作分配，若是超级管理员，仅可查看“我继承了离职成员的”客户/客户群。
![](https://wework.qpic.cn/wwpic3az/749941_CEoIy7LIRbCuLMW_1765332037/0)

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、成员已在通讯录被删除，但是已分配离职成员资产找不到这个离职成员，无法分配客户和客户群。

确认已在通讯录中删除成员，但在已分配离职成员资产找不到成员，原因是：
（1）已分配离职成员资产页面中展示的离职成员姓名目前仅展示成员离职前在通讯录的姓名，若设置了别名，则不会显示别名。而操作日志会展示姓名（别名）。因此会有可能误以为不是同个人。
（2）离职成员在离职前把客户群解散了、把客户删除了，则不会出现可分配列表中。

#### [](https://open.work.weixin.qq.com)2、分配离职成员群聊，提示当前分配群聊数量超过上限。

目前单次分配离职成员的群聊数量上限为500个，当分配的数量超出上限时，在分配页面可一键自动选择500个客户群分配（仅手机端支持，管理端暂不支持）。如需分配的群聊数量超过500个，建议分次操作分配。

#### [](https://open.work.weixin.qq.com)3、离职继承中“账号已注销”的成员无法离职继承，选项框置灰。

目前账号已注销的情况下暂不支持进行客户群离职继承。（离职继承客户不受影响）

#### [](https://open.work.weixin.qq.com)4、离职继承可以自动分配吗

暂不支持自动分配，需要管理员/负责人在成员离职后手动操作分配。

#### [](https://open.work.weixin.qq.com)5、分配离职成员的客户群时，待分配的群聊里不显示群聊

目前仅离职成员是群主的客户群支持分配，请确认离职成员是否有是群主的客户群。

#### [](https://open.work.weixin.qq.com)6、分配离职成员的客户/客户群，提示“XXX离职较久或离职前太久未登录企业微信，其客户群已无法被分配”

由于离职成员太久未登录企业微信，目前暂时不支持分配这类不活跃用户的群聊，我们将在后续优化该情况。
建议让离职成员使用企业微信账号历史绑定的手机号/微信号登录企业微信APP后再尝试分配。
