# 如何给好友添加标签

员工在【通讯录->外部联系人->标签】中至少新建一个标签后，即可开启个人标签，给外部联系人打上个性化标签。
![](https://wework.qpic.cn/wwpic3az/278527_gozPDj2uT6yZm89_1765354499/0)

也可以在外部联系人个人信息中给其打上标签。
![](https://wework.qpic.cn/wwpic3az/164754_DTjyD5RmRbK7bmz_1765354516/0)
