# 如何设置客户朋友圈权限

### [](https://open.work.weixin.qq.com)一、设置朋友圈使用权限

企业成员需要有朋友圈使用权限才能在手机端使用朋友圈功能。

超级管理员在【[管理后台](https://work.weixin.qq.com)->客户与上下游->客户联系->权限配置->朋友圈使用范围】
![](https://wework.qpic.cn/wwpic3az/924345_AeQXfYxISLSqIll_1753669154/0)
或【手机端->工作台->客户朋友圈->配置->配置使用范围和管理规则】入口给成员添加朋友圈的使用权限。
![](https://wework.qpic.cn/wwpic3az/768826_ICOF49YgQpScNFi_1765354359/0)

### [](https://open.work.weixin.qq.com)二、设置朋友圈负责人

1、客户朋友圈管理规则设置数量上限是2000个。
2、负责人管理权限可设置：查看成员全部的发表、发表到成员客户的朋友圈、配置封面和签名素材库。
![](https://wework.qpic.cn/wwpic3az/930435_JvFaxOcfRwCKOf4_1753669178/0)
3、设置入口：
超级管理员在【[管理后台](https://work.weixin.qq.com)->客户与上下游->客户联系->权限配置->朋友圈管理规则】
![](https://wework.qpic.cn/wwpic3az/408126_4pe6IMr-SZWBoc9_1753669199/0)

或【手机端->工作台->客户朋友圈->配置->配置使用范围和管理规则】入口中设置朋友圈负责人。
![](https://wework.qpic.cn/wwpic3az/493062_dauc0eI2RTqtBxe_1765273768/0)

### [](https://open.work.weixin.qq.com)三、关闭成员的客户朋友圈入口

超级管理员可在【手机端->工作台->管理企业->工作台显示->成员可见->客户联系】关闭，关闭后，仅超级管理员和具有使用权限的成员可见客户朋友圈入口、无权限的成员将无法在工作台上看到该入口。
![](https://wework.qpic.cn/wwpic3az/498524_XcXu-03tSV2qthm_1765273720/0)
