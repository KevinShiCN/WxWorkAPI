# 如何设置对外字段信息

如何设置头像、别名、联系方式？[点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488324188827072)

### [](https://open.work.weixin.qq.com)一、设置对外字段信息

企业成员可以选择是否对外展示自己的手机号、邮箱、职务、地址；对于其他[自定义的文本、网页、小程序](https://open.work.weixin.qq.com/help2/pc/17428)类型字段信息，需要联系管理员进行设置。

设置入口：【手机端->消息页面左上角“≡”->头像->个人信息->对外信息显示】
![](https://wework.qpic.cn/wwpic3az/712173_Nuys5dvlRjqU6C4_1765353521/0)

### [](https://open.work.weixin.qq.com)二、常见问题

1、在手机端设置对外信息显示是提示“管理员已设置该信息无法对外显示，如需显示，请联系管理员处理”
该提示是因为管理员设置了对应的信息不对外展示。
可联系管理员在【[管理后台](https://work.weixin.qq.com)->我的企业->通讯录管理->对外资料显示->修改->配置成员展示信息】勾选对应的信息后保存。设置后可在手机端重试。

2、修改对外信息显示后未生效
设置了对外信息后，外部联系人可重新点击查看企业微信联系人的个人信息，触发更新，若没及时更新可以等待96小时后重新查看。
