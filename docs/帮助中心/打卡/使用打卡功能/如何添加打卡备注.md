# 如何添加打卡备注

### [](https://open.work.weixin.qq.com)一、上下班打卡规则

1、若打卡规则**未开启允许补卡**，那么当打卡有异常时，可以通过点击打卡页面的「查看原因」按钮进行添加备注，打卡备注支持100字。
![](https://wework.qpic.cn/wwpic3az/805169_zCgsp2gAQV2h6ux_1765441686/0)
2、若打卡规则有**开启允许补卡**，可以进入打卡应用->统计->打卡记录->点击需要添加备注的记录/处理异常->添加备注。
![](https://wework.qpic.cn/wwpic3az/73444_AJMJlgLzTZmmfO7_1765441766/0)

### [](https://open.work.weixin.qq.com)二、外出打卡规则

成员可以在打卡之前点击「添加备注」进行添加，编辑好内容之后再进行打外出卡；
可以进入打卡应用->统计->打卡记录->点击需要添加备注的记录->添加备注。
![](https://wework.qpic.cn/wwpic3az/470456_10Pd577ASvSXy0J_1765441693/0)
