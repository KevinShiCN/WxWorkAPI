# 「审批打卡」使用介绍

### [](https://open.work.weixin.qq.com)一、什么是审批打卡

当成员因定位不准、系统识别异常等问题带来无法打卡时，可及时提交一个审批打卡审批单，管理员审批通过后打卡结果即可生效。
![](https://wework.qpic.cn/wwpic3az/75011_ZuJQf7npQyOPelr_1765426100/0)

### [](https://open.work.weixin.qq.com)二、如何使用审批打卡

1、打卡规则中配置：可在打卡规则中「补卡请假」设置中，开启/关闭审批打卡的开关。

设置入口：
【[管理后台](https://work.weixin.qq.com)->应用管理->打卡->设置规则->选择已有的打卡规则/添加规则->编辑->审批打卡】
![](https://wework.qpic.cn/wwpic3az/784096_-C7yRJhmStOsuoe_1721967744/0)
【手机端->工作台->打卡->管理->选择已有的打卡规则/添加规则->编辑->补卡和请假设置->审批打卡】
![](https://wework.qpic.cn/wwpic3az/11165_GwAcGwc4SNCcFkZ_1721982769/0)
2、成员打卡时触发：开启后，成员遇到范围外无法打卡、非常用设备禁止打卡、地点异常禁止打卡时，将记录其打卡时间、打卡地点、拍照备注、打卡备注等信息，提交审批打卡。
![](https://wework.qpic.cn/wwpic3az/440158_LbK_2dL4RgSJbcV_1721982780/0)
3、审批通过后生效：成员提交的审批打卡审批单，需管理员审批通过后其打卡结果才可以生效。
![](https://wework.qpic.cn/wwpic3az/668643_7oJC6wtlTrKn0eY_1765426108/0)

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、审批打卡可以从「打卡-申请」、「审批」应用中提交吗？

审批打卡开启后，仅可从范围外无法打卡、非常用设备禁止打卡、地点异常禁止打卡这几个特定情况进入提交；成员无法从「打卡-申请」、「审批」应用中提交审批打卡审批单。

#### [](https://open.work.weixin.qq.com)2、如何管理审批打卡审批模板？

（1）审批打卡是由「审批」应用提供的模板，可在「审批」应用中管理模板的申请记录、模板设置和规则设置；
（2）打卡时间、打卡班次、打卡地点、拍照备注、打卡备注字段也不可更改/删除，但企业可按需调整增加模板字段.

#### [](https://open.work.weixin.qq.com)3、未启用审批应用，还可以提交审批打卡吗？

目前没有启用审批应用，也可以发起审批打卡的。

#### [](https://open.work.weixin.qq.com)4、一个审批打卡申请单，可以打多次卡？

一张审批打卡单，只能打一次卡。
