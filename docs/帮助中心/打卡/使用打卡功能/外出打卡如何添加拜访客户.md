# 外出打卡如何添加拜访客户

v5.0.6版本开始，外出打卡时成员可添加拜访客户，支持水印拍照并汇总外出轨迹，管理员可查看团队外出情况统计，销售拜访、日常巡店等外出场景更方便。

1、外出打卡可添加拜访客户、水印拍照、打卡备注
路径：手机端->工作台->打卡->外出打卡
![](https://wework.qpic.cn/wwpic3az/834755_wU2PraR1Th-sLxC_1770713355/0)

2、支持查看外出打卡轨迹
查看路径：手机端->工作台->打卡->统计->外出打卡->选择对应的成员->打卡轨迹
![](https://wework.qpic.cn/wwpic3az/25581_Df3PEbYYRryQaez_1770713659/0)

3、外出统计优化，团队统计支持查看已外出人数，外出详情可查看客户等信息
注：统计页外出打卡部分，会露出三个外出成员的预览。

查看路径：手机端->工作台->打卡->统计->右上角可选择日、周、月->外出打卡->选择对应的成员->在外出详情页面点击客户，可查看客户信息。
