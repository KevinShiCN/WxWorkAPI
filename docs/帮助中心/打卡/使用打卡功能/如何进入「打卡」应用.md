# 如何进入「打卡」应用

### [](https://open.work.weixin.qq.com)一、操作路径

1、登录手机端企业微信后，进入工作台，点击打卡应用，可选择上下班打卡或外出打卡。
![](https://wework.qpic.cn/wwpic3az/907429_gGsTnkinTqillM4_1765426394/0)

2、在打卡界面会有打卡使用指引入口，点击后可以看到打卡功能的一些重要能力的介绍，企业可点击了解如何管理团队打卡。
![](https://wework.qpic.cn/wwpic3az/970982_rd4mWyGBR8GBJft_1765426398/0)

3、登录电脑端企业微信后，进入工作台，支持进入打卡应用。目前该入口仅做展示使用。
（1）超级管理员、分级管理员进入打卡应用首页，展示管理功能，可点击对应功能跳转到后台对应设置页面；
（2）其他人员则展示对应所属规则和当天打卡时间，实际需前往手机端打卡应用操作打卡，若对应成员无所属打卡规则，则会提示暂无打卡规则。
![](https://wework.qpic.cn/wwpic3az/731107_vcFj7nZ-TiaCg3C_1749799889/0)
![](https://wework.qpic.cn/wwpic3az/392404_paL1DeLQSTGjuut_1749800009/0)

### [](https://open.work.weixin.qq.com)二、常见问题

1、电脑端支持打卡吗？
因为打卡大部分企业需要地理位置打卡，手机才支持位置上报，所以目前只能在手机端操作打卡。
