# 打卡提示“今天不上班，好好休息”

1、当成员所属的打卡规则中工作日未勾选这一天或未在排班上下班中给成员设置当天的班次。则当天成员不需要打卡，在打卡页面会出现该提示。

2、若打卡规则勾选了「[同步法定节假日](https://open.work.weixin.qq.com/help2/pc/17508)」，同步中国节假日则节假日期间不用打卡。

3、若在打卡规则[特殊日期](https://open.work.weixin.qq.com/help2/pc/19698)中，当天被设置为「不用打卡的日期」则无需打卡。

4、成员当日所属打卡规则/[班次的打卡时段设置/打卡时间限制](https://open.work.weixin.qq.com/help2/pc/17505)，若未勾选上班需要打卡/下班需要打卡。关闭后，系统将自动在该时间点打卡，员工打卡状态显示为「无需打卡」。

5、请让成员在【手机端的消息界面，点击左上角“≡”】确认登录的企业与管理员设置打卡规则的企业是否为同一企业。
