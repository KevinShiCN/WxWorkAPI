# 打卡提示“当前时间不能打卡”

1、需要检查问题人员所在规则是否有设置[打卡时段限制](https://open.work.weixin.qq.com/help2/pc/17505)。
管理员可在【打卡规则->打卡时间->打卡时段】中查看打卡时间限制，当成员不在可打卡时间范围内点击打卡会出现这个提醒。
成员也可以自行点击“查看今日上下班安排”->点击打卡规则，查看可打卡时间。
![](https://wework.qpic.cn/wwpic3az/209791_Nqr2m5rJRveKgZo_1765771677/0)

2、排班上下班规则，未到可打卡时间会提醒“当前时间不能打卡”。成员可以点击“查看今日上下班安排”->点击打卡规则，查看对应班次的可打卡时间。

![](https://wework.qpic.cn/wwpic3az/478802_6ymPlMgRT5SnRyN_1765771515/0)
