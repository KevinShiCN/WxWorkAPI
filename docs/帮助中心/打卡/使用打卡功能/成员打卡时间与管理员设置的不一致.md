# 成员打卡时间与管理员设置的不一致

可能成员以单个成员和所属部门的形式，被设置了多个打卡规则，但生效的打卡规则不符合预期。

调整生效规则的办法：
1、在希望生效的规则中，以单个成员的形式重新添加该员工；
2、在不希望生效的规则中，将该员工加入到白名单。
