# 如何使用统计字段管理

### [](https://open.work.weixin.qq.com)一、功能介绍

v4.1.36版本开始，打卡统计支持管理员新增、自定义考勤统计字段，数据统计更方便，可管理上下班打卡、外出打卡统计字段的计算规则、单位和取整方式。

1、**不可编辑字段**：比如时间、姓名、部门、所属规则等，只能看到“查看”入口；
![](https://wework.qpic.cn/wwpic3az/481833_mY2Isj_wTCSQ4EY_1746697709/0)
2、**跳转编辑字段**：比如加班统计方式，可跳转打卡规则配置中进行管理；
![](https://wework.qpic.cn/wwpic3az/317693_j4yPqVIXQBy5SXX_1746697713/0)
3、**直接编辑字段**：（共6个）应出勤天数、实际出勤天数、实际工作时长、迟到时长、旷工时长、早退时长；
![](https://wework.qpic.cn/wwpic3az/271251_gfXJT62-QSOKgJ5_1746697716/0)
4、**历史数据兼容**：历史数据按照原有的统计方式，规则生效后，实时生效；混合新旧数据时，按照旧数据逻辑展示。

### [](https://open.work.weixin.qq.com)二、设置入口

【管理后台->应用管理->打卡->统计字段管理】
![](https://wework.qpic.cn/wwpic3az/150806_fuuRptSeS7CFyAo_1753767284/0)
![](https://wework.qpic.cn/wwpic3az/946562_5xAWgqQgSt-g0C1_1744597103/0)
或【管理后台->应用管理->打卡->上下班打卡->查看统计->日报/月报汇总->设置报表->统计字段管理】
![](https://wework.qpic.cn/wwpic3az/733622_EXezgiTlT2OtXPh_1744596804/0)
![](https://wework.qpic.cn/wwpic3az/634234_7qMVyyCaQ9qE2_6_1744596433/0)
