# 如何进行打卡拍照

#### [](https://open.work.weixin.qq.com)1、如何设置打卡拍照

可在[打卡规则](https://open.work.weixin.qq.com/help2/pc/17501)拍照与人脸识别中，勾选每次打卡均需拍照，成员点击打卡需要拍照。打卡拍照的照片会显示地址、时间水印。
![](https://wework.qpic.cn/wwpic3az/354332_gxwYj8V5TL-2LM8_1749799035/0)

#### [](https://open.work.weixin.qq.com)2、查看拍照打卡后的照片

打卡成员在【手机端->工作台->打卡->统计->打卡记录->打卡详情】可查看打卡照片。
![](https://wework.qpic.cn/wwpic3az/544662_QiDCCebFTLe2p7u_1766719454/0)
管理员在【手机端->工作台->统计->团队统计->上下班打卡-团队日明细->点击成员查看打卡明细】可查看打卡照片。
![](https://wework.qpic.cn/wwpic3az/514580_08BygM-0TPeyxr0_1766719475/0)
管理员在【[管理后台](https://work.weixin.qq.com)->应用管理->打卡->查看统计->日报汇总->点击成员打卡记录】可查看打卡照片。
![](https://wework.qpic.cn/wwpic3az/603587_G6TvIkMzR_qARcI_1766719493/0)
