# 如何选择打卡规则生效时间

编辑打卡规则后，在保存打卡规则时，可选择规则生效时间：「明日生效」、「立即生效」。

- 若选择了「明日生效」，则修改后的打卡规则会在第二天凌晨4点后生效。
- 若选择了「立即生效」，则当天在修改规则前已经打卡记录有可能会失效，无法统计。（今天之前的记录不影响）
