# 如何设置范围外打卡

打卡规则中支持设置「允许范围外打卡，记录为地点异常」、「允许范围外打卡，记录为正常外勤」、「不允许范围外打卡」。
默认设置：允许范围外打卡，记录为地点异常

**允许范围外打卡，记录为地点异常：**在范围外打卡，则打卡日报记录地点异常。
**允许范围外打卡，记录为正常外勤：**在范围外打卡，则打卡日报记录正常。
**不允许范围外打卡：**则在打卡范围外无法进行打卡。范围外打卡页面提示“不在公司范围内，当前位置不能打卡，请前往打卡地点”。（若开启了审批打卡，成员仍可打卡）
