# 如何设置节假日打卡

在打卡规则中可选择设置是否同步中国节假日，若勾选，法定节假日、调休期间不用打卡，补班日期需打卡。

勾选“中国法定节假日不用打卡”，支持查看法定节假日日历，同步排休或加班配置更清晰。

规则类型「排班上下班」暂不支持设置节假日。

操作路径：
1、管理后台->应用管理->打卡->上下班打卡->设置规则，点击需要设置的打卡规则，在“节假日”中勾选“中国法定节假日不用打卡”。
![](https://wework.qpic.cn/wwpic3az/366874_QzlNmQjtRYGbolB_1764036528/0)

2、手机端->工作台->打卡->管理，点击需要设置的打卡规则，在“节假日与特殊日期”中勾选“中国法定节假日不用打卡”。
![](https://wework.qpic.cn/wwpic3az/978196_iwDsI4IKRHOs2bF_1765770892/0)
