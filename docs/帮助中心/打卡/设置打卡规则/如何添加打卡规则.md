# 如何添加打卡规则

### [](https://open.work.weixin.qq.com)一、添加打卡规则入口

管理员支持在管理后台和手机端添加打卡规则。支持添加1000个打卡规则。
目前暂不支持批量新增/编辑打卡规则。

**操作路径如下：**
1、【[管理后台](https://work.weixin.qq.com)->应用管理->打卡->上下班打卡->设置规则->添加规则】
![](https://wework.qpic.cn/wwpic3az/632437_2kFu9dhqSkCsHkD_1753759941/0)
![](https://wework.qpic.cn/wwpic3az/688966_mFdP3aI0Q2q_nkF_1765510097/0)

2、【手机端->工作台->打卡->管理->上下班规则点击“+”添加打卡规则】
![](https://wework.qpic.cn/wwpic3az/457923_vZKJtg2BQXq14Du_1765510101/0)

### [](https://open.work.weixin.qq.com)二、筛选查看打卡规则

1、支持按照规则名称、部门/成员搜索打卡规则；
2、支持按照考勤规则、创建者筛选打卡规则；
3、支持查看打卡规则对应的打卡人数、创建者；
4、2020年之前创建的打卡规则，会不显示创建者（创建者为空）。
