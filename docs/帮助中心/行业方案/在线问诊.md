# 在线问诊

### [](https://open.work.weixin.qq.com)一、在线问诊入口

针对行业类型是医疗的企业，在管理端->客户联系->联系我功能中新增「通过在线问诊联系」功能。（手机端暂没有配置入口）

### [](https://open.work.weixin.qq.com)二、在线问诊的作用

1、使用「联系我」在线问诊功能，医院可以为居民提供专业的线上咨询筛查服务。医院可以配置提供咨询服务的医务人员范围，通过线上公众号菜单、线下物料海报等方式放置在线问诊入口。
![](https://wework.qpic.cn/wwpic3az/573132_vHzt3UxsSiivD3Q_1753410680/0)

2、患者可通过医院官方发布的问诊渠道 （公众号等 添加医生企业微信，医生通过企业微信与患者微信进行联系沟通。患者结束咨询24小时后，会话将自动断开，释放医疗资源为更多群体提供所需服务。
在线问诊多人二维码，若同一个患者多次扫码会出现同一个医生。
![](https://wework.qpic.cn/wwpic/954000_vxcVGvOnQFKa9kN_1644805903/0)

### [](https://open.work.weixin.qq.com)三、在线问诊二维码数量上限

在线问诊（单人+多人）二维码上限为1000个。
