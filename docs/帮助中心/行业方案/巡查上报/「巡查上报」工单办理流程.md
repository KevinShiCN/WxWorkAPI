# 「巡查上报」工单办理流程

[点击观看视频介绍1](https://v.qq.com/x/page/a32206cpuon.html)
[点击观看视频介绍2](https://v.qq.com/x/page/l3220owlahu.html)
1、 网格员/职能部门处理人在企业微信侧「巡查上报」应用中创建上报，可选择「我已办结」或「上报到网格」两种方式提交：

> 自己已经处理的工单选择「我已办结」，上报单为「已办结」状态，流程结束并关单
> 自己未处理的工单选择「上报到网格」，工单提交至网格中心，等待网格负责人或超管分派，上报单为「待分派」状态

2、网格中心负责人或超管可对上报到网格的「待分派」工单进行办理，可选择「我已办结」或「分派他人」

> 如果本人已办理，选择「我已办结」，此时上报单的状态由「待分派」变更为「已办结」
>  如果本人未办理，选择「分派他人」，此时可指定一位企业内人员作为该上报单的办理人，此时上报单的状态由「待受理」变更为「办理中」

3、 上报单的相关办理人，可将「办理中」的工单变更为「已办结」，或将网格中心分派给自己的工单转交给其他同事办理。当工单状态变为「已办结」时，流程结束并关单，通过应用消息反馈给提交上报的成员
