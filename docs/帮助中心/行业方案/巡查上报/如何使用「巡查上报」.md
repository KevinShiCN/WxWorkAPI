# 如何使用「巡查上报」

成员可将日常巡查中发现的问题进行上报，并由所属网格的负责人统一分配办理。通过流程化的工单分类处理机制帮助政府提升管理效率。该应用是内部应用，仅工单相关成员查看填写，不支持转发到群聊。
目前支持在手机端、win端使用。

### [](https://open.work.weixin.qq.com)一、角色分类

1、成员：网格员、职能部门处理人。
若成员点击巡查上报提示“暂时无使用权限，联系管理员开通”，需让超管要到巡查上报->配置->配置网格中把成员添加到网格员。
2、负责人：网格中心负责人，可分配网格员上报到网格中心的待分配工单。
3、超管：具有负责人的全部功能，此外还可配置「事件分类」、「网格和网格负责人」。
4、3.1.8版本开始，对巡查上报/居民上报权限进行优化，普通成员需要被设置为网格员或者是网格负责人以后才能进入巡查上报/居民上报内。
5、3.1.10版本开始，超管和网格负责人支持对待分派的工单进行催办，mac端暂不支持。
6、每个网格支持设置多个网格负责人、多个网格员。同一个成员支持同时被设置20个网格负责人，或多个网格的网格员，配置网格目前仅web端和手机端支持。

### [](https://open.work.weixin.qq.com)二、创建上报

1、可在手机端->工作台点击「巡查上报」应用的新建上报入口。
![](https://wework.qpic.cn/wwpic3az/735774_vrcXOD5TT6m6cN__1765175928/0)
2、可在手机端->工作台->巡查上报应用内点击“我的事项”右上角新建上报。
![](https://wework.qpic.cn/wwpic3az/384564_03Q82vv2REawVKk_1765175933/0)
3、填写必填字段后，可选择「我已办结」或「上报到网格」两种方式提交上报。其中自己已经处理的工单选择「我已办结」，自己未处理的工单选择「上报到网格」。
![](https://wework.qpic.cn/wwpic3az/869230_pwfXL4unR4m0lTZ_1765175939/0)
**必填字段如下：**
【事件类别】：必选字段，共两级分类，由超管在「配置」中进行设置，默认预置。
【事件描述】：必填字段（文本、附件必填其一），可发送纯文本、纯图片、纯视频、文本+图片、文本+视频，图片和视频不能同时被选中。文本字数上限为2000字符，图片最多选择9张，视频最大支持100M。
【紧急程度】：必选字段，3个选项依次为「一般」「重要」「紧急」，默认选中「一般」。
【事件类别】：必选字段，共两级分类，由超管在「配置」中进行设置。
【详细地址】：必填字段，可调起地图选择地点。
【所属网格】：必选字段，由超管在「配置」中进行设置，冷启动时默认预置。
【处理流程】：必选，有「我已办结」、「上报到网格」两种。如选「我已办结」，则必须填入处理结果（文字、附件必填其一）；如选「上报到网格」，则提示“你的上报将由网格中心负责人分配给相关人员处理”。
4、网格员首次提交工单时，系统弹出提醒，要求网格员选择网格，可通过直接点击或搜索的方式查找网格并选择。

### [](https://open.work.weixin.qq.com)三、查看上报与分派

1、【我的上报】：上报列表展示所有本人待办以及参与过的上报，包括本人待办的、本人创建的、本人转交的、本人办理的。
注：普通成员只有“我的上报”入口。
2、【网格中心】：上报单的办理状态共3种，分别为「待分配」、「办理中」、「已办结」。
3、【网格中心筛选】：筛选字段包括「事件类别」、「所属网格」、「紧急程度」、「办理人」（办理中或办结才支持筛选办理人）。筛选项默认都为全部。
4、网格负责人/超管可在「网格中心」将网格员上报到网格中心的「待分配」工单分配给具有巡查上报应用使用权限的处理人。
5、办理中的事件网格员（处理人）收到他人转交或网格中心分配的「办理中」反馈后，可以选择「转交他人」或「我已办结」。「转交他人」次数暂无上限。
6、支持批量分派功能，在网格中心待分派列表可以批量选择「待分派」工单进行分派。
7、支持对办理中的工单进行催办。
8、管理员支持筛选网格，分别查看各个网格的数据统计。支持查看每个网格中，有提交过上报成员的工单上报数量，或办结过工单的成员的工单办理数量。
9、在管理后台->应用管理->巡查上报->成员统计明细中，如果是筛选“网格”，是按照网格事件的关联人进行筛选的，并不是按照事件所属网格筛选。
关联人是指这个网格事件由这个成员上报或者办理过，那这个成员就是网格事件的关联人。
例如：A网格的张三创建上报了一个办理事件，此时这个事件的所属网格是A网格；然后B网格的李四也加入办理该事件，此时如果管理员在成员统计明细中筛选“B网格”，也会出现张三上报的这个事件，因为B网格的李四参与过这个事件。
10、负责人可以删除管理范围内的巡查上报工单，不规范的工单一键删除，方便留档备案

### [](https://open.work.weixin.qq.com)四、查看上报统计

1、仅网格负责人和超管可查看应用内的统计，负责人可查看自己管理的所有网格的数据，超管可查看全企业所有数据
2、统计一级页面分为3部分，分别是「上报概况」、「业务透视」、「分类统计」
（1）上报概况：统计项与工作台数据看板的统计项完全一致，包含4个统计项
【今日上报】：当日新增的上报数据
【待分牌】：当前全部待网格中心分派的上报
【办理中】：当前全部办理中的上报
【今日办结】：当日新增的办结数据
（2）业务透视：用漏斗形态展示「上报->办结」的业务流程，包含2个统计项和1个比率
【累计上报】：历史所有上报
【累计办结】：历史所有已办结
【办结率】：累计办结/累计上报（用%表示，保留两位小数）
（3）分类统计：用条形图展示全部一级分类，每个一级分类展示「上报总数」、「办结总数」，并在条形图上用不同颜色区分
【上报率】：该一级分类的上报总数|全部分类的上报总数
【办结率】：该一级分类的已办结总数|该一级分类的上报总数
展示每个二级分类的「上报总数」、「办结总数」，根据「上报总数」从多到少排序。
![](https://wework.qpic.cn/wwpic3az/462725_8uYabamKSiSbB4m_1765175950/0)

### [](https://open.work.weixin.qq.com)五、配置

1、配置只有超管可见，可配置「事件类别」和「网格中心」
2、【巡查上报】和【居民上报】两个应用是同一份数据，因此在一个应用配置或修改，另一个应用会同步展示。
3、如果因离职导致网格没有对应负责人，则默认添加超管为该网格负责人。
4、事件类别：可以只有一级分类没有二级分类，但是一级分类和二级分类不能全部为空。一级分类最多可配置50个，二级分类最多可配置50个。
![](https://wework.qpic.cn/wwpic3az/382418_ocInHZkdQveVM5s_1765175960/0)
5、网格负责人选人范围需在应用使用范围内，每个网格最多可以有20个负责人，目前不限制创建网格的数量。
![](https://wework.qpic.cn/wwpic3az/628171_me66yqtvSyiUJ6Q_1765175964/0)

### [](https://open.work.weixin.qq.com)六、巡查上报/居民上报的多级上报以及权限管理

1、巡查上报的网格支持添加下级网格；
2、超级管理员支持增删改任意网格以及该网格的下级网格，网格负责人则支持给自己的网格进行增删改；
3、网格中有网格员时无法进行删除（仅有网格负责人则支持）；
4、整个网格结构最多10层，每层网格最多添加20个网格负责人以及100个网格员；
5、同一个主网格下不允许添加同名下级网格；
6、网格名称最多20个中文汉字；
7、web端支持使用模板批量导入网格架构；
8、路径：
（1）web端：应用管理->居民上报/巡查上报->配置网格；
（2）客户端：工作台->居民上报/巡查上报->配置->配置网格。
![](https://wework.qpic.cn/wwpic3az/226099_1FctSAoZTTyyIfh_1765175972/0)

### [](https://open.work.weixin.qq.com)七、配置上报及班级权限

可以通过设置每个不同成员/负责人上报/分派的范围，从而达到逐级上报的目的。

（1）成员新建事件上报给
a、当前网格负责人：成员只能上报给本网格负责人
b、上级网格负责人：成员如果在子网格内，可以上报给子网格的任意一个上级网格的负责人
![](https://wework.qpic.cn/wwpic3az/432668_1u5C-vGsQOCDNdl_1765175977/0)
（2）负责人可分派/转交事件给
a、全部网格内的成员：网格负责人支持把事件分派/转交给任意一名网格员
b、管理网格内的成员：网格负责人只能把事件分派/转交给自己管理的网格内的网格员
![](https://wework.qpic.cn/wwpic3az/563548_VQrqVjvgRiy79vH_1765175981/0)
（3）成员可转交事件给
a、全部网格内的成员：成员支持把事件转交给任意网格员
b、所在网格内的成员：成员只能把事件转交给自己所属网格内的其他网格员
![](https://wework.qpic.cn/wwpic3az/467402_3NlgH1GUQ3qCyle_1765175987/0)

### [](https://open.work.weixin.qq.com)八、居民上报/巡查上报明细支持导出到微盘

管理后台->应用管理->巡查上报/居民上报->巡查上报记录/居民上报记录->查看并导出所有数据，导出的时间跨度是1年。
若巡查上报中带有图片，导出后图片在表格中是以链接的形式显示，可通过链接查到图片。
