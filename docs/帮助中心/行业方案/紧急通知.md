# 紧急通知

### [](https://open.work.weixin.qq.com)一、紧急通知介绍

目前仅支持行业类型为**医疗**的企业使用紧急通知功能。
针对紧急事件，确保在指定时间内让相关干系人认领任务并做出响应。紧急通知是在目前应用消息推送的基础上，通过自动语音电话播报功能，确保消息实时送达，成员通过语音电话播报提醒后，进行处理相关的提醒事件。医院遇到类似危急值提醒、重要会议召开、紧急预案启动等场景,可使用紧急通知功能。 

### [](https://open.work.weixin.qq.com)二、紧急通知的应用场景

医院内部有类似危机值推送、MDT、医患冲突等应急预案场景。

### [](https://open.work.weixin.qq.com)三、紧急通知的使用方法

#### [](https://open.work.weixin.qq.com)1、启用/禁用紧急通知

管理员进入企业微信【管理后台->应用管理->应用->紧急通知】启用应用，由管理员设置可发紧急通知的成员及发送范围。管理员还可查看并导出紧急通知的通话记录。

#### [](https://open.work.weixin.qq.com)2、员工如何发送紧急通知

支持在手机端和windows桌面端发送通知，通过电话提醒成员及时查看。发送后，可随时关注紧急通知的阅读状态，并对未读成员进行再次提醒。

#### [](https://open.work.weixin.qq.com)3、调用接口推送紧急通知

企业可结合应用的自建应用推送消息，调用这个接口进行拨打，接口指引可到【管理后台->应用管理->紧急通知->api->[查看api文档](https://open.work.weixin.qq.com/api/doc/16595)】查看具体的说明。
语音电话拨号支持一次性通知多个成员，一次性拨打人数暂无上限。

### [](https://open.work.weixin.qq.com)四、紧急通知功能的账户余额

使用公费电话和紧急通知，都将消耗余额中的分钟数。
（扣费方式、充值方式与公费电话扣费方式一致，公费电话和紧急通知公用一个账户余额）。

若一次性通知多个成员，未接听的成员不会扣除时长，仅扣除已接听的时长。

### [](https://open.work.weixin.qq.com)五、紧急通知的推送频率

1、每条通知一次性最多可以发给1000人。每个企业成员每天最多只能接收50个紧急通知（包括重复推送的通知），超过的会自动不推给这个人。
2、接口呼叫限制是24小时8次（接口呼叫推送，未接也会包含在呼叫限制中）；企业微信侧发送的推送次数暂无限制。
对应的接口指引：[点击查看](https://work.weixin.qq.com/api/doc/16821)

### [](https://open.work.weixin.qq.com)六、紧急通知的来电显示展示说明

显示为固话，若需要来电时显示企业名称，需要成员自行将固话保存到手机通讯录中，手动备注对应名称。

### [](https://open.work.weixin.qq.com)七、紧急通知的通话说明

1、成员接听了语音后，如果不挂机，语音在重复播报三遍后，会自动挂机。
2、余额不足的情况下，通知是可以通知成功。目前是允许欠费的。例如余额只有1分钟，是可以通知10个人的；但是余额为0，就不让发了，消息和电话都不推。

### [](https://open.work.weixin.qq.com)八、常见问题

#### [](https://open.work.weixin.qq.com)1、紧急通知收到语音通知电话企业名称与实际企业名称不一致

目前要展示为企业公费电话的名称，需要授权允许企业微信【写入联系人】权限，然后通过拨打公费电话的方式，触发写联系人。

#### [](https://open.work.weixin.qq.com)2、成员退出企业，但还能收到紧急通知？

目前成员主动退出企业，但管理员未从企业通讯录中将其删除，成员还是能收到紧急通知，可联系联系管理员将其删除，该情况我们将在后续优化。
