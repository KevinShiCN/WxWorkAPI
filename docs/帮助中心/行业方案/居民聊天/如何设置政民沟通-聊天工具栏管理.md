# 如何设置政民沟通-聊天工具栏管理

### [](https://open.work.weixin.qq.com)一、聊天工具栏基本介绍

企业可配置应用页面到聊天工具栏，方便成员在与居民的聊天中查看和使用，提高服务效率。
工具栏的应用可添加的上限：100个。

### [](https://open.work.weixin.qq.com)二、聊天工具栏配置路径

1、【[管理后台](https://work.weixin.qq.com)->政民沟通与上下游->政民沟通->聊天工具->聊天工具栏管理】
![](https://wework.qpic.cn/wwpic3az/759265_uyESA8TbTaitXng_1753753065/0)
【[管理后台](https://work.weixin.qq.com)->应用管理->自建应用->配置到聊天工具栏】
![](https://wework.qpic.cn/wwpic3az/358243_jtZJEH0jThW_cim_1753753070/0)
2、【手机端->工作台->居民联系->工具->聊天工具栏】
![](https://wework.qpic.cn/wwpic3az/777724_8zoEa6cyRFmHRJr_1765180075/0)

### [](https://open.work.weixin.qq.com)三、支持使用聊天工具栏的居民会话

1、与居民会话的单聊
2、企业微信侧发起的外部群聊（包括居民群、含互联企业人员的群聊）
