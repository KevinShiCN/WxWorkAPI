# 如何设置政民沟通功能的权限

### [](https://open.work.weixin.qq.com)一、什么是政民沟通使用范围

政民沟通使用范围中的成员可使用政民沟通功能，支持添加居民、创建居民群，他们添加的居民和建立的群聊由企业统一管理。

### [](https://open.work.weixin.qq.com)二、设置政民沟通使用范围人员

超级管理员支持在【手机端->工作台->居民联系->配置->使用范围和管理规则】
![](https://wework.qpic.cn/wwpic3az/734590__Oq49Wx8Sa-C5rb_1765244702/0)

或是在【[管理后台](https://work.weixin.qq.com)->政民沟通与上下游->政民联系->权限配置->政民联系和居民群】可修改使用范围。
![](https://wework.qpic.cn/wwpic3az/654670_59_kuIHFSPOvevc_1753752679/0)

### [](https://open.work.weixin.qq.com)三、超级管理员设置政民沟通负责人

#### [](https://open.work.weixin.qq.com)1、超级管理员在手机端设置政民沟通负责人

在手机端，超级管理员可以在【手机端->工作台->政民沟通->配置->使用范围和管理规则】中设置管理规则，并在管理规则中添加业务负责人、指定负责人的管理范围和管理权限。
![](https://wework.qpic.cn/wwpic3az/691782_dJ-TT1rHRSKUTr3_1765244707/0)

#### [](https://open.work.weixin.qq.com)2、超级管理员在web管理后台设置政民沟通负责人

在[管理后台](https://work.weixin.qq.com)，超级管理员可以在【政民沟通与上下游->政民沟通->权限配置->政民沟通和居民群->管理规则】中设置业务负责人，帮助分管成员添加的居民。
![](https://wework.qpic.cn/wwpic3az/53823_HG8KzHYlRmCq6Ch_1753752688/0)

#### [](https://open.work.weixin.qq.com)3、负责人设置政民沟通下级负责人

被设定为业务负责人的成员也能在自己的权限范围内新建下级负责人，让下级负责人帮助自己分管业务。
功能入口为：【手机端->工作台->居民联系->设置管理规则->设置下级管理规则】
![](https://wework.qpic.cn/wwpic3az/223504_CYu8PzmNSCSkmLS_1765244715/0)
