# 如何发表政民朋友圈

### [](https://open.work.weixin.qq.com)一、政府发表到居民的朋友圈

1、超级管理员或有发表到成员居民的朋友圈权限的负责人支持创建朋友圈内容，将活动信息、产品动态、专业知识等内容发表到居民的微信朋友圈，并与居民评论互动。
![](https://wework.qpic.cn/wwpic3az/602727_92PTg1qKRGCMVjp_1765172082/0)
2、超级管理员支持在管理后台【政民沟通与上下游->政民沟通->群发工具->政府发表到居民的朋友圈】进行新增朋友圈。
![](https://wework.qpic.cn/wwpic3az/701909_H_GHlbdGSaOu6l6_1753691361/0)
3、朋友圈内容支持文字或文字+图片/视频/网页。
![](https://wework.qpic.cn/wwpic3az/577681_ZkCjC2SjSEKmu2m_1753691364/0)
4、超级管理员或有发表到成员居民的朋友圈权限的负责人也可以在手机端【工作台->居民朋友圈->政府发表到居民的朋友圈->发表】进行新增企业朋友圈。
![](https://wework.qpic.cn/wwpic3az/25931_sqBMvgYEQyOjk81_1765172091/0)

5、超级管理员或应用负责人发送了企业发表到居民的朋友圈内容后，成员会在政民联系应用收到对应的发送提醒，可以在【朋友圈->企业通知/通知】中查看对应的内容，点击【发布】后，这条内容会发布到居民朋友圈。
![](https://wework.qpic.cn/wwpic3az/599725_KOBpcfK6Q-eaoe9_1733121561/0)

### [](https://open.work.weixin.qq.com)二、成员编辑内容发表到居民朋友圈

1、在手机端【工作台->政民朋友圈->我发表的】入口可新增个人朋友圈。成员编辑好内容后、可按标签选择要展示的居民，发表后，居民即可在微信朋友圈中查看、还可以点赞和评论成员发表的内容。
2、每个成员支持对同个居民展示3条朋友圈。这3条朋友圈可以是个人朋友圈，也可以是政府推送的朋友圈。

![](https://wework.qpic.cn/wwpic3az/137028_1e_IrWNxQleE7Sv_1765172096/0)

### [](https://open.work.weixin.qq.com)三、居民朋友圈可展示居民限制/发送朋友圈提示“当前没有满足发表条件的居民”

1、**朋友圈发送频率：**一个居民每天只能看到同个人发送的3条朋友圈，这三条朋友圈可以是企业发表的，也可以是成员发自己发表的。

2、**政民朋友圈可展示的居民数：**
**未验证/认证企业**，企业居民数最多只能添加100人，因此朋友圈内容最多只能对100个居民可见。
**已验证/认证企业**，可展示居民数为所有已添加居民；认证到期未续期，已发表的朋友圈不受影响，新发表的朋友圈最多可展示100个居民。

3、若发送朋友圈提示“当前没有满足发表条件的居民”，请更新企业微信到最新版本后重试。
若更新版本后仍未恢复，请确认：
（1）有可能是超过可发送的限制，一个居民每天只能看到同个人发送的3条朋友圈，这三条朋友圈可以是企业发表的，也可以是成员发自己发表的。（发表成功后再删除也会占用次数）
（2）成员需要有添加微信联系人，且联系人是居民，可查看【手机端->通讯录->我的居民】是否有微信居民。
（3）成员需要在[政民朋友圈权限](https://open.work.weixin.qq.com/help2/pc/19266)使用范围内。

### [](https://open.work.weixin.qq.com)四、设置朋友圈封面

1、支持在朋友圈页面设置封面及个性签名。
2、【手机端->工作台->政民朋友圈】支持设置封面素材库和签名素材库，有朋友圈管理权限的成员可配置素材，成员在设置封面及签名时可直接使用。
3、朋友圈背景推荐尺寸是1028 x1028。
![](https://wework.qpic.cn/wwpic3az/543424_yHS7aQe-TFepKRA_1765172108/0)

### [](https://open.work.weixin.qq.com)五、政民朋友圈如何停止发表

政府群发的朋友圈支持停止发表，未发表成员不会再收到发表提醒。

管理员可在【管理后台->政民沟通与上下游->群发工具->政府发表到居民的朋友圈->政府发表记录->选择对应的朋友圈记录->点击右侧“…”->停止发送】（仅支持停止发表一年内的朋友圈，大于一年的，需在手机端操作）
![](https://wework.qpic.cn/wwpic3az/157562_pI3xBBHFQhSadr1_1753691374/0)![](https://wework.qpic.cn/wwpic3az/378352_zvDXuQIATVW4C-__1753691374/0)
【手机端->工作台->政民朋友圈->统计->待成员发表的->选择对应的朋友圈记录进入->右上角“…”->停止发表】
![](https://wework.qpic.cn/wwpic3az/762072_kXG5CUiIQEOV3wV_1765172114/0)
