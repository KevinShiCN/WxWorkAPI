# 如何使用政民朋友圈统计

### [](https://open.work.weixin.qq.com)一、如何查看政民朋友圈统计

超级管理员和业务负责人在【手机端->工作台->政民朋友圈->统计->成员发表概况】中可以查看管理范围内当天成员发布朋友圈概况。
![](https://wework.qpic.cn/wwpic3az/854993_sw7GJ56iRsWn0gV_1765249725/0)

### [](https://open.work.weixin.qq.com)二、如何接收成员每日发表概况

仅超级管理员可在手机端【工作台->政民朋友圈->配置】开关开启【接收成员每日发表概况】，开关默认关闭。
开启后超管和业务负责人可以在每天18:00后在【朋友圈】应用收到成员每日发表概况推送。
![](https://wework.qpic.cn/wwpic3az/305111_0ayVuzwNTRCAu26_1765249721/0)
