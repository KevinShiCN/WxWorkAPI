# 居民通知单次群发居民群限制

1、选择「我添加的居民群」发送，单次最多可选1000个居民群进行发送；
2、选择「政府管理的的居民群」发送，单次最多可选1000个居民群进行发送；
3、支持发送图片；文字上限10000字符，图片最多支持10张，单张图片不能大于10M，单张图片的像素无要求。
