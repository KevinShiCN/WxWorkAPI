# 如何使用在职继承分配成员的居民群

### [](https://open.work.weixin.qq.com)一、什么是在职继承分配居民群

若在成员[创建了居民群](https://open.work.weixin.qq.com/help2/pc/19133)，成员业务发生变更的情况下，超级管理员可将成员的居民群批量分配给其他成员继续提供服务；也可将指定居民群分配给其他成员服务（新群主需是同企业人员且是居民联系和居民群的使用范围内的成员）。居民群群主将自动变更为接替的成员，原成员将退出群聊且无法继续服务群居民。

支持通过接口实现按业务场景分配在职成员的居民群，并查询居民群的分配记录。

### [](https://open.work.weixin.qq.com)二、分配在职成员的居民群操作方式

管理端入口：【管理后台->政民沟通与上下游->政民沟通->在职继承】，选择需要转接的居民群，点击待分配的群聊，选择转接的成员进行分配。
![](https://wework.qpic.cn/wwpic3az/762078_NOdw7LBqTvSOUQV_1753671179/0)
手机端入口：超级管理员/负责人可在【手机端->工作台->居民联系/居民群->在职继承】操作分配。超级管理员可以分配所有成员的居民群，负责人只能分配自己管理范围内成员的居民群。
![](https://wework.qpic.cn/wwpic3az/267379_aFPFOiYiRiuui-P_1765246705/0)
电脑端入口：【工作台->居民群->在职继承】，仅客户联系负责人在电脑端可见，若是超级管理员，则看不到“在职继承”入口。

### [](https://open.work.weixin.qq.com)三、在职继承居民群的频率限制

一个居民群90天内只能继承2次；
单次最多分配500个居民群。
注：居民群满500人时操作继承，会提示要随机移除一个居民，确认移除居民后邀请新群主加入群聊。
