# 如何在居民群设置快捷回复

### [](https://open.work.weixin.qq.com)一、什么是居民群快捷回复

企业可为成员统一配置快捷回复，成员还可自己添加，添加后，成员可在与居民的聊天中点击「快捷回复」直接选取内容回复居民，提高工作效率。

### [](https://open.work.weixin.qq.com)二、如何配置居民群快捷回复

超级管理员支持在【管理后台->政民沟通与上下游->政民沟通->聊天工具->快捷回复】配置企业成员的快捷回复，可将常用短语配置为快捷回复，快捷回复支持批量导入。
![](https://wework.qpic.cn/wwpic3az/2375_6G91uU39R8uRDYs_1753682821/0)
业务负责人和超管在【手机端->工作台->居民群->工具->快捷回复】也支持配置快捷回复。
![](https://wework.qpic.cn/wwpic3az/118581_6E-IHGY4STG24aD_1764834459/0)

### [](https://open.work.weixin.qq.com)三、如何删除居民群快捷回复

超级管理员支持在【[管理后台](https://work.weixin.qq.com)->政民沟通与上下游->政民沟通->聊天工具->快捷回复】中删除企业配置的快捷回复内容。
![](https://wework.qpic.cn/wwpic3az/837354_Ww0DrPYvTaugMo7_1753682828/0)
业务负责人和超管支持在【手机端->工作台->居民群->工具->快捷回复】中删除快捷回复内容。
管理员支持删除企业新增的快捷回复以及自己新增的快捷回复内容；成员不能删除企业新增的快捷回复内容，可以删除自己添加的快捷回复内容；业务负责人支持删除自己新增的快捷回复内容。
![](https://wework.qpic.cn/wwpic3az/4780_C5owIIiwRQC9ujI_1764834476/0)

### [](https://open.work.weixin.qq.com)四、居民群快捷回复上限说明

1、分组名最多支持12个汉字，单条快捷回复最多支持500个汉字；
2、超级管理员、业务负责人、成员分别可以创建100个分组；
3、一个分组里的快捷回复最多存在200条。
