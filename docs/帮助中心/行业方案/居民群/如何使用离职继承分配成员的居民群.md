# 如何使用离职继承分配成员的居民群

### [](https://open.work.weixin.qq.com)一、什么是离职继承分配居民群

若在成员[创建了居民群](https://open.work.weixin.qq.com/help2/pc/20628)，群主离职的情况下，居民群需由管理员分配新群主（新群主需是同企业人员且是居民联系和居民群的使用范围内的成员），若居民群群主离职，管理员未分配新的群主，则这个群的群主不会自动转让群主。

### [](https://open.work.weixin.qq.com)二、分配离职成员的居民群操作方式

#### [](https://open.work.weixin.qq.com)1、管理端入口

【[管理后台](https://work.weixin.qq.com)->政民沟通与上下游->政民沟通->离职继承】，选择需要查看的离职人员，点击待分配的群聊。
![](https://wework.qpic.cn/wwpic3az/572439_8ucJY984TRWkE4T_1753671674/0)
![](https://wework.qpic.cn/wwpic3az/685636_OSLIPGcJQX2JIhE_1753671677/0)

#### [](https://open.work.weixin.qq.com)2、手机端入口

管理员/负责人可在【手机端->工作台->离职继承】操作分配。管理员可以分配所有离职人员的居民群，负责人只能分配自己管理范围内离职人员的居民群。
![](https://wework.qpic.cn/wwpic3az/552716_x5xZGe4zTYON5Xr_1765246582/0)

#### [](https://open.work.weixin.qq.com)3、电脑端入口

【工作台->居民群->去分配】，仅客户联系负责人可在电脑端操作分配，若是超级管理员，仅可查看“我继承了离职成员的”居民/居民群。

### [](https://open.work.weixin.qq.com)三、常见问题

1、成员已在通讯录被删除，但是已分配离职成员资产找不到这个离职成员，无法分配居民和居民群。
确认已在通讯录中删除成员，但在已分配离职成员资产找不到成员，原因是：
（1）已分配离职成员资产页面中展示的离职成员姓名目前仅展示成员离职前在通讯录的姓名，若设置了别名，则不会显示别名。而操作日志会展示姓名（别名）。因此会有可能误以为不是同个人。
（2）离职成员在离职前把居民群解散了、把居民删除了，则不会出现可分配列表中。

2、分配离职成员群聊，提示当前分配群聊数量超过上限。
目前单次分配离职成员的群聊数量上限为500个，如需分配的群聊数量超过500个，建议分次操作分配。

3、离职继承中“账号已注销”的成员无法离职继承，选项框置灰。
目前账号已注销的情况下暂不支持进行居民群离职继承。（离职继承居民不受影响）
