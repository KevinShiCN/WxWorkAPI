# 如何查看居民群成员动态

支持在【手机端/电脑端企业微信->工作台->居民群】中查看最近有新增居民群的10位同事（头像、姓名、居民群数、群成员数）。
![](https://wework.qpic.cn/wwpic3az/180511_14kgDN7zSFWRPIf_1764906303/0)
![](https://wework.qpic.cn/wwpic3az/759807_YkbHcvb9Roeo93r_1750128662/0)

管理员还可以在【手机端企业微信->工作台->居民群->右上角的三个点->显示成员动态】中自定义开启或关闭显示。开启后，成员可以在居民群中查看到最近创建了居民群的同事。
![](https://wework.qpic.cn/wwpic3az/722506_9QVO0GmsTF-Nbzz_1764906306/0)
