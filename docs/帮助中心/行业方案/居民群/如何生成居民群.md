# 如何生成居民群

### [](https://open.work.weixin.qq.com)一、主动创建居民群

在[政民联系使用范围内](https://open.work.weixin.qq.com/help2/pc/19253)的成员，发起的外部群会默认为居民群。

方式一：在【消息界面右上角“+” ->发起群聊->选择我添加的居民】
![](https://wework.qpic.cn/wwpic3az/970697_Mnwwj4swTOe8alU_1764733430/0)
方式二：在【手机端->工作台->客户群->创建一个居民群】
![](https://wework.qpic.cn/wwpic3az/121292_gL57M_ZHS5Ok2FH_1764733435/0)

### [](https://open.work.weixin.qq.com)二、已有群聊确认为居民群

在企业微信侧创建的外部群，当群主把群主身份转让给在[政民联系使用范围内](https://open.work.weixin.qq.com/help2/pc/19253)内的成员，新群主可以在把群聊确认为居民群，设置入口【群管理->标记为居民群】。
若成员不在[政民联系使用范围内](https://open.work.weixin.qq.com/help2/pc/19253)，创建了外部群，然后给成员添加了居民联系功能使用权限后，成员支持把这类型的群确认为居民群，设置入口【群管理->标记为居民群】。
![](https://wework.qpic.cn/wwpic3az/124427_SoiKDPorQVuLhxS_1764733439/0)

### [](https://open.work.weixin.qq.com)三、创建居民群数量上限

未验证未认证的企业，所有外部群聊（包括居民群）加起来的总人数上限为100人；未满100人时，也仅支持创建20个群聊；
已验证/认证的企业，群聊个数无限制。
关于认证，可以参考：[企业微信认证介绍](https://open.work.weixin.qq.com/help2/pc/19729)
温馨提示：认证审核需要收取服务费，在支付之前，我们建议您详细阅读支付页面的相关条款。
