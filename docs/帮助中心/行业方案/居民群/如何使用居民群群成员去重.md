# 如何使用居民群群成员去重

### [](https://open.work.weixin.qq.com)一、群成员去重操作入口

【管理后台->政民沟通与上下游->政民沟通->安全管控->群成员去重】支持按群主筛选或按群名包含的关键词筛选去重。
![](https://wework.qpic.cn/wwpic3az/889499_8MMlUKM_QRGZbtD_1753676074/0)

【手机端->工作台->居民群->群成员去重】
![](https://wework.qpic.cn/wwpic3az/345246_lm7WdavMRhiy1qC_1764913193/0)

### [](https://open.work.weixin.qq.com)二、支持筛选关键词给对应的群成员去重

超级管理员、业务负责人支持根据群主和群名关键词筛选最近活跃的2000个居民群中是否有同一个居民，筛选后选择要踢出该居民的居民群通知服务人员。

### [](https://open.work.weixin.qq.com)三、管理员点击了去重后成员侧需点击确认去重

成员在【客户端->消息栏->居民群】收到通知去重，点击进入后可以操作移出居民。
![](https://wework.qpic.cn/wwpic3az/482723_-1eZVeLBRTyLnJO_1764913196/0)
