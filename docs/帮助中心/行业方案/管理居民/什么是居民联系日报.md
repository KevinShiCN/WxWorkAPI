# 什么是居民联系日报

### [](https://open.work.weixin.qq.com)一、什么是居民联系日报？

居民联系日报是 3.1.8 版本新增的功能，目前支持一级行业为政府的单位使用。

我们每天统计单位的网格治理数据，包括**「激活成员数」、「添加居民数」、「邀请居民进群数」**，以日报的方式发送给单位成员，支持分部门查看和排序。旨在帮助管理者随时掌握工作进度，加快工作进程。
![](https://wework.qpic.cn/wwpic3az/81828_JNgbbnWqRJSMMJT_1764913718/0)

日报数据可导出明细，明细会保存到【微盘->我的文件】中，日报页面支持分享到企业微信、微信聊天中。

### [](https://open.work.weixin.qq.com)二、居民联系日报常见问题

#### [](https://open.work.weixin.qq.com)1、何时开始发送日报？

3.1.8版本后创建的企业，添加第一名居民后，次日开始全企业成员都会收到居民日报推送。

#### [](https://open.work.weixin.qq.com)2、如何停止发送日报？

超级管理员可在【工作台->居民联系->配置->居民联系日报设置】或【日报右上角->设置】页面选择「不接收日报」。
![](https://wework.qpic.cn/wwpic3az/369269_Iw78IfdoTMmWNue_1764913728/0)

#### [](https://open.work.weixin.qq.com)3、如何修改日报接收权限？

超级管理员可在【工作台->居民联系->配置->居民联系日报设置】或【日报右上角->设置】页面设置日报接收权限为：
（1）所有企业成员接收日报
（2）仅超管接收日报
（3）指定成员接收日报
![](https://wework.qpic.cn/wwpic3az/795921_L9nlkYBqTUS_4fe_1764913723/0)

#### [](https://open.work.weixin.qq.com)4、如何在日报中移除某部门？

超级管理员可在【工作台->居民联系->配置->居民联系日报设置】或【日报右上角->设置】页面设置「不参加排名的部门」，设置后，不参加排名的部门不出现在居民联系日报中，成员也不能收到日报。
![](https://wework.qpic.cn/wwpic3az/289703_RGLLBQ8ESGCxfA__1764913735/0)

#### [](https://open.work.weixin.qq.com)5、日报可以记录多久的数据？

该推送日报只记录90天内的数据，点击90天以前推送的日报将无法查看。
