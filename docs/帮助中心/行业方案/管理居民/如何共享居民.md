# 如何共享居民

### [](https://open.work.weixin.qq.com)一、成员如何共享居民给同事？

成员可在查看居民信息时，点击右上角的“…”，选择共享给同事，操作共享居民给其他成员。
![](https://wework.qpic.cn/wwpic3az/698779_tdHdvu05SBqMK-d_1765181328/0)

### [](https://open.work.weixin.qq.com)二、管理员如何共享企业居民给企业成员？

管理员可在【管理后台->居民联系->居民->点击“总人数”->选择对应的居民】进入居民详情操作共享居民。
![](https://wework.qpic.cn/wwpic3az/880594_oT18O5NySzeBvVl_1753692211/0)
![](https://wework.qpic.cn/wwpic3az/466245_mRCxin-ARaWJ56X_1753692215/0)

### [](https://open.work.weixin.qq.com)三、哪些成员可以被共享居民？

成员需要有居民联系使用权限，才可被共享居民。

### [](https://open.work.weixin.qq.com)四、共享居民，添加成功后会同步居民详情的信息吗?例如备注名、标签、描述等。

共享客户后，原添加人对该名居民的备注名、企业标签、描述、手机号、企业名称等备注和描述信息，都将同步共享给接受共享的企业成员。

如果接受共享的企业成员已经添加了部分的备注名、企业标签、描述、手机号、企业名称等备注和描述字段，则优先以接受共享的成员自己添加的显示。
