# 如何使用政民沟通在职继承分配成员的居民

### [](https://open.work.weixin.qq.com)一、什么是在职继承

当成员或居民/居民群有变更时，企业可以将在职成员的居民/居民群分配给其他成员继续提供服务。
分配后，居民和接替的成员将在24小时后自动成为联系人；如果居民主动拒绝，接替成员还可选择主动添加。居民群群主将自动变更为接替的成员，同时原成员将退出群聊。

### [](https://open.work.weixin.qq.com)二、在职继承的分配指引

当成员或居民有变更时，超级管理员或业务负责人可在【工作台->居民联系->在职继承】
![](https://wework.qpic.cn/wwpic3az/305253_tyHUR4XaTDCK37F_1764831829/0)

或【[管理后台](https://work.weixin.qq.com)->政民沟通与上下游->政民沟通->在职继承】中需要转接的居民分配给其他成员。
![](https://wework.qpic.cn/wwpic3az/199700_6Gq62Z6JT2ilnWQ_1753693568/0)
分配后，居民将在微信中收到转接提醒。
![](https://wework.qpic.cn/wwpic/225610_pXeKxoSRTc61BF__1662606163/0)
居民和接替的成员默认将在24小时后自动成为联系人；如果居民主动拒绝，接替成员还可选择主动添加。
![](https://wework.qpic.cn/wwpic3az/464393_KABCdynPTOaBRn__1764831845/0)
注：为了居民不被重复打扰，如果新成员成功接替居民，则原跟进成员将无法再继续为该居民户提供服务，居民发送消息给原跟进成员，系统将会自动引导居民联系新的接替成员；
如果新成员接替失败，则原成员还可继续为居民提供服务

### [](https://open.work.weixin.qq.com)三、查看在职继承的分配记录

在【手机端->工作台->居民联系->在职继承->右上角“···”】
![](https://wework.qpic.cn/wwpic3az/613942_DqdBEo6bRAy9hps_1764831850/0)
或【管理后台->政民沟通与上下游->政民沟通->在职继承】中查看已分配的居民/居民群。
![](https://wework.qpic.cn/wwpic3az/755283_3VBeerIOQDqz4m3_1753693574/0)

### [](https://open.work.weixin.qq.com)四、在职继承的频率限制

在职成员的每位居民每90天内，最多可被转接2次。
注：
在职继承的次数等限制不影响离职继承功能的使用。
在职分配单次最高支持分配5000个居民。
在职分配后转接状态显示“转接失败”不占用90天2次次数。

### [](https://open.work.weixin.qq.com)五、在职继承同步居民详情信息场景

在职继承后，原添加人对该名居民的备注名、企业标签、描述、手机号、企业名称等备注和描述信息，都将同步共享给接受共享的企业成员。
注：
1、未到24小时自动分配，居民主动点击系统消息添加新成员，不会继承备注和描述等信息。
2、若新员工和接替的居民本身是好友关系，不会继承之前成员添加的备注信息（包括添加的标签）。
3、在职继承未到24小时自动分配，原成员新增或修改备注、描述等信息，在时间达到24小时后，新内容会一并同步给新成员。
4、如果接替的企业成员已经添加了部分的备注名、企业标签、描述、手机号、企业名称等备注和描述字段，则优先以接替的成员自己添加的显示。
5、若是在发起继承后成员主动添加了居民，也会继承之前成员添加的备注信息和标签。
