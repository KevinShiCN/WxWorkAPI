# 如何给居民添加标签

若企业已创建[企业居民标签](https://open.work.weixin.qq.com/help2/pc/19256)，且成员有[政民联系使用权限](https://open.work.weixin.qq.com/help2/pc/19253)，成员支持在居民信息中，给居民打上企业标签。

成员支持在【通讯录->我添加的居民->标签->右上角“+”】中创建个人标签，创建后，可在居民信息中打上个人标签。
