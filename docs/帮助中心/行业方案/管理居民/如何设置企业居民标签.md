# 如何设置企业居民标签

### [](https://open.work.weixin.qq.com)一、居民标签的作用

1、企业居民标签分为：
（1）超级管理员设置的居民标签；
（2）一级业务负责人设置的规则组标签。
2、超级管理员可以在管理端或手机端添加 「居民标签」 ，以便企业成员给添加的居民打上企业标签，方便分类查看。
3、一级业务负责人可以在手机端添加「规则组标签」 ，以便管理规则范围内的成员给其添加的居民打标签的时候使用。

### [](https://open.work.weixin.qq.com)二、居民标签的上限

1、每个企业的企业居民标签上限为10000个（包含企业标签和规则组标签合计数）。
2、标签组数目上限100组，每个组里面限制100个标签。
3、标签名称支持15个字符。

### [](https://open.work.weixin.qq.com)三、居民标签的添加入口

管理支持在管理后台和手机端设置标签，业务负责人支持在手机端设置标签。
【管理后台->政民沟通与上下游->政民沟通->政府管理的居民标签】；
![](https://wework.qpic.cn/wwpic3az/403442_IM41RtwmR0CMo2p_1753695861/0)
【手机端->居民联系->工具->政府管理的居民标签】。
![](https://wework.qpic.cn/wwpic3az/607575_Oxvr2bySSSmmGFQ_1764744909/0)

### [](https://open.work.weixin.qq.com)四、居民标签支持排序

超级管理员在编辑企业标签时可对企业标签进行排序。
![](https://wework.qpic.cn/wwpic3az/797255_AjyN0WnwS3exoDt_1753695866/0)

### [](https://open.work.weixin.qq.com)五、居民标签的规则组标签

#### [](https://open.work.weixin.qq.com)1、如何配置居民标签的规则组标签

（1）企业的超级管理员按照企业的各条业务线设置一级管理规则。
![](https://wework.qpic.cn/wwpic3az/693627_JtpyBtUiQRGPlfo_1764744918/0)

（2）一级业务负责人在【工作台->居民联系->政府管理的居民标签】中为当前的业务线配置规则组标签，该管理规则下的管理范围内的成员可使用。
![](https://wework.qpic.cn/wwpic3az/796595_yEZkZPumS2Krh5F_1764744923/0)

（3）管理规则范围内的成员在给居民打标签的时候，可以选择规则组标签使用。
![](https://wework.qpic.cn/wwpic3az/750094_OVx5o6CPSjq8G14_1764744929/0)

#### [](https://open.work.weixin.qq.com)2、如何管理居民标签的规则组标签

如需往规则组新增标签，可使用企业内部接口API进行调用：[查看接口](https://work.weixin.qq.com/api/doc/90000/90135/94882)
一级业务负责人可以直接在手机端新增、查看、修改、删除规则组标签。

#### [](https://open.work.weixin.qq.com)3、管理居民标签规则组标签常见问题

（1）为什么在政府居民标签中找不到配置规则组标签的入口？
①请确认操作人是一级业务负责人。
②请确认超级管理员在【管理规则->管理权限】中已经开启【配置政府居民标签】的权限开关。

（2）为什么成员在给居民打标签时，看不到规则组标签？
请确认成员在该规则组标签对应的管理规则的管理范围内。确认路径：【工作台->居民联系->配置->管理规则->管理范围】。

（3）为什么成员在标签筛选的时候无法筛选出规则组标签？
如果成员曾经在管理规则A的管理范围，后续转移到管理规则B的管理范围，则在标签筛选的时候，只能按照成员当前所在规则组B的标签进行筛选。
