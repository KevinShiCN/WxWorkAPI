# 如何设置学员会话聊天敏感词

### [](https://open.work.weixin.qq.com)一、什么是聊天敏感词

企业可设置敏感词，设置后将下载到成员的企业微信。当成员发送给学员的消息包含敏感词时，将予以提示或发送失败。

### [](https://open.work.weixin.qq.com)二、学员会话聊天敏感词设置入口

超级管理员可在【管理后台->学员与上下游->学员联系->安全管控->聊天敏感词】配置成员和学员聊天的敏感词。
支持手动新增或Excel批量导入，手机端暂不支持设置。
![](https://wework.qpic.cn/wwpic3az/604032_OYQRK58uQbakWbn_1753436107/0)

### [](https://open.work.weixin.qq.com)三、学员会话聊天敏感词的规则上限

最多设置10条规则，规则名称最多20个汉字；每条规则中最多100个敏感词，每个词最多32个字符。

### [](https://open.work.weixin.qq.com)四、学员会话聊天敏感词的拦截类型

敏感词（关键词）、手机号码、邮箱地址、红包（互通红包）。
![](https://wework.qpic.cn/wwpic3az/933308_u5CHqkLdQBi-iJw_1753436113/0)

### [](https://open.work.weixin.qq.com)五、学员会话聊天敏感词的拦截方式

【预警并拦截发送】：提示“因包含企业设置的敏感词，该消息无法发送”。
【仅发警告】：提示“因包含企业设置的敏感词，请确认是否发送”；成员可选择继续发送，或取消发送。
选择后，单聊、群聊、消息群发等场景，均会在成员发送时弹出拦截提示。客户端须3.0.36版本以上才会生效。
![](https://wework.qpic.cn/wwpic3az/631396_ukoWpfmiSoGs9sh_1753436124/0)
![](https://wework.qpic.cn/wwpic3az/57777_d6ZRU7OTTMOMqd-_1764751721/0)

### [](https://open.work.weixin.qq.com)六、如何查看学员会话聊天敏感词的拦截记录

保存最近45天的拦截记录，管理员可在后台查看或导出拦截记录，导出excel。
