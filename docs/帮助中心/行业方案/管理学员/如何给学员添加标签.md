# 如何给学员添加标签

若企业已创建[企业学员标签](https://open.work.weixin.qq.com/help2/pc/20646)，且成员有[学员联系使用权限](https://open.work.weixin.qq.com/help2/pc/17903)，成员支持在学员信息中，给学员打上企业标签。

成员支持在【通讯录->我的学员->标签->右上角“+”】中创建个人标签，创建后，可在学员信息中打上个人标签。
