# 如何查看学员联系-成员动态

1、普通成员视角：显示的是最近有添加学员的10位同事（头像、姓名、添加学员数）。按添加顺序倒叙排序（每次切换tab进入则会进行更新最新动态）。

![](https://wework.qpic.cn/wwpic3az/623333_Vx0FhBdrSO2a7SI_1764922949/0)

2、管理员视角/业务负责人：与成员视角差不多。管理员多了底部栏。管理员右上角“…”->有“显示成员动态”的按钮，可用来关闭“同事动态”入口，默认打开，仅超级管理员才有。
![](https://wework.qpic.cn/wwpic3az/757191_hkpYRXMCTfihLFk_1764922957/0)

3、同事动态不受通讯录规则的影响。
4、业务负责人可查看全企业全部成员的同事动态。
5、已在通讯录删除的成员，同事动态目前暂不会删除。
