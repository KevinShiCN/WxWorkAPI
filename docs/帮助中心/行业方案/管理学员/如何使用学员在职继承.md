# 如何使用学员在职继承

### [](https://open.work.weixin.qq.com)一、什么是在职继承

当成员或学员/学员群有变更时，企业可以将在职成员的学员和学员群分配给其他成员继续提供服务。
分配后，学员和接替的成员将在24小时后自动成为联系人；如果学员主动拒绝，接替成员还可选择主动添加。学员群群主将自动变更为接替的成员，同时原成员将退出群聊。

### [](https://open.work.weixin.qq.com)二、在职继承分配指引

当成员或学员有变更时，超级管理员或业务负责人可在【手机端->工作台->学员联系->在职继承】或【管理后台->学员联系与上下游->学员联系->在职继承】中需要转接的学员/学员群分配给其他成员。
![](https://wework.qpic.cn/wwpic3az/497814_hPH7Usc9QWeh-jH_1753435409/0)
分配后，学员将在微信中收到转接提醒
![](https://wework.qpic.cn/wwpic/56410_Mq81LkdIRpCDRY2_1663124793/0)
学员和接替的成员默认将在24小时后自动成为联系人；如果学员主动拒绝，接替成员还可选择主动添加。
![](https://wework.qpic.cn/wwpic3az/880562_qY8OzJtkSBCmHrU_1764753606/0)
注：为了学员不被重复打扰，如果新成员成功接替学员，则原跟进成员将无法再继续为该学员提供服务，学员发送消息给原跟进成员，系统将会自动引导学员联系新的接替成员；
如果新成员接替失败，则原成员还可继续为学员提供服务.

### [](https://open.work.weixin.qq.com)三、查看在职继承分配记录

在【手机端->工作台->学员联系->在职继承->右上角】或【管理后台->学员联系与上下游->学员联系->在职继承】中查看已分配的学员/学员群。
![](https://wework.qpic.cn/wwpic3az/115662_-XsfD9UDTKKBDgk_1764753615/0)
![](https://wework.qpic.cn/wwpic3az/181580_Pmz4W2bmRKq_9Po_1753435420/0)

### [](https://open.work.weixin.qq.com)四、在职继承的频率限制

在职成员的每位学员/学员群每90天内，最多可被转接2次。
注：
在职继承的次数等限制不影响离职继承功能的使用。
在职分配单次最高支持分配5000个学员。
在职分配后转接状态显示“转接失败”不占用90天2次次数。

### [](https://open.work.weixin.qq.com)五、在职继承同步学员详情信息场景

在职继承后，原添加人对该名学员的备注名、企业标签、描述、手机号、企业名称等备注和描述信息，都将同步共享给接受共享的企业成员。
注：
1、未到24小时自动分配，学员主动点击系统消息添加新成员，不会继承备注和描述等信息。
2、若新员工和接替的学员本身是好友关系，不会继承之前成员添加的备注信息（包括添加的标签）。
3、在职继承未到24小时自动分配，原成员新增或修改备注、描述等信息，在时间达到24小时后，新内容会一并同步给新成员。
4、如果接替的企业成员已经添加了部分的备注名、企业标签、描述、手机号、企业名称等备注和描述字段，则优先以接替的成员自己添加的显示。
5、若是在发起继承后成员主动添加了学员，也会继承之前成员添加的备注信息和标签。
