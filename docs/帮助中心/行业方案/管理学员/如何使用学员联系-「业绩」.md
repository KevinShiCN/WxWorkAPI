# 如何使用学员联系-「业绩」

**v4.1.32版本开始，学员联系「收款转化分析」升级为「业绩」。**

### [](https://open.work.weixin.qq.com)一、学员联系-业绩是什么？

业绩是企业微信提供的官方私域收款分析能力。旨在帮助企业更好地量化并了解企业通过企业微信向学员服务后的经营收款情况。

企业通过企业微信登录并选择需计入收款的微信支付商户号，成员服务的学员在对应商户号下线上线下渠道的交易，都可计入「业绩」。
●线上渠道包含：对外收款、小程序、网页、App、红包等其他
●线下渠道包含：线下门店（如POS收款、刷脸支付、刷掌支付等）

![](https://wework.qpic.cn/wwpic3az/113881_bKT1bQEAQIq1hRV_1765247438/0)

### [](https://open.work.weixin.qq.com)二、如何统计的？

企业在「业绩」中授权了微信支付商户号后，可配置适合自己业务场景的统计规则，选择学员被服务后1日或7日内的交易计入统计：
●成员为学员提供服务后，学员在1日或7日内在该商户号收款的线上、线下渠道产生交易，都将计入，并将归属给提供服务的成员。
●交易时间发生在服务时间之后的交易才会被计入。
●若有多位成员均为该学员提供过服务，则归属给距离交易时间最近的成员。
![](https://wework.qpic.cn/wwpic3az/230153_g49n4itOSpeM1YM_1765247442/0)

### [](https://open.work.weixin.qq.com)三、统计时间周期是什么？

每天24时统计当日的收款数据，并于第二日上午9时前更新展示。
若因网络原因产生数据延迟展示，可稍后查看。 

### [](https://open.work.weixin.qq.com)四、每个数据的定义是什么？

●**学员：**企业成员添加为好友的学员，和加入学员群的学员。
●**学员总数：**将所有学员汇总后去重计算。
●**付款学员：**仅统计学员的付款，非学员的付款不计入。
●**付款学员占比：**付款学员数/学员总数，四舍五入至小数点后2位。
●**单学员付款金额：**收款总金额/付款客户数，四舍五入至小数点后2位。

### [](https://open.work.weixin.qq.com)五、其他说明

1、若收款发生后至当天晚上24时，学员或群学员关系链发生变化（如在职继承、删除学员等），则收款归属收款时间前的关系链成员；
2、若收款发生后至当天晚上24时，成员被移除服务人员身份，或离职，则不做当日统计；
3、若管理规则下的成员发生变化，历史管理规则下的学员总数及付款学员数不会进行变化；
4、付款学员数按本规则统计所得，仅供参考，如与实际有出入，以实际情况为准。

### [](https://open.work.weixin.qq.com)六、如何在企业微信里统计商户号的业绩

可在【客户端->工作台->学员联系->企业本周业绩->计入业绩->去授权/邀请商户号管理员授权】授权绑定后，可查看商户号的收款情况。

管理员还可以在【管理后台->学员联系与上下游->学员联系->学员->计入业绩】使用企业微信或微信扫码授权商户号，计入业绩。

### [](https://open.work.weixin.qq.com)七、成员如何查看业绩

v4.1.32版本开始，支持成员自己在【客户端->工作台->学员联系->我的本周业绩】查看。

同时支持向管理员申请计入商户号的业绩。若成员自己是商户号管理员，可以自己授权商户号计入。![](https://wework.qpic.cn/wwpic3az/139748_S8V2fzgdRfORdg3_1765247449/0)
