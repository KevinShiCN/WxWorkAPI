# 企业如何查看成员添加的学员

### [](https://open.work.weixin.qq.com)一、如何查看成员添加的学员

企业的超级管理员可以查看企业学员列表，业务负责人可以看到自己管理范围内的成员所添加的学员列表。

在【手机端->工作台->学员联系->学员总数】中查看企业学员。
![](https://wework.qpic.cn/wwpic3az/923596_OCMZ9-3ITymI-sR_1764750370/0)

或在【管理后台->学员联系与上下游->学员->点击“总人数”】查看企业学员。
超级管理员暂不支持删除成员添加的学员，需要成员自主删除已添加的学员。
![](https://wework.qpic.cn/wwpic3az/104283_fDkf2c9JSc2vCi5_1753666423/0)

### [](https://open.work.weixin.qq.com)二、如何导出企业学员

目前超级管理员和业务负责人支持导出企业学员。管理员需要在管理后台导出。业务负责人可以在电脑端导出。

管理员可在【管理后台->学员联系与上下游->学员->点击“总人数”】可导出企业学员。
管理后台单次最多支持导出50万学员、筛选的“添加人”暂不支持超过13w。
![](https://wework.qpic.cn/wwpic3az/541597_c4Qksm-jR7-Ib_k_1753666426/0)
![](https://wework.qpic.cn/wwpic3az/294695_bLR4qP3aRNWv9rz_1753666432/0)

业务负责人可以在【电脑端->工作台->学员联系->学员概况】导出学员。（MAC暂不支持）
电脑端导出学员列表Excel表格上限: 最多50万行、最大2GB、单元格数最多200万 (最长列数*最长行数)。
![](https://wework.qpic.cn/wwpic3az/587172_ryz3alvOQ7-uLdq_1750055917/0)
