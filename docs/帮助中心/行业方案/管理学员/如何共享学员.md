# 如何共享学员

### [](https://open.work.weixin.qq.com)一、成员如何共享学员给同事？

成员可在查看学员信息时，点击右上角的“…”，选择共享给同事，操作共享学员给其他成员。
![](https://wework.qpic.cn/wwpic3az/802256_jo9oN6zFRPu8tsy_1765248756/0)

### [](https://open.work.weixin.qq.com)二、管理员如何共享企业学员给企业成员？

超级管理员可在【管理后台->学员联系与上下游->学员->点击“总人数”->选择对应的学员】进入学员详情操作共享学员。
![](https://wework.qpic.cn/wwpic3az/151205_mu73SAYFQtqQvKV_1753432239/0)
![](https://wework.qpic.cn/wwpic3az/980262_KAQRl1_4SVCI1nQ_1753432241/0)
![](https://wework.qpic.cn/wwpic3az/426224_arEQPZ9ITdmB_T4_1753432244/0)

### [](https://open.work.weixin.qq.com)三、哪些成员可以被共享学员？

成员需要有学员联系使用权限，才可被共享学员。

### [](https://open.work.weixin.qq.com)四、共享学员，添加成功后会同步学员详情的信息吗?例如备注名、标签、描述等。

共享学员后，原添加人对该名学员的备注名、企业标签、描述、手机号、企业名称等备注和描述信息，都将同步共享给接受共享的企业成员。
如果接受共享的企业成员已经添加了部分的备注名、企业标签、描述、手机号、企业名称等备注和描述字段，则优先以接受共享的成员自己添加的显示。
