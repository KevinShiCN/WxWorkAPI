# 如何查看学员流失情况

【学员流失情况】为企业提供流失数据，企业可根据流失数据来统计学员流失数量、原因和明细，根据学员流失情况及时调整获客方案。

常见的流失原因分为三种：
1、查看成员信息页后未添加：学员查看了成员的信息，没有操作添加成员；
（1）是否去重：A学员查看A成员不论多少次都只会计算为1位流失学员，A学员如果查看A成员后，又查看B成员，就会计算为2位流失学员，以此类推。
（2）学员查看了企业人员的信息后，发起好友请求，但企业人员没有通过好友请求，算流失学员。
（3）操作了在职继承/离职继承：学员当天查看了接替人的信息后，拒绝了继承，那么会计算流失。如果没有查看就拒绝了，不会计算流失。
（4）同一个学员在同一天内2次点击查看同一个成员，第一次只查看未添加，第二次查看且添加，这种不会计算流失学员。
（5）同一个学员在不同日期点击查看同一个成员，第一天只查看未添加，第二天查看且添加，这种只有第一天会计算1位流失学员。

2、成员账号异常导致添加失败：成员达到当日添加好友上限或账号被封禁，导致学员添加失败；
注：学员搜索企微用户手机号添加，但成员没有同意也会计算为学员流失情况。

3、企业当前外部联系人规模达到了上限：企业外部联系人规模达到上限，成员无法添加学员，导致添加学员失败。

超级管理员和业务负责人支持在【手机端/电脑端->工作台->学员联系->学员流失情况】看到管辖范围内的成员的数据。（Mac暂不支持）
