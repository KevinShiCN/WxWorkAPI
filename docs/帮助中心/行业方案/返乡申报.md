# 返乡申报

### [](https://open.work.weixin.qq.com)一、「返乡申报」功能简介

返乡申报是政务版的基础应用，可帮助基层政府快速**收集返乡人员行程信息和健康状况**，助力基层网格化疫情防控。

### [](https://open.work.weixin.qq.com)二、「返乡申报」功能的特点

1、 **居民直接在微信上填报**，简单方便
2、内置了常用模板，可一键启用
3、支持编辑题目，个性化收集信息（仅支持单位超管）
4、支持多种分享方式，**链接、二维码**均可
5、支持数据汇总和导出 excel，方便向上汇报

### [](https://open.work.weixin.qq.com)三、「返乡申报」功能设置入口

1、企业微信政务版->**工作台**->返乡申报
（1）打开「返乡申报」应用
（2）查看、修改模板
（3）分享链接给居民填写或保存并打印二维码，居民用微信扫码填写
（4）查看数据汇总
（5）导出数据，向上汇报
![](https://wework.qpic.cn/wwpic3az/992513_4VRpNV65TXSQGXV_1764659747/0)
2、企业微信政务版->**居民群->聊天工具栏**->返乡申报->发送申请表到群
