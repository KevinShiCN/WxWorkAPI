# 如何使用在职继承分配成员的学员群

### [](https://open.work.weixin.qq.com)一、什么是在职继承分配学员群

若在成员[创建了学员群](https://open.work.weixin.qq.com/help2/pc/20667)，成员业务发生变更的情况下，管理员可将成员的学员群批量分配给其他成员继续提供服务；也可将指定学员群分配给其他成员服务（新群主需是同企业人员且是学员联系和学员群的使用范围内的成员）。学员群群主将自动变更为接替的成员，原成员将退出群聊且无法继续服务群学员。

支持通过接口实现按业务场景分配在职成员的学员群，并查询学员群的分配记录。

### [](https://open.work.weixin.qq.com)二、分配在职成员的学员群操作方式

管理端入口：【管理后台->学员联系与上下游->学员联系->在职继承】，选择需要转接的学员群，点击待分配的群聊，选择转接的成员进行分配。
![](https://wework.qpic.cn/wwpic3az/463151_58_-Z92eRfyVbkh_1753413513/0)

手机端入口：管理员/负责人可在【手机端->工作台->学员联系->在职继承】操作分配。管理员可以分配所有成员的学员群，负责人只能分配自己管理范围内成员的学员群。
![](https://wework.qpic.cn/wwpic3az/13629_DAd0rtnjR8yUU8x_1765173471/0)
电脑端入口：【工作台->学员群->在职继承】，仅客户联系负责人在电脑端可见，若是超级管理员，则看不到“在职继承”入口。

### [](https://open.work.weixin.qq.com)三、在职继承学员群的频率限制

一个学员群90天内只能继承2次；
单次最多分配500个学员群。
注：学员群满500人时操作继承，会提示要随机移除一个学员，确认移除学员后邀请新群主加入群聊。
