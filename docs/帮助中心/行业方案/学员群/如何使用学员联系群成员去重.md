# 如何使用学员联系群成员去重

### [](https://open.work.weixin.qq.com)1、群成员去重操作入口

【管理后台->学员联系与上下游->学员联系->安全管控->群成员去重->开始去重】。
![](https://wework.qpic.cn/wwpic3az/308534_IOo2dCWrReuJ8us_1753424185/0)
![](https://wework.qpic.cn/wwpic3az/795161_tAfzrh-fTgmBjV6_1753424188/0)
【手机端->工作台->学员群->群成员去重->全部的学员群去重】。
![](https://wework.qpic.cn/wwpic3az/46350_dlEEVf9oRu2pXep_1764751323/0)

### [](https://open.work.weixin.qq.com)2、支持筛选关键词给对应的群成员去重

超管、业务负责人支持根据群主和群名关键词筛选最近活跃的2000个学员群中是否有同一个学员，筛选后选择要踢出该学员的学员群通知服务人员。

### [](https://open.work.weixin.qq.com)3、管理员点击了去重后成员侧需点击确认去重

成员在【客户端->消息页面的学员群应用通知->学员群收到通知去重】，点击进入后可以操作移出学员。
![](https://wework.qpic.cn/wwpic3az/554377_wdiDt1dfTf2A6qf_1764751329/0)
