# 如何查看学员群的成员动态

成员、超级管理员、学员联系业务负责人在【手机端/电脑端企业微信->工作台->学员群】中可查看最近有新增学员群的10位同事（头像、姓名、学员群数、群成员数）。
![](https://wework.qpic.cn/wwpic3az/418250_uGwo3vKTSgSofCl_1764747833/0)

超级管理员还可以在【手机端企业微信->工作台->学员群->右上角的三个点->显示成员动态】中自定义开启或关闭显示。开启后，成员可以在学员群中查看到最近创建了学员群的同事。

![](https://wework.qpic.cn/wwpic3az/67869_DH_snwb9SQuIB-f_1764747837/0)
