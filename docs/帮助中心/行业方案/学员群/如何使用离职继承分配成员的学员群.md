# 如何使用离职继承分配成员的学员群

若在成员[创建了学员群](https://open.work.weixin.qq.com/help2/pc/20667)，群主离职的情况下，学员群需由管理员分配新群主（新群主需是同企业人员且是学员联系和学员群的使用范围内的成员），若学员群群主离职，管理员未分配新的群主，则这个群的群主不会自动转让群主。

**分配路径：**
管理端入口：【管理后台->学员联系与上下游->学员联系->离职继承】，选择需要查看的离职人员，点击待分配的群聊。
![](https://wework.qpic.cn/wwpic3az/348010_0i_tRyOqRq2IA0e_1753412809/0)
![](https://wework.qpic.cn/wwpic3az/685233_7oF1twh5TC6D_Qw_1753433791/0)

手机端入口：管理员/负责人可在【手机端->工作台->离职继承】操作分配。管理员可以分配所有离职人员的学员群，负责人只能分配自己管理范围内离职人员的学员群。
![](https://wework.qpic.cn/wwpic3az/11725_pFdKwctoSjWZpBL_1765181622/0)

电脑端入口：【工作台->学员群->去分配】，仅客户联系负责人可在电脑端操作分配，若是超级管理员，仅可查看“我继承了离职成员的”学员/学员群。
