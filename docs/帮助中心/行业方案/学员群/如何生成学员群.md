# 如何生成学员群

### [](https://open.work.weixin.qq.com)一、主动创建学员群

在[学员联系使用范围内](https://open.work.weixin.qq.com/help2/pc/17903)的成员，发起的外部群会默认为学员群。
方式一：在【消息界面右上角“+” ->发起群聊->选择我添加的学员】
![](https://wework.qpic.cn/wwpic3az/74982_WZxEcnaHSSmaF9w_1764661118/0)
方式二：在【手机端->工作台->学员群->创建一个学员群】
![](https://wework.qpic.cn/wwpic3az/612726_6dVZKMewTu-mLFq_1764661122/0)

### [](https://open.work.weixin.qq.com)二、已有群聊确认为学员群

在企业微信侧创建的外部群，当群主把群主身份转让给在[学员联系使用范围内](https://open.work.weixin.qq.com/help2/pc/17903)的成员，新群主可以在把群聊确认为学员群，设置入口【群管理->标记为学员群】。
若成员不在[学员联系使用范围内](https://open.work.weixin.qq.com/help2/pc/17903)，创建了外部群，然后给成员添加了学员联系功能使用权限后，成员支持把这类型的群确认为学员群，设置入口【群管理->标记为学员群】。

### [](https://open.work.weixin.qq.com)三、创建学员群数量上限

未验证未认证的企业，所有外部群聊（包括学员群）加起来的总人数上限为100人；
未满100人时，也仅支持创建20个群聊；
已验证/认证的企业，群聊个数无限制。
