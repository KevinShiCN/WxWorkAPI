# 如何设置学员联系快捷回复

### [](https://open.work.weixin.qq.com)一、什么是学员群快捷回复

企业可为成员统一配置快捷回复，成员还可自己添加，添加后，成员可在与学员的聊天中点击“快捷回复”直接选取内容回复学员，提高工作效率。

### [](https://open.work.weixin.qq.com)二、如何配置学员群快捷回复

1、超级管理员支持在管理后台配置企业成员的快捷回复，可将常用短语配置为快捷回复，快捷回复支持批量导入。
管理后台操作路径：【管理后台->学员联系与上下游->学员联系->聊天工具->快捷回复】
![](https://wework.qpic.cn/wwpic3az/645372_4aZ9EqX3Si6uYpm_1753428874/0)
![](https://wework.qpic.cn/wwpic3az/644166_n6QHa_FKQ0C8uYh_1753428877/0)
2、业务负责人和超级管理员在手机端也支持配置快捷回复。
手机端操作路径：【手机端->工作台->学员联系/学员群->工具->快捷回复】
![](https://wework.qpic.cn/wwpic3az/837987_FJqFh2NDQkChWil_1764744207/0)

### [](https://open.work.weixin.qq.com)三、如何删除学员群快捷回复

超级管理员支持在【管理后台->学员联系与上下游->学员联系->聊天工具->快捷回复】中删除企业配置的快捷回复内容。
![](https://wework.qpic.cn/wwpic3az/745828_sDw-9NyZT9qkWLU_1753428883/0)
业务负责人和超级管理员支持在【手机端->工作台->学员联系/学员群->快捷回复】中删除快捷回复内容。
![](https://wework.qpic.cn/wwpic3az/505009_22AMI9x0RvSctDi_1764744304/0)
超级管理员支持删除企业新增的快捷回复以及自己新增的快捷回复内容；成员不能删除企业新增的快捷回复内容，可以删除自己添加的快捷回复内容；业务负责人支持删除自己新增的快捷回复内容。
![](https://wework.qpic.cn/wwpic3az/885120_8RVLh3f_S42zosn_1764744233/0)

### [](https://open.work.weixin.qq.com)四、学员群快捷回复上限说明

1、分组名最多支持12个汉字，单条快捷回复最多支持500个汉字；
2、超管、负责人、成员分别可以创建100个分组；
3、一个分组里的快捷回复最多存在200条。
