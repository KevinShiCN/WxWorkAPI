# 如何添加居民

### [](https://open.work.weixin.qq.com)一、添加居民方式

#### [](https://open.work.weixin.qq.com)1、成员主动添加居民的方式

成员主动添加居民支持通过面对面添加、搜索手机号添加、将名片分享到微信、互通群中点击群成员好友详情发起添加、通过好友名片等方式进行添加。（不支持搜索微信号添加）
![](https://wework.qpic.cn/wwpic3az/881477_0J6BE0CfSzS28Iv_1764916441/0)

#### [](https://open.work.weixin.qq.com)2、外部联系人转为居民

成员被设置在政民沟通权限范围前已添加的外部联系人会统一收纳到【通讯录->我的居民->其他外部联系人】中，需要手动点击确认为居民后，对应的外部联系人才会成为居民，成为居民后支持使用政民沟通功能，可被管理员查看。

确认为居民的方式：
【通讯录->我添加的居民->其他外部联系人->选择联系人->确认为居民】
![](https://wework.qpic.cn/wwpic3az/348530_1uXoiRF1S8aNYv8_1764916447/0)
【外部联系人个人信息页面->右上角的“…”->确认为居民】
![](https://wework.qpic.cn/wwpic3az/167989_FTSAC2bdTDO77CK_1764916520/0)

### [](https://open.work.weixin.qq.com)二、如何查看我的居民列表

在【通讯录->我的居民】中，可以查看到我添加的居民列表。
![](https://wework.qpic.cn/wwpic3az/863291_f3eH2yfHQkq3R4k_1764916524/0)

### [](https://open.work.weixin.qq.com)三、联系我

#### [](https://open.work.weixin.qq.com)1、居民联系「联系我」的添加联系方式

企业管理员可在【管理后台->政民沟通与上下游->政民沟通->联系我】中选择「联系我二维码联系」、「通过工卡联系」或「在小程序中联系」三种联系方式进行添加，并设置绑定的成员。
支持设置类型有：批量单人、单人、多人。
**单人：**单个二维码/小程序「使用成员」仅支持设置一个成员，设置后将生成这个成员的单人二维码。
**批量单人：**单个二维码/小程序「使用成员」支持设置多个成员/部门；设置后将批量生成这些成员的单人二维码，可通过压缩包形式下载。
**多人：**单个二维码/小程序「使用成员」最多可配置100个成员，多个居民扫码添加的时候，会将居民以同等概率分配给二维码中的服务人员。
![](https://wework.qpic.cn/wwpic3az/868107_Jfa4m4DVQp2gPSB_1753754097/0)
![](https://wework.qpic.cn/wwpic3az/222539_0jF4dXvrQkGWJ37_1753754101/0)

#### [](https://open.work.weixin.qq.com)2、政民沟通「联系我」的配置方式

完成联系方式的添加后，可根据企业需求将「联系我」二维码展示给居民，或将「联系我」按钮配置到小程序中。

#### [](https://open.work.weixin.qq.com)3、政民沟通「联系我」的使用方式

完成配置后，居民可通过扫描二维码或在小程序中点击按钮获得绑定成员的企业微信联系方式，企业可将添加方式设置为“无需成员确认自动成为好友”的模式。
![](https://wework.qpic.cn/wwpic3az/393233_IH2yGBPIRU-2nQe_1753754105/0)

#### [](https://open.work.weixin.qq.com)4、政民沟通多人「联系我」二维码的居民分配规则

政民沟通多人[联系我]二维码的居民分配规则是同等概率分配的。

### [](https://open.work.weixin.qq.com)四、政民沟通「欢迎语」的使用方式

#### [](https://open.work.weixin.qq.com)1、政民沟通-加居民自动下发欢迎语功能说明

欢迎语由管理员或业务负责人统一配置（成员无法自行修改），配置后，将对有「政民联系」使用权限的成员生效，居民添加成员为联系人后，也将收到该欢迎语。
支持欢迎语的消息类型为文本、图片、视频、文件、网页与小程序。附件支持添加9个图片/视频/文件/网页/小程序。
欢迎语字数上限为1000字，最多可配置100条欢迎语。

操作路径入口：
（1）企业管理员或业务负责人可在【手机端/电脑端->工作台->居民联系->欢迎语】中为成员配置欢迎语；
![](https://wework.qpic.cn/wwpic3az/499791_0v3zYMF6RAiMFd__1764916533/0)

（2）企业超级管理员登录【管理后台->政民沟通与上下游->政民沟通->添加居民->欢迎语】中进行配置。
![](https://wework.qpic.cn/wwpic3az/947111_JFGsTguwQXeCsyN_1753754111/0)
![](https://wework.qpic.cn/wwpic3az/484078_xhkMcD_2Qz2B7xH_1753754114/0)
![](https://wework.qpic.cn/wwpic3az/983007_U-GtPlQAS_yRkeJ_1753754116/0)

#### [](https://open.work.weixin.qq.com)2、政民沟通-加居民没有自动下发欢迎语的原因

（1）有可能在添加居民成功一刻，企业未设置欢迎语，在添加居民后新增的欢迎语对历史已添加的居民不会生效。

（2）有可能是欢迎语生效范围的原因，若是全部成员，需确保添加居民的成员在政民联系使用范围内；业务负责人添加的欢迎语，需确保添加居民的成员在业务负责人管理范围内，业务负责人自己也需要在自己的管理范围内才会生效自己设置的欢迎语。

（3）有可能成员登录的企业和管理员查看的企业不是同个企业，成员添加的居民实际是没设置欢迎语的。

（4）欢迎语可能会出现有5秒左右的延迟，当添加成功后在下发欢迎语之前成员给居民发了消息，欢迎语不会下发。

（5）成员和居民双方不是好友的情况下，好友添加成功后，添加成功后会收到欢迎语。
单向好友的情况， 如果是居民侧删除企业微信联系人，企业微信侧没有删除这个客户，重新添加会下发欢迎语； 如果是企业微信联系人删除了这个居民，居民侧没有删除这个企业微信联系人，则不会下发欢迎语。

#### [](https://open.work.weixin.qq.com)3、政民沟通欢迎语支持带参吗

便于企业给不同成员在欢迎语中配置不同的H5和小程序页面，提供链接中带成员UserID的参数。
比如：
小程序pagepath应该是这样的格式：pages/index orderid=123123123
所以也可以直接加上userID的参数：wxwork_userid=xxxxxx
