# 如何使用居民通知

### [](https://open.work.weixin.qq.com)一、居民通知功能介绍

1、针对一级行业类型为“政府”的企业，可见该功能，可群发通知给政府管理的居民或居民群，可统计居民阅读及签字确认情况并导出统计明细。
2、居民通知发送成功后，支持通过客户端和管理后台导出通知详情，包括：已读、未读；已确认、未确认；已签字、未签字的状态。mac暂不支持。
![](https://wework.qpic.cn/wwpic3az/698110_DDFH6RWCReG4ylv_1765418108/0)
3、应用入口：iOS、安卓、win端、管理后台均可见该功能入口，mac则暂不支持。
【手机端/win端->工作台->居民通知；管理后台->政民沟通与上下游->政民沟通->居民通知】
4、支持居民联系管理规则负责人管理居民通知，可创建居民通知，要求管理范围内的成员对居民群发通知。

### [](https://open.work.weixin.qq.com)二、发送给居民群

1、发送对象：居民群。
2、支持发送图片；文字上限10000字符，图片最多支持10张，单张图片不能大于10M，单张图片的像素无要求。
3、一天可发多条通知，暂无发送次数限制；
群发到“我管理的居民群”，单次上限1000个，分次无上限；
群发到“政府管理的居民群”，单次上限1000个群，分次均无上限。
注：管理端发表居民通知给居民群时，若超过1000个群聊，会提示服务器异常，请稍后再试。
4、成员仅可群发到自己是群主的居民群（即只能发送到“我管理的居民群”）。
5、超管/负责人可以发到自己是群主的居民群或管理范围内成员的居民群，发送时可选发送到“我管理的居民群”或者“政府管理的居民群”；若选择群发到“政府管理的居民群”，会通知群主确认，群主确认后才会下发到居民群。
6、支持按标签筛选发送范围。
7、发送前可以让居民确认，可选“仅需居民阅读通知”、“仅需居民点击确认”、“仅需居民确认签字”。
8、 发送前可以设置自动提醒或不提醒；也可以在发送后查看通知详情，手动提醒居民阅读/确认通知。
![](https://wework.qpic.cn/wwpic3az/347133_JsihGqg6TmGtm6O_1765418116/0)

### [](https://open.work.weixin.qq.com)三、发送给居民

1、发送对象：已添加的居民（仅居民联系范围内的才支持群发）。
2、支持发送图片；文字上限10000字符，图片最多支持10张，单张图片不能大于10M，单张图片的像素无要求。
3、一天可发多条通知，暂无发送次数限制；
群发到“我添加的居民”，单次上限200，分次无上限；
群发到“政府管理的居民”，单次支持10万，分次均无上限。
4、成员仅可群发给自己添加的居民（即只能发送到“我添加的居民”）。
5、超管/负责人可以发给自己添加的居民或管理范围内的居民，发送时可以选择发送到“我添加的居民”或者“政府管理的居民”；若选择群发到“政府管理的居民”，会通知成员确认，成员确认后才会下发给居民。
6、支持按群名关键字筛选发送范围。
7、发送前可以让居民确认，可选“仅需居民阅读通知”、“仅需居民点击确认”、“仅需居民确认签字”。
8、无自动提醒的设置，可以在发送后查看通知详情，手动提醒居民阅读/确认通知。
9、居民通知显示：备注->群昵称->微信昵称。
![](https://wework.qpic.cn/wwpic3az/404330_KO8NJZdVQuGgxbH_1765418121/0)

### [](https://open.work.weixin.qq.com)四、查看/修改/删除通知记录

居民通知的创建人可在手机端/win端/管理后台的居民通知中查看居民通知记录，同时支持在居民群/居民通知记录中，点击通知详情的右上角“···”，操作修改通知内容或删除已发送的通知。（删除居民通知暂无时间限制）
具体操作路径如下：
1、手机端：工作台->居民通知->查看群发记录
![](https://wework.qpic.cn/wwpic3az/75788_K_FbImP-R9GOF2W_1765418125/0)
2、电脑端：工作台->居民通知->查看群发记录

![](https://wework.qpic.cn/wwpic3az/982794_ZOdiH4o7SBOlQRZ_1750151519/0)
3、管理后台：政民沟通与上下游->政民沟通->居民通知->查看群发记录
