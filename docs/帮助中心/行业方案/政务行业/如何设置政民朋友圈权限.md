# 如何设置政民朋友圈权限

### [](https://open.work.weixin.qq.com)一、设置朋友圈使用权限

**需要有朋友圈使用权限的成员才能在手机端使用朋友圈功能。**
超级管理员在管理后台【政民沟通与上下游->政民沟通->权限配置->朋友圈使用范围】
![](https://wework.qpic.cn/wwpic3az/421371_Cy3nrvxOS_KwT3X_1753690115/0)
或手机端【工作台->政民朋友圈->配置->配置使用范围和管理规则】入口给成员添加朋友圈的使用权限。
![](https://wework.qpic.cn/wwpic3az/654425_NYsJLiUXToanH3t_1765245027/0)

### [](https://open.work.weixin.qq.com)二、设置朋友圈负责人

超级管理员在管理后台【政民沟通与上下游->政民沟通->权限配置->朋友圈管理规则】
![](https://wework.qpic.cn/wwpic3az/966706_Ip9vEl3bRciAxBy_1753690121/0)
或手机端【工作台->政民朋友圈->配置->配置使用范围和管理规则】入口给成员添加朋友圈的管理权限。
![](https://wework.qpic.cn/wwpic3az/514154_Yvm12bS8TZeoDMx_1765245037/0)
负责人管理权限可设置【查看成员全部的发表】、【发表到成员已服务居民的朋友圈】、【配置封面和签名素材库】。
![](https://wework.qpic.cn/wwpic3az/152627_IYY-GMsmTR-3ihq_1753690128/0)

### [](https://open.work.weixin.qq.com)三、关闭成员的政民朋友圈入口

超级管理员可在【手机端->工作台->管理企业->工作台显示->成员可见->居民联系】处打开开关，打开后，仅超级管理员和具有使用权限的成员可见政民朋友圈入口、无权限的成员将无法在工作台上看到该入口。
![](https://wework.qpic.cn/wwpic3az/609359_E6gdLcQVTLCYrsR_1765245045/0)
