# 如何群发消息到居民群

### [](https://open.work.weixin.qq.com)一、群发助手基本介绍

1、具有居民联系和居民群使用权限的成员，都可以使用「居民群群发」将消息批量发送到居民群；
2、超级管理员和业务负责人还可统一创建群发内容，通知员工选择居民群并发送成员每次仅可选择自己作为群主的、最近活跃的2000个居民群进行群发。最近活跃指最近创建或最近发过消息的居民群。
3、群发消息和发表朋友圈时，支持按标签筛选要发送或者不发送的居民。
4、群发额度是以居民/居民群维度进行计算的，个人群发和企业群发共用群发频率，支持企业统一创建发送，成员也可自己创建发送。
企业管理员支持在【手机端->工作台->群发助手->右上角“···”->设置群发规则】灵活配置群发规则为“每日可接收1条” 或 “每周可接收7条” 或 “每月可接收条数为当月天数”。
配置后，将根据配置的规则，每天可群发1条消息 / 每周可群发7条消息 / 每月可群发当月天数条消息给每位居民/每个居民群。
例如：
（1）管理员配置群发规则为“每日可接收1条”，企业每天可群发1条消息给每位居民/每个居民群；
（2）管理员在周一配置群发规则为“每周可接收7条”，当周每位居民/每个居民群可接收的群发消息数量为7条，每周一0点更新该数据；
（3）管理员在5月1日设置群发规则为“每月可接收条数为当月天数”，当月每位居民/每个居民群可接收的群发消息数量为31条。

### [](https://open.work.weixin.qq.com)二、群发助手发送入口

管理员支持在管理端或在手机端创建企业群发。
管理端路径：【管理后台->政民沟通与上下游->政民沟通->群发工具->群发消息到政府管理的居民群】。
![](https://wework.qpic.cn/wwpic3az/227539_MXK8LQE2SJWo2Mo_1753674452/0)
成员/管理员手机端操作群发路径：【手机端->工作台->群发助手】
![](https://wework.qpic.cn/wwpic3az/321222_8sBDAgg5TV2LES-_1765173277/0)

### [](https://open.work.weixin.qq.com)三、如何停止企业群发

4.0.8版本开始管理员可停止企业群发，停止后未发送的成员将无法继续发送该内容。
若企业消息已发送完毕，没有待发送的成员时，则不会有停止发送入口。
操作入口：
【工作台->群发助手->右上角“···”群发记录->查看对应记录详情->右上角“···”->停止发送】
![](https://wework.qpic.cn/wwpic3az/153246_WVX_ZBBtSxCo8PP_1765173282/0)

【管理后台->政民沟通与上下游->政民沟通->群发工具->群发消息给居民群->群发记录->查看对应记录详情->右边的三个点->停止发送】
![](https://wework.qpic.cn/wwpic3az/604387_oL9bVD19QpyRxSU_1753674457/0)

### [](https://open.work.weixin.qq.com)四、内容上限与格式

1、文字支持输入4000个汉字。

2、消息支持发送文字+图片/视频/网页/小程序/文件消息，图片+视频+网页+小程序+文件支持多种类型同时选择，加起来最多选9个，格式暂无限制。

3、群发图片的时候，不会因为尺寸、大小自动进行裁剪、压缩。

4、图片限制长宽 10000*20000（超过可能出现上传失败/一直显示正在上传）

5、当前Mac端只支持添加一个附件，该情况后续优化。

6、客户端视频支持100m，手机端图片大小没限制；管理端视频/图片/文件支持200m，管理端文件仅支持从本地上传，手机端文件支持从本地/微盘上传，本地上限200m，微盘上传无上限。
