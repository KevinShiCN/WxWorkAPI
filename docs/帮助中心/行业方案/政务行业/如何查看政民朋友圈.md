# 如何查看政民朋友圈

### [](https://open.work.weixin.qq.com)一、居民查看朋友圈

#### [](https://open.work.weixin.qq.com)1、查看入口

成员发布朋友圈后，微信居民在朋友圈以及查看成员个人详情可查看成员推送的朋友圈内容。
![](https://wework.qpic.cn/wwpic3az/544084_VCs1y5n3QBahJas_1765180427/0)

#### [](https://open.work.weixin.qq.com)2、查看不到居民朋友圈的原因

（1）一个居民每天只能看到同个人发送的3条朋友圈，这三条朋友圈可以是企业发表的，也可以是成员发自己发表的。若管理员对这个成员的居民在当日推送超过3条，则成员可以发送成功，居民则看不到这个朋友圈。
（2）若企业中的多个成员添加了同一个居民，管理员对这些添加了同一个居民的成员都推送了政府朋友圈，则居民侧会看见最先点击确认发送成员推送的企业朋友圈内容。后面点击确认的成员会显示发送成功，实际居民还是只能看见第一个点击确认推送的成员发表的内容。
（3）微信居民在查看到企业微信联系人发送的朋友圈后，对应的朋友圈消息右上角可“不感兴趣”：点击后对同一个服务人员的圈第一次点7天不可见，第二次点30天不可见，第三次点一年(自然年)不可见。
（4）发表朋友圈后，可能会有10分钟的延迟。
（5）居民的微信需要更新到微信最新版本，Android版本7.08版本或以上，ios版本需要7.09版本或以上。
（6）需要检查看不到朋友圈的这个微信是否与成员是双向好友，单向好友的情况看不到朋友圈。
（7）发布的朋友圈图片或视频大小不支持超过10M，若超过大小在微信侧朋友圈会看不到该内容。
（8）发送朋友圈时可通过输入#号加文字，添加微信话题，如果居民使用的微信版本低于7.0.17，则#号的话题内容不会显示蓝色。
（9）企业超级管理员在【管理后台->政民沟通与上下游->政民沟通->政民朋友圈->查看对应的政民朋友圈->查看可见的居民】是否显示“无可见居民”，显示无可见居民，对方是看不到的。
（10）若朋友圈带有图片/链接/视频，微信侧暂不支持展示heic/webp/exif类型的图片/链接/视频，建议重新编码后重试。我们该在后续优化该问题。
（11）由于企业未验证未认证，企业最多100位居民（包括非居民身份的外部联系人），所以朋友圈只能对这100居民可见。若需要添加更多居民，企业需认证。
已验证/认证企业，可展示居民数为所有已添加居民；认证到期未续期，已发表的朋友圈不受影响，新发表的朋友圈最多可展示100个居民。

### [](https://open.work.weixin.qq.com)二、成员查看朋友圈

成员支持在【手机端->工作台->政民朋友圈】页面查看已发表的朋友圈。
![](https://wework.qpic.cn/wwpic3az/153408_ujaqTFchQCWIUh5_1765180441/0)

### [](https://open.work.weixin.qq.com)三、负责人查看朋友圈

负责人支持在【手机端->工作台->政民朋友圈->我的发表】页面查看自己已发表的朋友圈。
可以在【工作台->政民朋友圈->统计->成员发表概括】查看管理范围内成员已发表的个人和政府朋友圈。
![](https://wework.qpic.cn/wwpic3az/901046_-Op5SqhNQoWFX3B_1765180413/0)

### [](https://open.work.weixin.qq.com)四、管理员查看朋友圈

超级管理员支持在管理后台和手机端查看企业成员发表的朋友圈，支持根据部门/成员、选择发布类型、时间进行筛选查看企业成员的朋友圈。
【手机端->工作台->政民朋友圈->统计->成员发表概况】
![](https://wework.qpic.cn/wwpic3az/600371_H7l3au9zQ6Swyvk_1765180396/0)
【管理后台->政民沟通与上下游->政民沟通->政民朋友圈】
![](https://wework.qpic.cn/wwpic3az/945422_Y43OtaHoSRWi0RH_1753690679/0)
