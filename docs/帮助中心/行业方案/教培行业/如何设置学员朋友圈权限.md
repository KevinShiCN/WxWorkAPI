# 如何设置学员朋友圈权限

### [](https://open.work.weixin.qq.com)一、设置朋友圈使用权限

超级管理员可在【管理后台->学员联系与上下游->学员联系->权限配置->朋友圈使用范围】
![](https://wework.qpic.cn/wwpic3az/983986_IYtc7SaaQBm6Kwo_1753431500/0)
或【手机端->工作台->学员朋友圈->配置->配置使用范围和管理规则】入口给成员添加朋友圈的使用权限。
![](https://wework.qpic.cn/wwpic3az/240294_1ipYYeb1Tp60tIp_1764755044/0)
需要有朋友圈使用权限的成员才能在手机端使用朋友圈功能。

### [](https://open.work.weixin.qq.com)二、设置朋友圈负责人

超级管理员可在【管理后台->学员联系与上下游->学员联系->权限配置->朋友圈管理规则】
![](https://wework.qpic.cn/wwpic3az/490814_zjwV4g1gT1WWuVB_1753431511/0)
或【手机端->工作台->学员朋友圈->配置->配置使用范围和管理规则】入口给成员添加朋友圈的管理权限。
![](https://wework.qpic.cn/wwpic3az/377652_GHdfi9ZgSVioQs5_1764755050/0)
负责人管理权限可设置【查看成员全部的发表】、【发表到成员服务的学员的朋友圈】、【配置封面和签名素材库】。
![](https://wework.qpic.cn/wwpic3az/167257_JWqaTFXqSiiN5qN_1753431518/0)

### [](https://open.work.weixin.qq.com)三、关闭成员的学员朋友圈入口

超级管理员可在【手机端->工作台->管理企业->工作台显示->成员可见->学员联系】关闭，关闭后，仅超级管理员和具有使用权限的成员可见学员朋友圈入口、无权限的成员将无法在工作台上看到该入口。
![](https://wework.qpic.cn/wwpic3az/32565_-ccoeI1bTo2X0Bu_1764755056/0)
