# 如何使用迁移微信中的学员群

### [](https://open.work.weixin.qq.com)一、如何使用「迁移微信中的学员群」

1、企业微信是 3.1.10 及以上版本
2、微信更新到了最新版本（iOS 7.0.14以上，安卓7.0.16以上）
3、企业微信绑定了微信
4、企业微信和微信当前登录的微信是一致的
5、行业为「教育->培训机构」行业
6、成员需在学员群的使用范围内，可让管理员在【手机端企业微信->工作台->学员群->配置】中查看使用范围。

操作入口：【手机端->工作台->学员群->迁移微信中的学员群】
![](https://wework.qpic.cn/wwpic3az/567269_WZ-voq49TWm7epx_1765182710/0)

### [](https://open.work.weixin.qq.com)二、可以迁移哪些群为「迁移微信中的学员群」

可以迁移用户自己是群主的所有微信群。

### [](https://open.work.weixin.qq.com)三、我双开微信，跳去微信报错怎么办？

重新登录微信账号，确保跳转的微信登录的是企业微信绑定的微信账号。

### [](https://open.work.weixin.qq.com)四、迁移群有哪些限制？

1、一次最多同时可以选择5个群聊进行迁移，如有多个群聊，可操作多次。

2、需要被迁移的微信群人数不大于499人。

3、未验证企业，最多可以迁移5个群聊，已验证/认证企业，最多可迁移数为使用企业微信的人数的10倍，企业可通过提高DAU峰值提升可迁移的群聊的总数上限。
注：DAU是指企业微信客户端使用人数。管理员可在【手机端->工作台->管理企业->成员使用分析】中查看企业微信客户端使用人数。（iOS暂不支持）
![](https://wework.qpic.cn/wwpic3az/574047_276o2HATTrGDv-H_1765875962/0)

或在【管理后台->安全与管理->管理工具->使用分析->成员使用统计->企业微信客户端】查看。
![](https://wework.qpic.cn/wwpic3az/15911_9zCfEhz2SGeG1fp_1765876466/0)

### [](https://open.work.weixin.qq.com)五、学员需要下载企业微信么？

不需要。迁移群后，学员留在微信上即可和企业微信上的工作人员消息互通。

### [](https://open.work.weixin.qq.com)六、如何避免两个 APP 的消息重复打扰？

迁移微信群到企业微信，当您的企业微信身份入群后，由于您的微信账号和企业微信账号同时在群内，所以会出现两个APP重复接收消息的情况，为避免重复提醒打扰，系统自动将您的微信号移出群聊。

### [](https://open.work.weixin.qq.com)七、如何恢复在微信上管理该群？

如果想要恢复群主为自己的微信号，您可以在企业微信的学员群中，点击【右上角->群管理->转让群主】，将群主转让回给自己的微信。 这样就可以继续在微信上以群主身份进行管理了。
注：转让群主给微信账号仅支持在微信管理群聊，不支持将群聊转回为普通微信群，且不支持去掉企业微信群聊标识。
