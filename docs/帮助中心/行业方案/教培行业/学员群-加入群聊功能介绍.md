# 学员群-加入群聊功能介绍

### [](https://open.work.weixin.qq.com)一、加入群聊功能基本说明

#### [](https://open.work.weixin.qq.com)1、如何获取进群方式

学员通过扫描群二维码、立牌或点击小程序上的按钮，即可获取入群方式，进入企业的学员群。

#### [](https://open.work.weixin.qq.com)2、加入群聊二维码的有效时间

设置加入群聊的二维码不删除的情况下，二维码一直有效，不会过期。

#### [](https://open.work.weixin.qq.com)3、加入群聊功能群上限问题

- 支持设置「群人数达上限时，自动创建新群」，当群聊人数达到200人时（扫码加入或手动邀请加入会触发），会以原群主身份裂变创建新的群聊。当群聊人数达到200后，可以让群里的人手动邀请人员加入群聊。学员群的人数上限还是500人。
- 二维码上限是1000个，小程序的上限是1000个。
- 一个二维码最多支持5个群。

#### [](https://open.work.weixin.qq.com)4、加入群聊功能使用限制

未验证企业可以在管理端配置加入群聊功能，手机端暂不支持。已验证/认证企业支持在管理端、电脑端和手机端配置。

### [](https://open.work.weixin.qq.com)二、加入群聊功能设置入口

1、超级管理员支持在【管理后台->学员联系与上下游->学员联系->添加学员->加入群聊】
![](https://wework.qpic.cn/wwpic3az/961204_22KqBvQDTA667ZT_1753417955/0)
![](https://wework.qpic.cn/wwpic3az/482233_wT3Eps6YTsah4vP_1753417959/0)
![](https://wework.qpic.cn/wwpic3az/750806_Xs6-mTySThmidK9_1753417961/0)
2、成员、超级管理员、负责人支持在【电脑端/手机端->工作台->学员群->加入群聊】进行设置
![](https://wework.qpic.cn/wwpic3az/473968_9MuGMfuyTCapkG5_1750057780/0)

### [](https://open.work.weixin.qq.com)三、新增学员联系加入群聊二维码可使用群模板

在新增学员加入群聊二维码时，可以在入群设置中点击【设置新群的群名称、群管理员等】进行选择对应的[群模板](https://open.work.weixin.qq.com/help2/pc/17963?person_id=1)，设置后，自动创建的群聊会默认使用群模板的内容进行配置。

同时也支持「加入群聊」自动创建新群时，单独设置新群的群名。
![](https://wework.qpic.cn/wwpic3az/560727_eERy9Xl_SOapP4O_1753417968/0)

### [](https://open.work.weixin.qq.com)四、扫【学员联系->加入群聊】二维码提示“群已满人”

1、若学员已在加入群聊二维码关联的群中。已在群中然后又扫码，所在的群满员时，则会出现对应提示。

2、当群聊人数未达200人，群主开启了“群聊邀请确认”开关，会出现这个提醒。建议联系群主关闭该开关，成员可以重新扫这个二维码加入群聊。

3、当加入群聊二维码仅有A群的情况下，A群满人了，自动创建了B群，B群没满人，但是管理员把B群移除了这个二维码，这时候扫加入群聊二维码会提示“群已满人”且不会自动创建新的群聊。可以尝试在管理后台新增一个群，待新增的群满人后，会自动创建新的群

4、电脑端创建的体验加入群聊小程序按钮，生成的二维码，暂不支持裂变新的群聊。群满200人后不支持扫码进群。

5、未验证/认证企业的学员群人数/个数达到上限，会导致裂变失败。

6、扫码的二维码需要是加入群聊该功能生成的二维码，非群聊本身的二维码。

7、创建加入群聊二维码的成员需要在外部沟通管理->成员使用权限内，若不在该范围，请添加至范围后重新创建二维码尝试。
