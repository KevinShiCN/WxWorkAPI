# 如何使用离职继承分配成员的学员

### [](https://open.work.weixin.qq.com)一、什么是离职继承

分配离职成员的学员，是企业微信提供的用于企业管理离职成员所添加学员的工具，支持管理端手动分配以及API接口分配。
成员离职后，如果他还有跟进中的学员，可以将学员分配给其他成员继续跟进。

### [](https://open.work.weixin.qq.com)二、分配离职成员的学员操作方式

从通讯录将成员删除后，如果他有已添加的学员，可以在【管理后台->学员联系与上下游->学员联系->离职继承】中分配给其他服务人员继续跟进；
也可以在【手机端->工作台->离职继承】中对离职成员的学员和学员群进行分配。

![](https://wework.qpic.cn/wwpic3az/345029_vp0ZwmU_Rq6TW5a_1764913879/0)
（2） 选择已离职成员的学员，可以分配给其他服务人员继续跟进，支持选择部分学员、全部学员和学员群进行分配。

### [](https://open.work.weixin.qq.com)三、学员侧的表现

分配后，离职成员的学员会收到一条通知，如果学员24小时内不主动点击「暂不添加」，将在24小时后自动添加新接替的成员为联系人。接替完成后，成员也可查看所有学员的添加情况。

若学员有服务需求，学员可以在提示页面点击接替成员名字主动操作添加发送好友验证，提前添加成员，服务不中断。

若学员点击了暂不添加，显示学员已拒绝后，接替者可以在【消息列表->离职继承->“学员继承完成”消息->学员已拒绝】找到对应学员重新添加，在【工作台->离职继承->我继承的学员与学员群】该入口会无法重新添加。
![](https://wework.qpic.cn/wwpic/613381_ymqyhKzOTHSA12t_1663125831/0)

### [](https://open.work.weixin.qq.com)四、离职继承的限制

支持在手机端、管理端批量分配离职成员的学员，单次最高支持分配5000个学员。当分配的数量超出上限时，在分配页面可一键自动选择5000个学员分配。若离职成员学员数超过5000个，可以尝试多次操作分配。

### [](https://open.work.weixin.qq.com)五、离职继承，添加成功后会同步学员详情的信息吗？例如备注名、标签、描述等。

离职继承后，原添加人对该名学员的备注名、企业标签、描述、手机号、企业名称等备注和描述信息，都将同步共享给接受共享的企业成员。
注：
1、未到24小时自动分配，学员主动点击系统消息添加新成员，不会继承备注和描述等信息。
2、若新员工和接替的学员本身是好友关系，不会继承之前成员添加的备注信息（包括添加的标签）。
3、如果接替的企业成员已经添加了部分的备注名、企业标签、描述、手机号、企业名称等备注和描述字段，则优先以接替的成员自己添加的显示。
4、若是在发起继承后成员主动添加了学员，也会继承之前成员添加的备注信息和标签。
