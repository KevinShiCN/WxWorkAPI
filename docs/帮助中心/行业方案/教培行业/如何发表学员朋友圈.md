# 如何发表学员朋友圈

### [](https://open.work.weixin.qq.com)一、企业发表到学员的朋友圈

1、超级管理员或有发表到成员学员的朋友圈权限的负责人支持创建朋友圈内容，将活动信息、产品动态、专业知识等内容发表到学员的微信朋友圈，并与学员评论互动。
![](https://wework.qpic.cn/wwpic3az/575463_bDqFoGciSJC3k5Z_1764914890/0)

2、超级管理员支持在管理后台【学员联系与上下游->学员联系->群发工具->企业发表到学员的朋友圈】进行新增朋友圈。
![](https://wework.qpic.cn/wwpic3az/934762_oDRWIjdeScuBjQd_1753430452/0)

3、一个学员每天只能看到同个人发送的3条朋友圈，这三条朋友圈可以是企业发表的，也可以是成员发自己发表的。内容支持文字或文字+图片/视频/网页。
![](https://wework.qpic.cn/wwpic3az/610340_F7vy06YaT2mL1AK_1753430464/0)
4、超级管理员或有发表到成员学员的朋友圈权限的负责人也可以在手机端【工作台->学员朋友圈->企业发表到学员的朋友圈->发表】进行新增企业朋友圈。
![](https://wework.qpic.cn/wwpic3az/757220_hyxqEt0YQ3eRhfL_1764914899/0)

5、超级管理员或应用负责人发送了企业发表到学员的朋友圈内容后，成员会在学员联系应用收到对应的发送提醒，可以在【朋友圈->企业通知/通知】中查看对应的内容，点击【发布】后，这条内容会发布到学员朋友圈。
![](https://wework.qpic.cn/wwpic3az/857972_XQqCQ7dmTwmIDK9_1764914905/0)

### [](https://open.work.weixin.qq.com)二、成员编辑内容发表到学员朋友圈

1、在手机端【工作台->学员朋友圈->我发表的】入口可新增个人朋友圈。成员编辑好内容后、可按标签选择要展示的学员，发表后，学员即可在微信朋友圈中查看、还可以点赞和评论成员发表的内容。

2、每个成员支持对同个学员每日发送3条个人朋友圈。一次支持选择发送朋友圈的学员数量是20000人。

![](https://wework.qpic.cn/wwpic3az/759160_TRnOHW5HRB-LWV0_1764914909/0)

### [](https://open.work.weixin.qq.com)三、学员朋友圈可展示学员限制/发送朋友圈提示“当前没有满足发表条件的学员”

1、**朋友圈发送频率：**一个学员每天只能看到同个人发送的3条朋友圈，这三条朋友圈可以是企业发表的，也可以是成员发自己发表的。

2、**学员朋友圈可展示的学员数：**
**未验证/认证企业**，企业学员数最多只能添加100人，因此朋友圈内容最多只能对100个学员可见。
**已验证/认证企业**，可展示学员数为所有已添加学员；认证到期未续期，已发表的朋友圈不受影响，新发表的朋友圈最多可展示100个学员。

3、若发送朋友圈提示“当前没有满足发表条件的学员”，请更新企业微信到最新版本后重试。
若更新版本后仍未恢复，请确认：
（1）有可能是超过可发送的限制，一个学员每天只能看到同个人发送的3条朋友圈，这三条朋友圈可以是企业发表的，也可以是成员发自己发表的。（发表成功后再删除也会占用次数）
（2）成员需要有添加微信联系人，且联系人是学员，可查看【手机端->通讯录->我的学员】是否有微信学员。
（3）成员需要在[学员朋友圈权限](https://open.work.weixin.qq.com/help2/pc/17954?person_id=1)使用范围内。

### [](https://open.work.weixin.qq.com)四、设置朋友圈封面

1、支持在朋友圈页面设置封面及个性签名。
2、【手机端->工作台->学员朋友圈】，新增封面素材库和签名素材库，有朋友圈管理权限的成员可配置素材，成员在设置封面及签名时可直接使用。
3、朋友圈背景推荐尺寸是1028 x1028。
![](https://wework.qpic.cn/wwpic3az/425147_sl6DL0k-RdmChhu_1764914914/0)

### [](https://open.work.weixin.qq.com)五、学员朋友圈如何停止发表

企业群发的朋友圈支持停止发表，未发表成员不会再收到发表提醒。

管理员可在【管理后台->学员联系与上下游->群发工具->企业发表到学员的朋友圈->企业发表记录->选择对应的朋友圈记录->点击右侧“…”->停止发送】（仅支持停止发表一年内的朋友圈，大于一年的，需在手机端操作）
![](https://wework.qpic.cn/wwpic3az/424470__XB7XitTQ_ectld_1753430475/0)
![](https://wework.qpic.cn/wwpic3az/263773_pJefyP-7SaS_tWp_1753430478/0)

【手机端->工作台->学员朋友圈->统计->待成员发表的->选择对应的朋友圈记录进入->右上角“…”->停止发表】
![](https://wework.qpic.cn/wwpic3az/773719_Nt2hNvw5Q0SrGt0_1764914921/0)
