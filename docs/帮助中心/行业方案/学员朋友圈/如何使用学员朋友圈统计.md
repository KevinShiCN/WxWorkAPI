# 如何使用学员朋友圈统计

### [](https://open.work.weixin.qq.com)一、如何查看朋友圈统计

超级管理员和业务负责人在【手机端->工作台->学员朋友圈->统计】中可以查看管理范围内当天成员发布朋友圈概况。
![](https://wework.qpic.cn/wwpic3az/111630_kPawbTtaQ8yMXM5_1765245411/0)

### [](https://open.work.weixin.qq.com)二、如何接收成员每日发表概况

超级管理员可以在【手机端->工作台->学员朋友圈->配置】中，开启【接收成员每日发表概况】，开关默认关闭。
开启后超管和业务负责人可以在每天18:00后在【朋友圈】应用收到成员每日发表概况推送。
![](https://wework.qpic.cn/wwpic3az/658389_OXVxjLTZSCKBQbm_1765245481/0)
