# 如何进行报餐管理

### [](https://open.work.weixin.qq.com)一、报餐基本说明

超级管理员、分级管理员可配置。配置内容修改后会立即生效。
1、 要统计的餐食：可设置每天需要收集报餐的餐食内容。默认为午餐、晚餐。用户可自定义编辑增加或删减选项。若修改了配置内容会立即生效，并将通知已填写过当前报餐的成员重新填写。
2、 每周几用餐：可设置每周需要报餐、需要收集用餐情况的天数。
3、 截止报餐时间：可设置报餐填写的截止时间。目前仅支持设置用餐前一天的具体时间，截止报餐时间需要晚于自动提醒成员中的提醒时间。

### [](https://open.work.weixin.qq.com)二、自动提醒成员报餐

配置内容修改后会立即生效。
1、 提醒时间：可设置自动给成员定时发送提醒的时间。目前仅支持设置用餐前一天的具体时间，提醒时间需要早于截止报餐时间。
2、 提醒谁报餐：可设置自动定时发送报餐提醒的成员范围。在提醒列表内的成员会在用餐前一天的提醒时间，收到提醒报餐的消息通知。目前手机端[Android、iOS]支持选择企业微信通讯录成员。选择后的部门和成员会排列显示在该区域的列表里。

### [](https://open.work.weixin.qq.com)三、如何管理和配置报餐内容

超级管理员、分级管理员可以在【手机端/Win端->工作台->报餐->报餐管理】进行配置或修改。包括自动提醒成员报餐、报餐内容配置（要统计的餐食、每周几用餐、截止报餐时间）。

### [](https://open.work.weixin.qq.com)四、如何查看报餐情况

1、 超级管理员、分级管理员可以在【手机端/Win端->工作台->报餐->报餐管理】查看当天对明日报餐收集结果的统计数据。并且支持切换到不同日期，查看历史收集统计结果。
2、 可以查看当前已提交报餐的人数，以及收集到需要统计餐食对应的份数。
3、 报餐截止时，会自动向管理员推送当天收集的明日用餐统计。
![](https://wework.qpic.cn/wwpic3az/919154_4RbcN1eUS2Glg8Y_1764301190/0)

### [](https://open.work.weixin.qq.com)五、报餐应用的可见范围设置

管理员可以在【工作台->管理企业->应用管理】中找到[报餐]应用，和【管理后台->应用管理->报餐】修改可见范围，或是停用/关闭应用。
