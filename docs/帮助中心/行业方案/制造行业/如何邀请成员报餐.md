# 如何邀请成员报餐

### [](https://open.work.weixin.qq.com)一、邀请企业微信中的成员报餐

超级管理员、分级管理员可以在【工作台->报餐->报餐管理】或【手机端->工作台->其他->报餐->报餐管理】中点击邀请同事报餐的蓝色按钮后，选择发送到会话，就可以发送当前的报餐邀请到企业微信会话中。
![](https://wework.qpic.cn/wwpic3az/96447_XA0fMXyiSteqqP5_1764920932/0)

### [](https://open.work.weixin.qq.com)二、邀请微信中的成员报餐

超级管理员、分级管理员可以在【工作台->报餐应用->报餐管理】或【手机端->工作台->其他->报餐->报餐管理】中点击邀请微信上的同事报餐的蓝色按钮后，选择邀请微信上的同事，就可以发送当前的报餐邀请到微信群中。

1、微信用户提交报餐会同时加入企业，支持在提交报餐后通过当前页面的按钮快捷下载企业微信客户端。

2、微信用户提交报餐后若在2个小时内没有下载并激活企业微信账号，系统会为成员自动关注企业的微信插件，支持在微信插件提交报餐。

3、成员加入企业后支持使用微信身份/企业微信身份提交报餐需求，系统会同时记录两个身份的报餐记录。

![](https://wework.qpic.cn/wwpic3az/297128_64NurdgnRrekk_w_1764920906/0)

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、报餐功能无法显示某一天的数据，推送显示是12号，点击进去是13号的数据，并且无法选择12号的数据。

默认配置的数据不展示，修改过配置后启用才可以在详情中查看。
注：报餐功能默认开启，报餐人员默认整个企业，调整任意配置后都可以在详情查看。

#### [](https://open.work.weixin.qq.com)2、导出报餐明细提示“无权限访问”

需确认操作人是否为超级管理员，目前仅超级管理员支持导出导出报餐。
