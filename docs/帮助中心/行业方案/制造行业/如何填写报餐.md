# 如何填写报餐

### [](https://open.work.weixin.qq.com)一、应用可见范围的企业微信成员、微信成员填写报餐入口

1、手机端/Win端工作台的报餐应用中，可以填写报餐；
2、微信侧关注了报餐应用消息的用户，可以填写报餐；
3、企业微信和微信会话中，接收到其他成员分享的邀请报餐链接，点击后可以填写。

注：目前仅支持提前一天收集次日的报餐情况，不支持成员填写当天的报餐情况

### [](https://open.work.weixin.qq.com)二、报餐填写对象

报餐应用默认在工作台为开启状态，无填写权限的限制。即所有在应用可见范围的企业微信成员、微信成员，能进入报餐填写入口，或者收到了邀请报餐通知的成员均可填写。

### [](https://open.work.weixin.qq.com)三、配置报餐内容

目前仅支持预定明日报餐的情况，即仅支持提前一天收集次日的报餐情况，不支持成员填写当天的报餐情况。报餐提交后支持修改。
超级管理员、分级管理员可以在工作台「报餐」应用的「报餐管理」，对报餐内容进行配置。关于配置内容的相关帮助，可以在同根目录的「报餐管理」子目录，中找到对应的问题。

### [](https://open.work.weixin.qq.com)四、填写报餐显示不在可见范围

如成员点击报餐填写后，显示不在应用可见范围，可以联系管理员在【工作台->管理企业->应用管理】中找到「报餐」应用，修改可见范围。
