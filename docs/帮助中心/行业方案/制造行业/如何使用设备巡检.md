# 如何使用设备巡检

1、设备巡检页面默认全员可见，管理员可以在【手机端->工作台->设备巡检->右上角->成员可管理设备->勾选仅负责的设备】，勾选后成员无法看见此页面。
注：若在工作台中看不到设备巡检入口，可以在【手机端->工作台->其他】中查看。
![](https://wework.qpic.cn/wwpic3az/588475_GOZeLGRqQW6PCOj_1765178836/0)
2、超级管理员、应用负责人、责任人支持查看自己管理范围内的所有设备当日的巡检情况。

3、页面上方显示今日巡检情况；支持点击当前页面巡检记录进入查看设备详情：设备当前状态、每一个巡检项目的异常次数统计、点击“查看设备信息”支持查看设备详情，若有设置设备责任人，可直接点击责任人姓名发起会话；点击“设备码”可查看该设备的设备码，支持保存到相册。
![](https://wework.qpic.cn/wwpic3az/363763_djKkcgrqToiv8Oj_1765178842/0)
4、页面最下方有“成员巡检记录”入口，点击可以查看所有的巡检记录；点击巡检记录也支持查看设备详情。
![](https://wework.qpic.cn/wwpic3az/493237_4tJF0qvdTTmfc06_1765178846/0)
5、设备状态分为“正常”和“异常”；若当前设备状态为异常，可见“修好了”和“填写巡检”按钮，点击修好了，设备状态更改为正常；若设备状态为正常，则不会有“修好了”按钮，支持继续填写巡检。

![](https://wework.qpic.cn/wwpic3az/512261_4nRvljxbSeGSXsE_1765178850/0)
6、异常设备点击“修好了”之后设备状态将变为正常，再次填写巡检时项目下方会有“上次巡检异常”标识。
![](https://wework.qpic.cn/wwpic3az/646588_PXcBz6hOTzqZx5l_1765178853/0)
7、若设备信息有误，支持点击右上角“…”进行修改。
![](https://wework.qpic.cn/wwpic3az/393662_hwnBjhj2Sp-2Bq0_1765178857/0)
