# 什么是我的巡检

1、成员支持在【手机端->工作台->设备巡检->我的巡检】对企业已添加的设备发起巡检；点击“直接填写”选择已有设备进行填写；选择“扫设备码”支持通过扫描二维码的方式进行填写；同时支持使用微信侧和企业微信侧右上角的扫一扫，扫码发起巡检。提交巡检后会生成巡检记录，记录仅支持查看，无法修改或删除。（电脑端暂不支持）
![](https://wework.qpic.cn/wwpic3az/497630_3CgUMlJaRquvyXc_1765165956/0)

2、管理员可查看所有巡检记录，成员仅支持查看自己填写的巡检记录。
3、管理员点击右上角支持设置应用负责人（负责人数量暂无上限），并对应用负责人设置管理范围，暂不支持对不同的应用负责人设置不同的管理范围。
4、支持通过右上角的入口下载企业已添加的所有设备的设备码，应用负责人仅支持下载可管理范围内的设备码。
5、点击“批量下载设备码”后，系统通过“设备巡检”以压缩包的形式下发设备码；每一张设备码以“设备名称+规格型号”的形式进行命名。
6、企业超级管理员和应用负责人会在每天17:00收到设备巡检日报。当企业设备数为0时，则不会推送。
