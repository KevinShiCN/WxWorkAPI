# 设备巡检-设备台账

1、设备台账页面默认全员可见，管理员可以在【手机端->工作台->设备巡检->右上角-成员可管理设备-勾选仅负责的设备】 ，勾选后成员无法看见此页面；若成员被设置为设备责任人，右下角可见“我负责的设备”入口。
![](https://wework.qpic.cn/wwpic3az/964263_YH9f78BaSaqwh6q_1765186680/0)
2、管理员和应用负责人支持在此页面添加新设备、查看企业已添加且自己有管理权限的设备；设备责任人仅支持查看设备，不支持添加新设备。

3、添加新设备时，设备名称为必填项，其他所有项目为选填项；设备名称可以和企业已有设备名称重名，但设备编号具有唯一性，不得填写企业内已添加的设备的设备编号，否则无法提交；一台设备支持设置最多100个责任人，每个成员支持被设置为责任人的设备数无上限；巡检项目支持从企业已有设备中复制；可添加的设备数暂无上限。

4、设备添加完成后生成设备码，支持保存至相册、继续添加新设备或关闭页面。

5、设备码永久且一台设备仅支持对应一个设备码，若添加的设备被删除则对应的设备码同时失效。

6、3.1.18版本开始，在设备台账页面支持按名称、编号、位置、型号搜索设备，以及在编辑已有设备时，支持对巡检项目进行排序。
