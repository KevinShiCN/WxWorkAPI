# 「居民上报」工单办理流程

1、居民在微信侧「居民上报」小程序中提交上报，提交后该上报单为待受理状态，此时负责第一层受理的成员将收到消息提醒

2、通过居民群内@机器人提交的上报首先由群主进行第一层受理，通过成员分享的小程序提交的上报首先由该成员进行第一层受理。第一层受理可选择「我来办理」或「上报到网格」
       如果本人可以办理，选择「我来办理」，此时上报单的状态由「待受理」变更为「办理中」
       如果本人无法办理，选择「上报到网格」，交由网格中心进行第二层受理，此时上报单的状态仍未「待受理」

3、网格中心负责人或超管可对上报到网格的「待受理」工单进行第二层受理，第二层受理可选择「不受理」、「我来办理」或「分派他人」
       如果上报单反馈信息不属实，可选择「不受理」，此时上报单的状态由「待受理」变更为「不受理」，办理流程结束并关单
       如果本人可以办理，选择「我来办理」，此时上报单的状态由「待受理」变更为「办理中」
       如果本人无法办理，选择「分派他人」，此时可指定一位企业内人员作为该上报单的办理人，此时上报单的状态由「待受理」变更为「办理中」

4、上报单的相关办理人，可将「办理中」的工单变更为「已办结」，或将网格中心分派给自己的工单转交给其他同事办理。当工单状态变为「已办结」时，流程结束并关单，通过小程序反馈给微信侧提交上报的居民
