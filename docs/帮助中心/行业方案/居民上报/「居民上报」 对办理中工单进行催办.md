# 「居民上报」 对办理中工单进行催办

1、「待受理、办理中」状态的工单可被催办。除办理人本人外，所有能够看到工单的人都可以催办。
2、工单状态由「办理中」变为「已办结」后，不显示催办按钮；如果办理人为本人，看到的「办理中」状态工单不显示催办按钮。
3、5分钟内不能重复催办。
4、点击催办按钮，成功则提示“已催办”。并给办理人在消息页面「居民上报」发送一条推送 “XX提醒您有一个待办理事项”。
