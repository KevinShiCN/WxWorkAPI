# 居民在微信中填写「居民上报」

### [](https://open.work.weixin.qq.com)1、在居民群中@助理机器人推送的小程序填写

居民可在居民群中@助理机器人，通过机器人发送的小程序提交上报
![](https://wework.qpic.cn/wwpic/20661_yHJAV8nhRKS8euF_1663055382/0)

### [](https://open.work.weixin.qq.com)2、企业微信侧居民群工具栏点击居民上报发送小程序到会话

成员可在企业微信侧居民群工具栏中，点击「居民上报」应用发送小程序到群聊中

![](https://wework.qpic.cn/wwpic3az/928353_59aUknnBRYKlKep_1764920037/0)

### [](https://open.work.weixin.qq.com)3、工作台「居民上报」应用分享小程序或二维码给微信用户填写

成员可点击工作台「居民上报」应用中的分享入口，发送小程序或二维码到微信侧会话中
![](https://wework.qpic.cn/wwpic3az/548729_G9ykY97uRI208Hy_1764920042/0)
