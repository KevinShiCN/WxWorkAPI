# 如何设置学员联系的权限

### [](https://open.work.weixin.qq.com)一、什么是学员联系使用范围

学员联系使用范围中的成员可使用学员联系功能，支持添加学员、创建学员群，他们添加的学员和建立的群聊由企业统一管理。

### [](https://open.work.weixin.qq.com)二、设置学员联系使用范围人员

超级管理员支持在【管理后台->学员联系与上下游->学员联系->权限配置->学员联系和学员群->使用范围】
![](https://wework.qpic.cn/wwpic3az/811698_sDl58motRCCSJCG_1753666831/0)
或是在【手机端->工作台->学员联系->配置->配置使用范围和管理规则->使用范围】
![](https://wework.qpic.cn/wwpic3az/130445_XAVJ3M27Qa-01p4_1765184747/0)

### [](https://open.work.weixin.qq.com)三、超级管理员设置学员联系负责人

#### [](https://open.work.weixin.qq.com)1、超级管理员在手机端设置学员联系负责人

超级管理员可以在【手机端->工作台->学员联系->配置->使用范围和管理规则】中设置管理规则，并在管理规则中添加业务负责人、指定负责人的管理范围和管理权限。
![](https://wework.qpic.cn/wwpic3az/81920_x-nG8gMfQ8u8TvB_1765184753/0)

#### [](https://open.work.weixin.qq.com)2、超级管理员在管理后台设置学员联系负责人

超级管理员可以在【管理后台->学员联系与上下游->学员联系->权限配置->学员联系和学员群->管理规则】中设置业务负责人，帮助分管成员添加的学员。
![](https://wework.qpic.cn/wwpic3az/568514_QqR2x_gFQrC_vjl_1753666842/0)

#### [](https://open.work.weixin.qq.com)3、负责人设置学员联系下级负责人

被设定为业务负责人的成员也能在自己的权限范围内新建下级负责人，让下级负责人帮助自己分管业务。功能入口为：【学员联系->配置->设置管理规则->设置下级管理规则】
![](https://wework.qpic.cn/wwpic3az/881972_daOa2mjLRDevpWw_1765184756/0)
