# 如何设置学员联系-聊天工具栏管理

### [](https://open.work.weixin.qq.com)一、聊天工具栏管理基本介绍

企业可配置应用页面到聊天工具栏，方便成员在与学员的聊天中查看和使用，提高服务效率。
1、工具栏的应用可添加的上限：100个。

2、支持在管理后台和手机端进行配置。具体操作路径如下：
管理后台配置路径：【管理后台->学员联系与上下游->学员联系->聊天工具->聊天工具栏管理】
（1）【管理后台->自建应用->配置到聊天工具栏】

![](https://wework.qpic.cn/wwpic3az/956525_PLjRtnJzSR2b0sC_1753667319/0)
（2）手机端配置路径：【工作台->学员联系/学员群->工具->聊天工具栏】
![](https://wework.qpic.cn/wwpic3az/244104_cKj-48D7RPOi3EY_1765177771/0)

（3）电脑端配置路径：【工作台->学员联系/学员群->工具->聊天工具栏】
