# 如何开始使用「上下游」？

> 查看详细的图文版[功能介绍](https://work.weixin.qq.com/nl/index/wxconnection?category=chaincorp&from=catDetail)

#### [](https://open.work.weixin.qq.com)**1、创建空间**

每一个企业都可以创建自己的上下游空间（最多20个），用以聚合、沉淀、管理企业的某类业务伙伴（例如经销商空间）。

#### [](https://open.work.weixin.qq.com)**2、邀请企业**

创建后，企业成员可以通过链接、二维码、迁移微信群、批量导入等多种方式邀请业务伙伴加入空间，组成「上下游通讯录」。

#### [](https://open.work.weixin.qq.com)**3、拉入同事**

已经加入上下游空间的本企业或其他企业成员，可以直接添加自己的同事进入上下游（超管可以添加“部门”），无需对方手动确认同意。

#### [](https://open.work.weixin.qq.com)**4、管理空间**

超级管理员可以管理自己企业创建的上下游空间，按分组整理加入的企业，分区域、分类别层层展示；按对接人设置不同企业的「上下游通讯录」查看权限。

#### [](https://open.work.weixin.qq.com)**5、在线沟通**

加入上下游空间的成员可以使用上下游通讯录直接跨企业联系，发消息可知对方已读未读，更有超大文件、会话存档、2000人大群等能力助力沟通。

#### [](https://open.work.weixin.qq.com)**6、群发消息**

在自己企业创建上下游空间内，可以群发产品动态、渠道政策等消息给其他企业的联系人，批量发送，即刻触达。

#### [](https://open.work.weixin.qq.com)**7、高效协作**

文档、微盘、日程、会议等协作工具全面支持上下游，你可以和上下游联系人共同编辑在线文档、共享微盘文件、共同参加在线会议。

#### [](https://open.work.weixin.qq.com)**8、共享应用**

企业可以共享应用给上下游联系人，打通客户资源，轻松实现跨企业业务协同；上下游联系人可以在自己企业的工作台上使用共享应用，完成业务流程闭环。
