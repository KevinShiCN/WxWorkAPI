# 如何修改已有上下游空间的名称？

**超级管理员可以修改自己企业创建的上下游空间的名称：**
1、在企业微信手机端打开工作台，进入「上下游」应用；
2、可通过右上角“切换”按钮，进入需要修改名称的上下游空间；
3、 进入空间首页后，点击下方的“管理此上下游”，即可修改上下游名称。

注：
（1）上下游的名称会展示给所有加入此空间的联系人，需要简洁、易懂；建议命名为「xxx的经销商」、「xxx的供应商」、「xxx的合作伙伴」等；
（2）上下游空间的名称具有全网唯一性，重名将不允许修改。
