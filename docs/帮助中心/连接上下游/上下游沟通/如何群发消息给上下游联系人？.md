# 如何群发消息给上下游联系人？

### [](https://open.work.weixin.qq.com)一、群发入口

1、打开企业微信工作台，进入「上下游」应用，点击“上下游群发”即可新建群发消息；
![](https://wework.qpic.cn/wwpic3az/96264_QjYL_fnFRjmu0KB_1755843276/0)
2、Mac端暂不支持“上下游群发”，可在手机端、Windows端、管理后台进行操作。

### [](https://open.work.weixin.qq.com)二、群发权限

1、管理员和成员可以向自己企业创建的上下游空间内的其他企业联系人群发消息；
2、不支持向其他企业创建的空间内的联系人群发消息；
3、不支持向自己企业的联系人群发消息。

### [](https://open.work.weixin.qq.com)三、群发限制

1、单日群发次数暂无限制；
2、单次最多可群发20000人。

### [](https://open.work.weixin.qq.com)四、群发内容

1、文字：最多支持4000个字符，支持emoji，支持换行；
2、附件：最多支持9个附件，支持照片、拍摄、视频号动态、文件、网页、小程序。

### [](https://open.work.weixin.qq.com)五、群发记录

1、管理员可以查看企业全部上下游群发记录；
2、成员只能查看自己的上下游群发记录。
