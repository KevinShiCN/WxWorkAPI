# 企业微信认证如何重填申请资料

1、认证提交审核后，若所填写的资料不符合要求，审核机构将会驳回企业的认证申请，驳回后，将会在企业微信中推送“企业微信认证审核需重填”的通知及最后提交期限的日期，可在通知底部点击“查看详情”入口，进入页面可查看需重填原因，以及“重新填写”入口。

![](https://wework.qpic.cn/wwpic3az/338767_2T0xIOGvRKWqKij_1702015881/0)
认证订单状态为“审核中”时如需重填/补充资料，或是对于重填需补充的资料有疑问，也可在“订单详情”找到审核公司的联系方式，与审核人员沟通；
若企业微信认证失败（包括资质审核失败、或名称审核失败），无法再提交相应认证资料，审核费用不予退还，只能重新付费提交企业微信认证。

2、超级管理员也可在【管理后台->我的企业->企业微信认证->认证订单->订单详情】中，查看具体需重填原因，以及“重新填写”入口。

**具体操作流程如下：**
第一步：登录[管理后台](https://work.weixin.qq.com)->我的企业->点击前往认证
![](https://wework.qpic.cn/wwpic3az/800315_hh44lku-SjqcVA5_1753348131/0)
第二步：点击认证记录
![](https://wework.qpic.cn/wwpic3az/487816_xgJ8TFKmQ_aFCsM_1753348332/0)
第三步：选择需要重填的认证记录，点击查看详情
![](https://wework.qpic.cn/wwpic3az/398641_w84CKbv3RheAFh4_1753847979/0)
第四步：查看具体需重填原因，并点击“重新填写”入口。
![](https://wework.qpic.cn/wwpic3az/537303_KnGCUyiLQA-i4o__1753848015/0)
