# 认证审核说明

### [](https://open.work.weixin.qq.com)一、如何联系认证审核公司

用户完成认证审核服务费用支付后，将会将该账号认证申请派发给第三方审核公司进行安全、严格的真实性认证。用户可在认证记录及认证详情中，查看审核状态及订单详情。在审核过程中，如有疑问可咨询客服或联系对应的审核机构。
若提交的认证资料准确无误，审核流程通常3-5个工作日即可完成。审核时长一般不超过10-15个工作日，将会通过「管理企业」消息、电话、邮箱等方式，告知用户审核结果。

查看路径：
1、【管理后台（https://work.weixin.qq.com） ->我的企业->前往认证->认证记录->认证详情】
![](https://wework.qpic.cn/wwpic3az/778861_O5hEhARvSlSTI-I_1766646415/0)
2、【手机端->工作台->管理企业->企业信息->重新认证->认证记录->认证详情】
![](https://wework.qpic.cn/wwpic3az/750310_0ncZWhKNQeW4LfL_1760519605/0)
温馨提示：若认证订单暂未分配审核公司，则不显示审核机构，可留意后续订单更新状态。

### [](https://open.work.weixin.qq.com)二、如何修改和补充认证资料

审核过程中，若有需要补充和重填的认证信息及材料，审核公司会主动联系用户，及时通知所需的操作、提交期限的日期及重新上传资料的入口。请关注审核结果并保持联系方式畅通。

企业微信认证审核过程中，从认证资料提交申请后，需要重填/补充资料，可联系审核公司沟通（比如让审核公司打回重填）；若企业微信认证失败（包括资质审核失败、或名称审核失败），无法再修改名称等认证资料，只能重新付费提交企业微信认证。
