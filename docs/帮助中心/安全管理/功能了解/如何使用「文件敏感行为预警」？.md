# 如何使用「文件敏感行为预警」？

### [](https://open.work.weixin.qq.com)一、功能简介

企业可以设置文件敏感行为规则，当成员命中规则时将发送预警通知，以降低企业信息泄漏风险。

### [](https://open.work.weixin.qq.com)二、使用流程

1、在管理后台->安全与管理->安全管理->文件敏感行为预警中，配置所需的预警规则。
![](https://wework.qpic.cn/wwpic3az/673546_qxxygR2oQuu7_Ef_1753758742/0)

2、预警规则包括 敏感文件、敏感行为、预警通知人 三部分。
![](https://wework.qpic.cn/wwpic3az/268119_DfTmqVsMTTalVgZ_1721806596/0)

3、敏感文件：支持自定义敏感文件大小、后缀类型。最多支持添加三种不同的敏感文件。
![](https://wework.qpic.cn/wwpic3az/900903_SbyMyjQRR3-M7u-_1721806690/0)

4、敏感行为：支持自定义文件 下载/导出、转发、上传 三种敏感行为。
![](https://wework.qpic.cn/wwpic3az/161931_m85s4ENvRSGciJ6_1721808415/0)

5、预警通知人：可选择将预警信息通知给 超级管理员、成员直属上级、其他成员。
![](https://wework.qpic.cn/wwpic3az/423329_qyXwvIa_QJGchUg_1721808433/0)

6、配置完预警规则后，可前往启用 文件敏感行为预警 功能。
![](https://wework.qpic.cn/wwpic3az/589223_7h1iLVnRRtG__5e_1753758763/0)

### [](https://open.work.weixin.qq.com)三、生效效果

当成员命中预警规则时，预警通知人将在「企业安全」会话中，收到相应通知。
![](https://wework.qpic.cn/wwpic3az/57937_DkcdCgbtQoGtQEe_1721808604/0)

此外，管理员也可在 敏感行为记录 中，查看企业内所有员工命中规则情况，以及命中的具体行为信息。
![](https://wework.qpic.cn/wwpic3az/589292_PFLAx14vS8OMaJ7_1721808632/0)
![](https://wework.qpic.cn/wwpic3az/579306_m2clfzl7S3uLW70_1721808652/0)

### [](https://open.work.weixin.qq.com)四、常见问题

#### [](https://open.work.weixin.qq.com)1、为什么启用了「文件敏感行为预警」，但是并没有命中记录？

「文件敏感行为预警」仅对开启「文件防泄漏」的成员生效。请确保同时启用这两个功能。

#### [](https://open.work.weixin.qq.com)2、「文件敏感行为预警」启用后，才开始匹配和记录成员行为吗？

是的。启用「文件敏感行为预警」后，将存储生效范围内成员的文件操作记录。当命中规则时，将推送预警。
