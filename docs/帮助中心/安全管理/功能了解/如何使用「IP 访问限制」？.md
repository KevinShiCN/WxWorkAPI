# 如何使用「IP 访问限制」？

### [](https://open.work.weixin.qq.com)一、功能简介

「IP 访问限制」可限制成员仅允许在指定的可信IP或IP段下访问邮件、微盘，以保障企业信息安全。

### [](https://open.work.weixin.qq.com)二、使用流程

1、企业管理员可从「管理后台->安全与管理->安全管理」进入「IP访问限制」模块。
![](https://wework.qpic.cn/wwpic3az/21257_hTXksDf4Qki59_n_1753695989/0)

2、将需要进行IP访问限制的成员设置进“生效范围”。
配置IP限制规则。可以填写IP或IP段，并将其设置为邮件或微盘的限制规则。
![](https://wework.qpic.cn/wwpic3az/411811_FnVOw1KPRKuRQpe_1753843795/0)

3、为帮助受限成员在正确的网络环境下访问邮件或微盘，管理员可以在管理后台配置成员受限时的指引资料链接，帮助成员了解限制原因和解除限制的方式。
![](https://wework.qpic.cn/wwpic3az/293622_SBl0Im86QpWgtFp_1753696165/0)
4、配置完成后，“开启”功能。

### [](https://open.work.weixin.qq.com)三、生效效果

1、根据企业配置的限制规则，生效范围内成员在非可信IP或IP段下，会被拦截访问。
2、若企业未配置指引资料链接，限制页面如下。
![](https://wework.qpic.cn/wwpic3az/982455_odVwHAPMRjGzyDU_1765358432/0)
3、若企业配置了指引资料链接， 限制页面如下。
![](https://wework.qpic.cn/wwpic3az/672087_TwpBJ31-T_CoXpW_1765358436/0)

### [](https://open.work.weixin.qq.com)四、常见问题

1、同时启用了安全管理、邮件、微盘中的IP访问限制，实际生效的是哪个？
安全管理、邮件、微盘中的IP访问限制规则将同时生效。例如，成员A被设置进「安全管理->IP 访问限制->邮件」生效范围，同时也在「邮件->IP 访问限制」白名单中，则最终成员A会生效「安全管理-> IP访问限制->邮件」限制规则。

2、开启「IP访问限制」后，为何生效范围内有成员没有生效？
生效范围内，仅拥有安全管理高级功能账号的成员，才可正常生效该功能。
