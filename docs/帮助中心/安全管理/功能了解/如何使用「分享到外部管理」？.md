# 如何使用「分享到外部管理」？

### [](https://open.work.weixin.qq.com)一、功能简介

支持企业设置禁止成员使用「分享到微信」、「用其他应用打开」、「保存邮件中的图片/文件」功能，进一步降低企业信息泄漏风险。
注：v4.1.32版本开始，支持设置禁止「保存邮件中的图片/文件」。

### [](https://open.work.weixin.qq.com)二、使用流程

企业管理员可从【管理后台->安全与管理->安全管理】进入【分享到外部管理模块】勾选**禁止成员使用「分享到微信」**、**禁止成员使用「用其他应用打开」**、**禁止「保存邮件中的图片/文件」**功能，添加生效范围。
![](https://wework.qpic.cn/wwpic3az/256598_w1CK1S69RKqSLGZ_1753757410/0)

### [](https://open.work.weixin.qq.com)三、生效效果

**禁止成员使用「分享到微信」**：开启后，在聊天、邮件、文档等所有场景，生效范围内的成员无法分享到微信。
![](https://wework.qpic.cn/wwpic3az/593879_bfO2EjnUTeqgzyr_1734944175/0)
**移动端禁止成员使用「用其他应用打开」**：开启后，在聊天、邮件、文档等所有场景，生效范围内的成员无法使用其他应用打开。
![](https://wework.qpic.cn/wwpic3az/596923_aUCs5_-0T--QNPM_1734944172/0)
**移动端禁止「保存邮件中的图片/文件」**：开启后，生效范围内的成员无法操作保存邮件正文图片、附件文件。
![](https://wework.qpic.cn/wwpic3az/72315_eSTG2yubSmyfrzQ_1734944179/0)
