# 如何使用「数据资产交接」

### [](https://open.work.weixin.qq.com)一、什么是「数据资产交接」

管理员把成员从通讯录中删除、或者成员主动的退出企业且超过了30天的保护期、或者成员注销账号且超过15天冷静期之后，该离职成员的数据资产可以在「数据资产交接」中被操作交接给其他成员。可将文档与微盘、日程、客户与客户群等数据资产交接给其他成员，避免企业资产流失。

**可交接的数据资产：**
（1）**文档与微盘**：离职成员放在【我的空间】中的所有文件和在线文档（文档需为公开文档）。
（2）**日程**：只支持纯日程型的交接，不包含会议型的日程（会议型日程需操作交接会议）。可交接未来日程，不支持交接已过期日程。交接的是组织人身份，不会改变日程共享状态。
（3）**客户和客户群**：需在【客户联系->离职继承】中操作分配。
（4）**会议及录制**：支持对离职员工会议数据资产发起交接，避免会议资产流失。

### [](https://open.work.weixin.qq.com)二、如何使用「数据资产交接」

#### [](https://open.work.weixin.qq.com)1、如何操作手动交接？

超级管理员可在【[管理后台](https://work.weixin.qq.com)->安全与管理->安全管理->数据资产交接->前往交接】中点击“去交接”查看到离职成员待交接的文档和微盘数量。

（1）可选择全部交接、部分交接、删除离职成员的文档、微盘文件。
**【全部交接给】**：需选择接手成员，交接后会将离职成员的所有资产
打包给这一位成员，该接手成员会收到一个“管理员已将离职成员XXX的资产交由你管理，请及时关注”的通知，打开可分别点击跳转至对应资产项查看详情。
**【部分交接】**：可选一个（或多个）资产项去交接给某位接手成员。
**【删除】**：可选一个（或多个）资产项去删除（注：客户和客户群目前暂不支持删除）。点击删除会有二次确认弹窗。

（2）可在【待交接的离职成员】列表中看到离职成员的姓名、所属部门、离职时间。

（3）可在【已交接的离职成员】列表中按部门、成员筛选查看到交接完成时间、交接详情。

（4）操作交接后，可查看到的交接状态有“空、部分交接、交接中、全部处理”

#### [](https://open.work.weixin.qq.com)2、如何使用离职资产自动交接？

购买安全高级功能后，管理员可以在【管理后台->安全与管理->安全管理->数据资产交接】设置离职资产自动交接。生效范围内的成员离职时，在文档、微盘、日程里的数据资产将可以被自动转交给其直属上级，保障企业数据资产安全。

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、「数据资产交接」中的客户和客户群交接后，是否可以在离职继承中看到交接记录？

可以的，双向打通，和微盘也双向打通。

#### [](https://open.work.weixin.qq.com)2、在数据资产交接，点击离职员工进入微盘交接详情删除数据提示“服务器异常，请稍后重试”。

可在微盘的离职成员文件转交列表中操作删除或进行数据交接，该情况会在后续进行优化。
路径：管理后台->协作->微盘->空间文件管理->离职成员文件转交。
注：微盘待转交的离职成员列表仅展示最近一年的数据。

#### [](https://open.work.weixin.qq.com)3、数据资产交接文档给新成员，新成员在“离职成员xxx的文档”中没有显示对应文档，而是显示在“离职成员xxx的文件”中。

目前文档如果是在个人空间，为了保持目录结构，转存到“xxx的离职文件”这个文件夹中。

#### [](https://open.work.weixin.qq.com)4、数据资产交接的离职成员是否有时间限制，超过限制就无法分配交接？

目前暂时没有时限限制，但后续可能会调整，建议尽快操作交接。
注：
（1）目前文档功能的“离职成员文档转交”和微盘的“离职成员文件离职转交”的时间限制为一年。若在文档和微盘中超过一年则无法操作转接，目前可以在「数据资产交接」中操作转交给其他人。
（2）「数据资产交接」的离职成员信息依赖于人事的离职表。人事助手是2022年后期才逐步放量使用。数据迁移依赖架构人事助手离职表的数据，并不是所有离职成员都会存在于人事离职表中，人事离助手职表不存在的话，一定不会存在于资产交接中。

#### [](https://open.work.weixin.qq.com)5、数据资产交接-资产自动交接中没有会议及录制

目前会议暂不支持资产自动交接，故资产自动交接中不显示会议及录制。

#### [](https://open.work.weixin.qq.com)6、在管理后台->离职成员资产交接中搜索离职成员姓名，显示暂无数据，实际列表中是可以看到有该成员

当前该位置的搜索框暂不支持英文搜索，只支持中文搜索。

#### [](https://open.work.weixin.qq.com)7、在数据资产交接中交接离职成员的微盘应用数据，提示“交接失败”

因目前资产交接侧暂无法处理微盘的共享空间数据，请在企业微信管理端后台->协作->微盘->空间文件管理->离职成员文件转交中进行操作。
