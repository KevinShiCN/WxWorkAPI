# 如何使用「截屏/录屏管理」？

如何防止聊天截图泄露？ [点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488324432827076_2)

### [](https://open.work.weixin.qq.com)一、功能简介

「截屏/录屏管理」可禁止或记录成员在企业微信客户端内的截屏、录屏操作，常用于对企业内接触敏感信息成员的行为管控，以保障企业信息安全。
目前仅支持移动端设备（iOS/Android）。

### [](https://open.work.weixin.qq.com)二、 使用流程

1、企业管理员可从「管理后台->安全与管理->安全管理」进入「截屏/录屏管理」模块。
![](https://wework.qpic.cn/wwpic3az/816730_VA9BpOstTLalLau_1753757594/0)
2、将需要进行截屏/录屏管理的成员设置进“生效范围”，随后“开启”功能。
3、支持在截屏记录中按照时间、部门、截屏内容（截屏的类型）、以及操作成员进行筛选查看记录。若成员截屏内容为文件，支持查看到文件名称，不支持查看文件内容。
![](https://wework.qpic.cn/wwpic3az/158634_c5J15rrLR9SKliT_1753757674/0)

### [](https://open.work.weixin.qq.com)三、生效效果

#### [](https://open.work.weixin.qq.com)1、Android

安卓手机在截屏、录屏时会被拦截，无法完成截屏、录屏操作。

#### [](https://open.work.weixin.qq.com)2、iOS

（1）iOS手机在截屏时，会记录截屏操作，并提示成员。管理员也可以在“管理后台->安全与管理->安全管理->截屏/录屏管理->截屏记录”中，查看截屏操作具体信息。
（2）iOS手机在录屏时，不会直接拦截，而是在录屏页面中，将企业微信页面展示为黑色背景提示。
![](https://wework.qpic.cn/wwpic3az/174887_HAdnUZKCTqGudrz_1734318458/0)

### [](https://open.work.weixin.qq.com)四、常见问题

1、开启「截屏/录屏管理」后，成员移动端是否能够截屏/录屏？
开启后，仅针对企业微信移动端会生效截屏/录屏的禁止或记录功能，成员依然可以截屏/录屏其他APP。

2、开启「截屏/录屏管理」后，是否影响企业微信内其他功能使用？
开启后，企业微信内与屏幕录制相关功能将无法使用，例如：会议/直播/VOIP时，无法共享企业微信内的页面。

3、开启「截屏/录屏管理」后，为何生效范围内有成员没有生效？
请检查以下条件是否满足：
（1）该功能需要成员移动端版本在4.1.8及以上。
启用功能后，可以点击「提醒成员升级」，成员将在移动端“企业安全”会话中，收到升级提醒，并可点击升级链接下载、升级最新版本。
（2）生效范围内，仅拥有安全管理高级功能账号的成员，才可正常生效该功能。
