# 行业资讯

### [](https://open.work.weixin.qq.com)一、什么是行业资讯功能

行业资讯功能每个工作日会推送行业的相关热文，帮助企业成员了解行业动态。资讯内容来自微信看一看的大数据智能推荐，目前功能随机邀请了部分用户体验行业资讯，帮助用户了解行业动态。

### [](https://open.work.weixin.qq.com)二、行业资讯功能说明

1、消息界面会推送一个信息列表，点击查看行业资讯，可以看见类似于像今日头条的这样一个信息流。
2、文章的排序是综合了文章本身的质量、发布的时间、文章在企业内被同事们阅读情况等进行综合评估排序的。
3、推送时间：

- 工作日每天8点-10点开始推送给企业内所有超管以及识别出来的关键角色（普通成员也支持被识别为关键角色）；收到后超级管理员可进行预览或删除，若有一个超管对文章操作了删除，企业内的成员不会收到当天的推送，其他超管暂不支持查看删除记录。
- 普通成员是每天10点推送，系统还未推送给普通成员时，普通成员也可以主动点击行业资讯应用看到这些文章。

### [](https://open.work.weixin.qq.com)三、关闭行业资讯功能

#### [](https://open.work.weixin.qq.com)1、关闭整个企业的行业资讯推送

企业超级管理员可以通过电脑登录管理后台，从【应用管理->行业资讯】进入操作关闭。关闭后，企业所有成员都不再会收到资讯推送。
![](https://wework.qpic.cn/wwpic3az/98355_uDmonOnXRh-Ejno_1753264707/0)

#### [](https://open.work.weixin.qq.com)2、成员关闭行业资讯的推送

可以进入行业资讯，点击右上角“…”进入，选择关闭“工作日推送”的开关。
![](https://wework.qpic.cn/wwpic3az/857234_LiMZrkO8Rc6NJp6_1732003113/0)

### [](https://open.work.weixin.qq.com)四、设置行业资讯偏好

管理员进入【手机端->工作台->行业资讯->右上角->企业资讯偏好】可设置企业资讯偏好，支持搜索公众号或主题并添加为偏好，系统会根据偏好推送相关文章。同时成员支持对管理员推荐偏好。
![](https://wework.qpic.cn/wwpic3az/722484_bhbt4LlzREOUTUY_1732002568/0)

### [](https://open.work.weixin.qq.com)五、设置允许发表评论

管理员可在【手机端->工作台->行业资讯->右上角“···”->允许发布评论】开启或者关闭，关闭后，管理员与成员均无法对行业资讯的文章进行评论，同时也无法查看以往的评论。
![](https://wework.qpic.cn/wwpic3az/922157_zzX1s0unSnKJIaA_1732002572/0)

### [](https://open.work.weixin.qq.com)六、屏蔽行业资讯文章

企业超级管理员可通过【手机端->工作台->行业资讯->点击某篇文章三个小点->点击管理或点击对应文章后->右上角三个点->屏蔽该资讯】

可对不适当的文章操作屏蔽，屏蔽后，全企业都不可见该篇文章，且不支持取消屏蔽。（可能会有5分钟延迟）
管理员每日会提前【半小时】收到推送，便于进行内容预览和管理。
![](https://wework.qpic.cn/wwpic3az/729804_ile33NJ2R7KTmYq_1732002576/0)

### [](https://open.work.weixin.qq.com)七、「在看」按钮的作用

成员在浏览行业资讯文章时，可在右下角点亮“在看”按钮，点亮后，行业资讯页面将会在对应文章下方展示点亮了“在看”按钮成员的头像。
成员再次点击已在看，将会取消点亮。
![](https://wework.qpic.cn/wwpic3az/34241_smfgAmGdRmK_5Sx_1732002580/0)

### [](https://open.work.weixin.qq.com)八、校园资讯配置公众号

行业类型为「教育->高等教育」的企业，支持在【管理后台->应用管理->行业资讯】进行添加绑定公众号（无需和企业微信主体一致，且被添加的公众号无需公众号管理员同意，可直接添加成功），支持批量搜索公众号，使分号或者逗号或者换行符进行隔开搜索，搜索界面会有推荐同主体的微信公众号。
![](https://wework.qpic.cn/wwpic3az/73031_U3B2kZ0xT6O8yS3_1753264713/0)
添加公众号后支持选择哪些公众号推送以及可以设置需要推送文章的展示范围。
![](https://wework.qpic.cn/wwpic3az/364784_aHGnN7KZToSQiZd_1753264719/0)
在后台行业资讯添加完公众号后，在手机端会点击行业资讯第四篇文章会显示校园资讯栏目，点击进入校园资讯会展示添加公众号24小时内发表的文章（最多展示50篇）。
若是24小时内发表的不足50章篇则会自动补齐公众号之前的历史发表文章（若公众号历史发表的文章没有超过50篇，就有多少补齐多少篇文章进行展示）
注：
（1）在行业资讯应用关闭允许成员评论后，校园资讯也会不能评论和查看之前评论，同时也支持屏蔽校园资讯的文章，也支持屏蔽校园资讯的文章，屏蔽后不支持添加回来）
（2）若用户添加了公众号后，行业资讯没有显示校园资讯栏目，需确认用户添加的公众号近7天是否有发表文章；若近期7天内公众号有发表文章刚添加公众号时候的有5分钟延迟，让用户等待5分钟左右在重新打开企业微信app重新尝试查看下。
（3）电脑端不支持查看校园资讯栏目。

### [](https://open.work.weixin.qq.com)九、常见问题

#### [](https://open.work.weixin.qq.com)1、历史推送的文章能继续查看吗？还是只能查看每天新推送的？

每天推送后，文章列表页才会更新一次内容。

#### [](https://open.work.weixin.qq.com)2、管理员在行业资讯中看不到“管理资讯内容”的设置入口

目前“管理资讯内容”的设置入口改到了文章“..->管理”中，该功能在陆续上线中，因此有可能出现部分企业需要在文章中点管理，有些企业可以在“行业资讯-管理资讯内容”入口管理。
![](https://wework.qpic.cn/wwpic3az/473618_-y2FvsoVTMmmqNO_1716803624/0)

#### [](https://open.work.weixin.qq.com)3、行业资讯翻到底部时，没有“点击查看更多资讯”入口，显示暂无更多内容。

当企业开启「个性化推荐」后，个性化文章只有60篇左右，滑到底就会显示暂无更多内容了。

#### [](https://open.work.weixin.qq.com)4、管理员能否删除企业成员在行业资讯文章里面的评论

暂不支持删除评论，只能删文章，文章删除后评论就没有了。
