# 同事在用小程序

### [](https://open.work.weixin.qq.com)一、什么是同事在用小程序

同事在用小程序，是推荐同事在企业微信侧使用过的小程序，发现更多工作时需要的小程序使用方式。

### [](https://open.work.weixin.qq.com)二、如何在工作台使用同事在用小程序

1、可在【手机端->工作台->用过的小程序】，可方便找到自己最近用过的小程序。（电脑端暂不支持）
（1）成员若没有使用过小程序，工作台会显示“同事在用”的小程序，点击进入后可以看到“同事在用”的小程序列表（需要是在企业微信侧使用过的小程序才会展示）；
（2）若成员有使用过小程序，会在工作台显示自己最近使用过的小程序，点击进入小程序后，可以看到自己使用过的小程序以及同事在用的小程序。
注：已关联到企业的小程序不会在“用过的小程序”中展示。
![](https://wework.qpic.cn/wwpic3az/561070_w8a4Z3lUQpiLj5c_1765267955/0)

2、小程序详情页，支持搜索更多小程序，还可发现更多同事在用的小程序。
3、企业自建关联的小程序，仍在企业应用分组内展示，不在该区域内出现。
4、对于“使用过的小程序”，支持长按小程序可以置顶或移除。
5、展示的小程序个数上限是30个。
6、管理员可在【[管理后台](https://work.weixin.qq.com)->应用管理->小程序】，查看到成员使用过的小程序，点击小程序可以选择禁用。已被禁用的小程序不会直接展示，需要点击右上角的“已被禁用的小程序”进行查看。点击“已被禁用的小程序”，管理员可以对小程序取消禁用。

### [](https://open.work.weixin.qq.com)三、如何在聊天内使用同事在用小程序

1、在【聊天附件栏->应用->用过的小程序】，可在聊天时快速发送最近用过的小程序。
2、打开小程序后，右上角菜单内即可选择发送到当前聊天。
3、首次点开，附件栏中的应用角标标红，再次进入不再显示红色角标，点击进入可查看“同事在用”小程序。
4、已添加小程序则会优先显示“最近使用”的小程序，下方显示已配置的应用。
5、点击右上角的编辑，可以选择移除或置顶当前小程序，已置顶的小程序图标右下角会有一个黄色的标志。
6、进入小程序后，可以点击右上角的【…】，选择把小程序发送到当前会话。
7、mac聊天附件栏暂无“应用”入口。

### [](https://open.work.weixin.qq.com)四、常见问题

#### [](https://open.work.weixin.qq.com)1、企业微信侧点击微信小程序无反应

需要检查手机端是否有安装微信，若无安装，建议下载安装微信app后重试。

#### [](https://open.work.weixin.qq.com)2、在B企业看到A企业的小程序

一般情况，B企业不应该出现A企业的小程序。会看见有可能是有成员同时在A/B企业使用对应的小程序（可能是通过别人发送等情况打开过），可联系小程序开发者确认下是不是有同时在A/B企业，打开过这个小程序。
