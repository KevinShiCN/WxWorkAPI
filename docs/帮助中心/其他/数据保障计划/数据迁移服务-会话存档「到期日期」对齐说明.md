# 数据迁移服务-会话存档「到期日期」对齐说明

当迁移多个同版本会话存档，而且 “到期日期” 不完全一致时，则需要先对齐到某个到期日期后，才能启动迁移。
选择不同的「到期日期」对齐，会导致**续期费用和权益损失**不一样。

具体如下：

### [](https://open.work.weixin.qq.com)一、迁移启动时：迁入企业未开通会话存档

- **取最长「到期日期」作为基准对齐**
**迁移前：**
![](https://wework.qpic.cn/wwpic/434344_cqTOF6xISyCsQ85_1689660324/0)
**迁移后：**
![](https://wework.qpic.cn/wwpic/627267_r7v1-ojUQAue4M4_1689661149/0)
- **取中间某个「到期日期」作为基准对齐**
**迁移前：**
![](https://wework.qpic.cn/wwpic/619526_00JHlJ0gRPCXq1x_1689660454/0)
**迁移后：**
![](https://wework.qpic.cn/wwpic/840292_DlBdYI9vT7OIxpK_1689661275/0)
*迁出企业C：由于选择了更短的到期日期对齐，会导致损失3个月的会话存档权益。
- **取最短「到期日期」作为基准对齐**
**迁移前：**
![](https://wework.qpic.cn/wwpic/649552_8sA1PQIlTYOyK2e_1689660591/0)
**迁移后：**
![](https://wework.qpic.cn/wwpic/923760_VMMFXbs8RNmu1pn_1689663643/0)
*迁出企业B/C：由于都选择了更短的到期日期对齐，会导致损失会话存档权益。
