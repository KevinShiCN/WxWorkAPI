# 数据迁移服务-企业迁移环境准备要求

默认支持按**账号粒度一对一迁移**客户数据，
请将贵司的迁移准备情况填写到（模板）跨企业迁移客户数据信息登记。

**迁移准备参考指引如下：**

### [](https://open.work.weixin.qq.com)一、在迁入企业微信创建迁入成员账号，建议账号ID和迁出成员账号ID保持一致

#### [](https://open.work.weixin.qq.com)1、如何查询账号ID？

登录企业微信管理后台，访问「通讯录」->「组织架构」，点击成员账号，即可查看账号ID。
![](https://wework.qpic.cn/wwpic3az/747914_Gpf5t7A0SlS6gcu_1753090011/0)

#### [](https://open.work.weixin.qq.com)2、如何配置账号ID？

- 在通讯录手动添加成员时，需填写成员账号，设定后不支持修改。
![](https://wework.qpic.cn/wwpic3az/878594_9d4zNvgqSj-h2em_1753090019/0)
- 如果成员账号是系统自动生成的情况下，在通讯录编辑成员信息时，支持手动修改一次。
![](https://wework.qpic.cn/wwpic3az/600109_nkHYLwNQRoyXHCk_1753090133/0)

### [](https://open.work.weixin.qq.com)二、确保所有参与迁移的账号均具备「外部沟通权限」和「客户联系权限」

#### [](https://open.work.weixin.qq.com)1、如何确认账号是否具备外部沟通权限？

- 企业微信移动端： 【工作台->管理企业->外部联系人权限设置】， 确认所有参与迁移账号都在外部沟通管理权限的授权范围内。
![](https://wework.qpic.cn/wwpic3az/991805_65xU_jyjRfyEpLN_1765268011/0)
- 企业微信管理后台：【我的企业->外部沟通管理->对外联系权限】，确认所有参与迁移账号都在外部沟通管理权限的授权范围内。
![](https://wework.qpic.cn/wwpic3az/440758_A8pkAgYAQNe_NMv_1753090156/0)

#### [](https://open.work.weixin.qq.com)2、如何确认账号是否具备客户联系权限？

- 企业微信移动端： 【工作台->客户联系->配置->使用范围和管理规则】， 配置了客户联系使用权限的服务人员视为具备客户联系权限。
![](https://wework.qpic.cn/wwpic3az/976222_3MugGRY7SwaIsnZ_1765268271/0)
- 企业微信管理后台：【客户与上下游->客户联系->权限配置->客户联系和客户群->使用范围】，配置了客户联系使用权限的服务人员视为具备客户联系权限。
![](https://wework.qpic.cn/wwpic3az/743731_a470alpaRH-8tAb_1753090162/0)

### [](https://open.work.weixin.qq.com)三、确保迁出和迁入企业员工账号为在职状态。

在职状态，即账号不能是未激活状态、退出企业状态、离职状态。

### [](https://open.work.weixin.qq.com)四、确保迁出和迁入企业员工账号，都完成绑定 微信或者手机号 授权

#### [](https://open.work.weixin.qq.com)1、如何判断企业员工账号已绑定微信或者手机号？

企业成员详情显示微信号或者手机号，则表示已完成绑定。
![](https://wework.qpic.cn/wwpic3az/648976_vAkY2k_UTWORrPo_1753090167/0)

#### [](https://open.work.weixin.qq.com)2、如何操作微信授权登录绑定？

**（1）迁出企业账号使用微信授权登录**
在企业微信登录界面，选择 “微信登录”并确认授权，确认后将自动跳转回企业微信界面，选择迁出企业进入。
![](https://wework.qpic.cn/wwpic/809090_ythLF83PTM--P7E_1671007228/0)
**（2）迁入企业账号先绑定微信，然后授权登录企业微信**
登录迁入账号所属企业微信后，在会话页左滑或点击左上角“≡”，找到齿轮入口并点击，进入“账号与安全”，确认迁入账号是否绑定微信；
![](https://wework.qpic.cn/wwpic3az/98259_ZGFgt1B1TU2a0AP_1765268355/0)
如发现迁入账号未绑定微信，需点击“微信”栏进入页面绑定微信。
绑定微信后，则可参考“迁出企业账号使用微信授权登录”指引，使用微信授权登录迁入账号的企业微信。

### [](https://open.work.weixin.qq.com)五、确保所有参与迁移的账号均为 “已实名”

#### [](https://open.work.weixin.qq.com)1、如何判断账号已实名？

成员可在【手机端->消息界面左上角“三”->头像->个人信息->姓名->实名认证】，查看账号是否已实名认证；
也可在外部群的群成员列表，点击自己名片详情，查看账号是否实名。

- 未实名标识如图
![](https://wework.qpic.cn/wwpic/912898_YVQkD51TSiqcZVD_1671002314/0)
- 已实名标识如图
![](https://wework.qpic.cn/wwpic/755226_M1xhaGVwR3SPYLq_1671002319/0)

#### [](https://open.work.weixin.qq.com)2、实名认证介绍指引

请查阅：[企业微信如何进行个人实名认证](https://open.work.weixin.qq.com/help2/pc/14958)

### [](https://open.work.weixin.qq.com)六、确保迁出账号需要迁移的客户群 “未满员”

迁移过程中，需要在客户群内，添加接替客户群的新群主；
为了避免因为群满员而导致不能添加新群主，建议参与迁移的客户群内腾出1-5个位置。

### [](https://open.work.weixin.qq.com)七、确保迁入主体有足量的外部联系人规模，以及参与迁移主体在迁移期间是已认证状态

1、在企业微信管理后台，依次通过「客户与上下游」->「客户联系」->「客户」，查看主体已使用的外部联系人规模总量。
迁入主体外部联系人规模余额需大于【客户总数】和【客户群中成员数】的总和。
外部联系人规模使用量可参考迁出主体中已服务的外部联系人数量。
![](https://wework.qpic.cn/wwpic3az/816426_UgFiZ0m4RH2amZR_1753090550/0)
2、查看主体认证过期时间
企业认证状态临近过期时，建议提前完成认证。可以在「我的企业」->「企业信息」，查看主体认证到期时间。

### [](https://open.work.weixin.qq.com)注意：

**在实施迁移期间，请周知所有企业管理员，保持不要对参与迁移的账号做任何改动操作，否则可能会影响迁移质量。**
