# 用户环境自查指引

如遇企业微信无法正常使用，并提示“网络异常，请检查网络或本机防火墙设置后重试”时，用户需要对使用环境进行自查。

### [](https://open.work.weixin.qq.com)一、自查流程

1、请尝试更换浏览器、网络、或设备重试。
2、请检查是否有配置网络代理；如果有，请检查代理或去掉代理重试。
3、请确认系统是否安装安全软件或防火墙；如果有，请检查拦截日志，尝试暂停或配置白名单。
4、浏览器环境下，请检查是否安装广告拦截插件，或安全检测插件；如果有，请关闭插件重试。
5、请打开[用户环境检测工具](https://work.weixin.qq.com/wework_admin/test/check_user_evn)检查域名是否有开放。若检测出异常的域名，需联系网络管理员进行开放。
6、请联系网络管理员确认内网是否有设置防火墙；若有，可尝试配置 ip 白名单，白名单可参考：[企业微信域名IP列表](https://work.weixin.qq.com/h5app/wework_domain_ip/latest)。
7、若已放开域名仍显示域名有异常/功能无法正常使用的，请使用[itango](https://itango.tencent.com/app/data/huatuo)检测对应域名是否可以正常访问。若可以正常访问，请提供检测结果截图和点击分享结果，将获取的链接联系企业微信人工客服进一步反馈。

### [](https://open.work.weixin.qq.com)二、具体操作指引

#### [](https://open.work.weixin.qq.com)1、更换网络

##### [](https://open.work.weixin.qq.com)Windows 使用手机热点网络

（1）在手机中开启热点功能。
（2）在电脑 WLAN 中找到并连接手机热点。
![](https://wework.qpic.cn/wwpic3az/918571_mcTLo8rDR86SXGi_1720445318/0)

##### [](https://open.work.weixin.qq.com)Mac OS 使用iPhone热点网络

以 Mac OS 使用iPhone 4G/5G 网络为例
![](https://wework.qpic.cn/wwpic3az/358527_G6r9VcyCTkCh__u_1720445496/0)

如果发现连接手机热点后仍无法访问互联网，可以尝试使用数据线连接 Mac 和手机之后重试。

#### [](https://open.work.weixin.qq.com)2、查看并取消系统代理

##### [](https://open.work.weixin.qq.com)Windows 系统

（1）打开电脑系统设置，选择“网络和 Internet”部分，点击“代理”。
![](https://wework.qpic.cn/wwpic3az/924457_TFh-R7CVRIeHMA0_1720445597/0)

（2）确认“使用设置脚本”与“使用代理服务器”均处于关闭状态。
![](https://wework.qpic.cn/wwpic3az/197430_F1lAdZzKTDWyLux_1720446603/0)

##### [](https://open.work.weixin.qq.com)Mac OS 系统

（1）打开系统设置，选择“网络”部分。
![](https://wework.qpic.cn/wwpic3az/442034_Ah02_gAPQKCvNFs_1720448844/0)

（2）点击“以太网”或“Wi-Fi”，查看详细信息。
![](https://wework.qpic.cn/wwpic3az/545487_wKPaxRYhRzamwpR_1720446622/0)
![](https://wework.qpic.cn/wwpic3az/139043_p9h-wuLpRHeySdg_1720446645/0)

（3）选择“代理”部分，关闭所有代理选项后点击“好”保存。
![](https://wework.qpic.cn/wwpic3az/751137_D2JeC8WJR2-kPom_1720446656/0)

#### [](https://open.work.weixin.qq.com)3、临时退出安全软件

以 QQ 电脑管家为例，在系统托盘中找到 QQ 电脑管家图标，右键图标，选择“退出管家”。
![](https://wework.qpic.cn/wwpic3az/952453_N2zoCUSmSoespCM_1720446669/0)
