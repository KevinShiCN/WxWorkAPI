# v4.1.30版本主要功能

# [](https://open.work.weixin.qq.com)效率工具与基础体验

#### [](https://open.work.weixin.qq.com)1、邮件

开通企业微信邮箱服务，配套企业微信使用，更稳定更安全；邮件基础功能免费，高级功能价格直降70%。管理员可在Web后台和客户端免费开通及咨询。
开通企业微信邮箱服务时，可以更快地[从原邮箱服务迁移历史邮件](https://open.work.weixin.qq.com/help2/pc/19809)，同时支持了迁移Office365邮箱。
企业微信邮箱服务已通过 IT 产品信息安全 EAL3 增强级认证，开通后享有更全面的邮件信息安全保护。

#### [](https://open.work.weixin.qq.com)2、智能表格

智能表格编辑体验升级，内容区域的右键菜单添加了图标，操作更直观。[对字段表头点击右键可快速进行筛选、分组和排序操作](https://open.work.weixin.qq.com/help2/pc/19684)。
智能表格编辑体验升级，[对人员、选项内容点击右键可快速筛选“包含此人”、“包含此选项”或“不包含此人”、“不包含此选项”的记录](https://open.work.weixin.qq.com/help2/pc/19684)。
智能表格中[使用筛选默认仅对自己生效](https://open.work.weixin.qq.com/help2/pc/19684)，多人查看时不会影响到其他人了。同时也可以将筛选设置的修改同步给所有人。
支持把常用的筛选项固定在工具栏了，筛选数据更方便。
智能表格编辑体验升级，对字段表头、选项等内容点击右键，[可通过“数据可视化”功能基于当前选中内容一键新建看板、添加图表](https://open.work.weixin.qq.com/help2/pc/19684)。
[收集表的收集结果可以关联到智能表格](https://open.work.weixin.qq.com/help2/pc/19684)了。智能表格可以按“所在部门”来分组展示结果，并在仪表盘中自动生成统计图表，查看数据更清晰。
在表格、甘特视图中支持通过拖拽冻结线，来快速调整冻结范围，操作更便捷。
优化了手机的记录详情查看体验，字段展示更紧凑，查看更高效。

#### [](https://open.work.weixin.qq.com)3、聊天和基础体验

[@ 我分组中可以设置是否包含 @ 所有人的未读消息](https://open.work.weixin.qq.com/help2/pc/21157)了，分拣和查看未读消息更便利。
[电脑端创建群聊时会根据所选成员推荐已有群聊](https://open.work.weixin.qq.com/help2/pc/17545)，减少重复创建相同群，聊天列表更简洁。
[移动端可以在「存储空间」中清理资源文件和其他企业聊天记录](https://open.work.weixin.qq.com/help2/pc/17282)了，为你的设备释放更多可用空间。

#### [](https://open.work.weixin.qq.com)4、搜索

[「全局搜索-聊天记录」内新增文字、图文和群公告三种消息类型的筛选](https://open.work.weixin.qq.com/help2/pc/21154)，查找历史消息更高效。

#### [](https://open.work.weixin.qq.com)5、管理后台

[查看操作日志时间可以精确到秒](https://open.work.weixin.qq.com/help2/pc/15160)了，还支持通过 API 导出，敏感操作行为监控更实时，安全审计更便捷。

#### [](https://open.work.weixin.qq.com)6、打卡

[支持添加如工号、入离职时间等，来自通讯录、人事助手的字段到打卡统计报表中](https://open.work.weixin.qq.com/help2/pc/17527)了。
[支持查看考勤机推荐型号并前往购买](https://open.work.weixin.qq.com/help2/pc/17114)，设备选购更便捷。
[考勤机可快捷创建打卡规则](https://open.work.weixin.qq.com/help2/pc/14854)，配置更简单。

#### [](https://open.work.weixin.qq.com)7、审批

[审批模板可以配置自定义打印格式](https://open.work.weixin.qq.com/help2/pc/21327)了，模板管理更灵活。
[导出审批单据时，可以导出审批人电子签名](https://open.work.weixin.qq.com/help2/pc/16582)了，信息留存更完整。

#### [](https://open.work.weixin.qq.com)8、人事

工资条导入时，支持表格文件内公式的识别，使用更方便。

# [](https://open.work.weixin.qq.com)连接微信

#### [](https://open.work.weixin.qq.com)1、客户群

外部群可以设置「[禁止群成员互相添加为联系人](https://open.work.weixin.qq.com/help2/pc/16749)」了，防止客户被其他企业加走，避免企业的客户资源流失。可以单个群设置，也可以通过群模板给批量设置。

#### [](https://open.work.weixin.qq.com)2、客户联系

客户联系新增「[收款转化分析](https://open.work.weixin.qq.com/help2/pc/21355)」，可清晰量化客户服务价值了。客户被服务后的7日/1日内的交易被计入，包括线上小程序、网页、App和线下门店的交易。还可查看门店和成员的收款明细。
针对成员发送的外部收款码，管理员和业务负责人可对企业认可的码进行无风险确认了，便于企业对收款风险进行精细化管理。

#### [](https://open.work.weixin.qq.com)3、基础体验优化

[可以收到和查看微信里的笔记](https://open.work.weixin.qq.com/help2/pc/14908)了，还可转发给微信客户，分享产品介绍、优惠活动、服务信息等。
[发送和接收的文件消息，最大可支持1G，可下载时长延长至14天了。](https://open.work.weixin.qq.com/help2/pc/14908)
[文件消息可撤回时间延长至3小时](https://open.work.weixin.qq.com/help2/pc/14923)了。

# [](https://open.work.weixin.qq.com)企业安全管理

文件防泄漏日志可以查看文件大小和MD5了，还新增展示申请下载的企业内成员账号，文件追溯更准确。
支持将截屏记录导出为本地表格，或通过API导出截屏记录，截屏行为留存更便捷。

# [](https://open.work.weixin.qq.com)高级功能

[上级可为需要使用高级功能的团队成员统一提交申请](https://open.work.weixin.qq.com/help2/pc/19837)，帮助管理员更好的处理成员需求，推进采购。

# [](https://open.work.weixin.qq.com)跨境沟通与境外用户体验

[境外同事可以通过拼音字母查看中文姓名](https://open.work.weixin.qq.com/help2/pc/17282)了，联系中国同事更便捷。
新建日程或预定会议时，可以看见与自己不同时区同事的当地时间，跨时区协作更方便。
与跨时区同事聊天时，可以看到对方是否处于深夜，跨境沟通更高效。

# [](https://open.work.weixin.qq.com)开放能力

#### [](https://open.work.weixin.qq.com)1、智能表格

第三方应用可通过接口新建智能表格，将应用中的数据写入到智能表格中，企业成员可查看与编辑。应用还可通过接口添加多种视图，方便企业查看与分析数据。应用创建的智能表格，可通过接口获取、更新智能表格中的数据，打通智能表格与应用数据。

#### [](https://open.work.weixin.qq.com)2、安全管理

支持通过接口获取防泄漏文件大小和MD5，还可获取申请下载的企业内成员账号。
支持通过接口导出截屏记录。
支持通过接口导出操作日志，安全审计更便捷。

#### [](https://open.work.weixin.qq.com)3、第三方

企业授权第三方应用的权限发生变更后，服务商可通过接口获取该事件通知。
