# v4.0.18版本主要功能

# [](https://open.work.weixin.qq.com)微信客服持续升级

1、微信客服
可[在视频号小店接入微信客服了](https://open.work.weixin.qq.com/help2/pc/19653)，接入后，微信客户可在小店商品详情页或订单页发起咨询。需登录「视频号小店管理后台」或进入「微信-视频号-创作者中心-带货中心-视频号小店」完成接入。
支持[在手机端查看客户咨询较多的热门问题](https://open.work.weixin.qq.com/help2/pc/19627)，并将其添加到知识库。
支持[在欢迎语添加热门问题](https://open.work.weixin.qq.com/help2/pc/19628)了。
可[自定义未命中知识库时机器人的回复](https://open.work.weixin.qq.com/help2/pc/19627)了，回复内容支持从知识库或热门问题中添加客户咨询较多的问题。
可[设置无效提问或未命中知识库时不自动转人工](https://open.work.weixin.qq.com/help2/pc/19627)了。

2、对外收款
成员也可以[创建商户号向客户收款](https://open.work.weixin.qq.com/help2/pc/14774)了。仅支持成员创建企业商户号，所收资金将直接提现到企业银行卡。

# [](https://open.work.weixin.qq.com)效率工具与基础优化

1、文档新增汇报
成员可按模板填写日报、周报和销售业绩等，向上汇报工作进展。
管理者可便捷查看和统计成员的汇报内容，了解成员的工作进展。
支持创建更多汇报模板，向成员收集工作汇报。

2、文档其他优化
PC端文档Tab支持[切换视图](https://open.work.weixin.qq.com/help2/pc/18196)，新视图下切换列表和查找文档更高效了。
PC端支持[通过导入和模板发送在线文档](https://open.work.weixin.qq.com/help2/pc/18195)了，和群成员协作更方便。
在PC端查看文档时，支持[在当前页面搜索](https://open.work.weixin.qq.com/help2/pc/18196)了，查找文档更高效。
可以[为企业内、外分别配置由管理员审批权限](https://open.work.weixin.qq.com/help2/pc/15301)了，管理文档更灵活。

3、会议
支持[自定义重复规则预定周期性会议](https://open.work.weixin.qq.com/help2/pc/18184)。
[新建和编辑会议时支持选择日历本](https://open.work.weixin.qq.com/help2/pc/18184)。
[会议列表支持搜索了，查找更方便。](https://open.work.weixin.qq.com/help2/pc/18184)
[会议消息和来电通知支持设置免打扰了。](https://open.work.weixin.qq.com/help2/pc/18184)

4、日程
[编辑场景下支持了日程和会议的互相切换。](https://open.work.weixin.qq.com/help2/pc/17695)

5、微盘
[在微盘相册中可以点赞和评论，与同事互动更方便了。](https://open.work.weixin.qq.com/help2/pc/15288)

6、邮箱
在移动端读信时，可以放大邮件正文字体了。
在移动端收到带多个附件的邮件时，可以一键下载全部附件了。
转发到聊天的邮件，打开后可以直接回复、转发了。

7、工作台
[工作台支持将应用添加到常用栏，使用应用更方便。](https://open.work.weixin.qq.com/help2/pc/17078)
[移动端工作台支持搜索，查找应用更快捷。](https://open.work.weixin.qq.com/help2/pc/17078)

8、安全管理
开启文件保密模式后，禁止对外分享文件[支持设置白名单](https://open.work.weixin.qq.com/help2/pc/19479)，禁止下载与导出文件[支持设置使用范围](https://open.work.weixin.qq.com/help2/pc/19479)，文件安全管理更灵活。

9、审批
明细控件支持复制已有明细和添加空白明细，[打印格式更清晰](https://open.work.weixin.qq.com/help2/pc/14895)。
管理后台支持按完成时间筛选单据了，核对单据更便捷。

10、打卡
[支持先审批再外出打卡了](https://open.work.weixin.qq.com/help2/pc/17500)。
[外出打卡记录可同步到上下班打卡中，核对考勤数据更方便。](https://open.work.weixin.qq.com/help2/pc/17527)

11、基础体验优化
Windows端支持打开多个独立窗口查看图片了，对比查看更便捷。
Mac端发送超长消息时，[支持快捷转为文档发送了](https://open.work.weixin.qq.com/help2/pc/14908)，阅读体验更佳。

# [](https://open.work.weixin.qq.com)持续助力在线教学

1、家校通讯录
[支持从管理后台统一导入学生手机号了](https://open.work.weixin.qq.com/help2/pc/15047)，管理学生信息更方便。

2、学生群
学生群中也可以使用[课外打卡](https://open.work.weixin.qq.com/help2/pc/17816)了，轻松帮助学生建立良好习惯。

3、班级作业
[PC端批改作业时支持对作业图片添加文字评语](https://open.work.weixin.qq.com/help2/pc/16910"PC端批改作业时支持对作业图片添加文字评语")了，可以给学生更细致的学习建议。

4、班级群学生群
[在班级群和学生群的工具栏上可以添加收集表使用了](https://open.work.weixin.qq.com/help2/pc/16906)，收集工作更简单。

# [](https://open.work.weixin.qq.com)开放能力升级

1、日程
支持通过接口创建公共日历和全员日历了。
支持通过接口更新和修改重复日程了。

2、微盘
第三方在微盘创建空间后，用户在该空间所做的文件操作支持回调给第三方了。

3、审批
支持通过接口将其他系统数据设置为审批外部选项了，业务申请更高效。

4、微信客服
不同服务商可同时在多个第三方应用关联同一个App了，用于在App里接入其他企业的微信客服。

5、家校沟通
支持通过接口导入学生手机号信息了。

6、上下游
支持通过接口变更上下游群成员了。

7、智慧硬件
第三方应用可通过接口获取企业下已绑定的考勤/门禁/测温设备的原始数据了。
