# v4.1.16版本主要功能

# [](https://open.work.weixin.qq.com)效率工具与基础体验优化

#### [](https://open.work.weixin.qq.com)1、邮件

电脑端新增了「仅显示邮件列表」的界面布局，可以快速分拣邮件。
免提醒邮件也支持自定义排序和对分组批量操作。
写邮件时，[支持设置段落格式和通过搜索快速选择字体](https://open.work.weixin.qq.com/help2/pc/20404)。
写邮件时，可以拖拽邮件列表里的邮件添加为附件。
[新增对外签名](https://open.work.weixin.qq.com/help2/pc/20339)，向收件人展示企业名、职务等信息。收件人还可点击查看企业微信名片。
[可以设置多个签名](https://open.work.weixin.qq.com/help2/pc/20339)，在写邮件时按需使用。

#### [](https://open.work.weixin.qq.com)2、日程

[日程支持设置时区](https://open.work.weixin.qq.com/help2/pc/17695)，与海外同事约会更方便。
从群聊新建日程时，[可以快捷添加群成员作为参与人](https://open.work.weixin.qq.com/help2/pc/17695)。
桌面端日程添加参与人，可以批量粘贴邮件地址，邀约更便捷。
在电脑端新建日程时，[支持新窗口打开](https://open.work.weixin.qq.com/help2/pc/17695)。
日程详情展示优化，可以更清晰地显示日程邀约内容。
[桌面端日程周视图可以自适应列宽了](https://open.work.weixin.qq.com/help2/pc/18206)，查询近期日程更清晰。

#### [](https://open.work.weixin.qq.com)3、会议

参会记录支持完整导出，会后可方便回溯参会成员名单。

# [](https://open.work.weixin.qq.com)连接微信持续升级

#### [](https://open.work.weixin.qq.com)1、对外收款

低至0.2%的支付费率，优惠时间已自动续期至2024年12月31日了。

# [](https://open.work.weixin.qq.com)开放能力升级

#### [](https://open.work.weixin.qq.com)1、打卡

支持通过接口设置自由上下班的新的一天开始时间、工作时长和加班规则配置。
