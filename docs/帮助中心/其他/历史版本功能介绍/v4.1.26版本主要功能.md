# v4.1.26版本主要功能

# [](https://open.work.weixin.qq.com)**效率工具与基础体验优化**

#### [](https://open.work.weixin.qq.com)1、聊天和基础体验

[支持按照未读、@我、单聊、群聊、内外部聊天、标记和标签等分组筛选消息列表](https://open.work.weixin.qq.com/help2/pc/21157)，处理消息更高效。
[手机端支持给聊天添加标签](https://open.work.weixin.qq.com/help2/pc/13352)。可以给不同项目的聊天打上标签，找回和管理更方便。
[手机端多窗口支持打开聊天中的图片和视频了](https://open.work.weixin.qq.com/help2/pc/21006)，处理工作更方便。
[支持在同一网络下手机端和电脑端互相迁移聊天记录了](https://open.work.weixin.qq.com/help2/pc/16757)。
支持将聊天消息、邮件和文件等便捷转发至其他企业的聊天。聊天为已读未读等。具体可以在「设置」-「快捷键」-「查看更多快捷键」中查看。

#### [](https://open.work.weixin.qq.com)2、境外用户体验优化

[境外成员可以使用邮箱注册登录企业微信](https://open.work.weixin.qq.com/help2/pc/17342)了。
境外成员的语音通话体验、文件传输速度、防泄漏模式下的预览速度提升。
境外企业可以使用汇报了，为境外成员提供便捷的汇报功能。

#### [](https://open.work.weixin.qq.com)3、智能表格

智能表格是一个比表格更强大的业务数据库，不仅能持续汇集业务数据，还支持以多种视图查看和分析数据，并将分析结果转化为直观、图形化的数据看板。关键数据变化时，自动化流程能按规则触发，自动发起群聊并发送消息提醒。

##### [](https://open.work.weixin.qq.com)（1）连接聊天和应用

智能表格可以发起群聊并向群聊发送消息了。关键数据变化时，自动化流程功能可以根据自定义的规则来发起群聊、通知成员。例如在任务管理中，你可为每个任务发起跟进群，群内实时同步进展，必要时还能自动拉入更多相关成员来协助处理。
智能表格中支持添加群聊名片。成员可在表格中记录任务跟进群、客户沟通群的名片，点击名片即可加入群聊。
应用可通过接口来新建智能表格，将内部系统的数据写入到智能表格中，邀请成员来查看与编辑。成员可以使用多种视图查看数据、搭建可视化数据看板。
应用创建的智能表格，可通过接口获取、更新智能表格的数据，打通智能表格与企业的其他数据系统。

##### [](https://open.work.weixin.qq.com)（2）汇集信息与数据

[可根据智能表格中的字段一键生成表单](https://open.work.weixin.qq.com/help2/pc/19681)，快速收集信息，成员在手机上也能方便地填写。
[表单可开启分享链接](https://open.work.weixin.qq.com/help2/pc/19681)，同事和微信上的客户都可通过链接填写，并且不会查看到其他人的提交结果，信息收集更安全。
[新增自动编号列类型](https://open.work.weixin.qq.com/help2/pc/19684)，可为每条记录自动生成唯一且递增的编号，还可自定义编号规则。适用于物资领用登记、进销存管理等场景。
[新增货币列类型](https://open.work.weixin.qq.com/help2/pc/19684)，支持多种国家/地区的货币符号，满足跨境业务的财务记录需求。
[新增地理位置列类型](https://open.work.weixin.qq.com/help2/pc/19684)，成员可填写地址信息。还支持设置“仅限手机自动定位”，适用于客户拜访、门店巡检等场景。

##### [](https://open.work.weixin.qq.com)（3）数据可视化

[新增仪表盘](https://open.work.weixin.qq.com/help2/pc/19684)，能将智能表格的数据展示为可视化的看板，在手机、电脑上都可以实时查看。相比传统表格，仪表盘只需简单的配置，就能搭出“任务进展看板”、“经营数据情况”等丰富的可视化报表，方便负责人了解全局状况。
[手机上可以用卡片模式来查看数据了](https://open.work.weixin.qq.com/help2/pc/19681)，卡片支持自定义展示的内容，关键信息一目了然。
[优化了手机上切换页面和视图的体验](https://open.work.weixin.qq.com/help2/pc/19681)。

##### [](https://open.work.weixin.qq.com)（4）其他

内容权限可以使用引用字段设置权限了，适用于从其他子表中引用负责人、部门等信息来设置权限。
支持配置视图权限，可设置仅指定成员能配置视图，防止视图被其他人误改。
自动化流程新增运行日志。当流程运行异常时，运行日志可辅助你解决问题。

#### [](https://open.work.weixin.qq.com)4、外部联系体验优化

[个人信息页可对商务伙伴展示更丰富的信息](https://open.work.weixin.qq.com/help2/pc/21158)了， 例如企业介绍、业务介绍等，支持添加图片/视频/文档/文件等丰富的格式，帮助商务伙伴快速了解自己。
[添加商务伙伴后，可自动递出名片](https://open.work.weixin.qq.com/help2/pc/21158)，沟通更专业。
[添加的商务伙伴，可在“通讯录-我的客户”中查看](https://open.work.weixin.qq.com/help2/pc/21158)，按企业显示，便于查看和管理自己的商务关系。
[成员可添加更多向商务伙伴展示的信息](https://open.work.weixin.qq.com/help2/pc/21158)，也可由管理员统一添加。

#### [](https://open.work.weixin.qq.com)5、邮件

新增邮件联系人，方便写邮件时，可以直接从邮件联系人中选择收件人。
可以批量导入邮件联系人。支持导入 Foxmail、Outlook中联系人导出的 xlsx / xls / csv 格式文件。
[邮件可以一键转发至聊天](https://open.work.weixin.qq.com/help2/pc/20332)，提醒同事查阅处理。

#### [](https://open.work.weixin.qq.com)6、打卡

[打卡防作弊管理新增支持识别虚拟定位](https://open.work.weixin.qq.com/help2/pc/20903)，有效防止地点作弊打卡。
加班申请优化，支持提交跨天加班。

#### [](https://open.work.weixin.qq.com)7、审批

可在手机端导出审批单，打印留存更方便。
申请跨天请假、出差、外出、加班时，可按需逐天修改时长明细，考勤统计更准确。

#### [](https://open.work.weixin.qq.com)8、硬件

绑定设备支持打卡门禁规则一站式配置，绑定管理更便捷。

#### [](https://open.work.weixin.qq.com)9、日程

跨时区日程全天事件展示优化。

# [](https://open.work.weixin.qq.com)**连接微信**

#### [](https://open.work.weixin.qq.com)1、群发助手

企业创建多个群发任务后，[成员可一键发送今日所有待发送的消息](https://open.work.weixin.qq.com/help2/pc/13260)了，精细化运营更方便。
[管理员可根据场景和需求配置群发规则](https://open.work.weixin.qq.com/help2/pc/13260)，每位客户/每个客户群可接收群发消息条数为“每日可接收1条”或“每周可接收7条”或“每月可接收条数为当月天数”。
发送群发消息后，[会话不会浮到消息列表最新位置](https://open.work.weixin.qq.com/help2/pc/13260)了。

#### [](https://open.work.weixin.qq.com)2、客户联系

[员工删除单向客户更好用](https://open.work.weixin.qq.com/help2/pc/20353)了，筛选出单向客户后，可左滑直接删除，还可多选删除。

#### [](https://open.work.weixin.qq.com)3、客户朋友圈

朋友圈发表的内容，[可展示给所有已添加的客户](https://open.work.weixin.qq.com/help2/pc/14767)了。

#### [](https://open.work.weixin.qq.com)4、客户群

[防骚扰功能支持踢发二维码、踢发语音、踢发聊天记录](https://open.work.weixin.qq.com/help2/pc/14737)。
有新的客户加入外部群时，[群主可点击进群消息进入该客户的个人信息页面](https://open.work.weixin.qq.com/help2/pc/14912)，查看客户相关信息；若发现不是企业真实的客户，还可快速设置他的防骚扰权限或者投诉等。
[「加入群聊」自动创建新群时，支持单独设置新群的群名](https://open.work.weixin.qq.com/help2/pc/14712)了，企业也可配合群模板使用，提高客户群的配置效率。
群模板支持配置群聊邀请确认，企业可批量配置到群。

# [](https://open.work.weixin.qq.com)**企业安全管理**

[文件防泄漏支持按需启用邮件场景](https://open.work.weixin.qq.com/help2/pc/19479)了，文件安全管控更灵活。

# [](https://open.work.weixin.qq.com)**高级功能**

[管理员可以在「管理后台-高级功能」管理成员申请的审批模板](https://open.work.weixin.qq.com/help2/pc/21155)，自定义申请流程更方便。
[企业充值后，可原路提现至付款账户](https://open.work.weixin.qq.com/help2/pc/20897)，资金管理更灵活。
[支持为成员设置企业充值余额的单笔使用上限](https://open.work.weixin.qq.com/help2/pc/20897)，便于企业管理资金使用。

# [](https://open.work.weixin.qq.com)**开放能力升级**

#### [](https://open.work.weixin.qq.com)1、高级功能

支持企业通过接口配置成员申请流程，可将成员申请与企业自建的审批流程对接，满足更灵活的申请流程管理需求。

#### [](https://open.work.weixin.qq.com)2、第三方

通过接口导入可信企业设备信息时，若与已导入设备重复，接口将返回冲突的设备编码，便于管理员核查。
