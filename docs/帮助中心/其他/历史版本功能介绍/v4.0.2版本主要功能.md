# v4.0.2版本主要功能

# [](https://open.work.weixin.qq.com)连接微信持续升级

1、可在客户群聊/客户单聊中接入微信客服了

当成员服务客户并需要客服帮助解答问题时，可在聊天中[主动推荐微信客服](https://open.work.weixin.qq.com/help2/pc/19187)，客户点击名片即可向客服发起咨询。
成员也可在客户群后直接[开启客服助理](https://open.work.weixin.qq.com/help2/pc/19187)，开启后，将邀请客服助理进群，客户在群内@客服助理，客服助理将自动发送微信客服名片，客户点击名片即可直接咨询微信客服。

2、微信客服

可以查看升级服务带来的客户数，包括添加专员的客户数和加入客户群的客户数。
接待人员在[Mac上也可以回复](https://open.work.weixin.qq.com/help2/pc/17961)客户咨询。

3、对外收款

支持管理员和应用负责人[用身份证快速开通商户号](https://open.work.weixin.qq.com/help2/pc/14774)，开通后成员即可收款，开通者还可在手机上快速提现。
企业微信用户也可[直接付款](https://open.work.weixin.qq.com/help2/pc/14778)。

4、视频号

可推荐视频号名片给客户，企业可以更好地传播品牌或活动。

# [](https://open.work.weixin.qq.com)效率工具与基础优化

1、日程

管理员可以在管理后台[统一配置成员的Exchange日历同步](https://open.work.weixin.qq.com/help2/pc/19006)，方便在企微上集中管理日程安排。
闲忙视图中可以展示员工添加其他日历账户下的日程。

2、会议

可以在[群聊右上角快速发起会议](https://open.work.weixin.qq.com/help2/pc/18185)。
桌面端可以[指定屏幕区域进行演示](https://open.work.weixin.qq.com/help2/pc/18174)。
共享屏幕时，支持[展示人像画中画](https://open.work.weixin.qq.com/help2/pc/18174)，会议讲解更生动。

3、会议室

管理员可以在管理后台[导出会议室使用明细](https://open.work.weixin.qq.com/help2/pc/19185)，方便对会议室使用情况进行统计管理。

4、文档

可以在文档成员列表修改成员的权限，管理文档更方便。
可以在文档面板创建和查看共享空间。

5、微盘

空间管理员可以开启[文件保密模式](https://open.work.weixin.qq.com/help2/pc/15301)，文件默认仅空间成员可见，存储更安全。
可以邀请外部企业成员和微信联系人加入空间。
对外分享链接支持批量下载文件。

6、其他

[Mac支持深色模式](https://open.work.weixin.qq.com/help2/pc/17282)。
打卡支持管理员[调整当天及历史班次](https://open.work.weixin.qq.com/help2/pc/14822)，班次修改更灵活。
补卡支持设置截止日期，考勤统计更准确。
[审批支持加签](https://open.work.weixin.qq.com/help2/pc/19182)，允许在审批单中新增审批人，共同审批。
公告支持发给上下游企业联系人，通知更便捷。

# [](https://open.work.weixin.qq.com)教育版应用能力增强，助力家校沟通

1、家长通知

学校老师发送的家长通知支持家长[签字确认](https://open.work.weixin.qq.com/help2/pc/16906)。
学校管理员发送的通知，班主任也可以查看并管理。

2、健康上报

健康上报支持设置含有[跳转逻辑的问题](https://open.work.weixin.qq.com/help2/pc/15350)。

3、直播上课

直播上课支持主播[老师对自己静音](https://open.work.weixin.qq.com/help2/pc/16946)。

# [](https://open.work.weixin.qq.com)政务版助力基层疫情防控

1、场所码

场所码支持快捷查找异常人员详情并导出记录。
管理员可以通过Excel导入批量创建场所码。

2、巡查上报

管理员可以分派离职成员的工单。

3、居民上报

管理员可以分派离职成员的工单。

# [](https://open.work.weixin.qq.com)开放能力

1、微信客服

开发者可以更灵活地调用微信客服API，在不通过API管理会话和消息时，也可以获取客户基础信息等。
服务商可通过自建应用代开发模式为企业提供微信客服相关服务。
可通过接口读取客服数据统计，更便捷地管理客服接待情况。
客户通过视频号咨询客服时，支持获取视频号名称，方便企业识别客户来源。

2、客户联系

客户在视频号中添加成员后，支持获取视频号名称，方便为客户提供更精细化的服务。

3、政民沟通

政府可通过接口查询场所码信息，包括场所码列表、上报问卷、上报明细，疫情防控更高效。

4、应用管理

企业和服务商可通过接口打开应用管理页面，应用管理更便捷。
服务商可通过接口打开应用评价页面，企业评价更方便。

5、智慧硬件

通过企业微信云打印机扫描的文件可以自动同步到微盘。
