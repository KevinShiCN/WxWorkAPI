# 如何获取AgentId

AgentId是授权方应用id，管理员可在管理后台进入应用详情获取应用的AgentId。

若过去90天未通过API secret调用接口的企业，企业管理后台API设置中的secret、Agentld、企业可信IP将不再展示，企业可通过自建应用的secret调用接口。
可在【[管理后台](https://work.weixin.qq.com)->应用管理->自建应用->Secret】，点击查看后通过客户端企业微信团队下发。

若过去90天有通过API secret调用接口的企业，企业管理后台API设置中的secret、Agentld、企业可信IP将保留，可以在【[管理后台](https://work.weixin.qq.com)->应用管理->对应的应用页面点击api查看】。

也可以通过[API操作](https://work.weixin.qq.com/api/doc/90000/90135/90665)。
