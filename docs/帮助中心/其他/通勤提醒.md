# 通勤提醒

### [](https://open.work.weixin.qq.com)一、什么是通勤提醒

通勤提醒主要帮助企业成员提供每日上班交通路线规划、实时公交、天气及气温信息，从而提升通勤效率。
![](https://wework.qpic.cn/wwpic/446912_XclozvQLStaCDzT_1663309168/0)

### [](https://open.work.weixin.qq.com)二、如何开通通勤提醒

【手机端->消息页面左上角“三”->点击姓名->通勤】在通勤提醒页面设置家庭住址、公司位置、通勤方式、提醒时间，完成设置后，将在每个工作日设置时间点收到当日最佳通勤方案；系统会自动判断上班所需的时间，提前发送消息提醒该出门上班了。

交通方式支持公交、驾车两种方式；在通勤线路卡片上可以筛选时间短/少拥堵/少收费的线路。

设置完成后，后续可以打开通勤提醒->点击右上角三个点进行修改交通方式、位置信息、提醒时间等信息。
![](https://wework.qpic.cn/wwpic3az/360544_Lb97EzEcTNmW-gk_1765264262/0)

### [](https://open.work.weixin.qq.com)三、如何关闭通勤提醒

【手机端->消息页面左上角“三”->点击姓名->通勤->右上角“···”】，进入界面后关闭「接收通勤消息」按钮，操作完成后你也不会再接收到任何有关企业微信推送的通勤消息。
![](https://wework.qpic.cn/wwpic/258588_2fvPk4hVRxSWyUP_1663309263/0)

### [](https://open.work.weixin.qq.com)四、常见问题

1、国外可以试用通勤提醒吗？
暂不支持。

2、公交能否自行选择公交线路？
不支持，系统会推荐用时最短的路线。

3、管理员可以统一关掉这个功能吗？
通勤提醒为个人功能，暂不支持由管理员统一关闭。
