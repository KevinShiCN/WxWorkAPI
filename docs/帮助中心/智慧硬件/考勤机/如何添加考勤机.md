# 如何添加考勤机

企业微信目前已支持多种主流品牌机型绑定（[查看支持机型](https://open.work.weixin.qq.com/help2/pc/20775)）。如企业已有可支持机型的设备，可通过扫描设备二维码、输入设备SN码、蓝牙匹配等多种方式添加。
一个打卡规则可以绑定多个考勤机（绑定考勤机数量暂无上限），一个考勤机也可以多个打卡规则。
一个考勤机只能同时对应一个企业，一个企业可以绑定多个考勤机。

#### [](https://open.work.weixin.qq.com)**第一步：确认操作者拥有添加设备权限**

在绑定前，需确认操作者是否有添加设备权限，具体可参考[如何分配考勤机管理权限](https://open.work.weixin.qq.com/help2/pc/15005)。

#### [](https://open.work.weixin.qq.com)**第二步：进入设备添加流程**

**路径1 首页扫一扫绑定**：移动端企业微信消息列表首页->右上角+号->扫一扫->扫描设备二维码
（设备二维码通常展示在设备屏幕、设备说明书等位置，不同品牌型号展示位置略有差异，若未找到设备二维码，可联系设备销售商或厂商咨询）

**路径2 打卡应用内绑定**：移动端企业微信->工作台->打卡应用->考勤机->添加考勤机，可选择扫二维码、输入SN码、蓝牙匹配等多种添加方式

**路径3 硬件管理**：移动端企业微信->工作台->管理企业->硬件管理->添加设备，可选择扫二维码、输入SN码、蓝牙匹配等多种添加方式
（本路径仅限超级管理员或分级管理员可操作，且分级管理员需配置添加硬件权限）

#### [](https://open.work.weixin.qq.com)**第三步：完成设备添加及规则配置**

通过扫码、SN码、蓝牙识别到具体设备信息后，进入设备绑定页面，可按需配置考勤机的可用范围、备注名称等。
添加成功后，还需配置对应打卡规则才可使用，请确保考勤机已配置对应的打卡规则。
