# 如何移除考勤机

超级管理员支持在【管理后台->应用管理->打卡->智慧考勤机->查看详情->移除设备】或【手机端->工作台->打卡->考勤机->点击需要移除的考勤机->设备详情】中移除考勤机。

注：如果企业解散了，考勤机会自动解绑。
