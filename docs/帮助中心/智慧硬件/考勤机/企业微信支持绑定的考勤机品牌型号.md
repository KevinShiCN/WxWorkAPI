# 企业微信支持绑定的考勤机品牌型号

目前与基础打卡应用和审批关联的考勤机有得力、海康威视、中控、宇泛、大华股份、科密、宇视科技、浩顺、商汤星云、安威士、10moons天敏部分考勤机。

支持已有的考勤机快速绑定，若企业未添加考勤机，可在【手机端企业微信->工作台->打卡->考勤机->添加考勤机->查看支持机型】中查看已支持绑定的品牌和机型。
![](https://wework.qpic.cn/wwpic3az/661428_h5JEms5pRRqSXSq_1760681254/0)
![](https://wework.qpic.cn/wwpic3az/367903_mBb1ytfITp2FrJR_1734934634/0)
企业已添加考勤机的，可以在【手机端企业微信->工作台->打卡->考勤机->添加考勤机->查看支持机型】中查看已支持绑定的品牌和机型。
![](https://wework.qpic.cn/wwpic3az/518489_c0fzlJ-2SdSWCj0_1734934884/0)
