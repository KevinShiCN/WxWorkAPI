# WX108/WX702 使用说明

### [](https://open.work.weixin.qq.com)二、WX108/WX702 联网设置：

1、wifi连接
（1）【考勤机菜单->通讯设置->网络设置】DHCP开启、
（2）【考勤机菜单->通讯设置->WiFi设置】在列表最底部选择「高级选项」按M/OK进入设置DHCP选择开启
（考勤机仅支持连接2.4GHZ网络频率，不支持连接需要网页验证的网络）

2、网线连接
（1）自动配置IP地址：【考勤机菜单->通讯设置->网络设置】DHCP开启，【考勤机菜单->通讯设置->WiFi设置】关闭
（2）手动配置IP地址：【考勤机菜单->通讯设置->网络设置】DHCP关闭，需手动配置IP地址，子网掩码，网关地址，DNS, TCP通讯端口需要确认是否有设置过路由器访问外网的端口，如果有设置过这个考勤机的通讯录端口需更改成路由器的一致。

### [](https://open.work.weixin.qq.com)三、WX108-** **输入密码

空白框可直接输入数字，输入错误按“向右键”删除，如密码只有数字输完后当前页面按“向下键”，选中“连接OK”，按“M/OK” 开始连接。

### [](https://open.work.weixin.qq.com)四、WX702-** **输入密码

输入单个字母，选中字母后 再按下OK，即输入单个字母。

### [](https://open.work.weixin.qq.com)五、WX108-** **切换输入法

按“向右键”切换输入法界面默认显示「Aa」为大小写混合输入,继续按“向左键”进行选择字母大小写、数字和符号；显示「A」输入大写, 「a」输入小写，「123」输入数字，「符号」在符号列表选择符号对应的数字输入；输完密码按“ESC”退出输入法界面，按“向下键”选中“连接OK”，按“M/OK” 开始连接。

### [](https://open.work.weixin.qq.com)六、WX108/WX702 管理员设置

考勤机菜单->系统设置->设备管理员（考勤机固件版本004以上）
设置密码，后续进入菜单需要输入密码 或 有管理权限的管理员扫码验证才能进入考勤机菜单。
密码设置为0即关闭管理员验证。
暂不支持恢复出厂设置，可以尝试断电重启。

### [](https://open.work.weixin.qq.com)七、WX108/WX702 如何绑定

设备联网后，企业微信创建人（超级管理员）/设备管理员打开企业微信扫一扫考勤机显示屏的绑定二维码，设置可用范围人员绑定设备使用；
可用范围人员指允许在这台设备打卡的人员，需要在设备打卡的人员则添加到可用范围内。
![](https://wework.qpic.cn/wwpic/79615_-b5ruzVxS3mQqRe_1614908938/0)

### [](https://open.work.weixin.qq.com)八、WX108/WX702录入指纹/人脸

1、管理员操作录入-在设备录入
设置入口：【手机端->工作台->管理企业->设备管理->设备详情->指纹信息/人脸信息】
在设备详情显示已连接状态下管理员在手机端操作；选择成员录入，在考勤机上根据提示录入指纹/人脸，支持一个人录入多个指纹（10个），第一个指纹录入后考勤机会提醒“是否 继续录入”，选择“是”则继续录入当前人员第二个指纹无需继续录入则“否”；
如需重新录入指纹/人脸或录入指纹完成后需录入第二个指纹，需先操作移除指纹再录入。
![](https://wework.qpic.cn/wwpic3az/941093_vXarb7xuTcW3GcE_1764655323/0)
2、员工录入-在设备录入
设置入口：【手机端->工作台->管理企业->设备管理->设备详情->员工录入】
管理员先在设备详情开启自助录入开关，开启自助录入后企业成员可自行使用企业微信扫一扫设备上的二维码，根据考勤机提示操作录入指纹/人脸。
![](https://wework.qpic.cn/wwpic3az/911872_hxASijQbRCWylaP_1764655756/0)

### [](https://open.work.weixin.qq.com)九、WX108/WX702管理设备

绑定智慧硬件后，超级管理员和具有硬件管理权限的分级管理员，可在手机端和管理后台查看和管理设备。
手机端设置入口：【手机端->工作台->管理企业->硬件管理】
管理端设置入口：【[管理后台](https://work.weixin.qq.com)->应用管理->硬件】
1、备注名称
用于区分不同设备，也会显示在与设备相关的应用当中。
2、升级固件
当智慧硬件的固件有新版本时，可在此更新；建议尽可能使用最新版的硬件固件，以修复历史缺陷、体验新功能。
3、关于此设备
了解设备名称、型号、SN编码、品牌、制造商、官网等信息，也可在此联系设备供应商客服。
4、移除此设备
移除后，设备将会与企业解绑，设备内的成员身份信息将被抹除，但不影响使用企业内其他包含身份信息的硬件。

### [](https://open.work.weixin.qq.com)十二、常见问题

1、WX108/WX702/F3369等型号考勤机人脸打卡时没问题，但指纹打卡时语音提示“请重按手指”/“验证失败”/“非法指纹，验证失败”，如何解决？

（1）手机端若显示已录入指纹，则需确认考勤机->用户管理->用户列表中，该打卡人员是否显示指纹图标。若是考勤机中有显示指纹图标，则可能是指纹不识别、匹配度较低，存在几种因素环境或指纹摩擦。

如：冬天手指表面皮肤容易变的很干燥，不利于指纹图像的采集，图像如果不清楚，就无法提取到指纹的特征；因此会出现指纹不灵敏打不上的情况。打不上卡时搓一下双手，或者往手指上哈气，让手指恢复下温度和湿度一般可以正常打卡。考勤机指纹识别的地方也要做清洁。

（2）手机端显示已录入，考勤机却没有指纹图标，需确认考勤机设备的指纹容量是否已满，若已满，可把考勤机设备的可用范围人员全部移除或移除部分可用范围人员后重新勾选人员，等待10分钟指纹会重新下发。

2、WX108考勤机打卡时提示“全员未登记”/“人员未登记”
考勤机通过人脸/指纹打卡时，语音提示“全员未登记”，需要管理员在手机端->工作台->打卡->考勤机->设备详情->移除设备，进行解绑。过两分钟后，重新绑定。绑定成功后，可等待5分钟，再打卡看下是否可以识别。

3、WX108录入指纹时提示重复录入或录入失败。
请尝试将录入提示重复的手指在考勤机打卡，确认是否打卡成功，若打卡成功则已录入过。
