# 如何设置考勤机单独用作打卡/门禁

若企业购买了多个考勤机同时用于打卡和门禁，管理员可以通过【手机端->工作台->打卡->考勤机->设备详情->配套应用详情->点击打卡/门禁-】关闭在当前设备上配套使用，即可单独设置用作打卡或者门禁，硬件使用更灵活。
