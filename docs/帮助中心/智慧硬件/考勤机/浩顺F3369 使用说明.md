# 浩顺F3369 使用说明

### [](https://open.work.weixin.qq.com)一、F3369按键操作说明

![](https://wework.qpic.cn/wwpic/187850_pCzJztr4TMiqa2W_1614905525/0)
考勤机设置视频指引：[点击观看视频 ](https://v.qq.com/x/page/m0967a3b6xr.html)

### [](https://open.work.weixin.qq.com)二、F3369联网设置：

**WIFI连接**
【考勤机菜单->高级设置->通讯设置->无线网络】选择网络
自动获取，“是”则自动分配IP ；“否”需手动设置IP ，网关，子网掩码，网关，DNS。
（考勤机仅支持连接2.4GHZ网络频率，不支持连接需要网页验证的网络）

### [](https://open.work.weixin.qq.com)三、F3369网线连接

【考勤机菜单->高级设置->通讯设置->以太网】选择网络
自动获取，“是”则自动分配IP ；“否”需手动设置IP ，网关，子网掩码，网关，DNS。

### [](https://open.work.weixin.qq.com)四、F3369机器管理功能（考勤机1.07版本以上支持）

【考勤机机器菜单->高级设置->机器设置->管理员密码】，设置密码为1-8位数字，设置成功后，用户进入菜单需要进行验证。设置为0则不需要验证。
解锁方式:
1、机器在线时，按一下MENU键，会弹出扫码解锁界面。可以用管理员扫码解锁。或者按OK键进入密码解锁，输入密码按OK键解锁。
2、机器未绑定或没在线时，按一下MENU键，会弹出密码解锁界面，输入密码按OK键解锁。

### [](https://open.work.weixin.qq.com)五、F3369管理设备

绑定智慧硬件后，超级管理员和具有硬件管理权限的分级管理员，可在手机端和管理后台查看和管理设备。
手机端设置入口：【手机端->工作台->管理企业->硬件管理】
管理端设置入口：【[管理后台](https://work.weixin.qq.com)->应用管理->硬件】
1、备注名称
用于区分不同设备，也会显示在与设备相关的应用当中。
2、关于此设备
了解设备名称、型号、SN编码、品牌、制造商、官网等信息，也可在此联系设备供应商客服。
3、移除此设备
移除后，设备将会与企业解绑，设备内的成员身份信息将被抹除，但不影响使用企业内其他包含身份信息的硬件。
