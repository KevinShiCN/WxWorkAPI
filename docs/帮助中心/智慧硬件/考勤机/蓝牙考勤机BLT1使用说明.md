# 蓝牙考勤机BLT1使用说明

![](https://wework.qpic.cn/wwpic/818541_7AWDcogaQ0GZs2h_1590652916/0)

### [](https://open.work.weixin.qq.com)一、设备开机方法

装2节7号电池，设备指示灯绿灯慢闪烁间隔1秒，则开机完成。

指示灯说明
绿灯慢闪：开机/重置，间隔1秒；设备正常运行，间隔60秒。
绿灯常亮：固件升级
红灯慢闪：低电量（电量低于10%），间隔30秒；

### [](https://open.work.weixin.qq.com)二、扫一扫绑定设备

企业微信创建人（超级管理员）/设备管理员在【移动端->消息->右上方“+”号】打开扫一扫设备上的二维码，为设备设置备注名称、可用范围后，绑定设备到企业。

### [](https://open.work.weixin.qq.com)三、设置可用范围

绑定设备时显示设备详情页，须在此设置设备可用范围。将需要在这台设备打卡的员工设置到可用范围当中。若企业还没有成员，请先 添加成员。
![](https://wework.qpic.cn/wwpic/798769_jLxCf9EfSh6SfBS_1590653243/0)

### [](https://open.work.weixin.qq.com)四、如何打卡

通过手机蓝牙接收考勤机蓝牙信号的方式打卡
1、管理员添加打卡规则，需成员在打卡设置中开启上/下班快捷打卡开关，请确保打卡规则、可用范围、身份信息设置正确；
2、打卡成员手机打开蓝牙开关，并允许“企业微信”访问蓝牙；
3、考勤机可用范围内的成员，在蓝牙考勤机附近（5米内）打开企业微信可自动打卡；在未开启上/下班快捷打卡开关状态下，需点击企业微信-工作台-打卡应用，进行手动打卡，打卡之后手机端自动推送蓝牙考勤机打卡成功提醒；
![](https://wework.qpic.cn/wwpic/947510_dm18XFkHTIKFn4Y_1590654784/0)

### [](https://open.work.weixin.qq.com)五、管理设备

绑定智慧硬件后，超级管理员和具有硬件管理权限的分级管理员，可在移动端和管理后台查看和管理设备。
移动端设置入口：【移动端->工作台->管理企业->硬件管理】
管理端设置入口：【[管理后台](https://work.weixin.qq.com)->应用管理->硬件】

1、升级固件（优化中）
通过蓝牙传输的方式进行固件更新，更新时请手机靠近设备，并保持手机蓝牙开启。当智慧硬件的固件有新版本时，可在此更新；建议尽可能使用最新版的硬件固件，以修复历史缺陷、体验新功能。
2、关于此设备
了解设备名称、型号、SN编码、品牌、制造商、官网等信息，也可在此联系设备供应商客服。
3、移除此设备
移除后，设备将会与企业解绑，设备内的成员身份信息将被抹除，但不影响使用企业内其他包含身份信息的硬件。

### [](https://open.work.weixin.qq.com)六、连接蓝牙考勤机后如何开启人脸识别

管理员可选择是否开启人脸识别验证功能。开启后，企业成员进行考勤打卡时，需先进行人脸识别验证，验证通过后才能完成考勤打卡。

**开启步骤如下：**

1、在打卡应用设置考勤规则，打卡方式选择“智慧考勤机”或“手机+智慧考勤机”。
![](https://wework.qpic.cn/wwpic3az/98338_FkYHrpL7RUSzDpH_1764657366/0)
2、在“更多设置”项选择“拍照与人脸识别”，在“拍照与人脸识别”页，打开开关，并引导企业成员录入人脸信息。
![](https://wework.qpic.cn/wwpic3az/499203_DJUvj_AyRhKnIkh_1764657369/0)

### [](https://open.work.weixin.qq.com)七、常见问题

1、连接蓝牙提示“无蓝牙搜索权限”/“设备当前未连接到网络”。
请您在手机系统设置->应用管理->企业微信的权限管理中检查“本地网络”和“附近的设备”权限是否已开启。另外可尝试换台手机进行绑定操作。

2、蓝牙考勤机BLT1电池一般可以使用多长时间？
6个月。

3、蓝牙考勤机打卡后企业微信原始记录没有记录。
目前蓝牙考勤机是通过手机蓝牙接收考勤机蓝牙信号的方式打卡，由企业微信app探测到蓝牙信号之后进入日报，不用上报考勤记录，故没有原始记录。
