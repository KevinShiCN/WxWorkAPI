# 如何购买考勤机

1、可进入京东进行购买企业微信考勤机设备：[京东购买入口](https://mall.jd.com/index-1000485773.html?from=pc)

2、【手机端->工作台->选应用->应用->浏览分类->智慧硬件】中点击跳转至“企业微信智能办公京东自营旗舰店”选购。
![](https://wework.qpic.cn/wwpic3az/554957_x449tGlMRpOlPUw_1764570882/0)
3、若企业未添加考勤机，在【手机端企业微信->工作台->打卡->考勤机->添加考勤机->选购考勤机】中支持查看考勤机推荐型号并前往购买，设备选购更便捷。
![](https://wework.qpic.cn/wwpic3az/725707_uNv8B9MgTZevbos_1764570423/0)
企业已添加考勤机的，可以在【手机端企业微信->工作台->打卡->考勤机->添加考勤机->选购考勤机】查看考勤机推荐型号并前往购买。
![](https://wework.qpic.cn/wwpic3az/907085_mvVquUmySvqNZuy_1764570427/0)
