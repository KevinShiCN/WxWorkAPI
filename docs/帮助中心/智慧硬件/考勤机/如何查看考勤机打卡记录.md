# 如何查看考勤机打卡记录

考勤机打卡记录首先存储在考勤机本地，部分考勤机支持在考勤机上查看考勤机打卡记录，一般路径为：考勤机菜单->记录查询。

1、考勤机在企业微信显示已连接状态打卡记录首先上传在“智慧考勤机”中，目前仅超级管理员可以看到该入口，仅支持查看近三个月的记录。

查看路径：【管理后台->应用管理->打卡->考勤机管理->右上角时间符号】查看/导出考勤机的原始记录。
![](https://wework.qpic.cn/wwpic3az/566942_iTosruvgQ2KYzl__1753348646/0)
![](https://wework.qpic.cn/wwpic3az/403250_Q8FU1qamQeCzH-g_1753348651/0)
![](https://wework.qpic.cn/wwpic3az/976302_Jd1C_dyrRaqzO6m_1753348653/0)

2、手机端查看路径：【手机端->工作台->打卡->统计->团队统计->考勤机打卡】
![](https://wework.qpic.cn/wwpic3az/126947_j_QKbObJSLKH-XY_1764574627/0)

3、上传的考勤机记录根据成员所在的打卡规则显示在日报。
查看路径：【[管理后台](https://work.weixin.qq.com)->应用管理->打卡->上下班打卡->查看统计】
![](https://wework.qpic.cn/wwpic3az/302373_UIrZukVGT16HXua_1764574631/0)
