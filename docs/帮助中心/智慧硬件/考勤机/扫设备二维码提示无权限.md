# 扫设备二维码提示无权限

1、一台设备仅支持绑定一个企业，一个企业可绑定多台设备无上限。企业绑定设备需由企业创建人、超级管理员、[有绑定权限](https://open.work.weixin.qq.com/help2/pc/15005?person_id=1)的分级管理员操作绑定。
若是普通成员需要绑定考勤机，可在【手机端->消息界面左上角“三”->姓名->个人信息->所属企业->创建人】联系企业创建人，由创建人操作绑定。

2、若**设备已绑定企业员工扫码**提示无权限：
（1）检查是否开启员工自助录入，设置入口：【手机端->工作台->管理企业->硬件管理->设备详情->员工录入】开启自助录入开关。
![](https://wework.qpic.cn/wwpic3az/762239_f_MJiVdSSBKWGo8_1764575325/0)
（2）请确认扫码人员是否在这台设备的可用范围内，可在【手机端->设备详情->可用范围】中查看；若不在设备可用范围内，需添加之后再尝试扫码。
![](https://wework.qpic.cn/wwpic3az/590274_JhGek0CKSOCCwsY_1764575503/0)
（3）请确认扫码人员登录的企业与考勤机的绑定企业是否一致，若被添加进多个企业，可以在【手机端->消息页面->左上角“三”】看到多个企业，可选择进入对应的企业。
![](https://wework.qpic.cn/wwpic3az/19614_gSedQ5IbQlKJ-oe_1764575257/0)
3、若已开启员工自助录入，成员扫码提示“请联系管理员开启自助录入功能”：
需检查成员是否在考勤机绑定的打卡规则中，不在考勤规则中的成员扫考勤机的二维码会出现该提示。
