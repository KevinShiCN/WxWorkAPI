# 没有考勤机还能使用打卡吗

企业微信打卡应用支持多种打卡方式，如果没有考勤机，可使用手机进行上下班打卡（现已支持手机定位打卡和Wi-Fi打卡）及外出打卡。

可在【手机端->工作台->打卡应用->打卡->“上下班打卡”或“外出打卡”】进行打卡。
