# 如何确认网络wifi频率

### [](https://open.work.weixin.qq.com)一、路由器的频率是什么

无线路由一般分为2.4GHz和5.0GHz两个发射频率，路由器可以分为单频和双频或者多频。单频无线路由器是指发射2.4GHZ一个频段；而双频无线路由器指的是发射2.4GHZ、5GHZ两个频段的无线路由器。

### [](https://open.work.weixin.qq.com)二、路由器频率5Ghz和2.4GH的区别

两者区别在于信号强度方面5GHz的频率较短，而频率越短的波长的穿透力就越弱5Ghz在近距离连接的时会传输更快，如果在居室这种有多重墙壁的环境中远距离2.4GHz信号覆盖更强，实际中2.4Ghz更实用。
干扰性方面，5GHz信号使用的设备比较少，而2.4GHz频段还有蓝牙等无线设备使用，因此干扰也会比较多。

### [](https://open.work.weixin.qq.com)三、如何确认考勤机所连接的网络是2.4ghz频率

1、安卓手机端查看（部分手机没有这个功能）
手机与考勤机连接同一wifi，在手机WiFi连接列表点开WiFi连接详情查看。
![](https://wework.qpic.cn/wwpic/181013_k3CtyGfzS7a8N7p_1672721280/0)
2、Windows电脑端查看
电脑与考勤机连接同一WiFi 点击电脑右下角，任务栏上面的WiFi连接图标，点击“属性”按钮，在设置页面到底部可以看到连接信息，或进入电脑设置-网络和internet-查看网络属性；（下图仅供参考）
![](https://wework.qpic.cn/wwpic/641018_I98RjkXWTESJvFw_1672721505/0)
3、其他方式
登录路由器终端查看，查看路由器背面信息，手机或电脑连接路由器WiFi，浏览器网址输入路由器终端配置地址，输入默认账号密码登录，登录提示错误需确认账号密码。【不同品牌路由终端显示界面是不一样的】参考下图示例：
![](https://wework.qpic.cn/wwpic/855400__JxU3VroSxeAjP6_1672721723/0)
2.4GHz和5.0GHz都开启表示双频合并wifi，部分考勤机仅支持连接单频2.4GHz频率，则需单独关闭5.0GHz。
