# 手机端考勤机设备显示“未连接”

考勤机使用过一段时间后，手机端考勤设备显示“未连接”可按以下方法操作：
1、可尝试更换手机热点连接考勤机网络；

2、重启wifi路由器，考勤机wifi关闭重新打开操作连接；

3、部分机型需要开启DHCP设置，具体有以下设备：
中控WX702 / WX108：考勤机菜单 -> 通讯设置 -> 网络设置 -> DHCP开启;
中控WX3960 / 中控W6：考勤机菜单 -> 网络设置 -> WiFi设置 -> WiFi列表界面按“0”进入高级选项DHCP左右键选择“开启”。
