# 如何分配考勤机管理权限

#### [](https://open.work.weixin.qq.com)1、给分级管理员分配权限

登录[管理后台](https://work.weixin.qq.com)，点击【我的企业->权限管理->分级管理员->编辑->硬件管理权限】，勾选绑定设备权限、需管理权限的设备。
![](https://wework.qpic.cn/wwpic3az/527784_lxCZcKUrSi2sGD9_1753350491/0)
![](https://wework.qpic.cn/wwpic3az/247434_EQjl4DwoRW2b8AE_1753350861/0)

#### [](https://open.work.weixin.qq.com)2、给成员分配权限

【管理后台->应用管理->硬件->设置->成员绑定硬件->勾选开启】，若取消开启，取消后，仅超级管理员和具有硬件绑定权限的分级管理员可绑定新的硬件设备。
