# 海康威视WX668 使用说明

![](https://wework.qpic.cn/wwpic/793600_Kf2mKnJ0SrWMnuh_1588039436/0)
考勤机设置视频指引-[点击观看视频](https://v.qq.com/x/page/d0963x9ut90.html)

### [](https://open.work.weixin.qq.com)一、WX668 如何激活设备

1、首次使用通电开机进入激活设备界面，设置考勤机密码输入两次，用于后续进入考勤机菜单配置考勤机功能
**（请谨慎输入，不支持修改/更换密码，如遇忘记密码功能请联系店铺客服）**
2、环境模式：选择环境模式「室内」或「室外」；注意室外应避免暴晒曝光淋雨等环境
3、选择网络：「有线」，直接插入网线进入下一步
4、选择网络：「无线」，wifi列表中找到您公司的wifi进行输入密码后连接。

### [](https://open.work.weixin.qq.com)二、WX668 考勤机主界面

进入菜单：待机界面点击左上角时间，输入密码或设备在线状态管理员扫码进入。

### [](https://open.work.weixin.qq.com)三、WX668 如何关联企业微信

设备联网后，企业微信创建人（超级管理员）/设备管理员打开企业微信扫一扫考勤机显示屏的绑定二维码，设置可用范围人员绑定设备使用；
可用范围人员指允许在这台设备打卡的人员，需要在设备打卡的人员则添加到可用范围内。
![](https://wework.qpic.cn/wwpic/499167_9j91JqEPR3ezUbo_1601271782/0)

### [](https://open.work.weixin.qq.com)四、WX668 录入人脸

1、管理员操作录入-在设备录入
在【工作台->企业管理->硬件管理】中选择设备，进入「设备详情」页，点击「人脸信息」，选择员工，在员工的人脸信息页点击「录入人脸信息」，选择「在设备上录入人脸」；
员工在设备前，根据设备屏幕上操作指引，等待设备录入。
![](https://wework.qpic.cn/wwpic3az/958395_SXNJvn8pROyFrde_1764656697/0)
2、员工录入-在设备录入
设置入口：【手机端->工作台->管理企业->硬件管理->设备详情->员工录入】
管理员先在设备详情开启自助录入开关，开启自助录入后在点击设备左上角时间进入选项选择自助录入，企业成员可自行使用企业微信扫一扫设备上的自助录入二维码，根据考勤机提示操作录入；
![](https://wework.qpic.cn/wwpic3az/974635_0i3axHhfQ82Ai2C_1764656621/0)

### [](https://open.work.weixin.qq.com)五、WX668 门禁使用

WX668考勤机（最新固件版本）可以配合门禁应用使用，进行远程开门。

管理员在【手机端->工作台->其他->门禁->设置规则】，可以设置门禁设备，通行时间，通行人员，远程开门人员。从而限制哪些人可以在什么时段通过哪个门禁。未在通行人员/时间内/未设置门禁规则，将会提示权限不足。

有远程开门权限的人员在门禁应用内可以查看到远程开门选项，点击后可以通过考勤机开启门禁，并会记录一次打卡数据。

### [](https://open.work.weixin.qq.com)六、WX668 管理设备

绑定智慧硬件后，超级管理员和具有硬件管理权限的分级管理员，可在手机端和管理后台查看和管理设备。
手机端设置入口：【手机端->工作台->管理企业->硬件管理】
管理端设置入口：【[管理后台](https://work.weixin.qq.com)->应用管理->硬件】
1、备注名称
用于区分不同设备，也会显示在与设备相关的应用当中。
2、升级固件
当智慧硬件的固件有新版本时，可在此更新；建议尽可能使用最新版的硬件固件，以修复历史缺陷、体验新功能。
3、关于此设备
了解设备名称、型号、SN编码、品牌、制造商、官网等信息，也可在此联系设备供应商客服。
4、移除此设备
移除后，设备将会与企业解绑，设备内的成员身份信息将被抹除，但不影响使用企业内其他包含身份信息的硬件。

### [](https://open.work.weixin.qq.com)七、常见问题

1、WX668如何初始化指纹？
目前暂不支持。

2、如何设置音量？
进入考勤机菜单->系统设置->系统信息->语音音量。

3、设置wifi密码提示“密码长度设置错误”。
WX668的密码设置长度上限为：8-16位字符密码（数字/字母/符号两种或两种以上组合）

4、WX668考勤机是否会自动补光？
目前暂时不支持自动补光，需要手动进入【考勤机菜单->系统设置->白光补光灯亮度】中进行设置。

5、WX668在连接到互联网时，需要使用到哪些端口？
仅支持连接2.4G  AP频率wifi，不支持连接需要验证的WiFi。暂无端口限制。

6、WX668已联网，扫码提示“设备绑定二维码过期”/WX668考勤机无法联网
请登录海康威视的官网，下载工具进行设备校时，较时后重试联网。
详情可参考：[如何给门禁设备校时？](https://hkrobot.hikvision.com/servlet/WBShow?action=sac&sysNum=145716889796196&wbId=-100&FromUserName=API_1726906206055874561&question=%26nbsp%3B%E5%A6%82%E4%BD%95%E7%BB%99%3Cstrong%3E%3Cspan+style%3D%22color%3A+rgb%28255%2C+0%2C+0%29%3B%22%3E%E9%97%A8%E7%A6%81%E8%AE%BE%E5%A4%87%3C%2Fspan%3E%3C%2Fstrong%3E%E6%A0%A1%E6%97%B6%EF%BC%9F&fid=&sId=460923&MsgType=workflow)

7、有海康威视其他型号考勤机，该如何使用？
可联系人工客服获取使用操作指引。
