# 如何连接网络硬件的WiFi

网络硬件需结合企业微信网络应用使用，企业管理员使用企业微信绑定设备成功后，支持以下几种方式连接WiFi：

#### [](https://open.work.weixin.qq.com)1、本机一键连网

企业成员进入网络应用后，确保手机已开启Wi-Fi与GPS，可点击“一键连网”按钮进行连网。
![](https://wework.qpic.cn/wwpic/357243_P8uKgisnR0KyGK6_1612531321/0)

#### [](https://open.work.weixin.qq.com)2、个人终端连网

企业成员可在网络应用「我的密码」中查看个人专属密码，使用此密码为个人的其他终端连接企业Wi-Fi。
![](https://wework.qpic.cn/wwpic/573380_5jwBk9gPQIOk8Jh_1612531329/0)

#### [](https://open.work.weixin.qq.com)3、访客设备连网

确保访客网络已开启的前提下，企业成员通过网络应用「访客连网」中查看访客密码，将该密码分享给访客后，访客可使用该密码连接企业访客网络。
![](https://wework.qpic.cn/wwpic/714685_mhhFsfZDTYaGnk0_1612531338/0)

#### [](https://open.work.weixin.qq.com)4、办公设备连网

企业管理员通过网络应用「办公设备连网」中查看办公设备专属密码，可在需要连网的办公终端-无线网列表中，使用此密码连接企业WiFi。
![](https://wework.qpic.cn/wwpic/565830_tInXLc8cQbuP2-u_1612531353/0)
