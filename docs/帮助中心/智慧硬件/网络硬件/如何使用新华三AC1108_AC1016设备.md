# 如何使用新华三AC1108/AC1016设备

新华三AC1108与AC1016设备都是网关控制器，集精细的用户控制管理、完善的射频资源管理、7×24小时无线管控、灵活的Qos控制、IPv4&IPv6双栈等多功能于一体，提供强大的有线、无线一体化的接入功能。

可按照以下步骤使用AC1108/AC1016设备：
第一步：连接设备电源
第二步：使用网线连接外网，一端接入设备WAN口，另一端接入外网网口
第三步：绑定设备
1、扫一扫绑定设备
企业管理员可使用企业微信app扫一扫AC设备上的二维码，为设备设置备注名称后，绑定设备到企业。
![](https://wework.qpic.cn/wwpic/729978_HVCKo5WhRra7Tcy_1612531160/0)
2、输入SN码绑定设备
企业管理员还可通过输入设备机身或包装上的SN编码找到设备，设置备注名称后，绑定设备到企业。
![](https://wework.qpic.cn/wwpic/53382_xMdpQ2ZbQSKvp2__1612531168/0)
