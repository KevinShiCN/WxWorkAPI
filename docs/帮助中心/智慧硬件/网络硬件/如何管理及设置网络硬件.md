# 如何管理及设置网络硬件

企业管理员使用企业微信绑定设备成功后，可在网络应用中管理以下内容：

#### [](https://open.work.weixin.qq.com)1、设置网络应用可见范围

企业管理员可在网络设备详情页中点击配套网络应用，在应用详情页内设置可见范围，可见范围内的成员即可在企业微信的工作台中看见并使用网络应用。
![](https://wework.qpic.cn/wwpic/551244_lZD9Z_IzRByb3r6_1612530319/0)

#### [](https://open.work.weixin.qq.com)2、查看网络使用数据

企业管理员在网络应用「企业网络管理」中，查看企业网络使用情况及联网终端情况。
![](https://wework.qpic.cn/wwpic/355300_phguaY0dSMumUn4_1612530387/0)

#### [](https://open.work.weixin.qq.com)3、管理联网终端

企业管理员可点击「终端数据」卡片，查看联网终端数据详情。支持搜索筛选员工终端、访客终端、办公终端以及禁用设备。
![](https://wework.qpic.cn/wwpic/995463_d8o0TVFoQh6ZU2K_1612530415/0)

#### [](https://open.work.weixin.qq.com)4、设置企业网络

企业管理员在「企业网络设置」页面中，支持查看修改企业网络名称、访客网络开关、访客网络名称及密码。
![](https://wework.qpic.cn/wwpic/747733_WsC3qulFS4WibZM_1612530459/0)

#### [](https://open.work.weixin.qq.com)5、管理网络设备

企业管理员在「网络设备管理」中查看设备详情，支持解绑和重启设备。
![](https://wework.qpic.cn/wwpic/538694_rx2gvzUJSF6L0a9_1612530497/0)
