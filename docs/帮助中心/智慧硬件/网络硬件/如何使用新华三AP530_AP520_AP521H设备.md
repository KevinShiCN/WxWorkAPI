# 如何使用新华三AP530/AP520/AP521H设备

新华三AP530/AP520/521H设备都是无线接入点，在WLAN与Internet之间起到连接作用，全面支持IPv4&IPv6双协议栈，接入速率高达1.934Gbps，可有效提升企业高密场景的无线接入能力。

可按照以下步骤使用AP530/AP520/AP521H设备：
第一步：连接设备电源
第二步：使用网线连接AC设备，一端连接AC设备LAN口，另一端连接AP设备GE口
第三步：当设备指示灯绿色常亮时，即可通过扫一扫或输入SN码绑定设备。

可点击查看[如何管理及设置网络硬件](https://open.work.weixin.qq.com/help2/pc/15018)
