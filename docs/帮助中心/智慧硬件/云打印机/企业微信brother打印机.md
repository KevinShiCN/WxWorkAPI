# 企业微信brother打印机

### [](https://open.work.weixin.qq.com)一、设备激活与联网

经销上门安装配置，激活联网后经销商提供绑定设备的二维码。

### [](https://open.work.weixin.qq.com)二、打印操作

1、企业微信工作台->打开任一文件浏览->右上方选择打印->显示打印->选择打印参数（页数/颜色（彩色或黑白））
2、在打印机面板上选择“企业微信”功能->扫描显示出来的二维码，再按OK输出打印作业

支持联网高速打印、复印、扫描、传真。

### [](https://open.work.weixin.qq.com)三、墨粉盒

墨粉盒中的粉(墨粉)是用来生成文字和图像的。

### [](https://open.work.weixin.qq.com)四、硒鼓单元

硒鼓单元是将墨粉转印到纸上。将激光打印机的墨粉盒放在硒鼓单元中称为感光鼓的辊上，然后通过加热和加压将墨粉定影在纸上来打印文字和图像。
Brother机器会自动检测到硒鼓单元打印质量的降低。如Brother机器上出现“更换硒鼓”的错误信息，则表示需要更换硒鼓单元。

### [](https://open.work.weixin.qq.com)五、接口相关

v4.0.20起，第三方应用内的文件可以直接使用企业下绑定的打印设备进行打印。（可参考[接口文档](https://developer.work.weixin.qq.com/document/path/97287)）
