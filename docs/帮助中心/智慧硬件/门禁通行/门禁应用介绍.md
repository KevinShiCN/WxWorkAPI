# 门禁应用介绍

### [](https://open.work.weixin.qq.com)一、如何设置门禁权限管理

管理员可在手机端进行门禁权限管理，企业成员可在手机端查看自己的门禁权限。
设置路径：【手机端->工作台->其他->门禁】
![](https://wework.qpic.cn/wwpic3az/240328_mbCKjMw1S0CGOgL_1764570908/0)

### [](https://open.work.weixin.qq.com)二、新建门禁规则

1、路径：【手机端->工作台->其他->门禁->门禁管理->添加门禁规则】
**规则名称**：必填；
**门禁设备**：必填，选择规则生效的设备；
**通行人员**：必填，在勾选范围内的人员才可以在设备前通行，否则会出现“权限不足”类似的提示 无法开门通行；
**通行时段**：必填，不在通行时段的时间内也无法开门通行。
![](https://wework.qpic.cn/wwpic3az/464810_cD6A_KQcRAGww0__1764570913/0)

2、路径：【手机端->工作台->其他->门禁->远程开门】
**远程开门**：非必填，远程开门权限可开放给部分成员。
![](https://wework.qpic.cn/wwpic3az/703152_u9K_jbU2RjCawOp_1764570922/0)

### [](https://open.work.weixin.qq.com)三、通行记录统计

支持在【手机端->工作台->其他->门禁->统计】
或【登录企业微信[管理后台](https://work.weixin.qq.com)->应用管理->门禁->门禁记录】查看通行记录，支持导出进出明细的门禁记录。
![](https://wework.qpic.cn/wwpic3az/43366_bH2piYBdSGSVcMs_1753346147/0)
