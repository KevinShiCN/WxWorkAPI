# 测温

### [](https://open.work.weixin.qq.com)一、什么是测温

所有行业类型的企业都支持使用「测温」应用，配合企业微信智慧测温设备，可以查看和导出员工或学生的温度测量记录，及时掌握员工或学生的发热状况。

可点击此处[购买设备](https://sale.jd.com/mall/GFSsevk0JAVMNrXh.html)。
管理员可在【管理后台->应用管理->测温】开启/停用应用。
![](https://wework.qpic.cn/wwpic3az/402744_i6SSYw6TSRqj45z_1753346387/0)

### [](https://open.work.weixin.qq.com)二、常见问题

#### [](https://open.work.weixin.qq.com)1、谁可以看到测温数据？

（1）企业内的成员使用了测温仪测温后，数据目前仅超级管理员可见。
（2）家校通讯录中的家长使用了测温仪测温后老师可以看到管理范围里的班级的数据统计情况。超级管理员可以看见所有学生的测温情况。

#### [](https://open.work.weixin.qq.com)2、如何设置家长侧收到测温成功的提醒？

管理后台暂不支持设置家长侧收到测温成功的提醒。可通过[接口](https://work.weixin.qq.com/api/doc/90002/90150/93992)实现。

#### [](https://open.work.weixin.qq.com)3、测温有数量限制吗？记录的是第一次体温吗？

一天支持多次测量；统计仅会取最后一次的体温。

#### [](https://open.work.weixin.qq.com)4、可以导出测温记录吗？

支持导出统计记录，目前一次仅支持导出一天的统计，以文档形式导出。
