# 硬件介绍

### [](https://open.work.weixin.qq.com)一、功能说明

企业管理员从硬件服务渠道购买硬件设备后，将设备绑定至企业、添加配套应用并设定应用可见范围后，企业内成员即可通过硬件设备及配套应用，获得软硬一体化的智慧服务。

### [](https://open.work.weixin.qq.com)二、企业使用硬件服务流程

1、企业管理员，从硬件服务商的线上渠道、线下渠道进行购买硬件设备；
![](https://wework.qpic.cn/wwpic3az/174394_07BpQndcSzGY2SO_1764568487/0)

2、企业管理员，开箱进行硬件设备和企业微信的关联匹配（可用企业微信扫一扫、蓝牙、SN编码输入的三种方式）；
![](https://wework.qpic.cn/wwpic3az/133882_SA9B4VsSS3SRAbD_1764568490/0)

3、企业管理员，确认绑定设备，并添加配套应用及选择配套应用的人员可见范围；
![](https://wework.qpic.cn/wwpic/340354_qWuWAXEtRjOIVnW_1663226898/0)

4、分配考勤机管理权限
（1）登录管理后台，点击【我的企业->权限管理->分级管理员】并编辑分级管理员；
![](https://wework.qpic.cn/wwpic3az/118986_OXy9uHlyTGSUfkJ_1753350885/0)
（2）【硬件管理权限】勾选绑定设备权限、需管理权限的设备。
![](https://wework.qpic.cn/wwpic3az/166323_Rx5MHbmcTqGLyV2_1753350889/0)
