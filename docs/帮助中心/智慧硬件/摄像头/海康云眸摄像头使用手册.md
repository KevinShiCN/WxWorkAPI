# 海康云眸摄像头使用手册

## [](https://open.work.weixin.qq.com)使用向导

欢迎使用海康云眸视频，这是一款视频监控服务类小程序。
通过海康云眸视频，您可轻松查看商铺、办公室、站点等场所的实时视频与历史录像。

### [](https://open.work.weixin.qq.com)一、设备配网与绑定

在企业微信添加设备成功后，配套小程序“海康云眸视频”即出现在工作台。（如果未出现可以手动添加）
![](https://wework.qpic.cn/wwpic3az/138015_MfevpRxISoy6PAT_1750052621/0)
点击进入小程序后，需将设备进行配网与绑定。
1、**「选择配网设备」**：在小程序首页，点击“去配置”，选择本次需要配网绑定的设备，点击“去绑定”，进入配置页。
2、**「配网准备」**：将手机连上可用WiFi后，确认设备处于通电状态，且指示蓝灯正在快速闪烁中，点击“下一步”（如指示蓝灯未快速闪烁，请尝试将设备进行RESET重启）。
![](https://wework.qpic.cn/wwpic3az/63806_6VXTVLe7QjS8-BN_1713866194/0)
3、**「输入WiFi密码」**：输入WiFi密码，点击“确定”，将WiFi信息配置给设备。
4、**「配网绑定成功」**：等待片刻后，小程序与设备均会进行配网成功与否的提示。如配网不成功，请按照指示进行重
新操作。
![](https://wework.qpic.cn/wwpic3az/740439_xAokGEZzQAe0ZS0_1713866292/0)
注：
1、选择WiFi时，请勿选择5G频段的WiFi。
2、配网与绑定成功后，如设备插有存储卡，请先对存储介质进行初始化，才能进行视频录像的存储。

### [](https://open.work.weixin.qq.com)二、预览与回放

配网与绑定成功的视频设备，可点击首页“视频卡片”，进入进行实时预览与录像回放。
1、**「实时预览」**：实时预览视频时，支持暂停、调整清晰度、画面最大化操作。
2、**「录像回放」**：进入录像回放，可选择录像日期、时间点进行回放；时间轴上的绿色区域代表该时段有录像。
![](https://wework.qpic.cn/wwpic3az/371395_wonib1fPRdKK17D_1713866485/0)

### [](https://open.work.weixin.qq.com)三、设备管理

点击首页“视频卡片”的“…”，进入该设备的设备管理页。

1、状态栏：
可查看设备的名称，型号，设备序列号，设备在离线状态。

2、操作栏：
可对“网络设置、设备录像设置、画面翻转、设备封面”进行设置。
**「网络设置」**：支持更换该设备的WiFi。
**「设备录像设置」**：如需进行设备录像存储，需在设备中配置SD存储卡。
录像形式：支持“告警录像”与“全天录像”。告警录像指仅存储设备移动侦测告警的录像片段；全天录像指连续不间
断对录像进行存储。录像存储根据存储卡容量循环覆盖。
初始化SD卡：支持对SD卡进行初始化（格式化）。
**「翻转」**：如因安装等原因，查看视频时发现画面颠倒，可通过翻转对画面进行翻转。
**「设备封面」**：支持对设备首页的封面图进行刷新。
![](https://wework.qpic.cn/wwpic3az/155001_ThiwYK8GRhufyiT_1750052701/0)
温馨提示：
1、设备名称请在企业微信-设备管理的设备备注名处进行修改。
2、存储卡首次使用时，需进行初始化才能进行录像存储。

### [](https://open.work.weixin.qq.com)四、团队管理

主管理员可对团队进行管理，选择顶部“团队”栏，实现子管理员的设置与主管理员身份的转让。
身份的变更，将通过企业微信消息告知相关成员。

子管理员设置：
主管理员支持从可见范围内，最多任命4名子管理员。

**「权限范围」**：子管理员在应用内，可进行视频的实时预览、录像回放、设备管理等功能。

主管理员转让
主管理员可将自己的身份，转让给可见范围内的其他成员。
![](https://wework.qpic.cn/wwpic3az/601717_PCPg4D5tQpWMO1b_1750052751/0)

### [](https://open.work.weixin.qq.com)五、常见问题

1、谁是首个主管理员？
首个关联“海康云眸视频”的企微管理员，自动成为小程序的首个主管理员。

2、“海康云眸视频”最多可管理几个设备？
“海康云眸视频”最多可管理20个设备，超出数量的设备无法进行管理与视频查看。设备按照添加至企业微信的顺序进行排序，如对前20个中的设备进行删除，则后续设备可依次进行替补。

3、哪些人可使用“海康云眸视频”？
本应用人员配置为：1个主管理员+4个子管理员，其他在可见范围内但非管理员的成员，无法使用本小程序。

4、什么设备可配合“海康云眸视频”使用？
“海康云眸视频”的设备从企业微信-设备管理同步设备，目前仅支持接入：萤石&企业微信联名的摄像头。
