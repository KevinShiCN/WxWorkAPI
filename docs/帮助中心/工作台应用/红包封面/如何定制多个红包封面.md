# 如何定制多个红包封面

企业定制“品牌红包封面”，成员领取后可在企业微信发红包时使用。
目前，企业可为每个品牌定制3个封面，默认每个企业可定制3个封面。
若企业存在多个品牌，可通过认证流程，基于公司简称/机构简称，或基于商标，为企业添加多个简称。

**认证多个简称后，企业可定制与简称数量相等数量的品牌封面**，以满足多个品牌的定制需要。
