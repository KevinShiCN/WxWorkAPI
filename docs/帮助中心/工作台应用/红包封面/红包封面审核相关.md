# 红包封面审核相关

### [](https://open.work.weixin.qq.com)**1、驳回原因：该红包封面不得出现除企业名称、企业自有商品及服务商标、春节祝福语以外的文字，如文字包含在商标里，请提供相关证明文件后重新提交。**

该原因主要是企业上传的背景图包含了广告语相关等非企业名称或祝福相关的文字，红包封面旨在企业提供给员工企业文化关怀，请勿将其当作广告封面来设计。ps：如企业有与企业名字不一致的子品牌、子产品名字，可提供相关商标证明后再次审核
![](https://wework.qpic.cn/wwpic/342475_jqkwrpGiSUO78ow_1644479169/0)

### [](https://open.work.weixin.qq.com)**2、驳回原因：封面限定区域内不能出现任何元素，请修改后重新提交。**

该原因主要是企业上传的封面，在“头像昵称、“祝福语”、“开”、“查看领取详情”位置（如下图红框位置）上有影响查看的图案。
![](https://wework.qpic.cn/wwpic/638886_Xlpqw5J0SI-g_mU_1644479189/0)

### [](https://open.work.weixin.qq.com)**3、驳回原因：封面底色色值需为#f35543（红色），请修改后重新提交。**

该原因主要是企业上传的背景图的背景色值不对，背景图底色色值需为#f35543（红色）
![](https://wework.qpic.cn/wwpic/442462_nBavrWHBRluzckc_1644479208/0)

### [](https://open.work.weixin.qq.com)**4、驳回原因：该红包封面不得出现人民币图样、场景和与人民币相关的信息，请修改后重新提交。**

该原因主要是企业上传的背景图包含了人名币相关的图样或符号
![](https://wework.qpic.cn/wwpic/533622_82J8CwBARiGz8nN_1644479256/0)

### [](https://open.work.weixin.qq.com)**5、驳回原因：该红包封面不得直接展示企业经营的商品实物图片或提供的服务实际场景，请修改后重新提交。**

该原因主要是企业上传的背景图直接包含了企业的商品或产品实物，但抽象化的商品或产品的表达是可以被接受的。
![](https://wework.qpic.cn/wwpic/977559_nfgKaQR_Tiqh2aq_1644479275/0)
