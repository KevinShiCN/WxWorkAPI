# 如何填写汇报

### [](https://open.work.weixin.qq.com)一、主动填写汇报

可从【手机端/电脑端->文档->新建汇报->填写汇报】页面进行填写汇报
![](https://wework.qpic.cn/wwpic3az/128287_Y5uVclAnTnOkncI_1766714362/0)

### [](https://open.work.weixin.qq.com)二、收到提醒填写通知填写汇报

【手机端/电脑端->文档->点击对应汇报提醒->填写汇报】
![](https://wework.qpic.cn/wwpic3az/754613_XehubwxTQGijTyw_1766714380/0)

### [](https://open.work.weixin.qq.com)三、从汇报应用填写

【手机端/电脑端->工作台->汇报->写汇报->填写汇报】
![](https://wework.qpic.cn/wwpic3az/967927_gOQd2trmSbmpWOk_1750645757/0)

### [](https://open.work.weixin.qq.com)四、周期内填写多份汇报

【手机端->填写汇报界面->点击右下方->写汇报】；
![](https://wework.qpic.cn/wwpic3az/510038_ZsP0iDZNRM6xZHG_1766714427/0)
【电脑端->填写汇报界面->点击右上方->填写】
![](https://wework.qpic.cn/wwpic3az/285567_ly_7kcOVRkKUITm_1750645852/0)
