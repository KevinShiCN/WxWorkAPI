# 如何添加汇报模板

如何创建汇报模板？[点击观看视频介绍](https://work.weixin.qq.com/nl/learning/24488325960827077_1)

### [](https://open.work.weixin.qq.com)一、汇报模板设置入口

1、手机端：【工作台-汇报->管理->发布汇报】
![](https://wework.qpic.cn/wwpic3az/517556_Xt2R7WdcROmVkIu_1766714120/0)
【文档->右上角“+”->新建汇报->发布汇报】
![](https://wework.qpic.cn/wwpic3az/695190_FBTYvMrjR_aq5Wy_1766714139/0)
2、电脑端：【工作台/文档->汇报->发布汇报】
![](https://wework.qpic.cn/wwpic3az/796352_naIqaF8oQOS2EO8_1750644911/0)

3、【管理后台->应用管理->汇报】暂不支持新建汇报模板。

4、新建方式：

- 支持新建空白汇报，自定义模板内容；
- 支持选择模板新建，快速设置日报、周报、月报等汇报；
- 支持从已有汇报复制，一键复制已有模板内容。

5、修改汇报模板入口：
超级管理员或汇报创建人，可在【手机端/电脑端->工作台->汇报->管理->点击对应模板->右上角“≡”->编辑内容】修改汇报模板。

### [](https://open.work.weixin.qq.com)二、汇报模板内容设置

**汇报模板支持的控件有：**
「文本」、「多行文本」、「单选框」、「多选框」、「下拉选择」、「数字」、「金额」、「图片」、「文件」、「地理位置」、「体温」、「签名题」、「表格题」
通讯录—「成员」、「部门」
时间—「日期」、「时间」、「时长」
位置—「地址」、「省份」、「自动定位」
联系方式—「手机号」、「微信号」、「邮箱」
填写人信息—「姓名」、「性别」、「年龄」、「身份证号」、「出生年月」、「服装尺寸」、「社会关系」

**模板设置内容上限：**

- 汇报模板标题：50个字符
- 汇报模板标题说明内容：暂无限制
- 问题标题：暂无限制
- 问题标题说明：暂无限制
- 问题标题说明插入图片：暂无限制
- 选项上限：暂无限制
- 选项字符上限：暂无限制
- 一个模板可添加控件问题上限：暂无限制
- 表格题可添加的子控件上限：暂无限制
- 设置特别关注的统计项：上限4个
- 单个模板内的题目数量：暂无限制，但考虑到数据导出，不建议题目超过2000条

![](https://wework.qpic.cn/wwpic3az/651479_SE8oJF76Th2bIW3_1750644919/0)

### [](https://open.work.weixin.qq.com)三、汇报模板上限

汇报模板上限为1000个。

### [](https://open.work.weixin.qq.com)四、转交汇报模板所有权

仅汇报模板创建者可以操作“转交所有权”。暂不支持转交给自己。

操作路径：
【手机端->工作台->汇报->管理】/【手机端->文档】选择对应的汇报->点击右上角“≡”->转交所有权。
![](https://wework.qpic.cn/wwpic3az/478405_jgnxYtb0QGy-3hS_1766714168/0)
【电脑端->工作台->汇报->统计】/【电脑端->文档】，选择对应的汇报->点击右上角“≡”->转交所有权。
![](https://wework.qpic.cn/wwpic3az/968029_YliIxQ8zSFC7JCT_1750644923/0)
