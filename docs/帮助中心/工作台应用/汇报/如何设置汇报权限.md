# 如何设置汇报权限

### [](https://open.work.weixin.qq.com)一、汇报应用管理员

支持给分级管理员添加汇报应用管理权限，拥有汇报应用管理权限的分级管理员，支持在管理员后台查看管理范围内的成员提交的汇报内容，分级管理员暂不支持管理汇报模板。

### [](https://open.work.weixin.qq.com)二、汇报创建权限

超级管理员支持设置【允许员工创建汇报】，可选择全部或指定成员创建，拥有创建汇报权限的成员支持在手机端创建和管理自己创建的汇报模板。

注：超级管理员在文档内创建的汇报模板汇报范围（填写人范围）可针对所有人，而成员创建的汇报模板最多500人。

设置路径：
【[管理后台](https://work.weixin.qq.com)->应用管理->汇报->权限设置】
