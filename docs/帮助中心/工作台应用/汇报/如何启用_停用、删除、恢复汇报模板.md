# 如何启用/停用、删除、恢复汇报模板

### [](https://open.work.weixin.qq.com)一、操作入口

汇报模板支持在【管理后台->应用管理->汇报->鼠标放至对应汇报模板->停用/启用、删除】
![](https://wework.qpic.cn/wwpic3az/4556_EyTNY1kWT3uu7GS_1753842424/0)

【客户端->打开对应汇报->右上角“≡”->停用/启用、删除】
![](https://wework.qpic.cn/wwpic3az/127431_bX--Ah0ZTKqZpzL_1766714218/0)
注：在【文档】或【微盘->我的文件】中删除了我创建的文档汇报或收集表汇报，对应的汇报模板会被删除。如果是不删除文档，仅删除提交汇报附件时同步到微盘的文件，不会删除汇报模板。
若是管理员在管理后台清空员工的个人空间，员工所创建的汇报模板也会被删除。

### [](https://open.work.weixin.qq.com)二、如何恢复被删除的汇报模板

**7天内删除**的汇报模板，模板创建人支持在【电脑端->文档->全部->回收站】；【手机端->工作台->微盘->微盘空间->回收站】中恢复/还原。
如模板创建人离职，也可以通过企业回收站进行恢复（企业回收站恢复需要成员是高级功能账号才支持恢复删除180天内的汇报模板，若离职前不是高级功能账号删除了汇报模板，没办法通过企业回收站恢复。）

注：

- v4.1.0版本之前删除的汇报模板暂不支持恢复。
- 提交人删除了自己提交的汇报记录暂不支持恢复。
- 被删除的汇报模板需要是模板的创建人才可以在回收站看到，不是模板创建人看不到（包含超级管理员）。
