# 如何在汇报中上传附件

### [](https://open.work.weixin.qq.com)1、管理员给汇报模板添加【附件】控件

管理员需要在【手机端/电脑端->工作台->汇报->点击对应模板->右上角“≡”->编辑内容->底部添加问题】中给对应汇报模板添加了【附件】控件后，成员在客户端提交汇报时，才可以上传附件，最多可以上传9个附件。
若对应汇报模板中没有添加【附件】控件，则不能上传附件。
![](https://wework.qpic.cn/wwpic3az/384433_RywlQeDNSXCNubh_1769415929/0)

### [](https://open.work.weixin.qq.com)2、在手机端提交汇报上传附件

在手机端提交汇报附件支持选择照片（拍照、从相册中添加）、从收藏选择、从微盘选择、选取文件（选取的是手机本地文件）。
![](https://wework.qpic.cn/wwpic3az/618844_uKEQ7YyzTbq8_JW_1769416338/0)

### [](https://open.work.weixin.qq.com)3、在电脑端端提交汇报上传附件

在电脑端提交汇报附件支持上传本地文件、从微盘中选择。
![](https://wework.qpic.cn/wwpic3az/352595_O3HatS4tToGv1X8_1769416344/0)
