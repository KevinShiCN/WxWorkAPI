# 填写汇报支持的功能

### [](https://open.work.weixin.qq.com)一、填写内容自动保存能力

● 填写汇报过程中退出会自动保存为草稿，草稿支持多端同步，下次进入内容不会丢失。
● 若主动点击右上角“x”关闭，则为删除当前正在填写的汇报。

### [](https://open.work.weixin.qq.com)二、支持导入上次填写内容

在填写汇报时，支持点击导入上篇，将上一篇汇报内容直接导入到这次汇报，也可进行修改。
![](https://wework.qpic.cn/wwpic3az/916914_l8Sbt-rkT4Wr0fc_1750645548/0)

### [](https://open.work.weixin.qq.com)三、填写汇报多行文本题目支持的样式

在填写多行文本时支持插入图片、表格、链接、文档、微盘文件、标题、加粗、斜体、下划线、删除线、对齐方式、数字编号、项目符号、缩进。

**汇报填写上限：**

- 文本控件的字数上限：4000个字符
- 多行文本控件的字数上限：5000个字
- 数字控件字数上限：4000个字符
- 评论字数上限：1000个字符
- 汇报附件限制：大小默认无限制，可设置最大限制3000MB，数量上限9个
- 汇报图片限制：大小默认无限制，可设置最大限制3000MB，数量上限9个
- 每一道多行文本题内添加的文字+图片加起来的大小不可大于2m；且单个汇报内所有多行文本题内添加的文字+图片加起来的大小不可大于10m。（注：若只是添加纯文字内容，不能超过5000字。）
