# 为什么看不到汇报模版

当成员未被设置成任何一个模板的汇报人员时候，在客户端进入汇报页面会提示“无汇报模板”

若无法查看到个别的汇报模板，请确认以下内容：

1、管理员在管理端是否可以看到该模板；

2、需管理员在管理后台确认是否可以看到对应的汇报模板，若管理后台也看不到对应模板，确认是否模板已被管理员/汇报创建人删除（管理后台删除的可查看操作日志）

2、在【管理后台->应用管理->汇报->进入汇报模板->规则设置->设置汇报人员】，即：模板的可见范围中查看成员使用有在可见范围内；

3、需检查企业内是否有同名成员，可能汇报人员添加的是另一个同名成员，需要把对应的人员添加到模板的汇报人员范围内；

4、需检查手机端和管理后台是否为同一个企业；

5、需检查查看的汇报模板是否已启用。

6、若是新增模板后，在管理端/手机端都看不到，需确认是否在原有的基础模板上更改，实际并未新增模板；所以看不到之前“新增”的模板；建议可以创建新模板进行使用。
