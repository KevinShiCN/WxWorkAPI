# 如何添加第三方应用

### [](https://open.work.weixin.qq.com)**方式1 从移动端添加：**

需企业超级管理员身份或有应用添加权限的分级管理员，在【手机端->工作台->选应用】；
![](https://wework.qpic.cn/wwpic3az/468638_PZ-LhCBySvO3sae_1766137060/0)

### [](https://open.work.weixin.qq.com)**方式2 从PC端企业管理后台添加：**

需企业超级管理员身份或有应用添加权限的分级管理员，[登录管理后台](https://work.weixin.qq.com/)，在【应用管理->应用->添加第三方应用】。
![](https://wework.qpic.cn/wwpic3az/827827_zAcitgu8R-O2HZV_1753778004/0)
