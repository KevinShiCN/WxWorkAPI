# 如何删除第三方应用

**方式1 在移动端删除：**需企业超级管理员身份或有应用添加权限的分级管理员，在【手机端->工作台->管理企业->应用管理->选择应用并删除】；
注：删除后，成员将无法使用该应用且该应用的历史数据将被删除。
![](https://wework.qpic.cn/wwpic3az/11077_GaySaOFiQYWYAcB_1766634602/0)

**方式2 在PC端企业管理后台删除：**需企业超级管理员身份或有应用添加权限的分级管理员，[登录管理后台](https://work.weixin.qq.com/)，在【应用管理->应用->选择应用->下拉至最下方“删除应用”】。
![](https://wework.qpic.cn/wwpic3az/324667_UK7-iFvWQDypZXa_1753776843/0)
