# 如何使用健康上报

### [](https://open.work.weixin.qq.com)一、如何创建健康上报

可在【手机端/电脑端->工作台->健康上报】创建。（Mac暂不支持）
接着进行内容编辑和上报设置，即先编辑上报名称和问题，下一步设置填写上报范围、上报给谁以及上报方式，同时可选择提醒员工上报并设置提醒时间，即完成创建开始收集。
![](https://wework.qpic.cn/wwpic3az/771266_OkXHQQlARv-45-Y_1766645518/0)

### [](https://open.work.weixin.qq.com)二、支持添加的控件

1、支持添加单选、多选、填空、地理位置、日期、收集文件/图片、收集单当日行程卡、签名。
![](https://wework.qpic.cn/wwpic3az/779380_YFQy0PPEQ3egfaP_1766645555/0)

2、支持设置题目的显示条件。
![](https://wework.qpic.cn/wwpic3az/116897_-G2YH71oSAGjBY7_1766645578/0)

### [](https://open.work.weixin.qq.com)三、查看健康上报的数据

1、上报填写通知下发后，即可实时查看员工的填写统计数据。
可在【手机端/电脑端->工作台->健康上报->统计】，选择需要查看的上报主题，点击进入，即可查看该上报的数据统计情况，在「数据统计」右侧可切换日期查看。
同时，还可在左下角点击「转发」继续督促未填写的员工上报，在右下角「查看上报明细」中了解每个员工的填写内容。
![](https://wework.qpic.cn/wwpic3az/710420_HAkMlfTaSM6VdiU_1766645601/0)

2、可在【手机端/电脑端->工作台->健康上报->统计->查看上报明细】导出上报结果。
导出后可以点击查看导出结果；后续也可以通过微文档查看已导出的上报明细。
健康上报收集的图片可在导出结果的文档中点击查看，发起人也可以在「微盘->我的文件」“健康上报”文件夹查看。
![](https://wework.qpic.cn/wwpic3az/538728_N8U8lSucQ7ibmb5_1766645626/0)

### [](https://open.work.weixin.qq.com)四、修改上报模板

新建了上报模板后，创建者可以在【健康上报->统计】页面点击对应的收集上报，点击右上角可以重新编辑上报问题和修改上报设置。
![](https://wework.qpic.cn/wwpic3az/559419_gPD1P7qWTOaOhDb_1766645652/0)

### [](https://open.work.weixin.qq.com)五、停止上报

可在【手机端->工作台->健康上报->统计->点击需要结束的上报->右上方->结束上报】
![](https://wework.qpic.cn/wwpic3az/733475_BqpYdaSFTXyXgwd_1766645664/0)

### [](https://open.work.weixin.qq.com)六、删除上报

超级管理员可以在【工作台->健康上报->统计->右上角三横杠->删除】其他人创建的健康上报。
![](https://wework.qpic.cn/wwpic3az/321542_PoNvAsXiSOSAorY_1766645698/0)

### [](https://open.work.weixin.qq.com)七、启用/停用健康上报

超级管理员可在【[管理后台](https://work.weixin.qq.com)->应用管理->健康上报->启用/停用】
![](https://wework.qpic.cn/wwpic3az/513691_VC_-evP1Tc-JiJY_1766645986/0)
或在【手机端->工作台->管理企业->应用管理->健康上报->停用/启用】操作启用/停用健康上报应用。
![](https://wework.qpic.cn/wwpic3az/441953_FdJgGomXTqao7kd_1766645737/0)

### [](https://open.work.weixin.qq.com)八、找不到需上报的模板/健康上报提示无权查看

1、每个健康上报模板都需要设置上报范围，只有上报模板的上报范围包含自己，才可以看到并且进行上报。若成员不在所以上报范围内，打开健康上报应用会提示“联系管理员创建”。
2、若已在上报的上报范围中，需要与管理员或者上报的创建者确认，该上报是否已经结束上报。
3、拥有家校沟通功能的企业，可以在上报范围中添加家长，这些上报支持在微信侧打开；非家校沟通功能的企业创建的上报，不支持在微信侧打开。
4、若是每日上报，成员A原本不在收集范围，当天修改了上报范围添加A成员后，次日健康上报才会生效，成员当天操作填写会提示暂无查看权限，可尝试次日进行填写。
