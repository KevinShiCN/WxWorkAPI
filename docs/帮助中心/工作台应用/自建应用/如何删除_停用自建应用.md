# 如何删除/停用自建应用

### [](https://open.work.weixin.qq.com)一、管理后台删除、停用应用

创建人/超级管理员可以在【管理后台->应用管理】进入到对应的自建应用页面，右上角操作启用/停用应用，左下角操作删除应用。
![](https://wework.qpic.cn/wwpic3az/144807_5dUCBODHSpSdbBB_1753779918/0)
![](https://wework.qpic.cn/wwpic3az/32929_iLkQ6XbWTA6RVNw_1753779921/0)

### [](https://open.work.weixin.qq.com)二、手机端删除、停用应用

创建人/超级管理员可以在【手机端->工作台->管理企业->应用管理->点击对应的应用】在应用详情底部停用/删除应用。
![](https://wework.qpic.cn/wwpic3az/444142_ukRexUzbRgyPdhX_1766634967/0)
