# 自建应用可信域名说明

### [](https://open.work.weixin.qq.com)一、可信域名的作用

1、[oauth2网页授权功能](https://work.weixin.qq.com/api/doc/90000/90135/91335)
2、[jssdk的调用](https://work.weixin.qq.com/api/doc/10029)
3、去掉微信那边的风险提示需设置可信域名

### [](https://open.work.weixin.qq.com)二、可信域名设置要求

1、设置的可信域名，不能包含协议头，不支持IP地址及短链域名，也不支持带下划线的域名。
2、域名支持一级域名、二级域名、三级域名；如一级域名进行了备案， 使用的是二级域名，则二级域名/三级域名不需要备案。
设置可信域名需要备案，去除微信侧的风险提示的话，也需要进行备案。
3、一个域名可以绑定不同应用。
4、校验可信域名提示“域名所有权不通过”，可点击[查看解决方法](https://open.work.weixin.qq.com/help2/pc/15370)。
5、域名备案成功，建议在备案成功后72小时再绑定使用。
6、绑定在应用中的可信域名需[工信网](http://beian.miit.gov.cn/publish/query/indexFirst.action)上可查到。
7、设置可信域名，“可作为应用OAuth2.0网页授权功能的回调域名”只能设置一个，“可调用JS-SDK、跳转小程序的可信域名”可以设置10个。
8、目前10080端口被列入浏览器的禁止访问名单，主流浏览器都暂不支持这个端口。若使用的是该端口，建议换个端口部署网站。
9、配置可信域名必须要有主体，需已验证/认证企业才可操作。
10、设置可信域名->可作为应用OAuth2.0网页授权功能的回调域名：支持中文域名转码配置。
设置接收消息->设置API接收->接收消息服务器配置->URL：支持中文域名转码配置。
注：自建应用->接收消息->设置API接收消息->接收消息服务器配置的URL不支持写ip地址。
如果域名是中文域名，建议使用[转码工具](https://stool.chinaz.com/urlencode)转码后的域名填写。
11、企业认证状态，配置可信域名后，企业变成未认证，暂不影响已绑定的可信域名使用。
12、域名所有权文件在校验后不能删除，删除后有可能影响使用。
13、每个二级域名需要做归属校验，需要将校验文件放到域名根目录下。
