# 设置可信域名时，提示“域名主体校验未通过，需配置备案主体与当前企业主体相同或有关联的域名”

1、第三方服务商主体的域名将无法在企业自建应用中进行相关配置。由服务商为客户企业代开发的自建应用，需要接入「企业微信自建应用代开发」模式。

需要您使用自建应用代开发模式，可参考文档：[代开发流程](https://open.work.weixin.qq.com/api/doc/90001/90143/94737) 、[自建应用第三方域名拦截说明](https://open.work.weixin.qq.com/wwopen/common/readDocument/32094)

2、出现该提示，是因为当前配置的域名与第三方服务商应用域名相同，并不是只看当前服务商账号/当前企业的配置。

3、例外情况：若是服务商自己的企业（测试企业或者关联主体的企业）可以配置服务商的域名，其他企业不支持配置服务商域名。若用服务商域名可以配置成功，建议找服务商确认是否被登记在服务商的测试企业。

温馨提示：
1、**未认证/未验证企业**支持配置非服务商使用的域名。
2、**已认证/验证企业**可以配置与域名**主体一致**或**与域名主体具备企业关联**的域名。若是配置出现拦截提示，可联系人工客服进一步咨询。
