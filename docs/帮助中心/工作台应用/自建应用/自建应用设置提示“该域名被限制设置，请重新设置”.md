# 自建应用设置提示“该域名被限制设置，请重新设置”

1、请检查设置的url中是否带有ip地址，目前接收服务器配置中不支持设置内网ip，请换成公网ip或域名后进行重试。
如未设置IP地址，可进入[企业微信开发者社区](https://developer.work.weixin.qq.com/community/question)进行提问，与企业微信技术同学进行交流。

2、请核查url的拼写是否正确，例如“http://”错写为“http:///”，多了个斜杠。
