# 如何切换汇报视图

### [](https://open.work.weixin.qq.com)一、支持的视图类型

在汇报中有三类视图，分别为：

- 「我提交的」：包括 我填写并提交的汇报 （我为填写人时，有此视图）；
- 「我收到的」：包括 我为接收人的汇报 （我为接收人时，有此视图）；
- 「其他」：包括 没有汇报给我，但我看过的汇报。

### [](https://open.work.weixin.qq.com)二、如何切换视图
