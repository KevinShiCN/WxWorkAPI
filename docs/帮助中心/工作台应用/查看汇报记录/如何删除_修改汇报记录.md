# 如何删除/修改汇报记录

管理员不支持删除、修改成员的汇报记录；
汇报者可以在【客户端->工作台->汇报->看汇报->我提交的->对应汇报旁三个点->删除、修改】进行删除、修改自己提交的汇报记录。

**注：删除后不支持恢复，需谨慎操作。**
