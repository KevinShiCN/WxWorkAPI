# 为什么管理后台看不到汇报记录

1、需确认汇报人员提交汇报时，是否有选择汇报对象，管理后台暂不支持查看未选择汇报对象的汇报记录。

未选择汇报人的汇报，创建这个汇报的成员支持在客户端查看这条汇报记录；管理员或汇报对象在手机端统计中会看见“已汇报”的统计，导出对应的汇报结果也会看见这条汇报记录。

2、在管理后台根据名字搜索汇报记录时，需在搜索结果的下拉框中选择对应的的成员进行查看。
![](https://wework.qpic.cn/wwpic3az/809501_0E9iR39-S2OVN1d_1753843443/0)
3、检查成员提交汇报使用的模板与管理员查看的是否一致，可能成员使用了错误的模板提交汇报。

4、目前汇报筛选时间范围需要控制在2年内，如果汇报记录的汇报时间大于2年，则会有可能看不到那个汇报记录。
例：需要查看2019年1月份的汇报记录，时间范围筛选了2017-2021年，则无法看到2019年的记录，只能看到2020-2021年的记录；可以调整筛选时间为2018-2019年或2019-2020年再进行查看。

5、汇报记录中的所在部门，是以成员提交汇报时的所在部门为准，若成员曾调整过部门，建议导出全部部门进行确认。

6、若是分级管理员看不到汇报记录，请确认分级管理员的管理范围是否包含成员曾经所在部门。

7、需检查看不到汇报记录的模板是否删除过，可能成员当前看的只是同名的新增模板/其他模板。

8、需检查成员是否提交成功，可能未点击提交/文档类型汇报，可能是草稿状态，还没点击提交。

9、需检查成员提交汇报时所在企业与管理员登录的管理端后台是否同一企业，需是同一个企业。
