# 如何预约上门服务

### [](https://open.work.weixin.qq.com)一、预约上门服务的方式

若有上门服务需求，超级管理员可在以下路径选择需要的讲解培训，支持在线咨询服务顾问或预约咨询顾问。

具体路径如下：
1、【手机端->工作台->管理企业->找服务】
![](https://wework.qpic.cn/wwpic3az/939579_TwHme5MoQlCSsHy_1766641427/0)

2、【手机端->工作台->找服务】
![](https://wework.qpic.cn/wwpic3az/355614_kvZviENVRGCX4ly_1766641446/0)

### [](https://open.work.weixin.qq.com)二、上门服务是否需要收费

目前上门服务需要收费。具体费用可以由超级管理员在【手机端->工作台->管理企业->找服务】或【手机端->工作台->找服务->讲解培训】中点击服务商信息查看，不同服务商提供的不同服务内容有不同的价位。

可先在线与服务商沟通，咨询完后如果有意愿购买再付费，付费后服务商提供正式服务。

若需费用发票开票，可联系提供服务的服务商开票。

### [](https://open.work.weixin.qq.com)三、如何取消预约

预约咨询顾问后，如需取消预约可在【手机端->工作台->找服务->点击对应预约的服务商->在线咨询】联系服务商沟通取消。
