# 如何设置应用页面/图片最小化

### [](https://open.work.weixin.qq.com)一、应用页面最小化

v4.1.22版本开始，支持在手机端临时最小化邮件、文档、文件、工作台应用和网页页面，多任务处理更方便。

操作方法：
1、iOS：使用返回手势（即手指右滑）将当前页面最小化。
![](https://wework.qpic.cn/wwpic3az/822454_uY62gg85SmWUdRf_1766644095/0)
2、安卓：点击页面右上角的“三”操作最小化。
![](https://wework.qpic.cn/wwpic3az/851222_OPxtKSPWTseWh-U_1766644901/0)
操作最小化后，可点击浮窗外侧的“X”关闭，或者是点击浮窗展开页面后，关闭企业微信app时将自动关闭所有最小化页面。
![](https://wework.qpic.cn/wwpic3az/16457_G7AU-IQFT-i6tsb_1766644298/0)

### [](https://open.work.weixin.qq.com)二、图片/视频最小化

v4.1.26版本开始，手机端多窗口支持打开聊天中的图片和视频，处理工作更方便。

操作方法：
1、iOS：点击图片/视频后，按住图片拉入“最小化”区域中，即可将图片最小化。
![](https://wework.qpic.cn/wwpic3az/229790__E8c7ElUTo6_dLr_1766644941/0)
2、安卓：长按图片，即可点击最小化。
操作最小化后，可点击浮窗查看所有图片/视频，点击浮窗外侧的“X”关闭，或者是点击浮窗展开页面后，关闭企业微信app时将自动关闭所有最小化页面。
![](https://wework.qpic.cn/wwpic3az/393411_bI4kvX3qSqOFUux_1766645092/0)
