# 人事助手-合同管理

#### [](https://open.work.weixin.qq.com)一、查看员工合同关键信息

可在【电脑端->工作台->人事助手->合同管理】，查看「已添加合同员工」列表、「未添加合同员工」列表，进行合同管理相关操作。

#### [](https://open.work.weixin.qq.com)二、添加合同

（1）可对已添加合同员工「续签合同」，续签后会为员工生成一条新的合同信息
![](https://wework.qpic.cn/wwpic3az/778823_rKJSovPvToG0Gpi_1754901802/0)
（2）可对未添加合同员工「添加合同」，添加后员工将展示在「已添加合同员工」列表
![](https://wework.qpic.cn/wwpic3az/392759_PKmD9P6gRhq9rUu_1754901808/0)
（3）合同到期提醒：支持设置合同到期前90、60、30个工作日进行提醒或不提醒。
路径：【电脑端/手机端->工作台->人事助手->提醒->合同到期提醒】
![](https://wework.qpic.cn/wwpic3az/490188_EGNZ1_6RT2i5jgE_1754901813/0)

### [](https://open.work.weixin.qq.com)三、常见问题

#### [](https://open.work.weixin.qq.com)1、在人事助手合同管理给成员添加了合同之后，可以删除员工的合同吗？

当添加了多份合同时，可以删除不需要的合同。但必须保留一份合同，目前无法清空所有合同。若只有一份合同时，暂不支持操作删除。

#### [](https://open.work.weixin.qq.com)2、上传合同附件提示“上传失败”

请确认您上传的附件是否符合要求：文件大小需在50m以内，支持格式包括jpeg、png、pdf、doc、xls和ppt等格式文件。

#### [](https://open.work.weixin.qq.com)3、无固定期限劳动合同设置

手机端：选择合同类型为「无固定期限劳动合同」后，默认选中「无合同到期日」，可「设置合同到期日」；若到期日中填有日期，可选择「修改合同到期日」/「无合同到期日」。
![](https://wework.qpic.cn/wwpic3az/275894_DRv-cyb2QkeZnef_1763713995/0)

电脑端：可不填写「无固定期限劳动合同」的到期日，将在手机端同步展示「无合同到期日」；若之前填写过合同到期日，可点击“x”清空。
![](https://wework.qpic.cn/wwpic3az/114895_JzGLl5emT8Wd1-g_1763714006/0)
