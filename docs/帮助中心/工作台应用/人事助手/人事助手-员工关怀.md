# 人事助手-员工关怀

### [](https://open.work.weixin.qq.com)一、什么是员工关怀

便捷查看/筛选/导出员工生日与入职周年信息，可配置祝福卡片，员工生日时自动推送。

### [](https://open.work.weixin.qq.com)二、如何使用员工关怀

#### [](https://open.work.weixin.qq.com)1、设置员工生日

管理员或应用负责人可在【手机端/电脑端->人事助手->花名册->编辑成员个人信息->出生日期】进行设置。
![](https://wework.qpic.cn/wwpic3az/320346_z-7sQ_kiS8yTZSI_1766634640/0)
![](https://wework.qpic.cn/wwpic3az/249449_Bt0_CntvQB6g9rm_1754903961/0)
![](https://wework.qpic.cn/wwpic3az/847557_-OKMXwTLRmWPII8_1754903964/0)
普通成员也可以在【人事助手->查看完整资料->出生日期】进行设置。
![](https://wework.qpic.cn/wwpic3az/801722_mqp30IGfSJ6jLYM_1766634693/0)

#### [](https://open.work.weixin.qq.com)2、查看和导出员工生日

可在【电脑端->工作台->人事助手->人事管理->员工关怀】
（1）选择「员工生日」，可查看全部员工生日统计、员工生日信息列表。
![](https://wework.qpic.cn/wwpic3az/635177_j18TWLirQS2esX9_1754904002/0)
（2）点击「筛选」可对列表员工的生日日期、部门、员工类型进行筛选查看，确认后列表仅显示符合筛选条件的员工，点击「清空」可取消所有筛选。
![](https://wework.qpic.cn/wwpic3az/763457_-JhykN3qQu-Gwc7_1754904006/0)
![](https://wework.qpic.cn/wwpic3az/719242_S0D_ucLvTRKampH_1754904011/0)
（3）点击「导出」可下载列表显示的员工生日信息表，或选择导出全部员工生日信息表。
![](https://wework.qpic.cn/wwpic3az/893155_r43PH7QHT3iKQ_I_1754904016/0)
![](https://wework.qpic.cn/wwpic3az/422641_e1dfGg53SviKyBK_1754904019/0)

#### [](https://open.work.weixin.qq.com)3、设置生日祝福

选择「员工生日」，点击右上角「生日祝福设置」，可打开生日祝福卡片开关和编辑卡片内容，员工生日当天，祝福卡片会在【手机端/电脑端->消息->人事助手->进行推送】，卡片落款显示的是企业简称。
注：成员需要在人事助手应用可见范围内，才会收到生日祝福，休息日或节假日，也会正常推送。
![](https://wework.qpic.cn/wwpic3az/117223_GJ_z3AxARd-8xHP_1754904024/0)

#### [](https://open.work.weixin.qq.com)4、设置入职日期

管理员或应用负责人可在【手机端/电脑端->人事助手->花名册->编辑成员在职信息->入职日期】进行设置。
![](https://wework.qpic.cn/wwpic3az/107917_QTt37Jj_QTiQjLC_1754904035/0)
![](https://wework.qpic.cn/wwpic3az/908738_22GHr1woSMOzOm6_1754904037/0)

#### [](https://open.work.weixin.qq.com)5、查看和导出员工入职周年信息

可在【电脑端->工作台->人事助手->人事管理->员工关怀->员工入职周年】
（1）选择「员工入职周年」，可查看全部员工近期内入职周年日统计、员工入职周年日期及司龄信息列表。
![](https://wework.qpic.cn/wwpic3az/101720_bkjBUS72RZ6E8Rr_1754904041/0)

（2）点击「筛选」可对列表员工的入职周年日期、部门、员工类型进行筛选查看，确认后列表仅显示符合筛选条件的员工，点击「清空」可取消所有筛选。
![](https://wework.qpic.cn/wwpic3az/980395_2h5tbrirTfODV0q_1754904048/0)
![](https://wework.qpic.cn/wwpic3az/139772_1qe0N8HHQdqX74-_1754904054/0)
（3）点击「导出」可下载列表显示的员工入职周年日期信息表，或选择导出全部员工入职周年日期信息表。
注：暂不支持推送入职周年祝福卡片给员工。
![](https://wework.qpic.cn/wwpic3az/54784_GfwbZgOnTu6XYl6_1754904059/0)
![](https://wework.qpic.cn/wwpic3az/922865_78HNUVnCRlm2tl5_1754904061/0)
