# 人事助手-员工统计

管理员/负责人可在人事助手-员工统计中查看企业总数和入离职趋势变化。

1、管理员/负责人可筛选半年或一年的查看范围。

2、自动生成每月员工人数、入离职人数统计图，直观展示企业员工人数按月变化趋势。

3、查看路径：
【电脑端/手机端->工作台->人事助手->员工统计】
![](https://wework.qpic.cn/wwpic3az/689343_7rAEURXdScGxc9Y_1754901817/0)

4、员工统计提醒
管理员/负责人会在每个月的第一个工作日收到一份人事统计小结。
注：应用负责人的管理范围需是全企业人员时，客户端才会有“员工统计”的入口。

可在【电脑端/手机端->工作台->人事助手->提醒->员工统计提醒】关闭提醒。
![](https://wework.qpic.cn/wwpic3az/381045_-D87uMssSTi6aV7_1754901820/0)

5、“X月员工统计小结”提醒，推送对象：企业超级管理员和有全企业管理范围的应用负责人。
