# 如何启用/停用人事助手

#### [](https://open.work.weixin.qq.com)1、管理后台操作路径

管理员可在【电脑端点击左上角头像->管理企业->进入企业微信管理后台 ->应用管理->人事助手->开启/停用】，停用后人事助手应用自动从工作台清除。开启后人事助手应用会自动添加到工作台，可前往工作台使用。

![](https://wework.qpic.cn/wwpic3az/341654_zFRhfO6FSCOl8-M_1753842925/0)
![](https://wework.qpic.cn/wwpic3az/897627_cCMbwW8XRaa-KbG_1753842804/0)

#### [](https://open.work.weixin.qq.com)2、手机端操作路径

【手机端->工作台->管理企业->应用管理->人事助手->停用/开启】
