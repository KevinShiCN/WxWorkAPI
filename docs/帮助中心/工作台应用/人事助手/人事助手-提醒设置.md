# 人事助手-提醒设置

支持设置人事流程到期自动提醒，例如：入职提醒、转正提醒、离职提醒、合同到期提醒、员工统计提醒、发布工资条提醒，避免错过重要事项。

操作路径如下：
1、【电脑端->工作台->人事助手->提醒】

![](https://wework.qpic.cn/wwpic3az/388652_wLubvtSISfCX8Zm_1754963770/0)
