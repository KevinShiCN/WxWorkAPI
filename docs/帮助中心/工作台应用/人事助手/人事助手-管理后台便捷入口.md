# 人事助手-管理后台便捷入口

### [](https://open.work.weixin.qq.com)一、假期管理

1、管理员从【电脑端->工作台->人事助手->设置->假期管理】进入审批的假期管理后台，可设置假期类型并查看假期余额，员工请假后根据请假审批单自动扣假。

2、详细介绍可参考：[使用假期管理](https://open.work.weixin.qq.com/help2/pc/18002)
![](https://wework.qpic.cn/wwpic3az/921105_Yhj1upSrSCKEqRU_1754964011/0)

### [](https://open.work.weixin.qq.com)二、审批管理

1、管理员从【电脑端->工作台->人事助手->设置->审批管理】进入审批管理后台，实现随时随地审批，可自定义审批模板，支持会签、或签、上级审批、条件审批，适应各种流程。

2、详细介绍可参考：[设置审批模板](https://open.work.weixin.qq.com/help2/pc/14528)、[设置审批流程 ](https://open.work.weixin.qq.com/help2/pc/17459)
