# 如何使用同事吧

### [](https://open.work.weixin.qq.com)一、如何开启/停用同事吧

1、管理员可以登录【[管理后台](https://work.weixin.qq.com)->应用管理->同事吧->开启/停用应用】。
![](https://wework.qpic.cn/wwpic3az/754326_fdTcTu4TT5KiWab_1753840656/0)
2、支持设置允许匿名发帖回帖、对外分享。
![](https://wework.qpic.cn/wwpic3az/206221_6_9Pf5RNRBqwj1Q_1753840662/0)
3、成员可以在【手机端->工作台->同事吧】进行发帖回帖。
![](https://wework.qpic.cn/wwpic3az/493130_xrbXlY-xTIa0qJB_1766646058/0)
4、同事吧的字数/图片大小上限
同事吧内容支持5000字、标题支持30个汉字、评论暂无上限、图片大小暂无上限。

5、目前仅10000人以内的企业，会有同事吧新帖消息推送。大于10000人的企业，同事吧有新帖子时，消息列表不会推送同事吧消息。

### [](https://open.work.weixin.qq.com)二、管理员权限

超级管理员可以在手机端删除所有成员发的帖子。

管理后台可以关闭同事吧，关闭后在成员手机端不会显示同事吧功能。
![](https://wework.qpic.cn/wwpic3az/858848_vaUuE1Q2Tz2t1bZ_1753840666/0)

### [](https://open.work.weixin.qq.com)三、如何发布帖子

在手机端【工作台->同事吧】支持发表、查看、评论帖子。电脑端暂不支持使用同事吧。
![](https://wework.qpic.cn/wwpic3az/197427__h5cP4d-Tt2M5TX_1766646085/0)
