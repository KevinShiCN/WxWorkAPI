# 学习园地功能使用说明

### [](https://open.work.weixin.qq.com)一、使用范围

教育行业、教育-培训机构、私立医院的企业暂不支持使用学习园地，因此成员无法在工作台上看到学习园地的入口。
其他行业的企业成员均可在工作台上看到学习园地的入口，企业管理员支持设置应用的可见范围。

设置路径如下：
【[管理后台](https://work.weixin.qq.com)->应用管理->学习园地->可见范围】
![](https://wework.qpic.cn/wwpic3az/317285_yisr7CjOS4OvdAf_1753842032/0)
【手机端->工作台->管理企业->应用管理->学习园地->可见范围】
![](https://wework.qpic.cn/wwpic3az/371632_hKwJYYzxRjmieJn_1766713645/0)

### [](https://open.work.weixin.qq.com)二、使用入口

管理员和成员都可以进入【工作台->学习园地】选择感兴趣的课程进行学习，还可以点击「让成员学习」，将课程发送到全员群、部门群等，群内员工将自动加入学习，可随时查看成员的学习进度。
![](https://wework.qpic.cn/wwpic3az/541281_6TWsbV0hQMSmNjR_1766713684/0)
