# 如何设置公告权限

1、管理员需要在【[管理后台](https://work.weixin.qq.com)->应用管理->公告应用】中给成员设置「发送公告权限」，拥有这个权限的成员，支持在移动端发公告。
![](https://wework.qpic.cn/wwpic3az/315071_68IMZPnWSDCpJLf_1753842453/0)
2、设置「发送公告权限」支持设置收发消息范围，范围支持设置部门或单个成员，暂时不支持设置标签。
![](https://wework.qpic.cn/wwpic3az/383955_9l8jNc4iQuSp7ev_1753842456/0)
