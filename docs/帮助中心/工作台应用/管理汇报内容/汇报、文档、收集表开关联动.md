# 汇报、文档、收集表开关联动

汇报、文档、收集表开关联动方式：
1、关闭文档，会联动关闭汇报。

2、可以单独关闭汇报，关闭后不影响文档开关，但在文档内会隐藏新建收集表/汇报的入口。（去填写汇报会提示应用已停用）

3、收集表关闭， 汇报打开，无法新建收集表/接龙/投票，但仍可以新建收集表汇报。汇报相关通知、卡片都能正常推送和打开；收集表/接龙/投票在应用关闭后无法填写、不发通知。
