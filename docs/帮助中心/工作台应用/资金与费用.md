# 资金与费用

### [](https://open.work.weixin.qq.com)一、企业支付资金提现

员工所付款项将进入企业所关联的微信支付商户号中。
微信支付商户提现是完全免费的。操作流程，可点击[查看详情](http://kf.qq.com/faq/161223NJBr2u161223Mfeqei.html)

### [](https://open.work.weixin.qq.com)二、企业支付手续费相关

向员工收款、对外收款需要手续费，查看费率请点击[查看详情](https://kf.qq.com/faq/220228IJb2UV220228uEjU3Q.html)
向员工付款、向员工发红包不需要手续费。

### [](https://open.work.weixin.qq.com)三、企业支付付款金额上限

向员工收款金额限额：取决于员工微信的支付限额。
微信支付银行卡交易限额：[查看详情](http://kf.qq.com/touch/faq/151210NZzmuY151210ZRj2y2.html)
向员工付款金额限制：单笔金额最低不能少于1元，最高不能超过200元，一个商户一天不能超过100000元。
