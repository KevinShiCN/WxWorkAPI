# 获取观看直播统计V2

通过该接口可以获取所有观看直播的人员统计

**请求方式：**POST（**HTTPS**）
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/school/living/get_watch_stat?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"livingid": "livingid1",
	"next_key": "NEXT_KEY"
}

```

    

**参数说明：**

        

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证 |
| livingid | 是 | 直播的id |
| next_key | 否 | 上一次调用时返回的next_key，初次调用可以填"0" |

**权限说明：**

- 「上课直播」应用可以获取用户所有的直播
- 第三方应用必须有直播权限和家校沟通权限
- 自建应用必须通知配置到「上课直播」和家长可以使用的应用中
- 自建应用和第三方应用只能获取本应用创建的直播

**参数说明：**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| ending | 是否结束。0：表示还有更多数据，需要继续拉取，1：表示已经拉取完所有数据。注意只能根据该字段判断是否已经拉完数据 |
| next_key | 当前数据最后一个key值，如果下次调用带上该值则从该key值往后拉，用于实现分页拉取 |
| stat_infoes | 统计信息列表 |
| stat_infoes.students | 观看直播的学生列表 |
| stat_infoes.students.student_userid | 学生的userid |
| stat_infoes.students.parent_userid | 家长的userid |
| stat_infoes.students.partyids | 学生所在的班级id列表 |
| stat_infoes.students.watch_time | 观看时长，单位为秒 |
| stat_infoes.students.enter_time | 首次进入直播时间 |
| stat_infoes.students.leave_time | 最后离开直播时间 |
| stat_infoes.students.is_comment | 是否评论，1表示评论，0表示没有评论 |
| stat_infoes.visitors | 观看直播的游客列表 |
| stat_infoes.visitors.nickname | 微信昵称 |
| stat_infoes.visitors.watch_time | 观看时长，单位为秒 |
| stat_infoes.visitors.enter_time | 首次进入直播时间 |
| stat_infoes.visitors.leave_time | 最后离开直播时间 |
| stat_infoes.visitors.is_comment | 是否评论，1表示评论，0表示没有评论 |
