# 获取配置的网格及网格负责人

**请求方式：** GET（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/report/patrol/get_grid_info?access_token=ACCESS_TOKEN

**参数说明：**

[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。“获取access_token” |

**权限说明：**

- 系统应用「巡查上报」默认可使用巡查上报接口
- 自建应用需要配置在“可调用接口的应用”里

**返回结果：**

```
{
   "errcode": 0,
   "errmsg": "ok",
   "grid_list":[
      {
         "grid_id": "grid_id",
         "grid_name": 1,
         "grid_admin": [
            "zhangsan",
            "lisi"
         ]
      }
   ]
}

```

**参数说明：**

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| grid_list.grid_id | 网格 ID |
| grid_list.grid_name | 网格名称 |
| grid_list.grid_admin | 网格管理员 userId 列表 |
