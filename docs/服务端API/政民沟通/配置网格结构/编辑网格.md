# 编辑网格

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/report/grid/update?access_token=ACCESS_TOKEN

**请求包体：**

```
{
    "grid_id": "grid_id",
    "grid_name": "grid_name",
    "grid_parent_id" : "grid_id",
    "grid_admin": ["zhangsan"],
    "grid_member": ["lisi", "invaliduser"]
}

```

**参数说明：**

[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。“获取access_token” |
| grid_id | 是 | 网格id |
| grid_name | 是 | 网格名称，不能超过30个字，同一个目标网格下，不能存在同名的同级子网格 |
| grid_parent_id | 是 | 父节点的id，网格结构层级最多支持10层 |
| grid_admin | 是 | 网格「负责人」userid列表，每个网格至少1个，最多20个负责人 |
| grid_member | 否 | 该节点的成员列表，不能超过100个，为空则表示清空所有的成员 |

**权限说明：**

- “居民联系” secret默认可使用网格接口
- 自建应用需要配置在“居民联系” 的“可调用接口的应用”中

**返回结果：**

```
{
   "errcode": 0,
   "errmsg": "ok",
   "invalid_userids": [
       "invaliduser"
   ]
}

```

**参数说明：**

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| invalid_userids | 不合法的userid |
