# 获取用户负责及参与的网格列表

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/report/grid/get_user_grid_info?access_token=ACCESS_TOKEN

**请求包体：**

```
{
    "userid": "zhangsan"
}

```

**参数说明：**

[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。“获取access_token” |
| userid | 是 | 需要查询的成员userid |

**权限说明：**

- “居民联系” secret默认可使用网格接口
- 自建应用需要配置在“居民联系” 的“可调用接口的应用”中

**返回结果：**

```
{
   "errcode": 0,
   "errmsg": "ok",
   "manage_grids":[
       {"grid_id": "grid_id1", "grid_name": "grid_name1"},
       {"grid_id": "grid_id2", "grid_name": "grid_name2"}
   ],
   "joined_grids":[
       {"grid_id": "grid_id1", "grid_name": "grid_name1"},
       {"grid_id": "grid_id2", "grid_name": "grid_name2"}
   ],
}

```

**参数说明：**

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| manage_grids | 作为管理员管理的网格信息 |
| manage_grids.grid_id | 网格id |
| manage_grids.grid_name | 网格名称 |
| joined_grids | 作为成员参与的grid信息 |
| joined_grids.grid_id | 网格id |
| joined_grids.grid_name | 网格名称 |
