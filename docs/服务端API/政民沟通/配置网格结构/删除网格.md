# 删除网格

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/report/grid/delete?access_token=ACCESS_TOKEN

**请求包体：**

```
{
    "grid_id": "grid_id"
}

```

**参数说明：**

[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。“获取access_token” |
| grid_id | 是 | 网格id，根节点可以填空 |

当目标网格及其子网格中不包含网格员时，可以删除目标网格及其全部子网格

**权限说明：**

- “居民联系” secret默认可使用网格接口
- 自建应用需要配置在“居民联系” 的“可调用接口的应用”中

**返回结果：**

```
{
   "errcode": 0,
   "errmsg": "ok"
}

```

**参数说明：**

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
