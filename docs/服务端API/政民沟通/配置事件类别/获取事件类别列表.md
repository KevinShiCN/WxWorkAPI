# 获取事件类别列表

可以获取已配置的一级或二级事件类别，网格员在提交「巡查上报」或「居民上报」时将填入配置的事件类别

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/report/grid/list_cata?access_token=ACCESS_TOKEN

**请求包体：**

**参数说明：**

[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。“获取access_token” |

**权限说明：**

- “居民联系” secret默认可使用网格接口
- 自建应用需要配置在“居民联系” 的“可调用接口的应用”中

**返回结果：**

```
{
   "errcode": 0,
   "errmsg": "ok",
   "category_list":[
      {
         "category_id": "category_id",
         "category_name": "2222",
         "level" : 1
      },
      {
         "category_id": "category_id",
         "category_name": "2222",
         "level" : 2,
         "parent_category_id": "parent_category_id"
      }
   ]
}

```

**参数说明：**

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| cata_list | 分类列表 |
| cata_list.category_id | 分类id |
| cata_list.category_name | 分类名称 |
| cata_list.level | 分类层级 |
| cata_list.parent_category_id | 分类层级为1时，该字段为空 |
