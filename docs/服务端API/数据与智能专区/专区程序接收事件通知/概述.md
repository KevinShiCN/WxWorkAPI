# 概述

调用流程参见[专区程序接收事件以及应用接收专区通知](https://developer.work.weixin.qq.com/document/path/99866#23-%E4%B8%93%E5%8C%BA%E7%A8%8B%E5%BA%8F%E6%8E%A5%E6%94%B6%E4%BA%8B%E4%BB%B6%E4%BB%A5%E5%8F%8A%E5%BA%94%E7%94%A8%E6%8E%A5%E6%94%B6%E4%B8%93%E5%8C%BA%E9%80%9A%E7%9F%A5)。

如果用户在企业微信操作，产生了相关事件，企业微信会通过http请求回调到专区。专区需要启动一个http服务，并监听**8080**端口。回调到该服务的根路径。
专区接收到请求后，需要将http请求的header以及body传给sdk解密接口

**http请求body：**
http服务接收到的http的body为加密的，需要调用sdk进行解密，解密后拿到消息原文，各个通知事件的协议参见本目录下各个文档。
