# 命中关键词规则通知

当企业会话内容授权范围内成员的会话命中关键词规则时，企业微信专区后台会向有企业会话内容权限的应用关联的程序推送相应的事件消息。

 

**事件示例：**

      
```javascript
{
    "event_type": "hit_keyword",
    "timestamp": 1408091189,
    "hit_keyword": {
        "token": "xxx"
    }
}
```

    
**参数说明：**

        
| 参数 | 说明 |
| --- | --- |
| event_type | 事件类型，此时固定为hit_keyword |
| timestamp | 时间戳 |
| token | 10分钟内有效，用于调用获取命中关键词规则的会话记录，不超过128字节 |
