# 删除公共邮箱

该接口用于删除已有的业务邮箱。

**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/delete?access_token=ACCESS_TOKEN

**请求包体**

      
```
{
  "id": 1
}
```

    

**参数说明**

        

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| id | uint32 | 是 | 业务邮箱ID |

**返回示例**

      
```
{
   "errcode": 0,
   "errmsg": "ok"
}
```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
