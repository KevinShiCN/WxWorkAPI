# 获取客户端专用密码列表

该接口用于获取公共邮箱创建的授权码列表，包括授权码ID、创建时间、最后使用时间、备注。需要注意的是，该接口不返回授权码，授权码仅会在创建时候返回，参考创建业务邮箱、更新业务邮箱

**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/get_auth_code_list?access_token=ACCESS_TOKEN

**请求包体**

      
```
{
  "id" : 1
}
```

    

**参数说明**

        

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| id | uint32 | 是 | 业务邮箱ID |

**返回示例**

      
```
{
    "errcode": 0,
    "errmsg": "ok",
    "list":[
      {
         "auth_code_id": 1,
         "create_time": 1724743240,
         "last_use_time": 1724743241,
         "remark": "授权码备注"
      },
      {
         "auth_code_id": 2,
         "create_time": 1724743240,
         "last_use_time": 1724743241,
         "remark": "授权码备注"
      }
    ]
}
```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
