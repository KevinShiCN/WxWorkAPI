# 获取用户功能属性

[TOC]
## 获取用户功能属性

该接口用于获取用户的功能属性

**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/exmail/useroption/get?access_token=ACCESS_TOKEN

**请求包体**

```
{
  "userid": "zhangsan",
  "type":[1,2,3]
}

```

**参数说明**

[
| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| userid | string | 是 | 用户UserID |
| type | uint32 | 是 | 功能设置属性类型 1: 强制启用安全登录 2: IMAP/SMTP服务 3: POP/SMTP服务 4: 是否启用安全登录 |

**返回示例**

```
{
  "errcode": 0,
  "errmsg": "ok",
  "option": {
    "list": [{
      "type": 1,
      "value": "0"
    }, {
      "type": 2,
      "value": "1"
    }, {
      "type": 3,
      "value": "0"
    }]
  }
}

```

**参数说明**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| type | int32 | 功能设置属性类型 1: 强制启用安全登录 2: IMAP/SMTP服务 3: POP/SMTP服务 4: 是否启用安全登录 |
| value | string | 1表示启用，0表示关闭 |

## 更改用户功能属性

该接口用于更新用户的功能属性

**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/exmail/useroption/update?access_token=ACCESS_TOKEN

**请求包体**

```
{
  "userid": "zhangsan",
  "option": {
    "list": [{
      "type": 1,
      "value": "0"
    }, {
      "type": 2,
      "value": "1"
    }, {
      "type": 3,
      "value": "0"
    }]
  }
}

```

**参数说明**

[
| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| userid | string | 是 | 用户UserID |
| type | int32 | 是 | 功能设置属性类型 1: 强制启用安全登录 2: IMAP/SMTP服务 3: POP/SMTP服务 4: 是否启用安全登录 |
| value | string | 是 | 1表示启用，0表示关闭 |

**返回示例**

```
{
  "errcode": 0,
  "errmsg": "ok"
}

```

**参数说明**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
