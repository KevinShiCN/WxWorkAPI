# 删除收集表事件

当文档管理员删除API创建的收集表后，触发该事件。

**示例**
xml请求示例：

      
```javascript
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[fromUser]]></FromUserName> 
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[doc_change]]></Event>
   <ChangeType><![CDATA[delete_form]]></ChangeType>
   <FormId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA]]></FormId>
   <FormId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzBBB]]></FormId>
</xml>
```

    
 

**说明**

        
| 参数 | 说明 |
| --- | --- |
| ToUserName | 企业微信CorpID |
| FromUserName | 成员UserID |
| CreateTime | 消息创建时间，unix时间戳 |
| MsgType | 消息类型，此时固定为：event |
| Event | 事件类型，此时固定为：doc_change |
| ChangeType | 事件类型，此时固定为：delete_form |
| FormId | 收集表ID列表 |
