# 创建收集表

## 创建收集表

该接口用于创建收集表。

**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/wedoc/create_collect?access_token=ACCESS_TOKEN   

**请求包体**

      
```
{
"spaceid": "SPACEID",
"fatherid": "FATHERID",
"form_info": {
"form_title": "FORM_TITLE",
"form_desc": "FORM_DESC",
"form_header": "FORM_HEADER",
"form_question": {
"items": [
{
"question_id": QUESTION_ID,
  "title": "TITLE",
  "pos": POS,
  "status": STATUS,
  "reply_type": REPLY_TYPE,
  "must_reply": MUST_REPLY,
  "note": "NOTE",
  "option_item": [
  {
  "key": KEY,
  "value": "VALUE",
  "status": STATUS
  }
  ]
  "placeholder": "PLACEHOLDER"
}
]
},
"form_setting": {
  "fill_out_auth": FILL_OUT_AUTH,
  "fill_in_range": {
  "userids": ["USER_1", "USER_2", "USER_3"],
  "departmentids": [10001, 10002, 10003]
  },
  "setting_manager_range": {
  "userids": ["USER_4", "USER_5", "USER_6"],
  "departmentids": [10004, 10005, 10006]
  },
  "timed_repeat_info": {
    "enable": ENABLE,
  "week_flag": WEEK_FLAG,
  "remind_time": REMIND_TIME,
  "repeat_type": REPEAT_TYPE,
  "skip_holiday: SKIP_HOLIDAY,
  "day_of_month": DAY_OF_MONTH,
  "fork_finish_type": FORK_FINISH_TYPE
  }
  "max_fill_cnt": MAX_FILL_CNT,
  "timed_finish": TIMED_FINISH,
  "can_anonymous": CAN_ANONYMOUS,
  "can_notify_submit": CAN_NOTIFY_SUBMIT
}
}
}
```

    

**参数说明**

        

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| spaceid | string | 否 | 空间spaceid |
| fatherid | string | 否 | 父目录fileid, 在根目录时为空间spaceid |
| form_title | string | 是 | 收集表标题 |
| form_desc | string | 否 | 收集表描述 |
| form_header | string | 否 | 收集表表头背景图链接 |
| form_question | object | 是 | 收集表的问题列表 |
| items | object[] | 是 | 问题数组 |
| question_id | uint32 | 是 | 问题id |
| title | string | 是 | 问题描述 |
| pos | uint32 | 是 | 问题序号 |
| status | uint32 | 是 | 问题状态。1：正常；2：被删除 |
| reply_type | uint32 | 是 | 回答类型。1：文本；2：单选；3：多选；5：位置；6：名字；7：部门；9：图片；10：文件；11：日期；14：时间；15：下拉列表；16：温度；17：签名 |
| must_reply | bool | 是 | 是否必答 |
| note | string | 否 | 问题备注 |
| placeholder | string | 否 | 编辑提示 |
| option_item | object[] | 是 | 单选或者多选题的选项列表 |
| key | uint32 | 是 | 选项key（1，2，3...） |
| value | string | 是 | 选项内容 |
| status | uint32 | 是 | 选项状态。1：正常；2：被删除 |
| form_setting | object | 否 | 收集表设置 |
| fill_out_auth | uint32 | 否 | 填写权限。0：所有人；1：企业内指定人/部门；默认为所有人可填写。 |
| fill_in_range | object | 否 | 指定的可填写的人/部门 |
| userids | string[] | 否 | 企业成员userid列表 |
| departmentids | uint64[] | 否 | 部门id列表 |
| setting_manager_range | object | 否 | 收集表管理员 |
| timed_repeat_info | object | 否 | 定时重复设置项 |
| enable | bool | 否 | 是否开启定时重复 |
| remind_time | uint32 | 否 | 提醒时间 |
| repeat_type | uint32 | 否 | 重复类型。0：每周；1：每天；2：每月 |
| week_flag | uint32 | 否 | 每周几重复，repeat_type = 0 时生效。1：星期一；2：星期二；4：星期三；8：星期四；16：星期五；32：星期六；64：星期日 |
| skip_holiday | bool | 否 | 自动跳过节假日，repeat_type = 1 时生效 |
| day_of_month | uint32 | 否 | 每月的第几天（1 - 31），repeat_type = 2时生效 |
| max_fill_cnt | uint32 | 否 | 最多填写数量，0为无限次。默认为无限次。 |
| timed_finish | uint32 | 否 | 定时关闭。定时重复与定时结束互斥，若都填，优先定时重复 |
| can_anonymous | bool | 否 | 是否支持匿名填写，默认不可匿名填写。 |
| can_notify_submit | bool | 否 | 是否有回复时提醒，默认不提醒。 |

**权限说明**

- 企业需要使用“文档”secret所获取的accesstoken来调用（accesstoken如何获取？）
- 第三方应用需具有“文档”权限
- 代开发自建应用需具有“文档”权限

**返回示例**

      
```
{
"errcode": 0,
"errmsg": "ok",
"formid": "FORMID"
}

```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| formid | string | 收集表id |

## 收集表的统计信息查询

该接口用于获取收集表的统计信息、已回答成员列表和未回答成员列表

**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/wedoc/get_form_statistic?access_token=ACCESS_TOKEN 

**请求包体**

      
```
// 仅获取统计结果
{
    "repeated_id":"REPEATED_ID1",
    "req_type":1,
    "start_time":1667395287,
    "end_time":1668418369,
    "limit":20
}
// 获取已提交的列表
{
    "repeated_id":"REPEATED_ID2",
    "req_type":2,
    "start_time":1667395287,
    "end_time":1668418369,
    "limit":20,
    "cursor":1
}
// 获取未提交的列表
{
    "repeated_id":"REPEATED_ID3",
    "req_type":3,
    "limit":20,
    "cursor":1
}
```

    

**参数说明**

        

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| repeated_id | string | 是 | 操作的收集表的repeated_id,来源于get_form_info的返回 |
| req_type | uint32 | 是 | 请求类型 1:只获取统计结果 2:获取已提交列表 3:获取未提交列表 |
| start_time | uint64 | 否 | 拉取已提交列表时必填，其余type不填。筛选开始时间，以当天的00:00:00开始筛选 |
| end_time | uint64 | 否 | 拉取已提交列表时必填，其余type不填。筛选结束时间，以当天的23:59:59结束筛选 |
| limit | uint64 | 否 | 分页拉取时批次大小，最大10000 |
| cursor | uint64 | 否 | 分页拉取的游标，首次不传 |

 

**权限说明**

- 企业需要使用“文档”secret所获取的accesstoken来调用（accesstoken如何获取？）
- 第三方应用需具有“文档”权限
- 代开发自建应用需具有“文档”权限
- 只能操作该应用创建的文档

**返回示例**

      
```
// req_type = 1 仅获取统计信息
{
    "errcode":0,
    "errmsg":"ok",
    "fill_cnt":1,
    "fill_user_cnt":1,
    "unfill_user_cnt":90
}
// req_type = 2,获取已提交列表
{
    "errcode":0,
    "errmsg":"ok",
    "fill_cnt":1,
    "fill_user_cnt":1,
    "unfill_user_cnt":90,
    "submit_users":[
        {
            "userid":"USERID1",
            "submit_time":1668418200,
            "answer_id":1,
            "user_name":"USER_NAME1"
        },
        {
            "external_userid":"EXTERNAL_USERID1",
            "submit_time":1668418200,
            "answer_id":2,
            "user_name":"USER_NAME2"
        },
        {
            "other_corp_userid":"OTHER_CORP_USERID1",
            "submit_time":1668418200,
            "answer_id":2,
            "user_name":"USER_NAME3"
        }
    ],
    "has_more":false,
    "cursor":1
}
// req_type = 3,获取未提交列表，仅当限制提交范围时有结果
{
    "errcode":0,
    "errmsg":"ok",
    "fill_cnt":1,
    "fill_user_cnt":1,
    "unfill_user_cnt":90,
    "unfill_users":[
        {
            "userid":"USERID1",
            "user_name":"USER_NAME1"
        }
    ],
    "has_more":false,
    "cursor":1
} 
```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| fill_cnt | uint64 | 已填写次数 |
| fill_user_cnt | uint64 | 已填写人数 |
| unfill_user_cnt | uint64 | 未填写人数 |
| submit_users | object[] | 已填写人列表 |
| external_userid | string | 外部联系人id，匿名填写不返回 |
| other_corp_userid | string | 外企业用户的userid，匿名填写不返回 |
| userid | string | 企业内成员的id，匿名填写不返回 |
| submit_time | uint64 | 提交时间 |
| answer_id | uint64 | 答案id |
| user_name | string | 名字，匿名填写不返回 |
| userid | string | 企业内成员的id，匿名填写不返回 |
| unfill_users | object[] | 未填写人列表 |
| user_name | string | 名字 |
| userid | string | 企业内成员的id |
| has_more | bool | 是否还有更多 |
| cursor | uint64 | 上次分页拉取返回的cursor |
