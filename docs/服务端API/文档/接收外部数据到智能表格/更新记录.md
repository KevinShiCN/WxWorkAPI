# 更新记录

## 更新记录

可通过智能表格工作表的 Webhook地址 更新一行或多行记录。
**注意**
暂不支持通过Webhook给公式、自动编号、查找引用、关联、创建人、创建时间、最后编辑人、最后编辑时间、群聊、文件字段更新记录。

**请求方式**：POST(HTTPS)
**请求地址**：https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartsheet/webhook?key=KEY

> 开启「接收外部数据」获取到的工作表Webhook地址即为请求地址，启用入口详见配置说明
请求包体写法请参考对应工作表提供的「示例数据」，示例数据位置详见配置说明

**请求包体**：

      
```json
{
    "update_records": [
        {
            "record_id": "RECORD_ID",
            "values": {
                "FIELD_ID": [
                    {
                        "type": "text",
                        "text": "文本内容"
                    }
                ]
            }
        }
    ]
}
```

    
**参数说明**

        
| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| update_records | Object[](UpdateRecord) | 是 | 由需要更新的记录组成的 JSON 数组 |

 

**返回示例**

      
```json
{
    "errcode": 0,
    "errmsg": "ok",
    "update_records": [
    ]
}
```

    
**参数说明**

        
| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| update_records | Object[](CommonRecord) | 由更新成功的记录的具体内容组成的 JSON 数组 |

## 参数详细说明

### UpdateRecord

        
| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| record_id | string | 记录 ID，记录 ID 的获取方式如下所示 |
| values | Object | 记录的具体内容，key 为字段 ID ，value 详见(Value) |

记录 ID 获取方式：
1. 通过 Webhook 新增记录，在请求返回体中会返回新增记录的record_id 
2. 通过文档管理智能表格内容查询子表API查询记录的record_id
3. 通过智能表格公式字段模板（如下图）获取记录ID
![](https://wework.qpic.cn/wwpic3az/859766_lsi6pWcAQmmHAse_1772281555/0)
