## 获取文档权限信息

该接口用于获取文档的加入规则、文档成员及权限、安全设置信息

**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/wedoc/doc_get_auth?access_token=ACCESS_TOKEN 

**请求包体**

      
```
{"docid":"DOCID"}
```

    

**参数说明**

        

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| docid | string | 是 | 文档id |

 

**权限说明**

- 企业需要使用“文档”secret所获取的accesstoken来调用（accesstoken如何获取？）
- 第三方应用需具有“文档”权限
- 代开发自建应用需具有“文档”权限
- 只能访问该应用创建的文档

**返回示例**

      
```
{
    "errcode":0,
    "errmsg":"ok",
    "access_rule":{
        "enable_corp_internal":true,
        "corp_internal_auth":1,
        "enable_corp_external":true,
        "corp_external_auth":1,
        "corp_internal_approve_only_by_admin":true,
        "corp_external_approve_only_by_admin":true,
        "ban_share_external":false
    },
    "secure_setting":{
        "enable_readonly_copy":false,
        "watermark":{
            "margin_type":2,
            "show_visitor_name":false,
            "show_text":false,
            "text":""
        },
        "enable_readonly_comment":false
    },
    "doc_member_list":[
        {
            "type":1,
            "userid":"USERID1",
            "auth":7
        },
        {
            "type":1,
            "external_userid":"USERID2",
            "auth":1
        }
    ],
    "co_auth_list":[
        {
            "type":2,
            "departmentid":{DEPARTMENTID1},
            "auth":1
        }
    ]
}
```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| access_rule | object | 文档的加入规则 |
| enable_corp_internal | bool | 是否允许企业内成员主动加入文档 |
| corp_internal_auth | uint32 | 企业内成员主动加入文档后获得的权限类型 1:只读 |
| enable_corp_external | bool | 是否允许企业外成员主动加入文档 |
| corp_external_auth | uint32 | 企业内成员主动加入文档后获得的权限类型 1:只读 |
| corp_internal_approve_only_by_admin | bool | 企业内成员加入文档是否必须由管理员审批，enable_corp_internal为false时，只能为true |
| corp_external_approve_only_by_admin | bool | 企业外成员加入文档是否必须由管理员审批，enable_corp_external和ban_share_external均为false时，该参数只能为true |
| ban_share_external | bool | 是否允许企业外成员加入文档 |
| enable_readonly_copy | bool | 仅浏览权限的成员是否允许导出、复制、打印 |
| watermark | object | 文档水印设置 |
| margin_type | uint32 | 水印密度 1:稀疏 2:紧密 |
| show_visitor_name | bool | 是否展示访问者名字 |
| show_text | bool | 是否展示水印文字 |
| text | bytes | 水印文字 |
| enable_readonly_comment | bool | 是否允许只读人员发表评论 |
| doc_member_list | obj[] | 文档成员及权限列表 |
| type | uint32 | 文档成员种类 1:user, 只支持成员 |
| userid | bytes | 企业成员的userid |
| external_userid | string | 外部联系人的userid |
| other_corp_userid | string | 外企业用户的userid |
| auth | uint32 | 该文档成员的权限 1:只读  7:管理员 |
| co_auth_list | object | 协作者列表，可以直接加入文档 |
| type | uint32 | 协作者类型 2:部门, 目前只支持部门 |
| departmentid | uint64 | 协作部门id |
| auth | uint32 | 协作者权限类型 1:只读, 目前只支持只读权限 |

 

## 修改文档加入规则

该接口用于修改文档加入规则。

**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/wedoc/mod_doc_join_rule?access_token=ACCESS_TOKEN 

**请求包体**

      
```
{
    "docid":"DOCID",
    "enable_corp_internal":true,
    "corp_internal_auth":1,
    "enable_corp_external":true,
    "corp_external_auth":1,
    "corp_internal_approve_only_by_admin":true,
    "corp_external_approve_only_by_admin":true,
    "ban_share_external":false,
    "update_co_auth_list":true,
    "co_auth_list":[
        {
            "departmentid":DEPARTMENTID1,
            "auth":1,
            "type":2
        }
    ]
}
```

    

**参数说明**

        

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| docid | string | 是 | 操作的docid |
| enable_corp_internal | bool | 否 | 是否允许企业内成员主动加入文档, 有值则覆盖 |
| corp_internal_auth | uint32 | 否 | 企业内成员主动加入文档后获得的权限类型 1:只读, 有值则覆盖 |
| enable_corp_external | uint32 | 否 | 是否允许企业外成员主动加入文档, 有值则覆盖 |
| corp_external_auth | uint32 | 否 | 企业外成员主动加入文档后获得的权限类型 1:只读, 有值则覆盖 |
| corp_internal_approve_only_by_admin | bool | 否 | 企业内成员加入文档是否必须由管理员审批，enable_corp_internal为false时，只能为true, 有值则覆盖 |
| corp_external_approve_only_by_admin | bool | 否 | 企业外成员加入文档是否必须由管理员审批，enable_corp_external和ban_share_external均为false时，该参数只能为true, 有值则覆盖 |
| ban_share_external | bool | 否 | 是否允许企业外成员访问, 有值则覆盖 |
| update_co_auth_list | bool | 否 | 是否更新协作者, true时更新协作者列表 |
| co_auth_list | object[] | 否 | 需要更新协作者部门时, 覆盖文档协作者列表, 特别的: 列表为空则清空 |
| departmentid | uint64 | 否 | 文档协作者部门id |
| auth | uint32 | 否 | 文档写作者权限 1:只读, 目前只支持只读权限 |
| type | uint32 | 否 | 文档写作者类型 2:部门, 目前只支持部门 |

 

**权限说明**

- 企业需要使用“文档”secret所获取的accesstoken来调用（accesstoken如何获取？）
- 第三方应用需具有“文档”权限
- 代开发自建应用需具有“文档”权限
- 只能操作该应用创建的文档

 

**返回示例**

      
```
{
    "errcode": 0,
    "errmsg": "ok"
}
```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

 

## 修改文档成员及权限

该接口用于修改文档成员列表，可以新增文档成员并设置成员权限、修改已有成员的权限以及删除文档成员

**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/wedoc/mod_doc_member?access_token=ACCESS_TOKEN 

**请求包体**

      
```
{
    "docid":"DOCID",
    "update_file_member_list":[
        {
            "type":1,
            "auth":7,
            "userid":"USERID1"
        },
        {
            "type":1,
            "auth":1,
            "external_userid":"EXTERNA_USERID1"
        }
    ],
    "del_file_member_list":[
        {
            "type":1,
            "userid":"USERID2"
        },
        {
            "type":1,
            "external_userid":"EXTERNA_USERID2"
        },
        {
            "type":1,
            "auth":1,
            "other_corp_userid":"OTHER_CORP_USERID1"
        }
    ]
}
```

    

**参数说明**

        

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| docid | string | 是 | 操作的文档id |
| update_file_member_list | obj[] | 否 | 更新文档成员的列表, 批次大小最大100 |
| type | uint32 | 是 | 文档成员的类型 1:人 文档成员仅支持按人配置 |
| auth | uint32 | 是 | 文档成员获得的权限 1:只读权限, 7:管理员权限，文档管理员最多三个 |
| userid | string | 否 | 企业内成员的ID |
| external_userid | string | 否 | 与userid互斥，企业外成员id |
| del_file_member_list | obj[] | 否 | 删除的文档成员列表，批次大小最大一百 |
| type | uint32 | 是 | 文档成员的类型 1:人 文档成员仅支持按人配置 |
| userid | string | 否 | 企业内成员的ID |
| external_userid | string | 否 | 与userid互斥，企业外成员id |
| other_corp_userid | string | 否 | 外企业用户的userid |

**权限说明**

- 企业需要使用“文档”secret所获取的accesstoken来调用（accesstoken如何获取？）
- 第三方应用需具有“文档”权限
- 代开发自建应用需具有“文档”权限
- 只能操作该应用创建的文档

**返回示例**

      
```
{
    "errcode": 0,
    "errmsg": "ok"
}
```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

 

## 修改文档安全设置

该接口用于修改文档的安全设置

**请求方式**：POST（**HTTPS**）
**请求地址**:  https://qyapi.weixin.qq.com/cgi-bin/wedoc/mod_doc_safty_setting?access_token=ACCESS_TOKEN 

**请求包体**

      
```
{
    "docid":"DOCID",
    "enable_readonly_copy":false,
    "watermark":{
        "margin_type":1,
        "show_visitor_name":true,
        "show_text":true,
        "text":"test mark"
    }
}
```

    

**参数说明**

        

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| docid | string | 是 | 操作的文档id |
| enable_readonly_copy | bool | 否 | 是否允许只读成员复制、下载文档，有值则覆盖 |
| watermark | object | 否 | 水印设置 |
| margin_type | uint32 | 否 | 水印疏密度，1:稀疏，2:紧密 |
| show_visitor_name | bool | 否 | 是否展示访问者名字水印，有值则覆盖 |
| show_text | bool | 否 | 是否展示文本水印，有值则覆盖 |
| text | string | 否 | 文字水印的文字，有值则覆盖 |

 

**权限说明**

- 企业需要使用“文档”secret所获取的accesstoken来调用（accesstoken如何获取？）
- 第三方应用需具有“文档”权限
- 代开发自建应用需具有“文档”权限
- 只能操作该应用创建的文档

**返回示例**

      
```
{
    "errcode": 0,
    "errmsg": "ok"
}
```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
