# 获取网络研讨会报名配置

查询网络研讨会的报名配置和报名问题，仅会议创建者可查询。

- 会议未开启报名时会返回未开启报名错误。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/webinars/get_enroll_config?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"meetingid": "MEETINGID1"
}
```

    

**参数说明：**

        

| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| meetingid | 是 | string | 会议 ID |

 

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"approve_type": 1,
	"is_collect_question": 1,
	"meeting_id": "MEETINGID1",
	"no_registration_needed_for_staff": true,
	"question_list": [{
        "is_required":1,
        "question_type":1,
        "question_title":"acd",
        "special_type":1,
        "option_list":[{
            "content":"asr"
        }]
    }]
}
```

    

**参数说明**

        

****
| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | Integer | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
| approve_type | Integer | 审批类型：1：自动审批2：手动审批，默认自动审批 |
| is_collect_question | Integer | 是否收集问题：1：不收集，默认值为不收集2：收集 |
| meeting_id | String | 网络研讨会唯一标识。 |
| no_registration_needed_for_staff | Boolean | 本企业用户无需报名：true：本企业用户无需报名false：默认配置，本企业用户及企业外用户需要报名 |
| question_list | Object 数组 | 报名问题列表，详见Question |

**Question 说明**

        

****

****
| 参数 | 类型 | 说明 |
| --- | --- | --- |
| is_required | Integer | 是否必填：1：否2：是 |
| option_list | Object 数组 | 问题选项列表，按传入的顺序排序，仅单选或多选时有效，最多8个选项，每个选项限40个汉字。详见QuestionOption。 |
| question_title | String | 问题标题，限制40个汉字 |
| question_type | Integer | 问题类型：1：单选2：多选3：简答 |
| special_type | Integer | 特殊问题类型：1：无2：手机号3：邮箱4：姓名5：公司名称说明：目前特殊问题均为简答题 |

**QuestionOption 说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| content | String | 问题选项 |
