# 更新网络研讨会嘉宾列表

修改指定的网络研讨会嘉宾列表，只有网络研讨会的创建者才有权限修改。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/webinars/update_guests?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"meetingid": "MEETINGID",
	"userid": "USERID",
	"guests": {
		"guest_type": 1,
		"userid": "USERID",
		"area": 86,
		"phone_number": "13333333333",
		"guest_name": "NAME",
		"email": "xxxxx@yy.com"
	}
}
```

    

**参数说明：**

        

| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| meeting_id | 是 | String | 网络研讨会唯一标识。 |
| userid | 是 | String | 网络研讨会创建者userid。 |
| guests | 是 | 详见GuestInfo | 网络研讨会嘉宾列表。 |

**GuestInfo说明**

        

| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| guest_type | 是 | Integer | 嘉宾类型：1：内部嘉宾2：外部嘉宾3：链接嘉宾 |
| userid | 否 | String | 嘉宾的userid。 |
| area | 否 | String | 国家/地区代码，外部嘉宾必填（例如：中国传86，不是+86）。 |
| phone_number | 否 | String | 手机号，外部嘉宾必填。通过嘉宾链接接入会的嘉宾无此字段。 |
| guest_name | 否 | String | 嘉宾名称，外部嘉宾必填（1~16位字符长度）。 |
| email | 否 | String | 邮箱，外部嘉宾选填。通过嘉宾链接接入会的嘉宾无此字段。 |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok"
}
```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | Integer | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
