# 取消网络研讨会

取消指定的网络研讨会，只有网络研讨会的创建者才有权限取消。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/cancel_webinars?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"meetingid": "MEETINGID",
	"userid": "USERID",
	"reason_code": 123,
	"reason_detail": "DETAIL"
}
```

    

**参数说明：**

        

| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| meetingid | 是 | String | 网络研讨会唯一标识。 |
| userid | 是 | String | 网络研讨会创建者userid。 |
| reason_code | 是 | Integer | 取消原因代码，可为用户自定义。 |
| reason_detail | 否 | String | 取消原因详细描述。 |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok"
}
```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | Integer | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
