# 审批网络研讨会报名信息

批量审批网络研讨会的报名信息，仅会议创建者可审批。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/webinars/approval_enroll_info?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"meetingid": "MEETINGID1",
	"enroll_id_list": [11312451, 11547631],
	"action": 3
}
```

    

**参数说明：**

        

| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| meetingid | 是 | string | 会议 ID |
| enroll_id_list | 是 | uint64数组 | 报名 ID 列表 |
| action | 是 | uint32 | 审批动作：1：取消批准2：拒绝3：批准取消批准后状态将变成待审批。 |

 

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"meetingid": "MEETINGID1",
	"handled_count": 2
}
```

    

**参数说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | Integer | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
| meetingid | String | 网络研讨会唯一标识 |
| handled_count | Integer | 成功处理的数量 |
