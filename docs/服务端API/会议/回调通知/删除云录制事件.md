# 删除云录制事件

会议创建者或者具有管理员权限的用户可以删除云录制文件，或者通过 API 接口删除云录制文件。
对于API创建的会议，当云录制文件被删除时，触发该事件。

 

**示例**
xml请求示例：

      
```
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[fromUser]]></FromUserName> 
   <FromUserType>2</FromUserType> 
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[meeting_change]]></Event>
   <ChangeType><![CDATA[recording_delete]]></ChangeType>
   <MeetingId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA]]></MeetingId>
</xml>
```

| 参数 | 说明 |
| --- | --- |
| ToUserName | 企业微信CorpID |
| FromUserName | 操作者UserID |
| FromUserType | 操作者UserID类型，2：userid/openid；4：ms_openid。 |
| CreateTime | 消息创建时间，unix时间戳 |
| MsgType | 消息类型，此时固定为：event |
| Event | 事件类型，此时固定为：meeting_change |
| ChangeType | 事件类型，此时固定为：recording_delete |
| MeetingId | 会议ID |
