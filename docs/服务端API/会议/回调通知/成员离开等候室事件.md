# 成员离开等候室事件

当与会者进入启用等候室的会议时，主持人可允许与会者入会，也可将与会者移出等候室。
对于API创建的会议，当主持人将与会者移出等候室或与会者主动离开等候室时，触发该事件。

 

**示例**
xml请求示例：

      
```
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[fromUser]]></FromUserName> 
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[meeting_change]]></Event>
   <ChangeType><![CDATA[meeting_participant_left_waiting_room]]></ChangeType>
   <MeetingId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA]]></MeetingId>
</xml>
```

| 参数 | 说明 |
| --- | --- |
| ToUserName | 企业微信CorpID |
