# 获取用户布局

- 根据会议 ID 和用户 ID 返回用户的布局设置信息。
- 布局优先级：用户个性布局 > 会议自定义布局（高级布局、基础布局）> 会议默认布局。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/advanced_layout/list_user_layout?access_token=ACCESS_TOKEN
 
**请求包体：**

      
```
{
	"meetingid": "MEETINGID",
	"query_tmp_openid": "TMPOPENID1",
	"query_instance_id": 1
}
```

    

**参数说明：**

        
[

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | 会议 ID |
| query_tmp_openid | 是 | 被操作用户临时身份 ID |
| query_instance_id | 是 | 被操作用户终端设备类型 ID：0：PSTN1：PC2：Mac3：Android4：iOS5：Web6：iPad7：Android Pad8：小程序9：voip、sip 设备10：linux20：Rooms for Touch Windows21：Rooms for Touch MacOS22：Rooms for Touch Android30：Controller for Touch Windows32：Controller for Touch Android33：Controller for Touch iOS |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"selected_layout_id": "LAYOUTID1",
	"layout_list": [{
        "layout_id": "LAYOUTID1",
        "layout_name": "testmictest",
		"layout_type": 2,
        "page_list": [{
            "layout_template_id": "3",
            "enable_polling": true,
            "polling_setting": {
                "polling_interval_unit": 1,
                "polling_interval": 10,
                "ignore_user_absence": false,
                "ignore_user_novideo": false
            },
            "user_seat_list": [{
                "grid_id": "1",
                "grid_type": 1,
                "video_type": 3,
                "user_list": [{
                    "tmp_openid": "TMPOPENID1"
                }]
            }]
        }]
    }]
}
```

    

**参数说明：**

        

****
| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| selected_layout_id | 会议应用的布局 ID |
| layout_name | 布局名称 |
| layout_type | 布局类型：0：默认布局1：基础自定义会议布局2：高级自定义会议布局3：个性布局 |
| page_list | 布局单页对象列表，详见LayoutPage |

**LayoutPage 说明**

        
********
| 参数 | 说明 |
| --- | --- |
| layout_template_id | 布局模板 ID |
| enable_polling | 开启或关闭轮询，默认关闭 |
| polling_setting | 轮询参数设置对象，详见PollingSetting |
| user_seat_list | 用户座次对象列表，详见UserSeat |

**PollingSetting 说明**

        

| 参数 | 说明 |
| --- | --- |
| polling_interval_unit | 轮询开启后设置参数。轮询间隔时间类型： 1：秒2：分钟 |
| polling_interval | 轮询开启后设置参数 轮询间隔时长，允许取值范围1～999999 |
| ignore_user_novideo | 轮询开启后设置参数，设置是否忽略没开启视频成员 |
| ignore_user_absence | 轮询开启后设置参数，设置是否忽略未入会成员 |

**UserSeat 说明**

        

****
| 参数 | 说明 |
| --- | --- |
| grid_id | 宫格 ID |
| grid_type | 宫格类型：1. 视频画面2. 共享画面 |
| video_type | 视频画面来源：1：演讲者2：自动填充3：指定人员。根据 user_list 的定义显示视频内容（此类型需传递 tmp_openid）。 |
| user_list | 宫格中的用户列表：轮询关闭，只有一个用户；轮询开启后，可以包含多个用户。详见GridUser |

**GridUser 说明**

        

| 参数 | 说明 |
| --- | --- |
| tmp_openid | 用户当前会议临时身份 ID，单场会议唯一 |
