# 获取会议投票列表

查询指定会议的投票列表。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/poll/get_poll_list?access_token=ACCESS_TOKEN

**请求包体**

      
```
{
	"meetingid": "ID"
}
```

    

**参数说明：**

        
[
| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | 是 | String | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | String | 会议ID |

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"polls_theme_info": [{
		"poll_theme_id": "theme1",
		"polls_info": [{
			"poll_id": "pollid",
			"poll_topic": "topic1",
			"status": 1,
			"is_shared": 1,
			"is_anony": 0
		}]
	}]
}
```

    

**参数说明：**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | String | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
| polls_theme_info | Object数组 | 投票主题信息列表，详见PollThemeInfo |

**PollThemeInfo**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| poll_theme_id | String | 投票主题 ID |
| polls_info | Object数组 | 投票信息列表 |

**投票信息**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| poll_id | String | 投票 ID。 |
| poll_topic | String | 投票标题。 |
| status | Integer | 投票状态。0：未开始1：投票中2：已结束 |
| is_shared | Integer | 共享状态。0：未共享1：已共享 |
| is_anony | Integer | 是否匿名。0：实名1：匿名 |
