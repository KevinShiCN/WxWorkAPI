# 获取会议投票主题信息

查投票模板的详细信息，包括问题、选项，但不包括投票结果。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/poll/get_theme_info?access_token=ACCESS_TOKEN

**请求包体**

      
```
{
	"meetingid":"ID",
	"poll_theme_id":"theme1"
}
```

    

**参数说明：**

        
[
| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | 是 | String | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 否 | String | 会议ID |
| poll_theme_id | 是 | String | 投票模板id |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"poll_topic": "TITLE",
	"poll_desc": "DESC",
	"is_anony": 1,
	"question_data": [{
		"question_desc": "desc",
		"question_type": 1,
		"option_info": [{
			"option_id": 1,
			"option_desc": "option"
		}]
	}]
}
```

    

**参数说明：**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | String | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
| poll_topic | String | 投票主题 |
| poll_desc | String | 投票描述 |
| is_anony | Integer | 是否匿名。0：实名1：匿名 |
| question_data | Object数组 | 投票问题数组，详见Question |

**Question说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| question_desc | String | 问题描述，最多50个字符 |
| question_type | Integer | 问题选择类型。0：单选1：多选 |
| option_info | Object数组 | 选项信息数组，详见Option |

**Option说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| option_id | Interge | 选项 ID |
| option_desc | String | 选项描述 |
