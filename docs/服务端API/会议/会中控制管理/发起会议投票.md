# 发起会议投票

使用已有的投票主题发起投票。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/poll/start?access_token=ACCESS_TOKEN

**请求包体**

      
```
{
	"meetingid":"ID",
	"poll_theme_id":"THEME1"
}
```

    

**参数说明：**

        
[
| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | 是 | String | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | String | 会议ID |
| poll_theme_id | 是 | String | 投票主题 ID |

**返回结果**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"poll_id":"POLLID"
}
```

    

**参数说明：**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | String | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
| poll_id | String | 投票ID |
