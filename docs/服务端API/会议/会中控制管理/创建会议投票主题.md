# 创建会议投票主题

为指定的会议创建投票。该接口投票支持多问题。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/poll/create_theme?access_token=ACCESS_TOKEN

**请求包体**

      
```
{
	"meetingid": "ID",
	"topic": "TOPIC",
	"description": "desc",
	"is_anony": 1,
	"questions": [{
		"type": 1,
		"description": "DESC",
		"option": ["OPTION1", "OPTION2"]
	}]
}
```

    

**参数说明：**

        
[

| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | 是 | String | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 否 | String | 会议ID |
| topic | String | 是 | 投票主题，最多50个字符。 |
| description | String | 是 | 投票主题描述，最多100个字符。 |
| is_anony | Interge | 否 | 是否匿名。0：实名，默认值1：匿名 |
| questions | Object数组 | 是 | 投票问题数组，每个投票支持添加10个问题。详见Question |

 

**Question**

        

| 参数 | 类型 | 必须 | 说明 |
| --- | --- | --- | --- |
| description | String | 必须 | 问题描述，最多50个字符。 |
| type | Number | 必须 | 问题选择类型。0：单选1：多选 |
| option | String数组 | 必须 | 每个问题支持添加10个选项，最少为1个选项。每个选项最多支持36个字符。 |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"poll_theme_id":"theme1"
}
```

    

**参数说明：**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | String | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
| poll_theme_id | String | 投票模板 ID |
