# 管理会中设置

设置会议中的会议属性，例如：全体静音、是否允许参会者聊天设置、锁定会议、隐藏会议号和密码、是否开启等候室等

 

**请求方式：**POST（**HTTPS**）
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/meeting/setting?access_token=ACCESS_TOKEN
**请求包体：**

      
```
{
	"meetingid":"MEETINGID",
	"mute_all": true,
	"allow_unmute_by_self": true,
	"enable_enter_mute": 1,
	"meeting_locked": true,
	"hide_meeting_code_password": true,
	"allow_chat": 1,
	"share_screen": true,
	"enable_red_envelope": true,
	"allow_external_user": true,
	"play_ivr_on_join": true,
	"auto_waiting_room": true
}
```

    

**参数说明：**

        

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| mute_all | 否 | 是否全体静音：true：全体静音false：关闭全体静音 |
| allow_unmute_by_self | 否 | 是否允许用户自己取消静音，请求参数 mute_all 必传，且 mute_all = true 时设置才生效。true：允许false：不允许 |
| enable_enter_mute | 否 | 成员入会静音：0：关闭静音1：开启静音2：超过6人自动开启静音 |
| meeting_locked | 否 | 是否锁定会议：true：锁定false：关闭锁定 |
| hide_meeting_code_password | 否 | 隐藏会议号和密码：true：隐藏false：不隐藏 |
| allow_chat | 否 | 允许参会者聊天设置：0：允许参会者自由聊天1：仅允许参会者公开聊天2：仅允许私聊主持人 |
| share_screen | 否 | 是否允许参会者发起屏幕共享：true：允许false：不允许 |
| enable_red_envelope | 否 | 是否允许参会者发送红包：true：允许false：不允许 |
| allow_external_user | 否 | 是否仅企业成员可入会：true：仅企业成员可入会false：不限制 |
| play_ivr_on_join | 否 | 成员入会是否播放提示音：true：成员入会播放提示音：false：不播放 |
| auto_waiting_room | 否 | 是否开启等候室：true：开启false：关闭 |

 

**返回结果：**

      
```
{
   "errcode": 0,
   "errmsg": "ok"
}
```

    

**参数说明：**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
