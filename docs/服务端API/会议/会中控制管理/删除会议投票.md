# 删除会议投票

删除会议投票

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/poll/delete?access_token=ACCESS_TOKEN

**请求包体**

      
```
{
	"meetingid":"MEETINGID",
	"poll_theme_id":"theme",
	"poll_id":"ID"
}
```

    

**参数说明：**

        
[

| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | 是 | String | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | String | 会议ID |
| poll_theme_id | 否 | String | 投票主题 ID，传入则代表删除投票主题，删除投票主题不影响投票实例。投票主题 ID 和投票 ID 二选一，如果都传入，会使用投票 ID。 |
| poll_id | 否 | String | 投票 ID，传入则代表删除投票实例。当主题下所有主题实例被删，则投票主题也被删除。投票主题 ID 和投票 ID 二选一，如果都传入，会使用投票 ID。 |

**返回结果**

      
```
{
	"errcode": 0,
	"errmsg": "ok"
}
```

    

**参数说明：**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | String | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
