# 修改会议投票主题

修改投票主题信息，目前仅支持全覆盖修改。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/poll/update_theme?access_token=ACCESS_TOKEN

**请求包体**

      
```
{
	"meetingid": "ID",
	"poll_theme_id":"theme1",
	"topic": "TOPIC",
	"description": "desc",
	"is_anony": 1,
	"questions": [{
		"type": 1,
		"description": "DESC",
		"option": ["OPTION1", "OPTION2"]
	}]
}
```

    

**参数说明：**

        
[

| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | 是 | String | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 否 | String | 会议ID |
| poll_theme_id | 是 | String | 投票主题id |
| topic | String | 是 | 投票主题，最多50个字符。 |
| description | String | 是 | 投票主题描述，最多100个字符。 |
| is_anony | Interge | 否 | 是否匿名。0：实名，默认值1：匿名 |
| questions | Object数组 | 是 | 投票问题数组，每个投票支持添加10个问题。详见Question |
