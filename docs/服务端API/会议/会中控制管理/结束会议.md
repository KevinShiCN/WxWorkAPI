# 结束会议

结束一个进行中的会议。

 

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/dismiss?access_token=ACCESS_TOKEN

 

**请求包体：**

      
```
{
	"meetingid": "meetingid11234",
	"reason_code": 1,
	"reason_detail": "test",
	"force_dismiss_meeting": 1,
	"retrieve_code": 1
}
```

    

 

**参数说明：**

        
[

****

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | 会议ID |
| reason_code | 是 | 取消原因代码，用户自定义 |
| reason_detail | 否 | 取消详细原因 |
| force_dismiss_meeting | 否 | 是否强制结束会议，默认值为1：0：不强制结束会议，会议中有参会者，则无法强制结束会议1 ：强制结束会议，会议中有参会者，也会强制结束会议 |
| retrieve_code | 否 | 是否回收会议号，默认值为0：0：不回收会议号，可以重新入会1： 回收会议号，不可重新入会说明：周期性会议如果还有子会议，需设置为不回收会议号，否则会导致后续子会议无法正常进行。此字段对快速会议不生效，快速会议会强制收回会议号。 |
