# 获取会议投票详情

查投票的详情，包括问题、选项、投票结果。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/poll/get_poll_detail?access_token=ACCESS_TOKEN

**请求包体**

      
```
{
	"meetingid":"ID",
	"poll_id":"POLLID"
}
```

    

**参数说明：**

        
[
| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | 是 | String | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | String | 会议ID |
| poll_id | 是 | String | 投票ID |

**返回结果**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"poll_theme_id": "theme1",
	"poll_topic": "topic",
	"poll_desc": "DESC",
	"is_anony": 1,
	"status": 1,
	"vote_total_num": 10,
	"question_data": [{
		"question_desc": "DESC1",
		"question_type": 1,
		"question_id": "id111",
		"option_info": [{
			"option_id": 1111,
			"option_desc": "nigjei",
			"option_num": 10,
			"rate": 10,
			"option_user": [{
				"userid": "NGIEO",
				"tmp_openid": "GNEIGEMM"
			}]
		}]
	}]
}
```

    

**参数说明：**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | String | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
| poll_id | String | 投票 ID。 |
| poll_topic | String | 投票主题。 |
| poll_desc | String | 投票描述。 |
| is_anony | Integer | 是否匿名。0：实名1：匿名 |
| status | Integer | 投票状态。0：未开始1：投票中2：已结束 |
| is_shared | Integer | 是否共享。 |
| vote_total_num | Integer | 投票人数。 |
| poll_question_data | Object数组 | 投票结果数组。详见QuestionData |

**QuestionData说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| question_desc | String | 问题描述。 |
| question_type | Integer | 问题类型。单选多选 |
| question_id | String | 问题 ID。 |
| option_info | Object数组 | 选项信息数组。详见OptionInfo |

**OptionInfo**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| option_id | Interge | 选项 ID。 |
| option_desc | String | 选项描述。 |
| option_num | Integer | 每个选项的投票数。 |
| rate | Integer | 投票比率。 |
| option_user | Object数组 | 用户信息数组。详见OptionUser |

**OptionUser说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| userid | String | 用户userid，同企业才展示 userid。如果非匿名，展示。 |
| tmp_openid | String | 会议临时ID，如果非匿名，展示。 |
