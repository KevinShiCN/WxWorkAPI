# 获取会议录制地址

查询会议录制地址，可获取会议云录制的播放地址和下载地址。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/get_record_url?access_token=ACCESS_TOKEN
 
**请求包体：**

      
```
{
"record_id": "record"
}
```

    

**参数说明：**

        
[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | 会议录制ID |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"record": {
		"record_id": "record",
		"meetingid": "MEETINGID",
		"meeting_code": "MEETINGOCDE",
		"title": "meetingtitle",
		"record_file_list": [{
			"record_file_id": "FILEID",
			"view_url": "https://view.url/124",
			"download_url": "https://download.url/234",
			"download_file_type": "mp4",
			"audio_download_url": "https://audio.download/342",
			"audio_file_type": "m4a",
			"summary": [{
				"download_url": "https://summary.url",
				"file_type": "txt"
			}]
		}]
	}
}
```

    

 

**参数说明：**

        

| 参数 | 说明 |  |
| --- | --- | --- |
| errcode | 返回码 |  |
| errmsg | 对返回码的文本描述内容 |  |
| record_id | 会议录制 ID。 |  |
| meetingid | 会议 ID。 |  |
| meeting_code | 会议 code。 |  |
| title |  | 会议主题。 |
| record_file_list | 直播录制文件列表，详见RecordFile |  |

 

**RecordFile说明**

        

| 参数 | 说明 |  |  |
| --- | --- | --- | --- |
| record_file_id | 录制文件 ID。 |  |  |
| view_url | 播放地址。 |  |  |
| download_url | 下载地址，默认6小时过期 |  |  |
| download_file_type | 下载视频文件格式，例如：mp4。 |  |  |
| audio_download_url | 音频下载地址。 |  |  |
| audio_file_type | 否 | String | 下载音频文件格式，例如：m4a。 |
| summary | 会议纪要文件列表，详见Summary |  |  |

 

**Summary说明**

        

| 参数名称 | 必选 | 参数类型 | 参数描述 |
| --- | --- | --- | --- |
| download_url | 下载文件地址。 |  |  |
| file_type | 下载文件类型，例如：txt、pdf、docs。 |  |  |
