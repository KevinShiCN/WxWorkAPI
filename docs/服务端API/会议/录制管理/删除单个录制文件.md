# 删除单个录制文件

删除单个录制文件，该接口支持从会议中删除指定的某个录制文件。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/delete_record_file?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"record_file_id": "fileid",
	"meetingid":"ID"
}
```

    

**参数说明：**

        
[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | 会议D |
| record_file_id | 录制文件ID |  |

**返回结果：**

      
```
{
"errcode": 0,
"errmsg": "ok"
}
```

    

**参数说明：**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
