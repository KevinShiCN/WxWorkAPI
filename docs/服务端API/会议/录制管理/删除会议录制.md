# 删除会议录制

删除会议的所有录制文件，该接口会删除会议录制 ID 里对应的所有云录制文件。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/delete_record?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"record_id": "record",
	"meetingid":"MEETINGID"
}
```

    

**参数说明：**

        
[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| record_id | 是 | 会议录制ID |
| meetingid | 是 | 会议ID |

**返回结果：**

      
```
{
"errcode": 0,
"errmsg": "ok"
}
```

    

 
**参数说明：**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
