# 修改会议录制共享设置

根据会议录制 ID 修改共享等配置，支持修改共享权限、共享密码、共享有效期等信息

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/record_setting?access_token=ACCESS_TOKEN
**请求包体：**

      
```
{
	"record_id": "record",
	"meetingid": "MEETINGID",
	"config": {
		"enable_sharing": true,
		"sharing_auth_type": 0,
		"enable_password": true,
		"password": "123352",
		"enable_sharing_expire": true,
		"sharing_expire": 1999999999,
		"allow_download": true
	}
}
```

    

**参数说明：**

        
[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| record_id | 是 | 会议录制ID |
| meetingid | 是 | 会议ID |
| config | 否 | 共享配置，详见Config |

**Config说明**
**共享配置对象**

        

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| enable_sharing | 否 | 共享开关，是否开启共享，默认为 true。true：开启false：未开启说明：- 未开启时不允许设置以下参数。- 修改为 false 关闭共享后，之前设置的共享设置将不保存。 |
| sharing_auth_type | 否 | 共享权限类型。0：仅允许登录用户查看1：仅企业用户成员可查看2：仅参会成员可查看 |
| enable_password | 否 | 是否开启密码，默认为 true。true：开启 |
| password | 否 | 共享密码，默认随机生成。说明：当 enable_password = true 时，必传；当 enable_password = false 时，不可传。 |
| enable_sharing_expire | 否 | 是否开启共享链接有效期，默认为 false。true：开启 |
| sharing_expire | 否 | 共享链接有效期，unix 时间戳（单位毫秒），默认为空。说明：当 enable_sharing_expire = true 时，必传；当 enable_sharing_expire = false 时，不可传。 |
| allow_download | 否 | 是否允许下载，默认为 false。true：允许下载 |

 

**返回结果：**

      
```
{
"errcode": 0,
"errmsg": "ok"
}
```

    

**参数说明：**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
