# 获取单个录制文件详情

查询单个云录制的详情信息，包括录制文件和会议纪要，并可获取播放地址和下载地址

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/get_record_detail?access_token=ACCESS_TOKEN
 
**请求包体：**

      
```
{
"record_id": "record1"
}
```

    

**参数说明：**

        
[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| record_id | 是 | 会议录制ID |

 

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"record": {
		"record_file_id": "FIELID",
		"meetingid": "MEETINGID",
		"meeting_code": "CODE",
		"view_url": "https://view.url",
		"download_url": "https://download.url",
		"download_file_type": "mp4",
		"audio_download_url": "url",
		"audio_file_type": "m4a",
		"summary": {
			"download_url": "https://download.url",
			"file_type": "pdf"
		}
	}
}
```

    

**参数说明：**

        

| 参数 | 说明 |  |
| --- | --- | --- |
| errcode | 返回码 |  |
| errmsg | 对返回码的文本描述内容 |  |
| record | 直播录制详情，详见RecordFile |  |

 

**RecordFile说明**

        

| 参数 | 说明 |
| --- | --- |
| record_file_id | 录制文件 ID。 |
| view_url | 播放地址。 |
| download_url | 下载地址，默认6小时过期 |
| download_file_type | 下载视频文件格式，例如：mp4。 |
| audio_download_url | 音频下载地址。 |
| audio_file_type | 下载音频文件格式，例如：m4a。 |
| summary | 会议纪要文件列表，详见Summary |

 

**Summary说明**

        

| 参数名称 | 必选 | 参数类型 | 参数描述 |
| --- | --- | --- | --- |
| download_url | 下载文件地址。 |  |  |
| file_type | 下载文件类型，例如：txt、pdf、docs。 |  |  |
