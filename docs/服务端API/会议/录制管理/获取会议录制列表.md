# 获取会议录制列表

获取用户云录制记录，根据用户 ID、会议 ID、会议 code 进行查询，支持根据时间区间分页获取。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/get_record_list?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"meetingid": "meetingid11234",
	"meeting_code": "MEETINGCODE",
	"user": {
		"id_type": 3,
		"id": "ROOMID"
	},
	"start_time": 16666666,
	"end_time": 178888888,
	"page_size": 10,
	"page": 1
}
```

    

**参数说明：**

        
[
****
****
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 否 | 会议 ID，不为空时优先根据会议 ID 查询。 |
| meeting_code | 否 | 会议 code，当 meeting_id 为空且 meeting_code 不为空时根据会议 code 查询。 |
| start_time | 是 | 查询起始时间戳，UNIX 时间戳（单位秒）。说明：时间区间不允许超过31天。 |
| end_time | 是 | 查询结束时间戳，UNIX 时间戳（单位秒）。说明：时间区间不允许超过31天。 |
| page_size | 否 | 分页大小，默认值为10，最大为20。 |
| page | 否 | 页码，从1开始，默认值为1。 |
| user | 否 | 查询对象，详见User |

**User说明**

        

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| id | 是 | 被操作者 ID，根据 operator_id_type 的值，使用不同的类型。 |
| id_type | 是 | 被操作者 ID 的类型：1：userid；2：open_id；3：rooms_id |

**返回结果：**

      
```
{
   "errcode": 0,
   "errmsg": "ok"
}
```

    

**参数说明：**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
