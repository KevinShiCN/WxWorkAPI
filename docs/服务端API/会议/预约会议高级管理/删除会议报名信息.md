# 删除会议报名信息

删除指定会议的报名信息，支持删除用户手动报名的信息和导入的报名信息。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/enroll/unregistration?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"meetingid": "meetingid11234",
	"enroll_id_list": [{
		"enroll_id": 123456
	}]
}
```

    

**参数说明：**

        

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | 会议ID |
| enroll_id_list | 是 | 报名ID列表。详见EnrollID。 |

**EnrollID说明**

        

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| enroll_id | 是 | 报名ID |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"total_count": 1
}
```

    

**参数说明**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| total_count | 成功删除的报名信息数量 |
