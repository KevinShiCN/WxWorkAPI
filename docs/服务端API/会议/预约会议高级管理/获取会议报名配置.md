# 获取会议报名配置

查询会议的报名配置和报名问题，仅会议创建者可查询。
说明
- 会议未开启报名时会返回未开启报名错误。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/enroll/get_config?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"meetingid": "meetingid11234",
	"userid": "USERID",
}
```

    

**参数说明：**

        

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | 会议ID |
| userid | 否 | 会议创建者的用户ID |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"approve_type": 1,
	"is_collect_question": 1,
	"question_list": [
		"is_required": 1,
		"option_list": [{
			"content": "CONTENT"
		}],
		"question_title": "TITLE",
		"question_type": 1,
		"special_type": 1
	],
	"no_registration_needed_for_staff": true
}
```

    

**参数说明**

        

| 参数 | 说明 |  |
| --- | --- | --- |
| errcode | 返回码 |  |
| errmsg | 对返回码的文本描述内容 |  |
| approve_type | 否 | 审批类型：1：自动审批，默认自动审批。2：手动审批。 |
| is_collect_question | 否 | 是否收集问题：1：不收集，默认不收集问题。2：收集。 |
| question_list | 否 | 报名问题列表，非特殊问题按传入的顺序排序，特殊问题会优先放在最前面，仅开启收集问题时有效。详见Question。 |
| no_registration_needed_for_staff | 否 | 本企业用户无需报名。true：本企业用户无需报名。false：默认配置，本企业用户及企业外用户需要报名。 |

**Question说明**

        

****

****

****
| 参数 | 说明 |
| --- | --- |
| is_required | 当场会议的用户临时 ID，适用于所有用户。 |
| option_list | 问题选项列表，按传入的顺序排序，仅单选和多选时有效，最多8个选项，每个选项限40个汉字。详见Option。 |
| question_title | 问题标题，限制40个字符。说明：special_type 为特殊问题时，该字段无效。 |
| question_type | 问题类型：1：单选2：多选3：简答说明：special_type 为特殊问题时，该字段无效。 |
| special_type | 特殊问题类型：1：无2：手机号3：邮箱4：姓名5：公司名称说明：目前特殊问题均为简答题。 |

**Option说明**

        

| 参数 | 说明 |
| --- | --- |
| content | 问题选项。每个选项限40个汉字。 |
