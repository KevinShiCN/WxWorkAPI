# 取消预约会议

该接口用于取消一个指定的预约会议。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/cancel?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
   "meetingid": "XXXXXXXXX"
}

```

    

**参数说明：**

        
[
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | 会议id，仅允许取消预约状态下的会议 |

**权限说明：**

- 仅允许取消当前应用创建的会议。
- 自建应用需要配置在“可调用接口的应用”列表
- 第三方服务商创建应用的时候，需要开启“会议接口权限”
- 代开发自建应用需要授权“会议接口权限”

**返回结果：**

      
```
{
   "errcode": 0,
   "errmsg": "ok"
}
```

    

**参数说明：**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
