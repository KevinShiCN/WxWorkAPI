# 获取会议健康度

查询已结束会议的会议及参会成员的健康度，付费开通该服务的企业管理员、超管可以查询，与是否为会议创建者、主持人或联席主持人无关。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/get_quality?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"meetingid": "MEETINGID",
	"sub_meeting_id": "SUB_MEETING_ID",
	"userid": "USERID",
	"start_time": 168888888,
	"cursor": "CURSOR",
	"limit": 20
}
```

    

**参数说明：**

        

****

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | 会议ID |
| sub_meeting_id | 否 | 周期性子会议ID说明：可通过查询用户的会议列表、查询会议接口获取返回的子会议 ID，即 current_sub_meeting_id；如果是周期性会议，此参数必传。 |
| userid | 是 | 操作者userid |
| start_time | 是 | 参会时间过滤起始时间，UNIX 时间戳（单位秒），可查询的时间区间为过去7天到现在。返回 meeting_id 对应会议房间下，开始时间大于等于 start_time 且离 start_time 最近的一个媒体房间数据（从第一个人入会到会中成员全部离开会议形成一个媒体房间，若同一会议号下再次有人入会则形成新的媒体房间）。如果同一会议号下有多个媒体房间，请先使用“获取账户级已结束会议列表”接口查询，获知需查询的媒体房间的 start_time。 |
| cursor | 否 | 分页查询用，将上一个请求返回的next_cursor字段传入。第一次查询时可不传值 |
| limit | 否 | 分页大小，默认50，最大50 |

 

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"quality": 0,
	"audio_quality": 1,
	"video_quality": 2,
	"screen_share_quality": 1,
	"network_quality": 1,
	"problems": [
		"PROBLEM1"
	],
	"attendees":[
		{
			"userid": "USERID",
			"tmp_openid": "TMP_OPENID",
			"instanceid": 1,
			"quality": 1,
			"audio_quality": 1,
			"video_quality": 1,
			"screen_share_quality": 1,
			"network_quality": 1,
			"problems":[
				"PROBLEM1"
			]
		}
	],
	"next_cursor": "NEXT_CURSOR",
	"has_more": true
}
```

    

**参数说明**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| quality | 健康度：0：无数据1：健康2：告警 |
| audio_quality | 音频质量：0：无数据1：好2：较好3：中4：差 |
| video_quality | 视频质量：0：无数据1：好2：较好3：中4：差 |
| screen_share_quality | 共享屏幕质量：0：无数据1：好2：较好3：中4：差 |
| problems | 告警的具体问题列表 |
| attendees | 参会人员健康度对象数组（按成员入会时间正序排列，入会越早的在越上面；成员使用不同端入会时平铺返回数据，instanceid 不同）。详见Attendee。 |
| next_cursor | 分页用，下一次拉取列表将该字段填入cursor字段 |
| has_more | 是否还有待拉取的列表 |

 

**Attendee说明**

        

| 参数 | 说明 |
| --- | --- |
| userid | 用户userid |
| ms_open_id | 当场会议的用户临时 ID，可用于会控操作，适用于所有用户。 |
| instanceid | 用户的终端设备类型：0：PSTN1：PC2：Mac3：Android4：iOS5：Web6：iPad7：Android Pad8：小程序9：voip、sip 设备10：linux20：Rooms for Touch Windows21：Rooms for Touch MacOS22：Rooms for Touch Android30：Controller for Touch Windows32：Controller for Touch Android33：Controller for Touch iOS |
| quality | 健康度：0：无数据1：健康2：告警 |
| audio_quality | 音频质量：0：无数据1：好2：较好3：中4：差 |
| video_quality | 视频质量：0：无数据1：好2：较好3：中4：差 |
| screen_share_quality | 共享屏幕质量：0：无数据1：好2：较好3：中4：差 |
| problems | 告警的具体问题列表 |
