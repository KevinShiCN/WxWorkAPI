# 审批会议报名信息

批量审批会议的报名信息，仅会议创建者可审批。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/enroll/approve?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"meetingid": "meetingid11234",
	"userid": "USERID",
	"action": 1,
	"enroll_id_list":[
		1233445
	]
}
```

    

**参数说明：**

        

****
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | 会议ID |
| userid | 否 | 会议创建者的用户ID |
| action | 否 | 审批动作：1：取消批准2：拒绝3：批准说明：取消批准后状态将变成待审批。 |
| enroll_id_list | 否 | 报名ID列表 |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"handled_count": 10
}
```

    

**参数说明**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| handled_count | 成功处理的数量。 |
