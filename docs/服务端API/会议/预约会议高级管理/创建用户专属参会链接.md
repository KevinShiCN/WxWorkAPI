# 创建用户专属参会链接

使用该接口，可用  
```
customer_data
```
  进行区分，为一场会议生成多个会议链接。通过用户入会、用户进入等候室等事件，或通过获取等候室成员列表的 API 查询到该参数。
说明
- 该接口不支持个人会议号会议、网络研讨会（Webinar）。支持企业品牌化链接。
- 参会者腾讯会议客户端版本需大于等于 3.2.0。

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/get_customer_short_url?access_token=ACCESS_TOKEN

**请求包体：**

      
```
{
	"meetingid": "meetingid11234",
	"customer_data": "CUSTOMERDATA"
}
```

    

**参数说明：**

        

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 是 | 会议ID |
| customer_data | 是 | 用户专属字段，长度不超过256 字节。 |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"meeting_short_url_customer_data": {
		"customer_data": "CUSTOMERDATA",
		"meeting_short_url": "URL"
	}
}
```

    

**参数说明**

        

| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| meeting_short_url_customer_data | 用户专属参会链接对象。详见CustomerData。 |

 

**CustomerData说明**

        

| 参数 | 说明 |
| --- | --- |
| customer_data | 用户专属字段 |
| meeting_short_url | 用户专属参会链接 |
