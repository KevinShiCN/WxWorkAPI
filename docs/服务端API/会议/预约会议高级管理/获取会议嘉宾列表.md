# 获取会议嘉宾列表

查询会议嘉宾列表

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/meeting/get_guest_list?access_token=ACCESS_TOKEN

**请求包体**

      
```
{
	"meetingid":"ID",
	"meeting_code":"CODE"
}
```

    

**参数说明：**

        
[
```

```

```

```

| 参数 | 必须 | 类型 | 说明 |
| --- | --- | --- | --- |
| access_token | 是 | String | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | 否 | String | 会议ID。meetingid和meeting_code必填一个，如果两个都填了以meetingid为准 |
| meeting_code | 否 | String | 会议入会码。meetingid和meeting_code必填一个，如果两个都填了以meetingid为准 |

**返回结果：**

      
```
{
	"errcode": 0,
	"errmsg": "ok",
	"meetingid": "15923312864334416762",
	"meeting_code": "653202434",
	"title": "jiabin",
	"guests": [{
		"area": "86",
		"phone_number": "xxxxxx",
		"guest_name": "test"
	}]
}
```

    

**参数说明：**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| errcode | String | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
| meetingid | String | 会议ID |
| meeting_code | String | 入会码 |
| guests | Object数组 | 嘉宾列表，详见Guest |

**Guest说明**

        

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| area | String | 国家/地区代码（例如：中国传86，不是+86，也不是0086）。 |
| phone_number | String | 手机号。 |
| guest_name | String | 嘉宾名称。 |
