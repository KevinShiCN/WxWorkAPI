# 批量获取汇报记录单号

企业可通过access_token调用本接口，以获取企业一段时间内企业微信“汇报应用”汇报记录编号，支持按汇报表单ID、申请人、部门等条件筛选。

一次拉取调用最多拉取100个汇报记录，可以通过多次拉取的方式来满足需求，但调用频率不可超过600次/分。

表单ID的获取方式：
管理后台—汇报应用—某个汇报的内容设置页—点击“汇报名称”，即可获取
