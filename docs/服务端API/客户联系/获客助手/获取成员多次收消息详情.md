# 获取成员多次收消息详情

## 获取成员多次收消息详情

企业和服务商可通过此接口获取成员多次收消息情况，如次数、客户id等信息。

**权限说明:**
调用接口的应用需要满足如下的权限：

        
| 应用类型 | 权限要求 |
| --- | --- |
| 自建应用 | 配置到「客户联系 可调用接口的应用」中 |
| 代开发应用 | 具有「查询使用量、创建和编辑获客链接、获取客户列表」权限 |
| 第三方应用 | 具有「获客助手」权限 |

 

**请求方式**: **POST(HTTP)**

**请求地址:**https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/get_chat_info?access_token=ACCESS_TOKEN

**请求示例**

      
```javascript
{
 	"chat_key":"CHAT_KEY"
}
```

    
**参数说明**:

        
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证 |
| chat_key | 是 | 成员多次收消息事件中回调的会话信息凭据ChatKey，回调后30分钟内有效 |

 

**返回结果**:

      
```javascript
{
	"errcode": 0,
	"errmsg": "ok",
	"userid":"zhangsan",
	"external_userid":"woAJ2GCAAAXtWyujaWJHDDGi0mACAAA",
	"chat_info":
	{
			"recv_msg_cnt":3,
			"link_id":"caXXXXXXX",
			"state":"STATE"
	}
}
```

    
**参数说明**:

        
| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
| userid | 成员的userid |
| external_userid | 客户id |
| chat_info.recv_msg_cnt | 成员收到的此客户的消息次数 |
| chat_info.link_id | 成员添加客户的获客链接id |
| chat_info.state | 成员添加客户的state |
