# 提交图片

填写材料需要用到的图片需要提前通过这个接口上传，获取对应的OpenWxPayMediaId用于填写材料

 

**请求方式：**POST(HTTPS)
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/corpecom/upload_file?access_token=ACCESS_TOKEN
