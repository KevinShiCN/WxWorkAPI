# 空间变更事件

当接口指定的管理员在企业微信客户端，对应用创建的空间进行变更，触发该事件。

**示例**
xml请求示例：

      
```javascript
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[sys]]></FromUserName> 
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[wedrive_space_change]]></Event>
   <ChangeType><![CDATA[dismiss_space]]></ChangeType>
   <SpaceId><![CDATA[spxxxxxxxxxxxxx]]></SpaceId>
</xml>
```

    
 

**说明**

        
| 参数 | 说明 |
| --- | --- |
| ToUserName | 企业微信CorpID |
| FromUserName | 此事件该值固定为sys，表示该消息由系统生成 |
| CreateTime | 消息创建时间，unix时间戳 |
| MsgType | 消息类型，此时固定为：event |
| Event | 事件类型，此时固定为：wedrive_space_change |
| ChangeType | 变更类型，此处目前固定为：dismiss_space |
| SpaceId | 空间spaceid |
