# 删除标签

**请求方式：**GET（**HTTPS**）
**请求地址：**https://qyapi.weixin.qq.com/cgi-bin/tag/delete?access_token=ACCESS_TOKEN&tagid=TAGID

 

**参数说明：**

        
| 参数 | 必须 | 说明 |
| --- | --- | --- |
| access_token | 是 | 调用接口凭证 |
| tagid | 是 | 标签ID |

**权限说明：**

调用的应用必须是指定标签的创建者。

**返回结果：**

      
```javascript
{
   "errcode": 0,
   "errmsg": "deleted"
}
```

    
**参数说明：**

        
| 参数 | 说明 |
| --- | --- |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
