# 修改自动升年级的配置

**请求方式：** POST（**HTTPS**）
**请求地址：** https://qyapi.weixin.qq.com/cgi-bin/school/set_upgrade_info?access_token=ACCESS_TOKEN

**请求包体：**

```
{
   "upgrade_time": 1594090969,
   "upgrade_switch": 2
}

```

**参数说明：**

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| upgrade_time | 否 | 自动升年级的时间，该时间戳只有月和日有效，不传则默认为传0，代表的1月1号 |
| upgrade_switch | 否 | 开启或关闭自动升年级。0：表示关闭，1：表示开启，不传默认关闭，传所有非1的值也视为关闭 |

**权限说明：**

- 学校需要使用“家校沟通”secret所获取的accesstoken来调用（[accesstoken如何获取？）；
- 第三方应用需拥有「家校沟通」使用和编辑权限。

**返回结果：**

```
{
   "next_upgrade_time": 1625587200,
   "errcode": 0,
   "errmsg": "ok"
}

```

**参数说明：**

| 参数 | 说明 |
| --- | --- |
| next_upgrade_time | 下次升级的时间，只有年月日有效，如果该学校今年已经升过年级，那本参数会给明年对应日期的时间戳 |
| errcode | 返回码 |
| errmsg | 对返回码的文本描述内容 |
